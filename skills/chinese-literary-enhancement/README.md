# 中文文学素养增强 Skill

## 简介

让 AI 输出的中文文本更具文化底蕴——善用成语、古诗词、名言警句、歇后语、俗语，写出引经据典、妙语连珠的文章。

## 适用场景

| 场景 | 效果 |
|------|------|
| 写作润色 | 将平淡表达升华为文采斐然的段落 |
| 文案撰写 | 标题、引言、结语更具感染力 |
| 正式邮件 | 措辞得体而有分量 |
| 演讲稿/发言 | 引经据典，增强说服力 |
| 书评/读后感 | 情感细腻，意境悠远 |
| 日常表达 | 穿插歇后语、俗语，增添趣味 |

## 核心原则

1. **引经据典** — 恰当引用使论述更有力
2. **善用成语** — 凝练四字，意蕴丰富
3. **雅俗共赏** — 文言白话因境而异
4. **以情动人** — 引用为情感服务，不炫技
5. **因境择词** — 励志用豪迈之语，感伤用婉约之辞

## 资源库

完整的成语、古诗词、名言、歇后语、俗语资源库见 [`resources.md`](./resources.md)。

## 使用方式

### 用户直接调用

```
/chinese-literary-enhancement <写作主题或需求>
```

### Agent 自动激活

当用户表达以下意图时，Agent 应自动加载本 Skill：
- 要求润色、修改、升华中文文本
- 要求"写得好一点""更有文采"
- 撰写文案、演讲稿、书评、正式邮件等
- 请求用成语、古诗词、名言表达某个意思

## 注意事项

- 引用须准确，不确定时宁可不用
- 密度适中，每 300-500 字 1-2 处引用为宜
- 好的文学输出让读者感受不到刻意，恰如"水中着盐，饮水方知"
