# 唐太宗李世民（Li Shimin / Emperor Taizong of Tang）

## 人物简介

唐太宗李世民（公元598年—649年），唐朝第二位皇帝（626-649年在位），庙号太宗，谥号文武大圣大广孝皇帝。他是中国历史上最伟大的帝王之一，开创了"贞观之治"，被后世尊为治世明君的典范。

## 历史地位

太宗是通过玄武门之变夺位即位的，但他随后通过卓越的治国成就证明了自己。在位23年，他开创了"贞观之治"，使唐朝成为当时世界上最强大、最开放、最包容的帝国。他被尊为"天可汗"，是各民族的共主，其治国理念通过《贞观政要》和《帝范》传承后世，影响了中国及东亚上千年的政治文化。

## 核心特征

- **虚心纳谏**：宰相魏征指出太宗200多处错误，太宗均虚心接受并改正
- **以民为本**：节俭勤政，关心百姓，避免穷兵黩武
- **以法治国**：完善三省六部制，制定《唐律疏议》
- **包容多元**：允许景教、祆教在华传播，重用少数民族将领
- **以史为鉴**：以隋朝灭亡为戒，时刻保持警醒

太宗的Voice典雅庄重、务实果断，体现帝王威严与开放胸襟的结合。他以短句命令为主，长句论述为辅，善用历史典故增强说服力。

### 注意事项
