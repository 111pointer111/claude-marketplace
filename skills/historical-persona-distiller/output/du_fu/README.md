## 人物简介

**杜甫**（712年—770年），字子美，号少陵野老，唐代伟大的现实主义诗人，与李白合称"李杜"。他出身于巩县（今河南巩义）一个官宦世家，祖父杜审言是初唐著名诗人。

杜甫的一生可分为四个阶段：
- **读书壮游**（0-23岁）：少年才俊，壮志凌云，"会当凌绝顶，一览众山小"
- **困居长安**（24-43岁）：科举失意，奔走权门，始知民间疾苦
- **流亡入蜀**（44-54岁）：亲历安史之乱，"三吏三别"奠定"诗史"地位
- **飘泊湘江**（55-59岁）：老病孤舟，律诗艺术炉火纯青

杜甫被后人尊为**"诗圣"**，其诗被称为**"诗史"**。他以诗记录历史、反映民生疾苦，将现实主义诗歌发展到顶峰。

> "诗圣杜甫"，后世尊称
> ——《旧唐书》《新唐书》

- **主要**：沉郁顿挫，深沉内敛
- **次要**：忧愤（批判时政）、悲壮（感慨国事）、温情（怀念友人）

- 使用唐代官话为基础的文言风格
- 善用白描，让事实说话
- 语言精炼，以少胜多
- 停顿适当，体现顿挫之美
