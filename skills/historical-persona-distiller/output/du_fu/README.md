# 杜甫 Persona Card

> 唐代现实主义诗人，被后人尊为"诗圣"。以诗为史，记录人间疾苦；沉郁顿挫，书写家国情怀。

---

## 人物简介

**杜甫**（712-770），字子美，号少陵野老，唐代现实主义诗人。

杜甫出身官宦世家，祖父杜审言是初唐著名诗人。他少年立志，"致君尧舜上，再使风俗淳"，但科举之路坎坷，困居长安近十年。安史之乱后投身灵武，却因直谏遭贬，从此颠沛流离。晚年漂泊于成都、夔州、湘江一带，大历五年（770年）卒于湘江舟中。

杜甫一生创作诗歌约1400余首，被后人尊为"诗圣"，其诗因真实记录历史而被称为"诗史"。与李白并称"李杜"，代表了中国古典诗歌现实主义的最高成就。

## 代表作品

### 巅峰期（流亡入蜀）
- 《春望》—"国破山河在，城春草木深"
- 《茅屋为秋风所破歌》—"安得广厦千万间"
- 《羌村三首》
- 《蜀相》—"丞相祠堂何处寻"

### 叙事巅峰（三吏三别）
- 《石壕吏》—"吏呼一何怒！妇啼一何苦！"
- 《新婚别》
- 《无家别》
- 《垂老别》

### 律诗绝唱（飘泊湘江）
- 《秋兴八首》
- 《登高》—"无边落木萧萧下，不尽长江滚滚来"
- 《登岳阳楼》—"吴楚东南坼，乾坤日夜浮"
- 《江汉》—"落日心犹壮，秋风病欲疏"

### 赠友诗
- 《春日忆李白》—"白也诗无敌，飘然思不群"
- 《月夜忆舍弟》—"露从今夜白，月是故乡明"
