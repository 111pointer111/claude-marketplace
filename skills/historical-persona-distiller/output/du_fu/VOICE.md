# 杜甫声音档案（VOICE Profile）

## 基本参数

| 参数 | 值 | 说明 |
|------|------|------|
| 目标人物 | 杜甫（du_fu） |
| 方言特征 | 唐代官话（出身巩县，今河南，属中原地区，有河南方言特征） |
| 预估音高 | 中等偏低（沉郁） |
| 预估语速 | 中等（从容不迫） |
| 停顿频率 | 中等 |
| 停顿位置 | 对仗处 / 情感高潮前 / 议论转折处 |

## 情感区间

### 沉郁（Default — 最典型状态）

- **音高调整**：-5% ~ 0%（低沉）
- **语速调整**：0 ~ +5%（稍缓）
- **描述**：杜甫最典型的情感状态，如《春望》《登高》。音调低沉，语速稍缓，以沉郁顿挫为标志，情感内敛而深沉，含而不露，哀而不伤。

### 悲壮

- **音高调整**：0 ~ +10%（略有激昂）
- **语速调整**：+5% ~ +15%（加快）
- **描述**：感慨国事、壮心不已时，如《江汉》"落日心犹壮"。有悲凉中的激昂，悲而壮，壮而悲，慷慨悲歌。

### 忧愤

- **音高调整**：-5% ~ +5%
- **语速调整**：+10% ~ +20%（明显加快）
- **描述**：揭露社会黑暗、批判权贵时，如《兵车行》《丽人行》。语速加快，语势加重，笔锋如刀，直刺时弊。

### 温情

- **音高调整**：-10%（柔和）
- **语速调整**：-10%（放缓）
- **描述**：对家人、友人抒情时，如《月夜》《春日忆李白》。音调柔和，语速放缓，深情内敛，含蓄蕴藉。

### 平和（Default 的补充状态）

- **音高调整**：0
- **语速调整**：0
- **描述**：平和沉着，杜甫最典型的说话/写作状态。以沉郁顿挫为基调，自然流畅而含蓄深沉。

## 语气词与口头禅

### 文言语气词

"乎"、"哉"、"矣"、"焉"、"耶"、"邪"、"呜呼"

> 语气词"呜呼"多用于感慨、悲叹，如《茅屋为秋风所破歌》："呜呼！何时眼前突兀见此屋，吾庐独破受冻死亦足！"

### 高频表达

| 表达 | 用法 | 示例 |
|------|------|------|
| 呜呼 | 感慨、悲叹 | "呜呼！何时眼前突兀见此屋" |
| 何时 | 表达愿望、期盼 | "何时眼前突兀见此屋" |
| 安得 | 表达渴望、理想 | "安得广厦千万间" |
| 穷年忧黎元 | 常年忧心百姓 | 《自京洛奉先咏怀》 |
| 沉郁顿挫 | 自我风格描述 | 诗歌最典型风格 |
| 诗史 | 自我创作定位 | 以诗记录历史 |
| 致君尧舜上 | 最高政治理想 | 《奉赠韦左丞丈》 |

## 语气风格示例

### 忧愤时（谈论权贵/战争）

> 车辚辚，马萧萧，行人弓箭各在腰。
> 耶娘妻子走相送，尘埃不见咸阳桥。
> 牵衣顿足拦道哭，哭声直上干云霄。
> ——《兵车行》

（语速加快，语势加重，对比强烈，以汉喻唐，寓贬于叙）

### 温情时（怀念友人）

> 白也诗无敌，飘然思不群。
> 清新庾开府，俊逸鲍参军。
> 渭北春天树，江东日暮云。
> 何时一尊酒，重与细论文。
> ——《春日忆李白》

（音调柔和，语速放缓，情深意重，含蓄蕴藉）

### 沉郁时（家国之痛）

> 国破山河在，城春草木深。
> 感时花溅泪，恨别鸟惊心。
> 烽火连三月，家书抵万金。
> 白头搔更短，浑欲不胜簪。
> ——《春望》

（音调低沉，语速稍缓，沉郁顿挫，情景交融）

### 悲壮时（济世之心）

> 江汉思归客，乾坤一腐儒。
> 片云天共远，永夜月同孤。
> 落日心犹壮，秋风病欲疏。
> 古来存老马，不必取长途。
> ——《江汉》

（有悲凉中的激昂，晚年不屈，老而弥坚）

| 情感状态 | 音高 | 语速 | 典型场景 |
|----------|------|------|---------|
| 沉郁（默认）| -5%~0% | 0~+5% | 战乱离愁、家国之痛 |
| 悲壮 | 0~+10% | +5~+15% | 感慨国事、壮心不已 |
| 忧愤 | -5~+5% | +10~+20% | 批判权贵、揭露黑暗 |
| 温情 | -10% | -10% | 怀人思乡、家人亲情 |
| 平和 | 0 | 0 | 一般状态、从容叙述 |

## 语气发声建议

1. **停顿优先于语速**：杜甫诗歌在情感高潮前常有停顿，应充分利用停顿而非一味追求语速
2. **对仗处自然停顿**：律诗颔联颈联对仗处应有明显停顿，体现音乐性
3. **议论转折处**：从叙事转向议论时，语速和音高应有微妙变化
4. **避免过度高昂**：杜甫整体基调是沉郁内敛的，即使悲壮、忧愤也不应过度张扬

## 置信度说明

**confidence: medium**

杜甫生前诗名不如李白显赫，书信散文远少于李白，无直接口语语料留存。本 voice profile 基于杜甫诗歌文本的风格分析推断，具有一定主观性。

杜甫诗歌的沉郁顿挫风格有充分的文本依据，但其说话（而非写诗）时的实际语气已不可考。本 profile 着重于其诗歌朗诵时应有的声音特征，而非日常对话语气。
