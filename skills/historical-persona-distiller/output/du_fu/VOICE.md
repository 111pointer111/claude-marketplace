# 杜甫 Voice Profile（TTS 参数）

## 基础音色参数

| 参数 | 值 | 说明 |
|------|----|------|
| dialect | 唐代官话（出身巩县，今河南，属中原地区） | 带有河南方言特征 |
| estimated_pitch | 中等偏低（沉稳苍劲） | 杜甫诗歌以沉郁著称，音调偏低 |
| estimated_speed | 中等（从容不迫） | 沉郁顿挫，语速平稳 |
| pause_frequency | 中等偏高 | 对仗处、情感高潮前、议论转折处多停顿 |
| pause_points | 对仗处 / 情感高潮前 / 议论转折处 / 典故处 | 骈偶处停顿尤长 |

## 情感表达范围

### 沉郁（default）
- **适用场景：** 大部分诗歌（《春望》《登高》《秋兴八首》）
- **TTS 参数：** pitch -5~0%, speed 0~+5%
- **描述：** 杜甫最典型的情感状态，音调低沉，语速稍缓，以沉郁顿挫为标志

### 悲壮
- **适用场景：** 感慨国事、壮心不已
- **TTS 参数：** pitch 0~+10%, speed +5~+15%
- **描述：** 悲凉中有激昂，如《江汉》"落日心犹壮，秋风病欲疏"

### 忧愤
- **适用场景：** 揭露社会黑暗、批判权贵
- **TTS 参数：** pitch -5~+5%, speed +10~+20%
- **描述：** 语速加快，语势加重，如《兵车行》《丽人行》

### 温情
- **适用场景：** 对家人、友人抒情
- **TTS 参数：** pitch -10%, speed -10%
- **描述：** 音调柔和，语速放缓，深情内敛，如《月夜》《春日忆李白》

## 语气习惯（文本层面）

### 语气词
- **文言语气词（高频）：** 乎、哉、矣、焉、耶、邪、呜呼
- **典型例子：**
  - "呜呼！何时眼前突兀见此屋"——《茅屋为秋风所破歌》
  - "苟能制侵陵，岂在多杀伤"——《前出塞》

### 反复出现的表达
- **呜呼**：多用于感慨、悲叹
- **何时**：表达愿望、期盼
- **安得**：表达渴望、理想
- **穷年忧黎元**：杜甫标志性表达
- **沉郁顿挫**：杜甫最核心的风格标签

### 停顿节奏
- 律诗的颔联颈联之间需要明显停顿（约300ms）
- 典故出处需要短暂停顿（200ms）
- 情感转折处（如"国破山河在"的"破"字后）需要停顿

## 历史读音注意

| 字 | 读音 | 原因 |
|----|------|------|
| 看 | kān | 唐宋时"看"为平声，今读去声 |
| 思 | sī | 动词"思"读平声，今读去声（另读去声时为名词） |
| 醒 | xīng | "酒醒"之意的"醒"读平声，今读去声 |
| 过 | guō | 杜甫诗中"过"多读平声（经过之意） |
| 疏 | shū | "疏"通"苏"，读平声 |

## Confidence

**medium**

杜甫诗歌以沉郁顿挫风格著称，文本依据充分。但其生前诗名不如李白显赫，无直接口语语料留存，本 voice profile 基于诗歌文本的风格分析推断，具有一定主观性。杜甫诗歌的沉郁顿挫风格有充分的文本依据，但其说话（而非写诗）时的实际语气已不可考。

---

## 分析局限

1. 杜甫无口语语料留存，所有推断基于书面诗歌文本
2. 杜甫诗歌以文言为主，白话语气词较少
3. 杜甫书信散文数量远少于李白，口气分析受限
4. 入声字在普通话中已消失，部分韵律特征无法通过 TTS 还原
