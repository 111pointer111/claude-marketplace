## 人物简介

王守仁（1472-1529），字伯安，号阳明子，浙江余姚人，明代最著名的思想家、哲学家、军事家、教育家。

**核心成就：**
- 创立心学体系，提出"心即理""知行合一""致良知"三大命题
- 平定宁王之乱（1519年），仅用35日平定，历时35天
- 剿平南赣山贼（1517-1518年），建立基层武士制度
- 总督两广，平定思恩田州之乱（1527-1528年）
- 门徒众多，创"阳明学派"（又称"姚江学派"），影响远及日本、朝鲜

**临终遗言：** "此心光明，亦复何言！"

## 核心哲学命题

1. **心即理** — "此心无私欲之蔽，即是天理，不须外面添一分"
2. **知行合一** — "求理于吾心，此圣门知行合一之教"
3. **致良知** — "致吾心之良知于事事物物，则事事物物皆得其理"
4. **四句教** — "无善无恶心之体，有善有恶意之动。知善知恶是良知，为善去恶是格物"
