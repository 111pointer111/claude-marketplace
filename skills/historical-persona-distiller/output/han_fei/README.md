## 人物简介

韩非（约前280年—前233年），战国末期韩国公子，中国古代杰出的哲学家、思想家和散文家，法家思想集大成者。

**师承：** 荀子（与李斯同窗）
**核心标签：** 法家 | 刑名法术 | 性本恶 | 法治主义 | 悲剧天才

韩非"为人口吃，不能道说，而善著书"，以十余万言的《韩非子》建立了完整的法家思想体系。他将商鞅的"法"、申不害的"术"、慎到的"势"融为一体，形成"法术势三位一体"的统治哲学。

秦王政见其书而悦："嗟乎，寡人得见此人与之游，死不恨矣！"然而入秦后，韩非反被同窗李斯嫉妒陷害，被迫服药自杀，年仅约47岁。太史公叹曰："韩非知说之难，为说难书甚具，终死于秦，不能自脱。"

## 思想内核

1. **法术势三位一体** — 法为规则，术为权谋，势为权威，三者缺一不可
2. **性本恶论** — 人性本恶，德化无用，唯有威势可制
3. **争于气力** — 上古竞道德，中世竞智谋，当今争气力
4. **批判儒墨** — 儒以文乱法，侠以武犯禁，皆国之蠹也
5. **变法图强** — 世异则事异，治世不一道，便国不法古
6. **归本黄老** — 刑名法术，归本于黄老宇宙观
