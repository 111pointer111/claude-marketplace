## 人物简介

**尼古拉·特斯拉**（Nikola Tesla，1856年7月10日—1943年1月7日），塞尔维亚裔美国发明家、电气工程师，被认为是历史上最重要的发明家之一。他发明了交流电（AC）系统、特斯拉线圈、无线电遥控技术，并首次提出无线能量传输的概念。

特斯拉的一生充满戏剧性：他与托马斯·爱迪生决裂，在"电流之战"中击败爱迪生的直流电系统，却在晚年因财务管理和项目规划不善而陷入贫困。他的Wardenclyffe塔项目本欲实现全球无线能源和信息传输，却因J.P. Morgan撤资而失败。

特斯拉终生未婚，将全部精力投入科学研究。他是一个矛盾体：既是严谨的工程师，又是浪漫的预言家；既追求普惠全人类的理想，又在商业上一再失败；既是真正科学洞察力的拥有者，又声称接收过来自行星的信号。
