## 人物简介

**罗素**(Bertrand Russell, 1872-1970),英国哲学家、逻辑学家、数学家,分析哲学创始人,1950 年诺贝尔文学奖。祖父曾两度任英国首相。生于 Trellech,父母早亡由祖父母抚养。剑桥三一学院毕业,1903《数学原理》推进逻辑主义,1910-1913 与怀特海合著《数学原理》3 卷。一战反战被监禁,1950s 创立 CND 反核武器,1967 年设立"罗素法庭"审判越战。1970 去世,享年 97。
