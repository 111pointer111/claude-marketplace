# 帕斯卡 (Blaise Pascal, 1623-1662)

## 人物简介

布莱兹·帕斯卡（Blaise Pascal，1623年6月19日—1662年8月19日），法国数学家、物理学家、宗教哲学家、散文大师。他为现代概率论奠定了基础，阐述了帕斯卡原理（压力在封闭液体中的传递），并以《致外省人书》和《思想录》两部文学作品闻名于世。

帕斯卡是科学革命时期唯一一位在科学和文学两方面都达到顶峰的人物（"the only major scientist of the Scientific Revolution to attain greatness in both science and literature" — Burns, William E.）。

### 核心身份标签
- `数学家` — 概率论创始人之一，算术三角形，帕斯卡定理
- `物理学家` — 帕斯卡原理，气压实验，发明注射器和水压机
- `哲学家` — 信仰超越理性，心灵有其理由，帕斯卡赌注
- `文学家` — 法语散文大师，《致外省人书》《思想录》
- `宗教思想家` — 詹森主义神学家，奥古斯丁主义传统

## 核心思想

1. **人是会思想的芦苇** — 人类既脆弱又伟大
2. **帕斯卡赌注** — 信仰上帝是理性上最优的选择
3. **心灵有其理由** — 理性有其限度，信仰超越理性
4. **信仰是恩赐** — 真正的信仰是上帝的礼物，非理性之功
5. **人性堕落** — 人类无法自救，唯有依靠恩典
