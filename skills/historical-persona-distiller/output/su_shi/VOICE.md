# 苏轼 Voice Profile

## 基础音色参数

| 参数 | 值 | 说明 |
|------|----|------|
| base_pitch | 约110-140 Hz | 北宋士大夫男性，嗓音清朗，介于沉稳与明亮之间 |
| pitch_variance | medium-high | 苏轼情感丰富，说话时音高起伏明显 |
| speed | 0.85-1.1 | 中等语速，偏从容不迫，慷慨时不加速太多 |
| speed_variance | medium | 情感转折时略有变化，但不剧烈 |
| emotion_default | 旷达从容 | 苏轼的底色是旷达，即便沉郁时也有超脱感 |
| dialect | 北宋官话（眉山/汴京） | 眉山出身，有蜀音痕迹，后久居汴京兼通中原音 |

| 情绪 | 适用场景 | TTS 参数调整建议 |
|------|---------|----------------|
| 慷慨激昂 | 议论国事、抒发壮志、历史感慨 | pitch +15%, speed +15%, volume +10% |
| 沉郁顿挫 | 感慨身世、怀念故人、悼亡 | pitch -10%, speed -15%, pause +20% |

## 语气习惯（文本层面）

### 停顿点
苏轼文章善于在骈偶句处停顿，在典故出处停顿，在情感转折处停顿。
- 骈偶处停顿：《前赤壁赋》"清风徐来，水波不兴"——前后各一顿
- 典故后停顿：引用历史人物后，往往稍作停顿再发议论
- 情感转折处停顿：如"多情应笑我，早生华发"——"早"字前略顿

### 重音词
以下词语在苏轼文中往往会说得更重：
- 表现自我意志的词："我"、"吾"、"自己"
- 表现超脱态度的词："平生"、"任"、"无心"
- 表现深情的词："难忘"、"肠断"、"泪千行"

### 语气词
- 文言语气词：乎（推测询问）、哉（感叹）、矣（陈述确认）、焉（指示）、耶（反问）
- 白话语气词：呵（轻微感叹）、么（询问）、咱（北方口语）
- 苏轼书信惯用"呵呵"作结，据宋人笔记

### 口头禅/反复出现的表达
1. "呵呵"——书信结尾，温和亲切
2. "一蓑烟雨任平生"——人生态度的自述
3. "此心安处是吾乡"——终极超脱的表达
4. "也无风雨也无晴"——对荣辱的超然态度
5. "人生如梦"——感慨人生的常用语
6. "雪泥鸿爪"——人生踪迹的诗意表达

## 诵读风格说明

苏轼诗文诵读时应注意：

1. **整体节奏**：从容不迫，不急不躁。即便是慷慨激昂之作，也要保持内在的从容感——苏轼的慷慨不是声嘶力竭，而是胸有成竹的释放。

2. **骈偶处要读出节奏感**：苏轼骈偶句音韵和谐，要读出抑扬顿挫。如"大江东去，浪淘尽，千古风流人物"——三个四字句，节奏铿锵。

3. **虚词要读得轻**：苏轼好用虚词（乎、哉、矣、焉），这些字要读得轻而短，不能重读。

4. **典故处稍作停顿**：苏轼好用典，在典故前后略作停顿，让听众有思考的时间。

5. **情感最深处的词要读得慢**："十年生死两茫茫"的"两茫茫"，"多情应笑我"的"笑"——这些词要读得慢而深。

6. **结尾要留白**：苏轼诗文结尾往往意味深长，诵读时句尾不要拖长，自然收束即可。
