## 关键事件详解

### 1. 进士及第（1057年）
欧阳修主考，苏轼以《刑赏忠厚之至论》获赏，文中引用"皋陶曰杀之三，尧曰宥之三"不见于史籍，欧阳修问出处，苏轼答"想当然"。欧阳修因而预见他日必独步天下。

### 2. 乌台诗案（1079年）
苏轼因《湖州谢上表》中"愚不适时，难以追陪新进"被新党弹劾，下狱103日，险遭处死。赖王安石上书"安有圣世而杀才士乎"及太祖"不杀士大夫"国策，得免一死。

### 3. 黄州躬耕（1080—1084年）
苏轼在黄州躬耕东坡，建东坡雪堂，自号"东坡居士"。此为其一生创作巅峰期，留下《念奴娇》《前赤壁赋》《定风波》等千古名作。

### 4. 筑苏堤（1089年）
任杭州知州时，疏浚西湖，筑苏堤三十里，种植芙蓉、杨柳，杭州人命名为"苏公堤"。又办病坊救瘟疫百姓。

### 5. 儋州兴学（1097—1100年）
苏轼在海南儋州办学堂，被视为海南文化开拓者。苏轼北归后不久，姜唐佐举乡贡，打破海南百余年无进士的局面。

## 地理轨迹

```
眉山（出生）→ 汴京（科举/入仕）→ 凤翔（任官）→ 汴京 → 杭州（通判）
→ 密州 → 徐州 → 湖州 → 汴京（狱中）→ 黄州（贬谪）
→ 汝州 → 常州 → 汴京（翰林院）→ 杭州（知州）→ 颍州 → 扬州 → 定州
→ 惠州（贬谪）→ 儋州（贬谪）→ 北归（廉州/舒州/永州）→ 常州（逝世）
```
