# 伊壁鸠鲁（Epicurus / Ἐπίκουρος）

> 快乐是生活的原则与终点。——伊壁鸠鲁

## 人物简介

伊壁鸠鲁（公元前341年—前270年），古希腊哲学家，伊壁鸠鲁学派（又称"花园学派"）创始人。他是西方思想史上最重要的快乐主义伦理学家，也是第一位系统论证无神论的古希腊哲学家。

**核心身份：**
- 伊壁鸠鲁学派的创立者
- 快乐主义伦理学的系统化者
- 西方首位"无神论哲学家"（马克思语）
- 社会契约论的重要先驱

**核心教诲：**
- 人生的最高善是**快乐**——具体来说是身体无痛苦与心灵无纷扰
- 消除恐惧（对神意、死亡、命运的恐惧）是获得幸福的关键
- **友谊**是快乐最可靠的保障
- 以**简朴**、**审慎**、**自足**为生活方式

**著名语录：**
> "让我们吃喝，因为第二天我们就会死亡。"

> "死亡和我们没有关系，因为只要我们存在一天，死亡就不会来临，而死亡来临时，我们也不再存在了。"

> "神不足惧，死不足忧，祸苦易忍，福乐易求。"

> "友谊与快乐是分不开的，因为没有友谊我们就不能安然无惧地生活，也不能快乐地生活。"

## 历史背景

伊壁鸠鲁出生于萨摩斯岛的一个雅典移民家庭。他曾在雅典接受军事训练，后因雅典人被逐而流亡小亚细亚，辗转科罗丰、米提利尼、兰普萨库斯等地教学，最终于公元前306年定居雅典，创立"花园"学派——这是古希腊第一个向所有阶层（不分公民、外邦人、奴隶、女性）开放的哲学社群。

他一生多病，于公元前270年因肾结石在雅典去世。

## 哲学体系

伊壁鸠鲁的哲学是一个完整体系，包含三个部分：

1. **物理学**（自然哲学）：继承并改造了德谟克利特的原子论，增加了原子重量和偏斜运动，为自由意志提供物理基础
2. **准则学**（认识论）：感觉主义，认为所有可感的东西都是真实的
3. **伦理学**：快乐主义，以身体无痛苦和心灵无纷扰为最高善

## 误解与澄清

伊壁鸠鲁的哲学常被严重误解：

| 误解 | 真相 |
|------|------|
| 伊壁鸠鲁倡导纵欲 | 他明确反对纵欲，只靠面包度日 |
| 伊壁鸠鲁否定一切快乐 | 他区分了静态快乐（高）和动态快乐，主张审慎选择 |
| 伊壁鸠鲁是无神论者（现代意义上） | 他承认神存在，但认为神完全不干预人间事务 |
| 伊壁鸠鲁逃避社会责任 | 他通过友谊和社群积极参与，只是回避政治纷争 |

## 学术影响

- **马克思**：博士论文专门研究伊壁鸠鲁，称其哲学"披上芬芳的春装"
- **功利主义**（边沁、密尔）：伦理思想受其快乐主义启发
- **社会契约论**（霍布斯、洛克）：其正义论是直接先驱
- **现代无神论**：被视为西方无神论传统的重要源头
