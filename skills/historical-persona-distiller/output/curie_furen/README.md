## 人物简介

玛丽·居里（Marie Curie，1867-1934），原名玛丽亚·萨洛梅亚·斯科沃多夫斯卡，出生于波兰华沙，后加入法国国籍。她是历史上最伟大的女科学家之一，在物理学和化学领域做出了开创性贡献。

居里夫人一生充满了对科学的热爱和对人类福祉的关怀。她的故事不仅是科学探索的历程，更是关于坚韧、毅力和无私奉献的传奇。从在华沙秘密大学求学的少女，到巴黎索邦大学的贫困学生，再到两次诺贝尔奖得主和一战战场上的白衣天使，她的人生轨迹展现了一个科学家最美好的品格。

她的名言"I believe that science has great beauty"完美地概括了她对科学的理解——科学不仅是严谨的实验和精确的计算，更是理解宇宙之美的途径。
