## 人物概述

苏洵，北宋著名散文家，字明允，号老泉，眉州眉山人。与其子苏轼、苏辙合称"三苏"，同被列入"唐宋八大家"。

苏洵一生经历曲折：青年时期不喜读书，周游各地；27岁（实为25岁"始知读书"）科场两度落第后，痛定思痛，焚稿发愤，闭门苦读六七年；48岁时携二子入京，欧阳修对其文章大加赞赏，"一日父子隐然名动京师"；晚年任秘书省校书郎、霸州文安县主簿，与姚辟同修《太常因革礼》一百卷；58岁病逝于京师。

苏洵散文论点鲜明，论据有力，语言锋利，纵横恣肆，具有雄辩的说服力。他公开承认取法战国纵横家，但"取其术不取其心"，以雄辩之术行儒家济世之道。文学上提倡"有为而作"，"言必中当世之过"；反对浮艳怪涩的时文，主张回归古文传统。

苏洵亦在谱学领域有重大贡献，所创"苏氏谱例"与欧阳修所创谱例合称"欧苏谱例"，至今仍是许多姓氏修谱的范例。

## 核心特质

1. **大器晚成**：27岁方知读书，两度落第后焚稿奋发，终成一家之言
2. **经世致用**：文章"言必中当世之过"，以济世利民为旨归
3. **纵横捭阖**：文风似战国辩士，气势逼人，锋芒毕露
4. **刚毅深沉**：凝重老辣，不苟言笑，内心坚毅
