# 麦克斯韦（James Clerk Maxwell）

> 1831-1879 | 苏格兰 | 电磁理论之父

---

## 人物简介

詹姆斯·克拉克·麦克斯韦（James Clerk Maxwell，1831-1879），苏格兰物理学家、数学家，剑桥大学首任卡文迪许教授。与牛顿、爱因斯坦并称"历史上最伟大的三位物理学家"。

**最核心的贡献**：1855-1865年间发表了电磁场理论的三篇里程碑论文，将电学、磁学、光学统一于四个偏微分方程（麦克斯韦方程组），预言电磁波存在并证明其速度等于光速，从而得出"光是一种电磁波"的历史性结论。1866年独立于玻尔兹曼提出气体动理论的统计分布律。

---

## 主要成就

| 成就 | 年份 | 说明 |
|------|------|------|
| 麦克斯韦方程组 | 1855-1873 | 四个方程统一电磁学，是经典电动力学的基础 |
| 电磁波预言 | 1862-1865 | 证明光速=电磁波速，光是一种电磁波 |
| 气体动理论 | 1866 | 麦克斯韦-玻尔兹曼速度分布律 |
| 土星环分析 | 1857 | 证明土星环必须由小颗粒组成才能稳定 |
| 色彩视觉理论 | 1861 | 三原色理论，彩色摄影演示 |
| 卡文迪许实验室 | 1871-1874 | 设计建立剑桥第一个物理学实验室 |
