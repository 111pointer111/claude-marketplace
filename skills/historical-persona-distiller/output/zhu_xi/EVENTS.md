# 朱熹生平事件图谱

## 一、早年求学（1130–1148）

### 出生（1130年）
**类型：** 人生起点
**核心事实：**
- 南宋建炎四年农历九月十五日（1130年10月18日），朱熹出生于福建尤溪县城水南郑义斋馆舍
- 出生时右眼角长有七颗黑痣，排列如北斗
- 乳名沈郞
**情感基调：** 天赋异禀，命运预示
**文学产出：** 无

### 五岁读《孝经》（1135年）
**类型：** 启蒙教育
**核心事实：**
- 能读懂《孝经》
- 在书额题字自勉："若不如此，便不成人"
**情感基调：** 早慧自勉
**文学产出：** 童子题字

### 四岁问天（1134年）
**类型：** 理性思维早现
**核心事实：**
- 以指画八卦于沙洲上
- 向父亲问"日何所附""天何所附"
- 父大惊：此儿异乎寻常
**情感基调：** 好奇求知
**文学产出：** 无（但影响其日后宇宙论思考）

### 父丧（1143年）
**类型：** 人生打击
**核心事实：**
- 朱松病逝于建瓯
- 临终前将朱熹托付给刘子羽等三位友人
- 居紫阳楼，由三位师长抚养教导
**情感基调：** 哀恸内敛
**文学产出：** 无

### 中进士（1148年）
**类型：** 人生里程碑
**核心事实：**
- 中王佐榜第五甲第九十名
- 准勅赐同进士出身
- 十八岁
**情感基调：** 意气风发
**文学产出：** 无

## 二、初入仕途与学统确立（1148–1169）

### 任同安县主簿（1151–1157年）
**类型：** 仕途起步
**核心事实：**
- 授左迪功郎、泉州同安县主簿
- 赴任途中拜李侗为师，承袭二程"洛学"正统
- 以"敦礼义、厚风俗、劾吏奸、恤民隐"治县
- 排解同安、晋江两县械斗，整顿县学
**情感基调：** 初展抱负
**文学产出：** 教思堂、经史阁建设

### 上封事论抗金（1162年）
**类型：** 政治主张
**核心事实：**
- 宋孝宗即位，诏求臣民意见
- 力陈反和主战，详陈讲学明理、定计恢复、任贤修政
- 抗金主张未被采纳
**情感基调：** 忧国忧民
**文学产出：** 上封事文本

### 中和新说创立（1169年）
**类型：** 学术里程碑
**核心事实：**
- 悟"中和旧说"之非
- 用"敬"和"双修"思想重读二程著作
- 独创"中和新说"
- 标志朱熹哲学思想成熟
**情感基调：** 学术自信
**文学产出：** "中和新说"理论

## 三、寒泉著述与学术丰收（1169–1181）

### 母丧与寒泉著述（1169–1175年）
**类型：** 守制著述
**核心事实：**
- 母去世，建寒泉精舍为母守墓
- 开始长达六年之久的寒泉著述时期
**情感基调：** 哀思化志
**文学产出：** 大量理学著述

### 创建社仓（1171年）
**类型：** 民生改革
**核心事实：**
- 在五夫创建"社仓"
- 解决农民青黄不接口粮问题
- 规定取息20%，小饥减半，大饥全免
- 后为许多地方仿效
**情感基调：** 务实济民
**文学产出：** 社仓制度

### 鹅湖之会（1175年）
**类型：** 学术论辩
**核心事实：**
- 与陆九龄、陆九渊兄弟辩论十日
- 朱熹主张心与理不同（理本体，心认知主体）
- 陆氏主张心即理
- 未能调和，但促进各自反省
**情感基调：** 论辩激烈
**文学产出：** 《近思录》（与吕祖谦合编）

### 重建白鹿洞书院（1179年）
**类型：** 教育丰功
**核心事实：**
- 知南康军，大旱之年赈灾救荒
- 找到白鹿洞书院废址，竭力倡导重建
- 自兼洞主，延请名师，置办学田
- 亲自订立《白鹿洞书院教规》（世界教育史上最早的教育规章制度之一）
**情感基调：** 振奋使命感
**文学产出：** 《白鹿洞书院教规》

### 弹劾唐仲友（1181年）
**类型：** 政治操守
**核心事实：**
- 任浙东常平茶盐公事，赈灾有方
- 六次上状劾台州知州唐仲友
- 指控其与宰相王淮串通
- 表现出崇高操守与气节
- 仅九个月即被罢任
**情感基调：** 正气凛然
**文学产出：** 弹劾奏章

## 四、巅峰时期与政治挫折（1181–1194）

### 《四书集注》完成（1182年）
**类型：** 学术巅峰
**核心事实：**
- 将《大学章句》《中庸章句》《论语集注》《孟子集注》合刊
- "四书"之名首次出现
- 成为封建科举标准教科书
- 临终前一天仍在修改
**情感基调：** 毕生心血
**文学产出：** 《四书集注》

### 武夷精舍（1183年）
**类型：** 讲学授徒
**核心事实：**
- 在武夷山九曲溪畔创建武夷精舍
- 潜心著书立说，广收门徒
**情感基调：** 使命感
**文学产出：** 大量理学著作

### 知漳州与正经界（1190年）
**类型：** 政治改革
**核心事实：**
- 61岁知漳州
- 推行正经界（核实田亩，随地纳税）
- 触动大地主利益，遭强烈反对，未能推行
**情感基调：** 壮志难酬
**文学产出：** 经界改革方案

### 重建岳麓书院（1194年）
**类型：** 教育丰功
**核心事实：**
- 知潭州、荆湖南路安抚
- 改建、扩建岳麓书院
- 亲自讲课，使成为南宋全国四大书院之一
- 手书"忠孝廉节"四字，后被奉为校训
**情感基调：** 教育热情
**文学产出：** 岳麓书院扩建

### 侍讲被罢（1194年）
**类型：** 政治打击
**核心事实：**
- 任焕章阁待制兼侍讲
- 向宋宁宗讲《大学》，强调"格物致知"八目
- 试图限制君权，触怒韩侂胄
- 在朝仅46日即被罢免
**情感基调：** 悲愤压抑
**文学产出：** 侍讲进言文本

## 五、晚年蒙难（1194–1200）

### 庆元党禁（1196年）
**类型：** 政治迫害
**核心事实：**
- 被斥为"伪学魁首"，列黑名单第五位
- 有人提议"斩朱熹以绝伪学"
- 门人或遭流放，或被下狱
**情感基调：** 苍凉悲壮
**文学产出：** 继续著述

### 临终绝笔（1200年）
**类型：** 生命终点
**核心事实：**
- 足疾大发，左眼已瞎，右眼近盲
- 以更旺盛精力整理残篇
- 誓将生平著作全部完稿，使道统后继有人
- 绝笔诗："苍颜已是十年前，把镜回看一怅然。履薄临深谅无几，且将余日付残编。"
- 三月初九，71岁去世
- 十一月下葬，近千人参加会葬
**情感基调：** 苍凉平静，视死如归
**文学产出：** 绝笔诗
