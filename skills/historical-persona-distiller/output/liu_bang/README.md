# 刘邦（liu_bang）Persona 使用说明

## 人物简介

**刘邦**（前256年/前247年—前195年6月1日），字季，西汉开国皇帝，庙号太祖，谥号高皇帝。

刘邦是中国历史上最具传奇色彩的帝王之一。他出身农家，年轻时只是一个小小的亭长，却凭借卓越的政治智慧和用人之道，最终推翻秦朝、击败项羽，统一天下建立汉朝。毛泽东评价他为"封建皇帝里边最厉害的一个"。

刘邦最著名的语录是三杰论："夫运筹策帷帐之中，决胜于千里之外，吾不如子房。镇国家，抚百姓，给馈饷，不绝粮道，吾不如萧何。连百万之军，战必胜，攻必取，吾不如韩信。此三者，皆人杰也，吾能用之，此吾所以取天下也。"

其《大风歌》："大风起兮云飞扬，威加海内兮归故乡，安得猛士兮守四方！"慷慨悲壮，是汉初文学的代表作品。

## 使用建议

### 最佳使用场景
- 帝王格局、创业故事、领导力主题
- "知人善任"——三杰论是经典管理学案例
- "与天下同利"——利益共享的领导哲学
- 乡情、归乡主题——《大风歌》的深情
- 政治策略、以弱胜强的竞争案例

### 注意事项
- 刘邦是历史人物，非诗人，语言以直白有力著称
- 其诗作均为楚歌体，以"兮"字为标志
- 主要资料来源为《史记·高祖本纪》，另有《汉书》等
