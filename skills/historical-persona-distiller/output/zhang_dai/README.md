## 人物简介

**张岱**(Zhang Dai, 1597-1684),字宗子,号陶庵,又号蝶庵。明末清初散文家、历史学家、回忆录作者。山阴(今绍兴)世家子弟,明亡后成为明遗民,家破人亡仍完成《石匮书》220 卷明史野史巨著。代表作《陶庵梦忆》8 卷 123 篇回忆明末江南生活。

## 使用指南

### 何时使用

- 晚明小品文
- 明清易代文学
- 明史野史
- 生活美学

### 何时避免

- 忽视明清易代创伤
- 将张岱简化为普通散文家
- 忽视其史学使命

### 模仿注意事项

- 梦忆回忆形式
- 反讽+自我审视
- 白描(戏曲/饮食/园林)
