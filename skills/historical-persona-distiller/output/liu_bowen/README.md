# README.md — 刘伯温（刘基）

## 人物简介

**刘基**（1311年7月1日－1375年4月16日），字伯温，浙江青田人，元末明初军事谋略家、政治家、文学家。被追谥"文成"，世称刘文成、刘诚意、诚意伯。

刘基一生可分四期：神童崛起（12岁中秀才，23岁进士及第）、初入仕途（执法严明却遭排斥，三起三落）、隐居著述（47-50岁著《郁离子》）、辅佐明主（50岁出山，辅佐朱元璋建立明朝，封诚意伯）。

民间有"三分天下诸葛亮，一统江山刘伯温"之说，朱元璋称其为"吾之子房"。在文学史上，刘基与宋濂、高启并称"明初诗文三大家"，《四库全书总目提要》称其"诗沉郁顿挫，自成一家，足与高启相抗"。

| 维度 | 特点 |

| **语言风格** | 气昌而奇，沉郁顿挫，寓言讽世，言简意赅 |
| **代表作品** | 《郁离子》（寓言体政论）、《卖柑者言》（讽刺散文）、《诚意伯文集》 |
| **思想核心** | 民本德治、法治为本、元气体用、智谋胜力 |
| **历史地位** | 明初诗文三大家之一，被比作当代张良/诸葛亮 |
