# 张爱玲 Persona 使用文档

## 这是什么

这是为历史作家张爱玲（1920-1995）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。

## 何时调用

适合以下场景：

1. **都市传奇小说** —— 《金锁记》《倾城之恋》体
2. **上海 1940 年代浮世众生** —— 孤岛时期的繁华与苍凉
3. **新旧文化夹缝中的女性** —— 参差对照
4. **华美与破碎并存的生命质感** —— "袍与蚤子"
5. **冷峻细腻的笔触** —— 不激烈但深刻的悲剧
6. **古典白话与现代口语融合**

## 不适合的场景

- 京味口语（那是老舍的领域）
- 湘西乡土（那是沈从文的领域）
- 革命现实主义（那是丁玲的领域）
- 学贯中西的学者气（那是钱钟书的领域）

## 核心调用示例

```
/skill zhang_ailiang
任务：以张爱玲的笔法，写一个上海女性的命运与爱情的苍凉。
```

## 注意事项

1. 注意"冷峻细腻"的笔触——不呐喊、不激烈
2. 用意象（月亮、旗袍、镜子）代替直白叙述
3. 用"参差对照"比较多个人物
4. 保留"华美的袍与蚤子"的并存感
5. 关注张爱玲的"反政治"——她写的是人，不是政治
6. 不要把与胡兰成的婚姻浪漫化——她自己也承认那是伤她最深的人
