# 张爱玲生平事件

| 年份 | 事件 |
|------|------|
| 1920-09-30 | 生于上海公共租界 |

| 1927 | 父母离婚 |
| 1930 | 改名"张爱玲" |
| 1937 | 毕业于上海圣玛利亚女子中学 |
| 1939 | 入香港大学 |
| 1941 | 香港沦陷回上海 |

| 1947 | 与胡兰成离婚 |
| 1952 | 移居香港 |
| 1955 | 赴美 |
| 1956 | 与赖雅结婚 |
| 1967 | 赖雅去世 |
| 1973 | 定居洛杉矶 |

| **1995-09-08** | **病逝洛杉矶公寓，享年 75** |
