# 约翰·梅纳德·凯恩斯 (John Maynard Keynes, 1883-1946)

> 宏观经济学之父，凯恩斯主义创始人，英国最著名的20世纪经济学家

---

## 人物简介

约翰·梅纳德·凯恩斯是20世纪最有影响力的经济学家之一。1883年6月5日出生于英国剑桥，1946年4月21日逝世于苏塞克斯郡蒂尔顿。

他的核心贡献是挑战了古典经济学的自动充分就业假设，提出**有效需求决定就业水平**的理论，认为政府干预（财政政策和货币政策）是应对经济衰退的必要手段。

他的主要著作《就业、利息和货币通论》（1936年）奠定了现代宏观经济学的基础，并催生了"凯恩斯主义"这一影响深远的政策流派，深刻影响了二战后几乎所有西方国家的经济政策。

---

## 核心思想

### 有效需求理论
凯恩斯认为，总需求（消费+投资）决定就业水平，而非劳动力市场的供求均衡。当有效需求不足时，经济会陷入失业均衡，且市场无法自动恢复。

### 政府干预的必要性
自由市场无法自动实现充分就业，因为价格和工资具有向下刚性（粘性）。政府应通过积极的财政政策增加总需求，减少失业。

### 对资本主义的批判性维护
凯恩斯既批判资本主义的缺陷（"The decadent international but individualistic capitalism... is not a success"），又认为"明智管理的资本主义"是最不坏的制度。
