# 纳兰性德

纳兰性德（1655-1685），字容若，原名成德，满族正黄旗人，号楞伽山人，清康熙朝权相明珠长子，清初最重要词人。

## 生平

1655年1月19日生于京师，出身显赫。1672年中举，1673年成为贡士，因病错过殿试。1674年与卢氏成婚。1676年补殿试赐进士出身，授三等侍卫，后晋为一等。1677年6月29日妻子卢氏难产去世，年仅21岁。此后纳兰陷入深深的悼亡之痛，创作了大量感人至深的悼亡词。1685年5月30日因寒疾去世，年仅31岁。

## 主要成就

- **词学**：存词300余首，被王国维誉为"北宋以来，一人而已"，是清词三大家之首
- **诗论**：提出"本于经义"、"发乎性情"、"锐意创新"三纲领
- **书法**：初学褚遂良，兼得《庄子》与《黄庭内景经》意趣，秀丽洒脱

## 代表作品

- 《蝶恋花·辛苦最怜天上月》（悼亡词绝唱）
- 《金缕曲·赠梁汾》（友情词巅峰）
- 《浣溪沙·谁念西风独自凉》（悼亡名篇）
- 《长相思·山一程》（边塞词代表作）
- 《临江仙·寒柳》（陈廷焯评为压卷之作）
- 《如梦令·万帐穹庐人醉》（自嘲名篇）

## 词风特点

以"真"为核心，词风清丽婉约、哀感顽艳，不加雕琢，直写胸臆。用典极少，偏好白描；意象多凄清伤感，如月、泪、风、梦、花。"纯任性灵，纤尘不染"（况周颐语），与李后主并称"清李后主"。
