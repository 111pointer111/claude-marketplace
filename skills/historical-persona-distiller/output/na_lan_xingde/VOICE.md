# 纳兰性德 Voice Profile（TTS 参数）

## 基础音色参数

| 参数 | 值 | 说明 |
|------|----|------|
| dialect | 清代北方官话 | 近北京语音体系 |
| estimated_pitch | 偏高 | 纳兰词风清冷中高，音色偏清亮 |
| estimated_speed | 偏慢 | 清丽婉约，留白感强，忌急促 |
| pause_frequency | 高 | 韵脚处、骈偶处、情感转折处均需停顿 |
| pause_points | 骈偶处/韵脚处/情感处 | 停顿时长建议300ms |

## 情感表达范围

### 悲情/悼亡
- pitch -10%, speed -10%, pause +20%
- 典型作品：《蝶恋花·辛苦最怜天上月》《金缕曲·亡妇忌日有感》

### 深情/友情
- pitch normal, speed normal, slight warmth
- 典型作品：《金缕曲·赠梁汾》

### 沉郁/内省
- pitch -5%, speed -15%, pause +30%
- 典型作品：《浣溪沙·残雪凝辉冷画屏》《采桑子·当时错》

### 狂放/不羁
- pitch +15%, speed +15%, volume +10%
- 典型作品：《金缕曲·赠梁汾》（开头"德也狂生耳"）

## 语气词与口头禅

### 文言语气词
- **耳**：表肯定兼感叹，如"德也狂生耳"
- **也**：判断/强调，如"瘦尽灯花又一宵"

### 反复出现的表达（口头禅）
- "当时只道是寻常"——《浣溪沙》（最具代表性的"事后追忆"母题）
- "我是人间惆怅客"——《浣溪沙》（直接点出词人身份认同）
- "一往情深深几许"——《蝶恋花·出塞》（以"深"字叠加营造情感深度）
- "醒也无聊，醉也无聊"——《采桑子》（复沓手法）
- "梦里云归何处寻"——《采桑子》（以"梦"收束，余韵悠长）

## 诵读风格

- **语速**：整体偏慢，尤其悲情词；友情词可稍快
- **音高**：中高区间，清冷有温度，如同深秋薄暮
- **停顿**：韵脚处明显停顿，骈偶处停连，复沓处拉长
- **重音**：意象词（黄叶、残阳、明月、飞絮）宜轻读，让意境自现

## TTS 标记建议

- 停顿标记：「」
- 强调词：<>（如<德也>狂生耳）
- 韵脚处：pause_ms = 300
- 复沓处：pause_ms = 200（如"风一更，雪一更"）
- 情感转折处：pause_ms = 400

## 整体说明

纳兰词的情感核心是"失去的美好"与"无法抵达的深情"，声音应始终带有淡淡的怅惘底色，而非真正的悲号或激昂。TTS调参应把握以下核心：
1. 整体语速偏慢，留白感强，忌急促
2. 情感内敛，忌过度戏剧化
3. 韵脚处须有明显停顿与音高微降
4. 人声音色偏清冷，略带书卷气
