# 乔伊斯生平事件图谱

## 1882-1902：都柏林成长与教育

| 年份 | 事件 |
|------|------|
| 1882年2月2日 | 出生于爱尔兰都柏林 |
| 1888年（6岁） | 进入克朗戈维斯Wood学院（Jesuit寄宿学校） |
| 1891年 | 因父亲经济困境，家庭陷入贫困，离开克朗戈维斯 |
| 1891-1893年 | 在家自学两年 |
| 1893年 | 进入贝尔维德尔学院（Jesuit语法学校） |
| 1893-1902年 | 在贝尔维德尔学院学习，失去天主教信仰 |
| 1900年（18岁） | 在《伦敦双周评论》发表关于易卜生的文章 |
| 1901年10月 | 发表论文《平民的暴动》攻击爱尔兰文学剧院 |
| 1902年10月31日 | 获得都柏林大学学院学士学位 |

## 1904-1915：流亡与自我放逐

| 年份 | 事件 |
|------|------|
| 1904年6月16日 | 遇见诺拉·巴纳克尔（Bloomsday的由来） |
| 1904年10月 | 与诺拉私奔离开爱尔兰，定居奥地利帝国普拉 |
| 1905年 | 迁至的里雅斯特，弟弟斯坦尼斯劳斯加入 |
| 1906-07年 | 在罗马一家银行工作8个月 |
| 1909年 | 两次回爱尔兰试图出版《都柏林人》和建立电影院，均失败 |
| 1904-1907年 | 创作《都柏林人》短篇 |
| 1914年 | 《都柏林人》出版 |

## 1915-1922：战争流亡与《尤利西斯》

| 年份 | 事件 |
|------|------|
| 1915年 | 意大利参战，弟弟被扣为战俘；获准前往瑞士苏黎世 |
| 1916年 | 《一个青年艺术家的画像》出版 |
| 1917-1930年 | 经历25次眼部手术（虹膜炎、青光眼、白内障） |
| 1918-1920年 | 《尤利西斯》部分在《小型评论》连载，后被禁 |
| 1922年 | 《尤利西斯》由莎士比亚书店的西尔维亚·比奇在巴黎出版 |

## 1922-1941：《芬尼根守灵夜》创作与晚年

| 年份 | 事件 |
|------|------|
| 1928-1937年 | 《芬尼根守灵夜》以"进行中的工作"形式连载 |
| 1931年 | 与诺拉正式结婚（同居27年后） |
| 1934年 | 女儿露西亚被诊断为精神分裂症 |
| 1939年 | 《芬尼根守灵夜》完整出版 |
| 1941年1月13日 | 在苏黎世去世，葬于苏黎世弗卢姆利公墓 |

## 关键引语

> "Welcome, O life! I go to encounter for the millionth time the reality of experience and to forge in the smithy of my soul the uncreated conscience of my race."
> ——《一个青年艺术家的画像》

> "I am really one of the greatest engineers….I am making an engine with only one wheel. No spokes of course. The wheel is a perfect square….No, it's a wheel, I tell the world. And it's all square."
> ——1927年，谈论《芬尼根守灵夜》
