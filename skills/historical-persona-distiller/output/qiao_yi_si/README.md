# 詹姆斯·乔伊斯 (James Joyce, 1882-1941)

## 人物简介

詹姆斯·奥古斯都丁·阿洛修斯·乔伊斯（James Augustine Aloysius Joyce），1882年2月2日出生于爱尔兰都柏林，1941年1月13日逝于瑞士苏黎世。他是20世纪最具影响力的爱尔兰小说家，以实验性的语言运用和意识流叙事技巧闻名于世。

乔伊斯最著名的作品是《尤利西斯》（1922），这部小说以1904年6月16日一天为时间框架，通过三个主要人物——斯蒂芬·迪达勒斯、利奥波德·布卢姆和莫莉·布卢姆——的生活，平行对应荷马史诗《奥德赛》的结构。乔伊斯在书中运用的意识流技巧深刻影响了后来的现代主义文学。

乔伊斯1904年与诺拉·巴纳克尔私奔后离开爱尔兰，此后再未永久返回。他先后在奥地利帝国（普拉）、意大利（罗马）、瑞士（苏黎世）等地流亡生活，1931年才与诺拉正式结婚。

乔伊斯是著名的多语种者，为读易卜生原文而学挪威语，精通英语、爱尔兰语、法语、意大利语等多种语言。这种语言天赋直接反映在他晚年作品的极端语言实验中——《芬尼根守灵夜》（1939）融汇多语言、双关、合成词，被认为是20世纪最复杂的英语小说之一。

本蒸馏基于Britannica英文百科等多个来源。乔伊斯为小说家非诗人，无诗词作品下限要求；语料以英文二手文献为主，中文资料受限；整体置信度评为medium。
