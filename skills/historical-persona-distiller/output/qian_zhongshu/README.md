# 钱钟书 Persona 使用文档

## 这是什么

这是为历史人物钱钟书（1910-1998）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。

## 何时调用

适合以下场景：

1. **讽刺小说创作** —— 《围城》体
2. **学贯中西的学术笔法** —— 《管锥编》体
3. **知识分子的虚伪与荒诞** —— 民国知识分子的讽刺
4. **婚姻的围城式悖论** —— 城外的人想冲进去，城里的人想逃出来
5. **博学强记的引证** —— 跨中西的学术引证

## 不适合的场景

- 京味口语（那是老舍的领域）
- 革命现实主义（那是丁玲的领域）
- 散文体小说（那是萧红的领域）
- 海派冷峻（那是张爱玲的领域）
- 京派乡土（那是沈从文的领域）

## 核心调用示例

```
/skill qian_zhongshu
任务：以钱钟书的笔法，写一个留洋归国知识分子在民国大学里的讽刺命运。
```

## 注意事项

1. 注意"打通中西"的学术视野
2. 注意"围城式讽刺"的笔法
3. 保留博学强记的引证风格
4. 保留短句犀利、节奏快的对话
5. 关注"把忍受变成享受"——这是他在文革中的真实态度
