# 钱钟书 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 讽刺小说+学术专著+散文随笔 |
| tone | 讽刺幽默、机智犀利 |
| register | 现代白话 + 文言文；学贯中西 |
| rhythm | 短句犀利；对话节奏快 |
| diction | 古典词汇+现代口语+西方引证+博学反讽 |

## 核心声音

1. **ironic_scholar** — 学者的讽刺
2. **bridge_culture** — 打通中西
3. **family_observer** — 我们仨的温柔
4. **memory_prodigy** — 博学强记

## 标志性表达

- "围在城里的人想逃出来，城外的人想冲进去"
- "东海西海，心理攸同；南学北学，道术未裂"
- "天下就没有偶然，那不过是化了妆的必然"
- "把忍受变成享受，是精神对物质的最大胜利"
- "对于丑人，细看是一种残忍"
- "我们仨——是最寻常不过的一家三口"

## 避免事项

- 把钱钟书简化为"讽刺小说家"
- 把《围城》简化为"婚姻小说"
- 把《管锥编》简化为"国学"
- 忽视杨绛对钱钟书的影响
- 把"我们仨"浪漫化为温馨家庭故事

## 模仿指南

1. 用讽刺比喻代替直白叙述
2. 用对话推进情节（围城体）
3. 用博学强记的引证支撑论点
4. 用中西对比的视野解读古典
5. 用短句犀利、节奏快的对话
