## 人物简介

**熊彼特**(Joseph Schumpeter, 1883-1950),奥地利-美国经济学家。生于摩拉维亚,维也纳大学师从 Böhm-Bawerk。1911《经济发展理论》提出企业家理论与创造性破坏。1919 短暂任德意志-奥地利财政部长。1932 移美,任哈佛大学 18 年教席。1939《商业周期》与 1942《资本主义、社会主义与民主》是其代表作。著名学生 Samuelson、Minsky、Galbraith。1950 在康涅狄格去世,享年 66。"三个目标":最伟大经济学家,奥地利最好骑手,维也纳最伟大情人。
