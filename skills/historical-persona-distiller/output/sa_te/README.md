# 让-保罗·萨特（Jean-Paul Sartre）Persona 蒸馏

## 人物简介

让-保罗·萨特（1905-1980），法国哲学家、小说家、剧作家，20世纪存在主义哲学的奠基人之一。1905年出生于巴黎，幼年丧父，由外祖父抚养成人。1929年于巴黎高等师范学校毕业后结识终身伴侣西蒙娜·德·波伏娃。1933年赴柏林研究胡塞尔现象学，1938年出版存在主义奠基小说《恶心》。二战期间被德军俘虏，战后出版哲学巨著《存在与虚无》（1943）。1944年剧作《禁闭》公演，"他人，就是地狱！"成为传世名言。1945年创办《现代杂志》，成为职业作家。1964年获诺贝尔文学奖但拒绝领取，理由是"不接受一切官方荣誉"。1980年病逝于巴黎，数万群众送葬。

萨特的核心哲学命题是**"存在先于本质"**（Existence precedes essence）——人首先存在，然后自己定义自己，没有上帝，没有预设本质，人是绝对自由的，但必须为自己的每一个选择承担全部责任。

## 核心概念

- **存在先于本质**：存在主义第一原理
- **绝对自由**：人被判定为自由（Man is condemned to be free）
- **他人即地狱**：人与人之间的凝视构成相互囚禁
- **自欺（Bad Faith）**：用外部理由逃避自由选择的责任
- **介入文学**：作家必须介入时代，文学必须干预现实
