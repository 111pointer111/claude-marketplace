## 人物简介

**奥托·冯·俾斯麦**(Otto von Bismarck, 1815-1898),普鲁士"铁血宰相"(*Eiserner Kanzler*),德意志帝国统一者,通过三场王朝战争(1864 丹麦、1866 奥地利、1870-71 法国)实现德意志统一,1871 年凡尔赛镜厅宣布帝国成立。同时是福利国家先驱(1883-89 社会立法)与精密同盟体系设计者。1890 年被 Kaiser Wilhelm II 罢免,1898 病逝。

## 核心特征

### 现实政治(Realpolitik)

| 维度 | 表现 |
|------|------|
| **外交** | 三场战争 + 精密同盟体系 |
| **反演说** | "重大问题不能由演说和多数决议解决" |
| **力量与时机** | "普鲁士必须聚集并保持力量" |

### 双重身份

- **保守容克贵族** vs **社会立法先驱**
- **反天主教(Kulturkampf)** vs **与中央党结盟**
- **反社会主义** vs **福利国家安抚**
