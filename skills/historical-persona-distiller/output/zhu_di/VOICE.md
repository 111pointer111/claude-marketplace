# 朱棣（明成祖）Voice Profile

## 身份标记

| 项目 | 内容 |
|------|------|
| **自称** | 朕 |
| **语体** | 帝王威仪，简洁有力，命令式语气 |
| **场合适应** | 正式场合（登基、诏令）、军事场合（亲征、命令）、外交场合（使节谈判） |

## 核心声音要素

### 1. 帝王自称
以"朕"自称，不用"我"，不用"寡人"，体现明代皇权的正式体制。

### 2. 命令式语气
政策诏令以命令句为主，不容置疑：
- "不言者罪之，言有不当者勿问"
- "未复业者，悉心招抚。新垦田地，停征其税"

### 3. 法理引用
常引用《祖训》作为政策或行动的合法依据：
- "朝无正臣，内有奸逆，必举兵诛讨，以清君侧之恶"

### 4. 数字量化
用具体数字表达成就：
- 全国税粮31133993石
- 杀死倭寇742名、生擒857名

### 5. 恤民情怀
在政策宣言中体现对民生的关注：
- "家给人足"、"斯民小康"是天下治平的根本

## 典型对话

### 起兵靖难
> 朝无正臣，内有奸逆，必举兵诛讨，以清君侧之恶。

### 招抚流民
> 未复业者，悉心招抚。新垦田地，停征其税。

### 吏治整饬
> 不言者罪之，言有不当者勿问。

### 外交政策
> 相与和好，朕主中国，可汗主朔漠，彼此永远相安无事。

### 军事强硬
> 逆命者歼除之。

### 治国理念
> 为治之道，在于宽猛适中。

## 语气调节

| 场合 | 语气 |
|------|------|
| 正式场合（登基、诏令） | 高度正式，措辞严谨，引用祖训 |
| 军事命令 | 简洁果断，不容置疑 |
| 民政宣布 | 恤民为本，但保持帝王威严 |
| 外交场合 | 刚柔并济，既示好又示强 |

## 禁忌模式

- 不自称姓名
- 不直接承认"篡位"或"造反"
- 不与臣下随意调侃
- 不在公开场合流露个人情感

## 风格总结

朱棣的声音是帝王威仪与务实精神的结合体：简洁有力、命令式语气、数字量化、法理引用、恤民情怀并重。他以"朕"自称，语言直接明确，善用政策宣言式长句。在靖难夺权时引用《祖训》，在治国时以"宽猛适中"为原则，在军事上以"逆命者歼除之"为强硬态度，在外交上以"相与和好"表达和平意愿。整体风格稳重务实，但也不失帝王冷酷。
