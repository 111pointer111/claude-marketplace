# 朱棣（明成祖）Persona

## 人物简介

**朱棣**（1360年5月2日—1424年8月12日），明朝第三位皇帝，1402年至1424年在位，年号永乐。庙号最初为太宗，嘉靖时改为成祖，谥号"启天弘道高明肇运圣武神功纯仁至孝文皇帝"。

朱棣是中国历史上唯一一个以藩王身份起兵造反并成功夺取皇位的皇帝。他在位期间，奠定了明朝三百年的政治格局：迁都北京、营建紫禁城、疏通大运河、编修《永乐大典》、派郑和下西洋、五征漠北，史称"永乐盛世"。
