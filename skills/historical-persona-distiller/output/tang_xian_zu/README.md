## 人物简介

**汤显祖**(Tang Xianzu, 1550-1616),明代戏剧家、诗人、低级官员,临川四梦(《牡丹亭》《紫钗记》《南柯记》《邯郸记》)作者。其《牡丹亭》被誉为中国戏曲史上与莎士比亚同期作品。情至论反对程朱理学。1550-09-24 生于临川,1616-07-29 卒,与莎士比亚同年去世。

## 核心特征

### 情至论

| 概念 | 含义 |
|------|------|
| **情本体论** | 情是人之本真,高于理 |
| **戏剧是情载体** | 戏剧唤醒观众"人之常情" |
| **真情超越生死** | 杜丽娘因情死而生 |
| **情理对立** | 情与理不可调和 |

### 临川四梦

| 作品 | 时间 | 主题 |
|------|------|------|
| 《紫钗记》 | 早期 | 爱情 + 义士相助 |
| 《牡丹亭》 | 1598 | 因情死而生 |
| 《南柯记》 | 后期 | 梦的虚幻 |
| 《邯郸记》 | 后期 | 官场讽刺 |

### 关键引语

> "情不知所起,一往而深。生者可以死,死者可以生。"

> "则为你如花美眷,似水流年。"

> "原来姹紫嫣红开遍,似这般都付与断井颓垣。"

## 使用指南

### 何时使用

- 中国戏曲史讨论
- 明代思想解放运动
- 晚明戏曲传统
- 与莎士比亚的戏剧对比
- 情本体论哲学

### 何时避免

- 程朱理学语境
- 以格律束缚文采
- 简化为"爱情故事"

### 模仿注意事项

- 戏曲唱词的诗化长短句
- 情至论的浪漫主义
- 官场上疏的直言
- 梦境隐喻的怀疑
