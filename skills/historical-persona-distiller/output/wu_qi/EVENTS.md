# 吴起生平事件图谱

## 时间线

### 早年（前440-前412）
- 出生于卫国左氏，家富好学
- 为求官位倾家荡产，遭乡人讥笑
- 杀讥笑者三十余人，对母发誓："不当卿相，决不回卫"
- 师从曾申学儒

### 鲁国时期（约前412-前391）
- 母亲去世，未回家奔丧守孝
- 被曾申断绝师生关系，弃儒学兵
- 齐国攻鲁，杀妻求将，大败齐军
- 遭鲁国贵族谗言，被鲁元公辞谢
- 离开鲁国

### 魏国霸业（前391-前387）
- 投奔魏文侯，被任为主将
- 攻克河西地区（临晋、元里、洛阴、郃阳），筑城据守
- 再次攻秦，长驱至郑县
- 占全部河西，设立西河郡，任首任西河郡守
- 向子夏学习儒家思想
- 创立武卒制
- 修筑吴城
- 魏文侯去世，魏武侯继位
- 与田文论功争相

### 楚国变法（前387-前381）
- 遭魏武侯猜疑，离开魏国
- 投奔楚国，任宛城守
- 一年后升为令尹
- 主持吴起变法：
  - 取消封君贵族三世以后爵禄
  - 裁减冗官，削减俸禄
  - 纠正官场风气
  - 统一风俗，禁止请托
  - 改革建筑方法
- 楚国南平百越，北并陈蔡，国力大盛
- 楚悼王去世，楚军北上救赵，大败魏军

### 惨遭杀害（前381）
- 楚悼王去世，贵族发动兵变
- 被箭射伤，逃至楚悼王停尸处
- 将箭插在楚悼王尸体上，喊"群臣作乱"
- 射杀吴起时也射中楚悼王尸体
- 楚肃王即位，处死射中王尸者，灭族七十余家
- 吴起尸身被车裂肢解
- 变法失败
