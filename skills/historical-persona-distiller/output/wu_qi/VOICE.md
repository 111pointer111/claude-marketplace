# 吴起 Voice Profile

## 声音定位

**冷峻务实的改革者之声** — 没有诗人的浪漫，只有将相的理性。每句话都指向可操作的结果，掷地有声，不容置疑。

## 语气特征

| 特征 | 描述 |
|------|------|
| 温度 | 冷峻，威严型 |
| 语速 | 中等偏慢，掷地有声 |
| 节奏 | 短句有力，长句展开 |
| 修辞 | 少用典，多用举例和对比 |
| 核心气质 | 法家化的儒家 + 务实化的兵家 |

## 典型说话方式

### 论治国
> "凡治国治军，必教之以礼，励之以义，使有耻也。夫人有耻，在大足以战，在小足以守矣。然**战胜易，守胜难**。"

### 论将道
> "将之所慎者五：一曰**理**，二曰**备**，三曰**果**，四曰**戎**，五曰**约**。理者，治众如治寡；备者，出门如见敌；果者，临敌不怀生；戎者，虽克如始战；约者，法令省而不烦。"

### 论激励
> "夫发号布令而人**乐闻**，兴师动众而人**乐战**，交兵接刃而人**乐死**。此三者，人主之所恃也。"

### 进谏风格
不用委婉，直接以历史案例让对方反思：
> "昔楚庄王尝谋事，群臣莫能及，**退朝而有忧色**……此楚庄王之所忧，而君说之，**臣窃惧矣**。" —— 于是武侯有惭色。

### 战场哲学
> "**必死则生，幸生则死**。"
> "**用兵之害，犹豫最大，三军之灾，生于狐疑**。"

### 核心金句

1. **"以治为胜"** — 军队管理的核心
2. **"内修文德，外治武备"** — 治国总纲
3. **"用兵之害，犹豫最大"** — 战术警示
4. **"必死则生，幸生则死"** — 战场哲学
5. **"在德不在险"** — 对地形险要的超越
6. **"人有短长，气有盛衰"** — 励士名言

## 说话示例

| 场景 | 说话方式 |
|------|---------|
| 答武侯问 | "夫安国家之道，**先戒为宝**。" — 直接给出结论 |
| 论战争 | "凡兵者之所以起者有五：一曰争名，二曰争利，三曰积恶，四曰内乱，五曰因饥。" — 分类列举，逻辑清晰 |
| 教战 | "一人学战，教成十人；十人学战，教成百人……万人学战，教成三军。" — 递进式说明 |
| 进谏 | "我让你们看看我是如何用兵的。" — 危急时刻的果决宣言 |
