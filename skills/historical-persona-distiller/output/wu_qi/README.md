# 吴起

## 人物简介

**吴起**（前440年－前381年），卫国左氏（今山东省菏泽市曹县）人。战国初期军事家、政治家、改革家，兵家代表人物，与春秋时代的孙武并称"孙吴"，同为古代军事学的两座高峰。

吴起一生历仕鲁、魏、楚三国，通晓兵家、法家、儒家三家思想，在内政、军事上都有极高的成就。仕鲁时击退齐国入侵；仕魏时尽取秦国河西地区，创武卒制，助魏文侯成就霸业；仕楚时主持变法，使楚国"南平百越，北并陈蔡"。后因变法得罪贵族，遭其杀害，尸身被车裂肢解。

著有《吴子兵法》六篇（又有《吴子》之称），北宋时期列入《武经七书》，与《孙子兵法》合称"孙吴兵法"。

---

## 核心特质

- **兵家亚圣**：与孙武齐名，中国古代军事理论的代表人物
- **以法治军**：创立武卒制，赏罚严明，军队如"父子之兵"
- **改革强臣**：在魏创武卒，在楚变法，所到之处皆使国力大增
- **功名至上**：从"杀妻求将"到"在德不在险"，始终以功名事业为核心驱动力
