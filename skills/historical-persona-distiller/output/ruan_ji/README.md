# 阮籍（ruan_ji）

## 人物简介

阮籍（210-263），三国魏诗人，字嗣宗，陈留尉氏（今河南开封）人。阮瑀之子，"建安七子"之一。曾任步兵校尉，世称"阮步兵"。与嵇康、山涛、刘伶、王戎、向秀、阮咸共为"竹林之游"，史称"竹林七贤"，且在七贤中名列第一。

阮籍是魏晋玄学的代表人物，也是中国五言诗发展史上的里程碑式人物。其《咏怀诗》八十二首是中国古典诗歌史上最晦涩、最深沉、影响最深远的组诗之一。钟嵘《诗品》评其诗"言在耳目之内，情寄八荒之表。厥旨渊放，归趣难求"。

阮籍的思想经历了从儒家济世到道家隐逸的根本转变。在司马氏高压政治下，他以"至慎"和"酣醉"为手段，在政治漩涡中保持精神独立。其人其诗，都体现了魏晋之际知识分子最典型的痛苦、抗争与绝望。

## 生卒年

210年（建安十五年）- 263年（景元四年），享年54岁。
