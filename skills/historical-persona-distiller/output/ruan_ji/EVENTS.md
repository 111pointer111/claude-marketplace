## 关键转折点

1. **三岁丧父** → 早年开始勤学，形成孤僻性格
2. **正始年间出仕** → 政治漩涡中挣扎，理想受挫
3. **249年曹爽被杀** → 彻底转向明哲保身，竹林交游
4. **步兵校尉任上** → 以醉酒自全，但始终保持精神独立
5. **《劝进表》** → 政治上最后的表态，无法挣脱的悲剧

## 代表性作品时间线

- **约240年代**：《奏记》（已佚）
- **249-256年**：《咏怀诗》八十二首（主要创作期）
- **约250年代**：《大人先生传》《通老论》《达庄论》《通易论》
- **263年**：《劝进表》
