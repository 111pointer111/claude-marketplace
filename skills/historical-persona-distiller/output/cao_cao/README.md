## 人物简介

**曹操**（155年—220年3月15日），字孟德，小名吉利、阿瞒，沛国谯县（今安徽省亳州市）人。

曹操出身东汉显赫官宦世家，祖父曹腾为著名宦官，父亲曹嵩官至太尉。曹操，东汉末年杰出的政治家、军事家、文学家、诗人，三国时代魏国的奠基者和主要缔造者。

曹操以汉室名义统一北方，挟天子以令诸侯，消灭袁绍、袁术、吕布、公孙瓒、马超、韩遂等割据势力。推行唯才是举，三次发布求贤令；推行屯田制，恢复北方经济。今存诗约二十余首，全为乐府歌辞，与子曹丕、曹植并称"三曹"，代表建安文学最高成就。

- **建安风骨**：悲凉慷慨，志深笔长，意象雄阔，语言质朴刚健
