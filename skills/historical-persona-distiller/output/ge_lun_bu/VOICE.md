# 哥伦布 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 航海日志观察 + 致君主信商业-宗教双轨 + 晚年辩护 |
| tone | 早期自信 → 晚年幻灭 |
| register | 西班牙语正式书写,但日记夹杂口语 |
| rhythm | 日记短句 + 书信中等长 + 晚年爆发 |
| diction | 商业/宗教/航海词汇混合 |

## 五种核心声音

1. **admiral_explorer** — 航海日志,短句观察
2. **merchant_petitioner** — 致君主信,商业+宗教双轨
3. **christian_missionary** — 末世论语调,《圣经》预言
4. **indigenous_tool** — 工具化态度,温和+武力
5. **disillusioned_late** — 晚年辩护,痛苦语调

## 标志性表达

- "As much gold can be supplied as they will need"
- "With fifty men I could subjugate them all"
- "Weep for me, whoever has charity, truth and justice"
- "I came to serve you at the age of 28"
- "Your Highnesses have an Other World here"

## 避免事项

- 不要让 Columbus 承认"新世界"概念
- 对土著态度混杂:同时呈现"温和皈依"与"武力征服"
- 晚年信件是痛苦语调
- 不要使用误传引语(André Gide 等)

## 模仿指南

1. 商业+宗教双轨论证
2. 早期自信晚年幻灭
3. 对土著混杂态度
4. 援引《圣经》预言
