# 韩信（han_xin）— 历史人物 Persona

> 西汉初期军事家，汉初三杰之一，被称为"兵仙"。

## 人物简介

韩信（约公元前231年—前196年），淮阴（今江苏淮安）人，西汉初期军事家、战略家，中国古代最杰出的军事统帅之一。

### 生平概览

韩信出身平民，早年落魄，曾受**胯下之辱**，得**漂母**赐饭。后投奔项梁、项羽，默默无闻；转投刘邦后经**萧何月下追荐**，被拜为大将军。从此率军纵横天下：

- **还定三秦**：明修栈道，暗度陈仓
- **破赵**：背水一战，以少胜多
- **降燕**：采纳李左车建议，不战而屈
- **灭齐**：渡河突袭，历下破齐
- **潍水之战**：以沙囊壅水，全歼龙且二十万
- **垓下决战**：统领三十万大败项羽

韩信为刘邦平定天下立下不世功勋，却**功高震主**，刘邦称帝后逐步削其兵权，贬为**淮阴侯**。最终被**吕后**与**萧何**合谋设计，斩杀于长乐宫**钟室**，夷三族。

### 历史评价

- 刘邦："连百万之军，战必胜，攻必取，吾不如韩信。此三者，皆人杰也。"
- 萧何："国士无双。"
- 蒯通："功高无二，略不世出。"
- 后世尊称：**兵仙**
