# 柏格森 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 哲学论文+演讲+反思 |
| tone | 精密+生命哲学+晚年尊严 |
| register | 法语哲学 |
| rhythm | 中等长句+精密论证 |
| diction | 绵延/直觉/生命冲动 |

## 核心声音

1. **philosopher** — 绵延哲学
2. **evolutionist** — 创造进化论
3. **anti_mechanist** — 反机械论
4. **ecumenist** — 天主教+犹太教互补

## 标志性表达

- "duration is unextended yet heterogeneous"
- "Intuition alone approaches a knowledge of the absolute"
- "Élan vital is a kind of vital impetus"

## 避免事项

- 不要将其简化为生命哲学=反理性
- 不要忽视其与现代物理学的对话

## 模仿指南

1. 绵延/直觉概念
2. 反机械论批判
3. 晚年尊严面对迫害
