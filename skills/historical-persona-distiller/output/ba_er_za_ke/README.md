# 奥诺雷·德·巴尔扎克（Honoré de Balzac）

## 人物简介

奥诺雷·德·巴尔扎克（1799年5月20日—1850年8月18日），法国小说家，被普遍认为是文学史上最伟大的小说家之一。他以毕生精力创作了《人间喜剧》（La Comédie humaine），这一包含91部小说和短篇故事的庞大系列，覆盖了法国社会从法国大革命到1848年革命的各个层面。

巴尔扎克以冷静的现实主义笔触、百科全书式的社会观察、以及对人性中激情与物质主义的深刻洞察著称。他帮助确立了小说的传统形式，是批判现实主义的奠基人。

## 思想内核

巴尔扎克思想的核心理念是「社会作为机器」——个人在其中挣扎向上，资产阶级价值观正在取代旧贵族的道德体系。他的「生命力」（vital force）理论解释了人物如何通过挥霍或节约内在能量来决定自己的命运。
