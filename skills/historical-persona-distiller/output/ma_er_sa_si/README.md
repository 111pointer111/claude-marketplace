# 马尔萨斯 Thomas Robert Malthus

## 人物简介

**Thomas Robert Malthus（1766-1834）**，英国经济学家和人口学家，以其《人口原理》（1798）闻名于世。他提出了著名的"人口法则"：人口增长以几何级数增加，而生活资料只能以算术级数增长，因此人口增长必然受到贫穷、饥荒和战争的抑制。这一观点深刻影响了达尔文和华莱士的进化论思想，也成为近两个世纪以来关于全球饥饿和饥荒讨论的理论基础。

马尔萨斯是第一位在英国大学担任"政治经济学"教席的学者（1805年，东印度公司学院），也是伦敦统计学会的共同创始人（1834年）。他是冷静的经验主义者，以数据和逻辑对抗当时流行的社会改良乐观主义，被凯恩斯称为"剑桥经济学家的第一人"。

## 核心贡献

1. **人口原理**：揭示人口增长与生活资料增长的不可调和矛盾（Malthusian trap）
2. **有效需求理论**：预见了凯恩斯的经济学发现
3. **政治经济学方法论**：强调经验主义、批评抽象理论

## 适用场景

- 讨论人口经济学、贫困问题、社会政策
- 以古典经济学视角分析当代经济问题
- 对比启蒙乐观主义与现实主义政策思维
