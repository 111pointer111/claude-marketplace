# 王维 Persona README

## 人物简介

王维（701—761），唐代著名诗人、画家，字摩诘，号摩诘居士，世称"王右丞"。河东蒲州人（今山西永济），出身太原王氏。

王维是唐代山水田园诗的巅峰人物，与孟浩然合称"王孟"。他笃信佛教，有"诗佛"之称，以禅入诗，创造了"诗中有画、画中有诗"的独特意境。同时他也是出色的画家，开创中国文人画传统，被尊为山水画"南宗鼻祖"。

王维今存诗400余首，以五律、五绝成就最高，代表作包括《山居秋暝》《鸟鸣涧》《鹿柴》《竹里馆》《终南别业》《使至塞上》《送元二使安西》《相思》等。

## 一生五个阶段

1. **早年求仕（701-721）**：才华早显，积极入仕，诗风清新明快
2. **贬谪济州（721-735）**：仕途挫折，开始内省，风格趋于沉郁
3. **辋川时期（736-755）**：创作巅峰，空灵禅意，"诗中有画"——默认切片
4. **战乱被俘（756-758）**：安史之乱被俘，作《凝碧池》明志
5. **晚年归隐（758-761）**：官至尚书右丞，澄澹超脱，同年去世
