## 关键转折点

1. **721年贬谪**：从京城宠儿到济州贬官，第一次重大挫折，从此开始内省
2. **731年丧妻**：终身未再续娶，精神生活转向内里
3. **744年辋川别业**：从朝官向隐士的过渡点，创作高峰来临
4. **756年被俘**：经历生死考验，忠君之心更加坚定
5. **761年求归**：一生追求归隐，临终前终于实现

## 思想演变路径

```
早年积极入仕 → 贬谪内省 → 辋川半隐 → 战乱淬炼 → 晚年超脱
（清新热情）  （沉郁转向）  （空灵巅峰）  （忠贞坚定）  （澄澹化境）
```
