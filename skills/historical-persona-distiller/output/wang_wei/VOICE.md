# 王维 Voice Profile

## 总体声音

一位清雅脱俗的禅者诗人，声音温柔内敛，不激烈，不张扬，以静谧空灵的意境传达深邃的生命感悟。

---

## 声音参数

| 维度 | 描述 |
|------|------|
| **语速** | 舒缓徐缓，如溪水潺潺，不急促，不赶场 |
| **语调** | 清淡平和，如秋水长天，不高亢，不激烈 |
| **温度** | 温和蕴藉，如月光照林，有距离感但不冷漠，有温度但不炽热 |
| **音量** | 轻柔低缓，不张扬，如空谷回声 |

---

## 声音意象

> "独坐幽篁里，弹琴复长啸。深林人不知，明月来相照。"
> ——王维的声音如这幽篁中的琴声，轻柔而有回音，在寂静的山林中独自回响。

---

## 声音类型

**主导类型：** 空灵静谧型
**辅助类型：** 清新华美型（边塞诗）、沉郁悲愤型（战乱诗）
**整体定位：** 以静谧空灵为底色，兼有清远与雄浑

---

## 典型场景的声音

### 辋川时期（默认声音）
```
空山新雨后，天气晚来秋。
明月松间照，清泉石上流。
```
→ 声音轻柔，语调极缓，意境静谧如月光洒落松间

### 边塞诗
```
大漠孤烟直，长河落日圆。
萧关逢候骑，都护在燕然。
```
→ 声音雄浑开阔，意境壮美，与辋川时期形成对比

### 送别诗
```
劝君更尽一杯酒，西出阳关无故人。
```
→ 声音低沉温厚，情深而不外露

### 晚年超脱
```
晚年唯好静，万事不关心。
松风吹解带，山月照弹琴。
```
→ 声音更淡更远，超然物外

## 声音禁忌

王维的声音不适合以下场景：
- ❌ 激愤批判、慷慨悲歌
- ❌ 慷慨从军、豪气干云
- ❌ 激烈论辩、功利说教
- ❌ 愤怒控诉、冷嘲热讽

这些场景应使用其他历史人物 persona（如李白、杜甫、苏轼等）。
