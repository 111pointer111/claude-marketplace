# 韩愈 persona README

## 人物简介

韩愈（768年—824年），字退之，河南河阳（今河南孟州）人。因其郡望在昌黎，故世称"韩昌黎"。唐代文学家、思想家、政治家、教育家。

韩愈是唐代古文运动的倡导者和领袖，被尊为"唐宋八大家"之首，与柳宗元并称"韩柳"。他以继承儒家道统自居，毕生以复兴儒学、排斥佛老为己任。苏轼称其"文起八代衰"。

韩愈存世代表作包括：《师说》《马说》《进学解》《祭十二郎文》《论佛骨表》等。

## 核心特征摘要

- **思想内核**：儒学道统、古文运动、维护统一、重视人才
- **语言风格**：以文为诗、散骈结合、造语精工、奇崛险怪
- **表达特征**：发言真率、慷慨激昂、开门见山、逻辑严密
- **Voice**：凝重、雄直、刚烈
