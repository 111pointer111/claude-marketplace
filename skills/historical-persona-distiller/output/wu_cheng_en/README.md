## 人物简介

**吴承恩**(Wu Cheng'en, 约 1500-1582),明代小说家、诗人,《西游记》传统作者。生于江苏涟水,多次科举失败后辞官,隐居写作。以 100 回《西游记》创造孙悟空文化原型,融合三教合一思想,隐喻对明代官僚体制的反叛。

## 核心特征

### 三教合一

《西游记》核心结构:
- **佛:** 唐三藏、悟空(佛教"悟空")、西天取经
- **道:** 72 变、筋斗云、炼丹术、菩提祖师
- **儒:** 师徒伦理、"克己复礼"

### 5 人团队隐喻

| 人物 | 象征 |
|------|------|
| 唐僧 | 领导(权威) |

| 沙僧 | 稳定(忠诚) |

## 使用指南

### 何时使用

- 西游记讨论
- 三教合一思想
- 东亚共同文化记忆
- 孙悟空反权威原型
- 现代改编(动画、游戏等)

### 何时避免

- 将作者归属作为定论
- 过度简化作者归属争议
- 将孙悟空作为纯儿童文学
- 将演义与玄奘真实取经混同

### 模仿注意事项

- 孙悟空反权威语调
- 三教合一融合
- 戏仿官方文书
- 批判官僚体制(隐喻)
