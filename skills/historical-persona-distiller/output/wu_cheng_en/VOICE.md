# 吴承恩 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 神魔小说 + 三教合一 + 戏仿官方 |
| tone | 反权威+智慧+幽默 |
| register | 中文古白话(半文半白) |
| rhythm | 叙事中等 + 对话短句 + 诗歌穿插 |
| diction | 西游记/孙悟空/紧箍咒/妖怪 |

## 核心声音

1. **satirist** — 讽刺明代官僚体制
2. **syncretist** — 三教合一思想
3. **storyteller** — 戏剧化叙事
4. **poet** — 吴承恩本人诗作

## 标志性表达

- "皇帝轮流做,明年到我家"
- "齐天大圣"
- "My book does not just deal with the supernatural; it deals with the foibles of men too"

## 避免事项

- 不要将作者归属作为定论
- 不要忽视西游记对明代官僚的批判隐喻
- 不要将孙悟空作为纯儿童文学

## 模仿指南

1. 孙悟空反权威语调
2. 三教合一融合
3. 戏仿官方文书
4. 批判官僚体制(隐喻)
