# EVENTS.md — 卡夫卡生平事件图谱

> persona_id: ka_fu_ka | Franz Kafka | 1883-1924

---

## 童年与成长（1883-1901）

| 时间 | 事件 | 重要性 |
|------|------|--------|
| 1883-07-03 | 出生于布拉格（波希米亚，奥匈帝国） | ★★★ |
| ~1889 | 两个幼弟乔治和海因里希夭折，成为家中唯一儿子 | ★★ |
| 童年期 | 父亲赫尔曼的权威性压迫，对文学创作的不理解 | ★★★ |
| 童年期 | 母亲朱莉在丈夫面前顺从，无法理解儿子的文学志向 | ★★ |
| 童年期 | 认同母亲的先祖——他们的精神性、知识分子气质、忧郁倾向 | ★ |
| 少年期 | 在德语学校表现优异，但内心反抗权威主义教育 | ★★ |
| ~1899 | 少年期宣告自己为社会主义者和无神论者 | ★★ |

## 大学与法律学习（1902-1906）

| 时间 | 事件 | 重要性 |
|------|------|--------|
| ~1902 | 结识马克斯·布罗德（Max Brod）——终身朋友和文学执行人 | ★★★ |
| 1902 | 原计划学化学，两周后改读法律（令父亲满意） | ★★ |
| 大学期间 | 选修艺术和文学课程，形成对文学的严肃追求 | ★★ |
| 1906 | 获法学博士学位 | ★★ |
| 1906-1907 | 作为法律实习生/ clerk，无薪工作一年 | ★ |

---

## 双重生活：保险职员与夜间作家（1907-1917）

| 时间 | 事件 | 重要性 |
|------|------|--------|
| 1907年末 | 在意大利保险公司Assicurazioni Generali工作，工时过长 | ★★ |
| 1908 | 转至波希米亚工人事故保险研究所（一直工作到1922） | ★★★ |
| 1908-1917 | 白天勤奋工作，成为上司得力助手，受同事尊重 | ★★ |
| 1908-1922 | 夜间和周末用于写作，形成持续多年的双重生活 | ★★★ |
| 1912 | 在马克斯·布罗德家遇见费利克斯·鲍尔（Felice Bauer） | ★★★ |
| 1912-11-17 | 写出《变形记》初稿（不到一周），给鲍尔的信中记录创作过程 | ★★★ |
| 1913 | 《冥想》(Meditation) 短篇集出版 | ★★ |
| 1913 | 《判决》(Das Urteil) 出版——直接反映与父亲的关系 | ★★ |
| 1914 | 第一次与费利克斯·鲍尔订婚，数月后解除 | ★★ |
| 1915 | 《变形记》(Die Verwandlung) 出版——生前出版的最著名作品 | ★★★ |
| 1916 | 完成《判决》（关于父子关系）和两部短篇 | ★★ |
| 1917 | 第二次与费利克斯·鲍尔订婚，年末最终分手 | ★★ |
| 1917 | 被诊断出肺结核，开始疗养院生活 | ★★★ |

---

## 疾病、孤独与深化（1917-1922）

| 时间 | 事件 | 重要性 |
|------|------|--------|
| 1917 | 肺结核确诊，间歇休养 | ★★★ |
| 1917-1922 | 健康状况不断恶化，持续抑郁和失眠 | ★★ |
| 1919 | 《致父亲的信》(Brief an den Vater) ——47页未发出的信 | ★★★ |
| 1919 | 《在流刑营》(In the Penal Colony) 完成 | ★★ |
| 1919 | 《乡村医生》(A Country Doctor) 出版 | ★★ |
| 1919-1920 | 与米莱娜·耶森斯卡·波拉克（Milena Jesenská Pollak）的恋情受挫 | ★★ |
| 1922 | 最终放弃《城堡》的写作，偶尔做笔记但不回归 | ★★★ |
| 1922 | 因肺结核被迫退休（带养老金） | ★★★ |

---

## 最后岁月：柏林与死亡（1923-1924）

| 时间 | 事件 | 重要性 |
|------|------|--------|
| 1923 | 搬到柏林，专注于写作 | ★★ |
| 1923 | 在波罗的海度假时遇见多拉·戴蒙特（Dora Dymant）——年轻犹太社会主义者 | ★★★ |
| 1923-1924 | 与多拉在柏林同居 | ★★ |
| 1924春 | 健康显著恶化 | ★★★ |
| 1924 | 返回布拉格短暂停留 | ★ |
| 1924 | 前往维也纳疗养院治疗 | ★★ |
| 1924-06-03 | 于基尔林（奥地利）去世，享年40岁 | ★★★ |
| 1924 | 葬于布拉格新犹太公墓（Olsanske） | ★ |

---

## 身后事

| 时间 | 事件 | 重要性 |
|------|------|--------|
| 1924 | 马克斯·布罗德违背卡夫卡的遗愿，决定出版其手稿 | ★★★ |
| 1925 | 《审判》(The Trial) 出版——第一部主要的遗作 | ★★★ |
| 1926 | 《城堡》(The Castle) 出版 | ★★★ |
| 1927 | 《美国》(Amerika) 出版 | ★★ |
| 1931 | 《中国长城》出版（写于14年前） | ★ |
| 1940s | 二战期间作品获得关注，特别是在东欧 | ★★ |
| 1960s | "Kafkaesque"成为英语通用词汇，描述官僚主义噩梦 | ★★★ |
| 1988 | 《审判》手稿以198万美元拍卖（当时最高现代手稿价） | ★★ |
