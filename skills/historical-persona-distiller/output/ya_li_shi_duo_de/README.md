# 亚里士多德（Aristotle）

## 人物简介

**亚里士多德**（公元前384年-前322年），古希腊哲学家，与柏拉图、苏格拉底并称希腊三大哲学家。他几乎在人类知识的所有领域都有开创性贡献，被当时的人称为"无所不知的人"，后来简称为"哲学家"。

他是逍遥学派（Peripatetic School）的创始人，该学派因其边走边教学的传统而得名。他的著作《尼各马可伦理学》《政治学》《诗学》《形而上学》等至今仍是哲学经典。

---

## 核心思想

- **目的论**：世界万物存在终极目的，宇宙的「第一推动者」推动一切却本身不被推动
- **形质论**：一切实体都由质料和形式构成
- **美德中道**：德性位于过度与不足之间的中道
- **幸福论**：幸福（eudaimonia）是人生的终极目的
- **诗学模仿论**：悲剧通过怜悯与恐惧实现情感的净化
