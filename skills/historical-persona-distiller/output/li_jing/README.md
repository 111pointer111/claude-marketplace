# 李靖（571年—649年）

> 唐初军事家，凌烟阁二十四功臣之一，武庙十哲之一。
> 兼资文武，出将入相，功高震主而善终。
> 以三千精骑灭东突厥，以两个月平萧铣，被评为"唐初名将推李靖为第一人"。

---

## 人物简介

李靖（571年—649年），字药师，京兆三原（今陕西三原）人，唐朝初期卓越的军事家。

### 生平概要

- **571年**：出生于陇西李氏丹杨房，祖父李崇义为北周殷州刺史
- **少年时期**：舅父隋朝名将韩擒虎称其"可与论孙、吴之术者"
- **隋朝仕途**：历任长安县功曹、殿内直长、驾部员外郎，获牛弘、杨素赏识
- **617年**：察觉李渊异动，欲向隋炀帝告密，被俘后归唐
- **618年**：被李世民召入幕府，用为三卫
- **621年**：平萧铣，献十策，率军沿三峡东下，以疑兵之计下江陵
- **621年—623年**：安抚岭南，平定辅公祏
- **625年**：抗突厥，以少胜多，保全江淮兵
- **630年**：灭东突厥，擒颉利可汗，任尚书右仆射
- **635年**：平吐谷浑
- **636年后**：阖门自守，不问政事
- **649年**：病逝，享年七十九岁，李世民亲临病榻

### 核心成就

| 类型 | 成就 |
|------|------|
| 平定江南 | 两个月灭萧铣，平定辅公祏，招抚岭南九十六州 |
| 抵御外侵 | 抗突厥侵略，保全江淮兵 |
| 开疆拓土 | 灭东突厥、平吐谷浑，扩展唐朝疆域 |
| 军事理论 | 奇正相变、知己知彼、地利为宝、赏罚并重 |
| 政治地位 | 凌烟阁二十四功臣第八名，尚书右仆射 |

> "可与论孙、吴之术者，惟斯人矣。" ——韩擒虎

> "李靖是萧铣、辅公祏膏肓，古之名将韩、白、卫、霍，岂能及也！" ——李渊
