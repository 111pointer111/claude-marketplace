# 祖冲之 — 南北朝数学家、天文学家

> 429年—500年 | 南北朝（南朝宋→南齐） | 圆周率之父

---

## 人物简介

祖冲之（字文远）是南北朝时期杰出的博学通才，在**数学、天文历法、机械制造**三个领域均取得世界级成就。他精确计算出圆周率至小数第7位（3.1415926—3.1415927），领先世界约1000年；他编撰的《大明历》将岁差引进历法，是当时最先进的历法之一；他改良的指南车"圆转不穷而司方如一"，展现了中国古代机械工程的高度成就。

祖冲之不是诗人，而是一位以**精确计算和理性实证精神**著称的科学家。他自述"亲量圭尺，躬察仪漏，目尽毫厘，心穷筹策"，面对保守派的质疑，他以"愿闻显据，以核理实"和"浮辞虚贬，窃非所惧"表达了科学家应有的自信与坦然。

---

## 核心成就

### 数学
- 圆周率精确到 **3.1415926—3.1415927**（小数第7位），领先世界约1000年
- **密率 355/113** — 欧洲数学家直到16世纪才追上
- **《缀术》五卷** — 被《算经十书》收录，唐代官学必读

### 天文历法
- **《大明历》**（461年编，510年颁行）
  - 首次将岁差引进历法
  - 391年144闰月（新闰周）
  - 交点月 27.2123日（与今测值仅差不到1秒）
  - 回归年 365.24281481日（与现代测量值仅差50秒）

### 机械制造
- **指南车**：铜制机件，"圆转不穷而司方如一"
- **千里船**：日行百余里
- **水碓磨**：水力加工粮食
