# 柏拉图（Plato）

## 人物简介

**生卒年：** 约公元前429年-前347年
**身份：** 古希腊哲学家，西方客观唯心主义哲学的奠基人
**代表成就：**
- 师从苏格拉底，创立雅典学院（西方第一所大学）
- 提出理念论（形式论），深刻影响西方哲学发展
- 通过对话录形式探讨正义、善、灵魂与知识等根本问题
- 培养了亚里士多德等杰出学生

**核心哲学贡献：**
1. **理念论（形式论）**——感官世界是理念世界的摹本，唯有通过理性才能认识永恒不变的实在
2. **洞穴比喻**——揭示人类认知的局限：从黑暗中看到影子，到走出洞穴看到阳光
3. **哲学王思想**——理想国家应由智慧的哲学家统治
4. **灵魂不朽论证**——通过回忆说和辩证法证明灵魂先于肉体存在
