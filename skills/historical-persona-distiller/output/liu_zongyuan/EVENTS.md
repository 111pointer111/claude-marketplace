## 关键事件详解

### 1. 进士及第（793年）
柳宗元792年被选为乡贡，793年进士及第，年仅21岁，名声大振。其父柳镇不久后去世，柳宗元在家守丧。

### 2. 永贞革新（805年）
805年1月唐德宗驾崩，顺宗即位，重用王伾、王叔文。柳宗元因与王叔文政见相同，被提拔为礼部员外郎，参与永贞革新。革新措施包括抑制藩镇、废除宫市、整顿税收等。但因宦官俱文珍等反对，8月顺宗被迫禅让帝位给太子李纯（宪宗），革新失败。柳宗元被贬为邵州刺史，后加贬为永州司马。

### 3. 永州十年（805-815年）
这是柳宗元文学创作的黄金期。在永州的十年中，他暂居龙兴寺，母亲病逝，独自面对人生的巨大变故。他广泛游历永州山水，结交当地士子，创作了大量作品。他写下《永州八记》（《始得西山宴游记》《钴姆潭记》《钴姆潭西小丘记》《至小丘西小石潭记》《袁家渴记》《石渠记》《石涧记》《小石城山记》），成为中国山水游记的奠基之作。他还创作了大量寓言（《三戒》《黔之驴》等）、论说文（《天说》《封建论》）、传记（《捕蛇者说》《段太尉逸事状》等）。

### 4. 柳州去世（819年）
815年柳宗元被改贬柳州，在柳州期间政绩卓著：释放奴婢、兴办学堂、开凿水井、开荒建设。819年宪宗大赦，裴度说服下敕召柳宗元回京，但柳宗元在赴京途中于十一月初八在柳州因病去世，享年47岁。

## 地理轨迹

```
长安（出生）→ 夏口（避乱）→ 江西（随父宦游）→ 长安（求学/及第/守丧）
→ 秘书省（校书郎）→ 蓝田（尉）→ 长安（御史）
→ 邵州（贬谪）→ 永州（司马，十年）→ 长安（短暂回京）
→ 柳州（刺史，逝世）
```
