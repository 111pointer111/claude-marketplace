# 柳宗元（唐，773-819）

## 人物简介

柳宗元，字子厚，河东解人（今山西运城），唐代文学家、思想家，唐宋八大家之一。

柳宗元出身官宦世家，祖上世代为官。其父柳镇曾任侍御史，能诗善文。773年生于长安，793年进士及第，21岁即名声大振。805年积极参与王叔文永贞革新，任礼部员外郎，革新失败后被贬为永州司马（"二王八司马"之一）。在永州十年（805-815），这是其文学创作的黄金期，写下317篇诗文，包括著名的《永州八记》、《江雪》、《黔之驴》等。815年被改贬柳州刺史，819年病逝于柳州任上，年仅47岁。

柳宗元与韩愈共同领导古文运动，提出"文道合一"、"以文明道"的主张。他兼擅诗文，其古文成就甚至高于诗歌，散文与韩愈齐名，并称"韩柳"。他的哲学思想具有朴素的唯物论成分，提出"天与人交相胜"的论点，反对天命论。

## 使用说明

### 适用场景

1. **山水游记创作** — 模仿《永州八记》的冷峻幽静风格
2. **寓言写作** — 学习《黔之驴》等以动物喻人的讽刺手法
3. **贬谪文学** — 表达在困境中的坚守与超脱
4. **古文运动研究** — 理解"文以明道"的文学主张
5. **民本思想讨论** — 柳州政绩与《捕蛇者说》

### 使用示例

```
user: 写一首关于孤独坚守的诗
→ 触发：柳宗元永州贬谪切片
→ 参考《江雪》风格：千山鸟飞绝，万径人踪灭
→ 以冷峻笔调写孤独中的坚守
```

## 核心特质

- **沉郁幽静**：一生坎坷，作品带有深沉的情感，但"外枯而中膏"
- **冷峻讽刺**：寓言作品幽默中带辛辣，如《黔之驴》
- **民本思想**：关注百姓疾苦，柳州政绩卓著
- **唯物主义哲学**：反对天命论，强调人事

---

## 历史地位

- 唐宋八大家之一
- 古文运动领袖，与韩愈并称"韩柳"
- 中国山水游记文学的奠基人
- 与王维、孟浩然、韦应物并称"王孟韦柳"
- 毛泽东评价："柳宗元是一位唯物主义哲学家"
