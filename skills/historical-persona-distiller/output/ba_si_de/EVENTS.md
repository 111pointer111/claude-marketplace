# 巴斯德 EVENTS.md — 生平事件图谱

## Stage 1: 早年求学与化学基础（1822-1847）

| 年份 | 事件 | 意义 |
|------|------|------|
| 1822.12.27 | 出生于法国汝拉地区Dole | 生命起点 |
| 早年 | 父亲Jean-Joseph Pasteur——鞣革匠，拿破仑战争期间荣获荣誉军团勋章的中士 | 家庭教育：荣誉感与爱国心 |
| 早年 | 学业平平，但擅长绘画和油画 | 艺术天赋 |
| ~1835-1840 | 在阿尔布瓦小学、贝桑松中学就读 | 基础教育 |
| 1840 | 获得文学学士（贝桑松皇家学院） | 第一个学位 |
| 1842 | 获得理学学士（贝桑松皇家学院） | 科学启蒙 |
| 1847 | 获得博士学位（巴黎高等师范学校） | 学术资格确立 |
| 1847 | 酒石酸晶体研究——发现两种镜像晶体，奠定立体化学基础 | 科学突破：立体化学诞生 |

## Stage 2: 斯特拉斯堡与里尔时期（1848-1865）

| 年份 | 事件 | 意义 |
|------|------|------|
| 1848 | 成为斯特拉斯堡大学化学教授 | 学术职位 |
| 1848-1849 | 在此结识大学校长女儿Marie Laurent | 人生伴侣 |
| 1849.5.29 | 与Marie Laurent结婚 | 家庭建立 |
| 婚后 | 育有五子女，仅二存活 | 个人悲剧 |
| 1854 | 被任命为里尔大学化学教授兼科学学院院长 | 事业高峰 |
| 1854-1860s | 研究酒精发酵和细菌致病理论 | 核心研究方向确立 |
| 1862.4.20 | 完成首次巴氏消毒法测试 | 关键技术发明 |
| 1862 | 当选法国科学院（Académie des Sciences）院士 | 最高学术荣誉 |

## Stage 3: 巴氏消毒法推广与丝绸业拯救（1862-1879）

| 年份 | 事件 | 意义 |
|------|------|------|
| 1862-1865 | 巴氏消毒法推广至酿酒业、牛奶产业 | 产业革命 |
| 1865 | 开始研究蚕病问题 | 新研究方向 |
| 1865-1870 | 证明微生物攻击蚕卵导致疾病，开发防污染方法 | 拯救法国丝绸产业 |
| 1868 | 严重脑卒中，半身瘫痪 | 健康危机 |
| 1868-1895 | 半身瘫痪但继续研究 | 意志力象征 |
| 1873 | 当选医学科学院（Académie de Médecine）合作成员 | 医学界认可 |
| 1879 | 意外发现鸡霍乱疫苗——弱化病毒可产生免疫 | 疫苗学突破 |

## Stage 4: 疫苗学突破与狂犬病（1879-1885）

| 年份 | 事件 | 意义 |
|------|------|------|
| 1881 | 开发炭疽疫苗，公开演示成功 | 疫苗学里程碑 |
| 1881 | 建立在科赫1876年分离炭疽杆菌纯培养的基础上 | 与科赫科学对话 |
| 1882 | 入选法兰西学院（Académie Française） | 法国最高文化荣誉 |
| 1882 | 决定专注于狂犬病问题 | 研究方向聚焦 |
| 1885.7.6 | 为9岁约瑟夫·梅斯特（Joseph Meister）接种狂犬病疫苗成功 | 国际声誉巅峰 |
| 1885 | 推动国际筹款建立巴斯德研究所 | 制度性遗产 |

## Stage 5: 晚年与遗产（1885-1895）

| 年份 | 事件 | 意义 |
|------|------|------|
| 1888.11.14 | 巴斯德研究所在巴黎落成 | 世界级研究中心 |
| 70岁生日 | 在索邦大学庆祝，包括李斯特在内的多位科学家出席 | 荣誉顶峰 |
| 1892 | 70岁生日庆典 | 人生巅峰 |
| 1895.9.28 | 逝世于Marnes-la-Coquette | 生命终点 |
| 1896 | 遗体移至巴斯德研究所新拜占庭式地下室 | 永恒纪念 |

## 关键转折点

1. **立体化学研究（1847）**：从化学走向生物学，为后续微生物研究奠定基础
2. **巴氏消毒法（1862）**：从理论走向实践，服务产业
3. **半身瘫痪（1868）**：健康危机，但未停止研究
4. **鸡霍乱疫苗（1879）**：从治已病到防未病
5. **狂犬病疫苗（1885）**：从动物到人类，科学巅峰
