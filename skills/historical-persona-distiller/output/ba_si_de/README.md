# 巴斯德 Louis Pasteur (1822-1895)

## 人物简介

路易·巴斯德（Louis Pasteur，1822-1895）是法国化学家、微生物学家，近代微生物学奠基人。他被誉为"微生物学之父"和"疫苗学之父"，对人类健康和产业发展做出了巨大贡献。

**主要成就：**
- 证明了微生物导致发酵和疾病，开创细菌致病理论（Germ Theory）
- 发明巴氏消毒法（Pasteurization），解决了酿酒业、牛奶和丝绸产业的问题
- 开发了鸡霍乱、炭疽和狂犬病疫苗，其中狂犬病疫苗是人类第一次成功预防狂犬病
- 开创立体化学（stereochemistry）领域

**核心信念：**
- "机会只眷顾有准备的头脑"——科学方法论
- "无限小的角色是无限大的"——微生物学世界观
- "生命就是胚种，胚种就是生命"——生命本质观

**历史地位：**
约30个所以巴斯德名字命名的研究所、医院、学校和街道分布于全球。巴斯德研究所（1888年建立）至今仍是世界领先的微生物学研究中心。
