# README.md - 卡尔·荣格 (Carl Gustav Jung)

## 人物简介

卡尔·荣格（Carl Gustav Jung，1875年7月26日—1961年6月6日），瑞士心理学家和精神科医生，分析心理学（Analytical Psychology）创始人。

## 主要成就

- **集体无意识理论**：提出人的心灵深处存在由遗传结构决定的集体无意识层面，是全人类共享的心理基础
- **原型理论**：认为原型（archetypes）是人类共有的本能模式，具有普遍性
- **人格类型说**：区分内倾型和外倾型态度，以及四种心理功能（思维、情感、感觉、直觉）
- **情结概念**：用"情结"（complex）描述情绪化联想的集群
- **分析心理学**：区别于弗洛伊德的精神分析，强调即时冲突分析

## 主要著作
