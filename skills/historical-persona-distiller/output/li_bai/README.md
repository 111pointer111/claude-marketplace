## 人物简介

**李白**（Li Bai，701年–762年），字太白，号青莲居士，唐代伟大的浪漫主义诗人。

李白是盛唐诗歌的代表性人物，被后世尊为"诗仙"，与杜甫并称"大李杜"。其诗豪迈奔放、清新飘逸，想象丰富、意境奇妙，是屈原以来最具个性特色和浪漫主义精神的诗人。他的诗歌代表作包括《静夜思》《将进酒》《蜀道难》《早发白帝城》《黄鹤楼送孟浩然之广陵》等，几乎每个中国人都能背诵其中名句。

## 必读名篇

1. **《将进酒》** — 李白酒神精神的最高表达
2. **《蜀道难》** — 浪漫主义巅峰之作
3. **《静夜思》** — 五言绝句的最高典范
4. **《早发白帝城》** — 遇赦后的轻快昂扬
5. **《梦游天姥吟留别》** — 梦中仙境的奇幻描写
6. **《月下独酌四首》** — 孤独与超脱的完美融合
7. **《侠客行》** — 武侠豪情的诗化表达
8. **《把酒问月》** — 对宇宙人生的哲学思考
