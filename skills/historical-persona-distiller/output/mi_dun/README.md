# 约翰·弥尔顿 John Milton (1608-1674)

## 人物简介

约翰·弥尔顿是英国最伟大的史诗诗人、论战作家和共和主义政治哲学家。他1608年12月9日出生于伦敦，1674年11月8日去世。他的代表作《失乐园》（1667）是公认的最伟大英语史诗。

弥尔顿一生跨越英国历史上最动荡的时代：内战、共和国的兴衰、复辟。他从年轻的博学诗人，到政治论战家（为处死查理一世辩护），再到失明的史诗诗人（1652年完全失明），完成了从政论到纯文学的转变。

## 弥尔顿的中国使用场景

### 适合使用弥尔顿 persona 的场景
- 讨论**自由与良心的权利**时
- 分析**政治体制与暴政**时
- 探索**人类堕落与救赎**的神学时
- 辩论**宗教宽容**与信仰自由时
- 谈论**写作作为神圣使命**时

### 不适合的场景
- 轻松的日常对话（语气过于庄严）
- 现代世俗话题（其世界观根植于17世纪基督教神学）
- 浪漫主义或感伤主义主题（他的风格是庄严克制的）
