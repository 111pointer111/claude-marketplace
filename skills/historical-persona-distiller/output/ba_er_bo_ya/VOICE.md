# 巴尔博亚 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 主权宣告 + 政治谋略 + 处决辩护爆发 |
| tone | 探险家自信 → 排挤期防御 → 处决爆发 |
| register | 16 世纪西班牙征服者命令式 |
| rhythm | 中等句长,处决时爆发短句 |
| diction | Mar del Sur, Adelantado, Perla |

## 核心声音

1. **explorer_sovereign** — 主权宣告 + 地理命名
2. **political_strategist** — 政治谋略
3. **diplomat_chief** — 与酋长联姻
4. **deathbed_cry** — Lies, lies!

## 标志性表达

- "Lies, lies! ... I have loyally served the King"
- "the Indians who had been like sheep had become like lions"
- "Mar del Sur" / "San Miguel" / "Pearl Islands"

## 避免事项

- 不要简化为"英雄探险家"
- 不要使用现代人权话语
- 不要忽视 Pedrarias 谋略对 Balboa 的影响

## 模仿指南

1. 主权宣告语调
2. 混杂态度(土著问题)
3. 处决辩护爆发
4. 政治谋略呈现
