## 人物简介

**马克·吐温**（Mark Twain，1835-1910），本名萨缪尔·兰亨·克莱门（Samuel Langhorne Clemens），美国幽默作家、记者、演说家、小说家，美国现实主义文学奠基人之一。

他出生于密苏里州汉尼拔镇的贫困家庭，11岁丧父后辍学打工，做过印刷学徒、水手、矿工、记者等多种职业。以笔名"马克·吐温"（源于河流术语，意为12英尺深）发表《卡拉韦拉斯县驰名的跳蛙》（1865）后一举成名。此后周游世界，发表旅行文学作品，又以《汤姆·索亚历险记》（1876）和《哈克贝利·费恩历险记》（1884）奠定文学史地位。其作品以口语化幽默、少年视角和尖锐的社会批判著称。晚年因家庭不幸和投资失败，性格趋于悲观，接连创作《傻瓜威尔逊》《败坏了哈德莱堡的人》等讽刺作品。威廉·豪威尔斯称他为"美国文学中的林肯"；威廉·福克纳称其为"第一位真正的美国作家"。

吐温以其独特的**口语化幽默**、**少年叙事视角**和**讽刺批判**闻名：

- 短句为主，对话驱动，口语化
- 第一人称"老实人"叙事者：天真善良，在腐败社会中碰壁
- 幽默里含讽刺，揭示社会问题
- 从早期欢快幽默演变为晚期尖刻悲观
- 亲身经历为素材，真实感强
