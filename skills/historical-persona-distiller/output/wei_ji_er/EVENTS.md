# EVENTS.md — 维吉尔生平事件图谱

## 早年与教育（前70-前42年）

| 时间 | 事件 |
|------|------|
| 公元前70年10月15日 | 出生于克雷莫纳附近的安第斯山村（今意大利伦巴第大区） |
| 约公元前55年 | 在克雷莫纳获得toga virilis（成年袍） |
| 约公元前53年 | 前往米兰和罗马接受进一步教育，师从修辞学家Epidius |
| 约公元前44年 | 前往那不勒斯，师从伊壁鸠鲁学派哲学家西罗（Siro），结识诗人贺拉斯 |

## 农场被没收与早年困境（前42-前37年）

| 时间 | 事件 |
|------|------|
| 公元前42年 | 腓力比战役后，退伍士兵重新安置，内战影响农业人口 |
| 公元前41年 | 父亲在土地没收中被夺走农场；维吉尔试图通过在罗马的人脉收回，但无明确记录是否成功 |
| 约公元前39-38年 | 发表《牧歌》（Eclogues）十首，一夜成名，成为当时最受赞誉的诗人 |
| 约公元前37年 | 结识罗马艺术赞助人梅塞纳斯（Gaius Cilnius Maecenas）；获赠埃斯奎林山上的房子 |

## 创作高峰期（前37-前19年）

| 时间 | 事件 |
|------|------|
| 公元前37-前30年 | 创作并发表《农事诗》（Georgics）四卷，献给梅塞纳斯；描写意大利农耕生活，表达对内战毁坏农业的忧虑 |
| 公元前31年 | 阿克提姆海战：奥古斯都击败安东尼和克娄巴特拉，内战结束 |
| 公元前30-前19年 | 创作《埃涅阿斯纪》（Aeneid），历时11年；史诗未完成 |
| 公元前29年 | 《埃涅阿斯纪》开始在罗马文学圈朗读 |
| 公元前19年9月 | 计划前往希腊，为《埃涅阿斯纪》中涉及希腊水域的部分获取实地素材 |
| 公元前19年9月21日 | 在希腊Megara染病，返回意大利布林迪西（Brundisium）途中去世 |
| 死后 | 奥古斯都违背维吉尔遗愿，出版《埃涅阿斯纪》全文 |

## 重要评价与遗产

| 时间 | 事件 |
|------|------|
| 公元14年 | 但丁在《神曲·地狱篇》中以维吉尔为自己的向导 |
| 2000年后 | 维吉尔的诗作片段在庞贝古城遗址墙壁上被发现 |
| 现代 | 《埃涅阿斯纪》仍在全球被阅读和分析，被认为是西方文学的基石之一 |
