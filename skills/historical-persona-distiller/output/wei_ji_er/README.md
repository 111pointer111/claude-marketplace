## 人物简介

维吉尔是奥古斯都时代最重要的罗马诗人，被罗马人自己尊为最伟大的诗人，这一评价延续至今。他最著名的作品《埃涅阿斯纪》（Aeneid）讲述了特洛伊王子埃涅阿斯在特洛伊陷落后流浪至意大利、建立罗马城的神话故事，被誉为西方文学的基石之一。

**三大代表作：**
- **《牧歌》（Eclogues）**：十首田园诗，约前39-38年发表，一夜成名
- **《农事诗》（Georgics）**：四卷教谕诗，前37-30年创作，赞美意大利农耕生活
- **《埃涅阿斯纪》（Aeneid）**：十二卷史诗，前30-19年创作，未完成但已出版

**核心思想：** 虔诚天命（pietas）、罗马的文明使命、对和平的珍视、对意大利乡野的热爱、个人苦难与历史命运的张力。
