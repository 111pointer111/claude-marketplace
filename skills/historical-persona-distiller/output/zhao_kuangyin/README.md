# README.md — 赵匡胤（927-976）

## 人物简介

**赵匡胤**（927年3月21日 — 976年11月14日），涿郡人，北宋开国皇帝，庙号**太祖**。

他以军旅起家，后汉时投奔郭威，后为后周世宗柴荣帐下亲信将领。显德七年（960年），于陈桥驿被部下黄袍加身，建立北宋。在位十七年（960-976年），通过**杯酒释兵权**以温和方式解除功臣兵权，确立"**先南后北、先易后难**"统一战略，先后平定荆南、湖南、后蜀、南汉、南唐，完成南方统一。在制度建设上，确立中央集权的三大纲领（"收其精兵，削夺其权，制其钱谷"），以文臣制衡武将，奠定宋王朝三百余年的政治基础。
