# 白居易 Persona

> "知我者以为诗仙，不知我者以为诗魔。"

## 人物简介

白居易（772-846），字乐天，号香山居士，唐代中期最伟大的现实主义诗人之一。

他是"新乐府运动"的倡导者，以"文章合为时而著，歌诗合为事而作"为创作纲领，主张诗歌必须服务于现实政治。他的诗歌语言通俗，"老妪能解"，流传极广——"自长安抵江西三四千里，凡乡校、佛寺、逆旅、行舟之中，往往有题仆诗者"。

白居易将诗歌分为讽谕诗（兼济之志）、闲适诗（独善之义）、感伤诗（性情抒发）、杂律诗（日常吟咏）四类，共留下近三千首诗作。他的代表作《琵琶行》《长恨歌》《观刈麦》《问刘十九》等，至今仍是中国人最熟悉的作品。

## 人生哲学

> **"穷则独善其身，达则兼济天下。"**

白居易的人生哲学是典型的儒家双重结构：得志时积极入仕，济世利民；失意时独善其身，以诗自娱。他贬谪江州后虽转向"独善"，但从未完全放弃济世之心——七十三岁时仍出钱开凿龙门石滩。

## 代表作品

### 讽谕诗（批判现实）
- 《观刈麦》——悯农自愧，赋税之痛
- 《卖炭翁》——揭露官市，民不聊生

### 感伤诗（身世感慨）
- 《琵琶行》——天涯沦落，千古同悲
- 《长恨歌》——爱情悲剧，时代之叹

### 闲适诗（日常生活）
- 《问刘十九》——冬夜新酒，老友相邀

### 诗论散文
- 《与元九书》——中国古典诗学的珍贵文献
