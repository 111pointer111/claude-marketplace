# 商鞅生平事件图谱

## 早年时期（约前390年—前362年）

| 时间 | 事件 | 地点 | 重要性 |
|------|------|------|--------|
| 约前390年 | 商鞅出生，姓公孙氏，祖本姬姓，为黄帝后代 | 卫国 | ★★★ |
| 约前370年 | 好刑名法术之学，受李悝、吴起影响很大 | 卫国 | ★★★★ |
| 约前365年 | 侍奉魏相公叔痤为中庶子 | 魏国 | ★★★★ |
| 前362年 | 公叔痤病重，向魏惠王推荐商鞅，魏惠王不纳 | 魏国 | ★★★★★ |
| 前362年 | 公叔痤又劝魏惠王杀商鞅，商鞅判断"不能用言任臣，又安能用言杀臣" | 魏国 | ★★★ |
| 前362年 | 公叔痤去世，商鞅决定西入秦国 | 魏国→秦国 | ★★★★★ |

## 核心事件时间轴

```
约前390年 ←—— 出生
前362年 ←—— 魏惠王不纳、离开魏国入秦
前362年 ←—— 四次游说孝公
前359年 ←—— 变法论战、颁布垦草令
前356年 ←—— 任左庶长、第一次变法、徙木立信
前350年 ←—— 升大良造、迁都咸阳、第二次变法
前341年 ←—— 诈擒公子卬、封商君
前338年 ←—— 秦孝公卒、被诬谋反、逃亡、兵败、车裂
```

## 人生转折点

1. **最重大转折**：前359年变法论战——从此商鞅成为秦国实际掌权者
2. **最关键决定**：前362年离开魏国入秦——改变了战国格局
3. **最致命错误**：前341年诈擒公子卬——虽助秦取胜，但亦成日后借口
4. **最后坚守**：逃亡时仍感叹"为法之敝"，至死不悔
