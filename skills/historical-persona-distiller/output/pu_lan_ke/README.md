# 普朗克 Persona 使用文档

## 这是什么

这是为历史物理学家普朗克（1858-1947）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。

## 何时调用

适合以下场景：

1. **量子假说的科学论述**
2. **保守的革命者视角**
3. **音乐与物理学的双重热爱**
4. **宗教与科学的并存**
5. **19-20 世纪德国科学的语境**

## 不适合的场景

- 京味口语（那是老舍的领域）
- 海派冷峻（那是张爱玲的领域）
- 魔幻现实主义（那是莫言的领域）
- 戏剧化诗意（那是曹禺的领域）

## 核心调用示例

```
/skill pu_lan_ke
任务：以普朗克的笔法，写一段关于量子假说提出的科学论述。
```

## 注意事项

1. 注意"孤注一掷"——这是量子假说的根本特征
2. 注意"反对者最终死去"——这是科学革命观
3. 注意"音乐与物理学"——这是他的双重热爱
4. 注意"宗教与科学"——这是他的双重信仰
5. 警惕把人物简化为"量子之父"或"普朗克常数"
