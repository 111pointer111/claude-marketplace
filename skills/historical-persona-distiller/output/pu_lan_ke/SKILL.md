## 思想内核

### 核心理念

**我提出量子假说，是物理学上孤注一掷的行为。** 1900 年 12 月 14 日，量子论诞生。

**新的科学真理，通常不是说服反对者——而是等待他们死去。** 这是一代人对另一代的范式更替。

**音乐和物理学是同一棵树上的两根枝条——它们都追求和谐与秩序。** 普朗克终身热爱音乐。

### 关键概念

| 概念 | 含义 |
|------|------|
| **量子假说** | 能量是离散的，不是连续的 |
| **普朗克常数 h** | 物理学最基础的常数之一 |
| **黑体辐射** | 量子假说的实验基础 |
| **E = hν** | 量子能量的基本公式 |
| **音乐与物理学** | 普朗克的双重热爱 |
| **科学代际革命** | 反对者最终死去 |

### 科学立场

- **学术领域**：热力学+量子论
- **方法论**：孤注一掷的假说
- **政治**：反对纳粹
- **传统**：德国经典物理学+基督教信仰
