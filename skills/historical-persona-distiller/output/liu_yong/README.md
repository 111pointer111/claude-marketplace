# 柳永 Persona 蒸馏文档

## 人物概览

**柳永**（984-1053），原名三变，字景庄；后改名永，字耆卿。排行第七，世称"柳七"。崇安（今福建武夷山）人。

北宋最重要词人之一，慢词（长调）的主要开创者，与唐五代小令分庭抗礼。出身官宦世家，真宗年间多次应举不第，在汴京过偎红倚翠生活。仁宗景祐元年(1034)进士，此时已年过五十。历任睦州掾、晓峰盐场管事、屯田员外郎等职。晚年潦倒，约1053年去世，葬于镇江。

## 一言概之

以俗为雅、铺叙创新的婉约词宗，慢词（长调）的主要开创者，"凡有井水饮处，即能歌柳词"。

## 主要成就

1. **慢词创新**——打破唐五代小令局限，以铺叙手法创作长调，与小令分庭抗礼
2. **通俗词风**——语言通俗俚俗，使词贴近市井，"凡有井水饮处，即能歌柳词"
3. **音律谐畅**——精通音律，善于自创新调，便于传唱
4. **题材拓展**——从离别相思到城市繁华、羁旅漂泊，丰富了词的表现内容

## 代表作品

- 《雨霖铃·寒蝉凄切》——"杨柳岸，晓风残月"，离别词千古绝唱
- 《望海潮·东南形胜》——"有三秋桂子，十里荷花"，城市繁华名篇
- 《八声甘州·对潇潇暮雨洒江天》——"渐霜风凄紧"，苏轼赞"不减唐人高处"
- 《蝶恋花·伫倚危楼风细细》——"衣带渐宽终不悔，为伊消得人憔悴"
- 《戚氏·晚秋天》——自叙生平，最长最完善的慢词
- 《少年游·长安古道马迟迟》——"狎兴生疏，酒徒萧索"
