# EVENTS.md — Walt Whitman 生平事件图谱

## 早年 (1819–1840)

| 年份 | 事件 | 关键词 |
|------|------|--------|
| 1819.05.31 | 出生于纽约长岛西山区(West Hills) | 荷兰血统母亲/英国血统父亲 |
| 1823 | 全家迁往布鲁克林，父亲做建筑承包商 | 城市扩张期 |
| 1831.12 | 12岁开始印刷学徒，爱上文字 | 印刷工 |
| 1836 | 17岁在长岛单间校舍任教 | 教师 |
| 1838 | 19岁租稳定阁楼，创办报纸 Long-Islander | 出版人 |
| 1840 | 21岁迁回曼哈顿，为民主党政治刊物撰稿演讲 | 政治活动 |

## 新闻全盛期 (1841–1848)

| 年份 | 事件 | 关键词 |
|------|------|--------|
| 1841 | 进入纽约World报纸担任排版员 | 职业记者 |
| 1843 | 担任曼哈顿 Aurora 日报主编，发行量增加 | 报纸编辑 |
| 1845 | 返回布鲁克林，为Star报报道剧院和文学 | 剧评/书评 |
| 1846 | 27岁任布鲁克林鹰报编辑，撰写 Emerson/Carlyle/Melville 书评 | 文学评论 |
| 1846–1848 | 支持民主党/反对美墨战争，后被解雇 | 政治立场转变 |
| 1848.02 | 任新奥尔良 Crescent 报纸编辑仅3个月 | 南方经历 |
| 1848.05 | 回到布鲁克林，共同创办自由土壤报 Brooklyn Freeman | 废奴主义起点 |
| 1848.秋 | 亲眼目睹新奥尔良奴隶拍卖，返回后成废奴主义者 | 见证奴隶制 |

## 《草叶集》创造期 (1850–1861)

| 年份 | 事件 | 关键词 |
|------|------|--------|
| 1850–1855 | 布鲁克林做建筑承包商/包工头，同时酝酿新诗风 | 边缘生活 |
| 1855.07 | 自资出版《草叶集》初版（12首，无题诗+序言），无出版社无署名 | 独立出版 |
| 1855.07 | 收到 Emerson 来信称赞："美国迄今贡献的最非凡的智慧与机智之作" | 文学认可 |
| 1856 | 第二版，33首诗，附 Emerson 公开信 + Whitman 公开复信 | 版式扩张 |
| 1857–1859 | 任布鲁克林时报编辑，诗歌创作间歇 | 报纸编辑 |
| 1860 | 第三版（Boston），大量扩充；含 Calamus 诗系列（同性爱情危机诗） | 诗歌高峰 |

## 内战期 (1861–1865)

| 年份 | 事件 | 关键词 |
|------|------|--------|
| 1861 | 美国内战爆发，Whitman 45岁，发誓过"净化"生活 | 战争爆发 |
| 1862.12 | 弟弟 George 受伤，赴华盛顿，访问医院伤兵 | 战地看护 |
| 1862–1872 | 在华盛顿医院做志愿护工11年（联邦和南方士兵同等对待） | 医院志工 |
| 1865.01 | 任内政部职员（Department of the Interior） | 联邦雇员 |
| 1865.05 | 升职；6月被内政部长 James Harlan 因《草叶集》"不体面"解雇 | 被解雇 |
| 1865 | William O'Connor 发表《好灰诗人》为惠特曼辩护 | 舆论支持 |
| 1865.04.17 | 林肯遇刺后立即写诗 Hush'd Be The Camps To-day | 林肯之死 |
| 1865–1866 | 写 O Captain! My Captain! 和 When Lilacs Last in the Dooryard Bloom'd | 经典挽歌 |
| 1865 | Drum-Taps 出版（内战诗歌集） | 战时诗集 |

## 晚年 (1865–1892)

| 年份 | 事件 | 关键词 |
|------|------|--------|
| 1865–1872 | 留任司法部长办公室职员，华盛顿生活 | 政府文员 |
| 1873 | 第一次中风，半身不遂，搬至新泽西州卡姆登弟弟家 | 中风 |
| 1873–1882 | 在卡姆登照顾母亲（1873年母亲去世） | 家庭看护 |
| 1882 | 出版自传散文《典型日子》(Specimen Days) | 自传散文 |
| 1882 | 《草叶集》终于盈利，用版税在卡姆登买房子 | 经济独立 |
| 1891–1892 | 最终版《草叶集》(David McKay, 1891–92)；Good-Bye My Fancy (1891) | 最后出版 |
| 1892.03.26 | 死于肺气肿（肺气肿导致左肺完全萎缩），葬于自建墓穴 | 去世 |
