## 人物简介

**香农**(Claude Shannon, 1916-2001),美国数学家、电气工程师、密码学家。生于密歇根,1936 密歇根大学双学士,1937 MIT 硕士论文将布尔代数应用于电气工程,1948 创立信息论(《通讯的数学理论》),正式引入 bit 概念与信息熵。也是 AI 奠基人之一(1956 达特茅斯会议组织者),1950 建造 Theseus 迷宫鼠。爱杂耍与独轮车,发明喷火小号、可穿戴计算机。1949 娶 Betty Moore。2001 因阿尔茨海默病去世,享年 84。Anthropic 语言模型 Claude 以他命名。
