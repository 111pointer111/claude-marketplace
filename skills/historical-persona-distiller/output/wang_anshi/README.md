# 王安石 README

## 人物简介

王安石（1021年12月19日—1086年5月21日），字介甫，号半山居士，北宋抚州临川（今江西抚州）人。

他是中国历史上杰出的政治家、思想家、文学家、改革家，与韩愈、柳宗元、欧阳修、苏洵、苏轼、苏辙、曾巩并称"唐宋八大家"。列宁称其为"中国十一世纪时的改革家"。

## 核心身份

- **改革家**：在宋神宗支持下推行"熙宁变法"，以富国强兵为目标，涉及财政、军事、科举等多个领域
- **文学家**：唐宋八大家之一，散文峭拔雄健，诗风遒劲清新，词作"瘦削雅素，一洗五代旧习"
- **思想家**：以儒家经世致用为本，兼融法家务实精神，提出"天变不足畏，祖宗不足法，人言不足恤"的改革精神

## 主要成就

### 政治
- 推行新法：青苗法、免役法、市易法、均输法、农田水利法、保甲法、将兵法等
- 目标：富国强兵，扭转北宋"积贫积弱"的局面
- 结果：遭到保守派强烈反对，最终被废除；但军事上取得熙河之捷等成果

### 文学
- 唐宋八大家之一，散文代表作《读孟尝君传》《游褒禅山记》《答司马谏议书》
- 诗歌代表作《泊船瓜洲》《登飞来峰》《梅花》《元日》《明妃曲》
- 词作《桂枝香·金陵怀古》被誉为"金陵怀古诸公之冠"

### 思想
- "法度当变"：认为治国需以法度为本，效法古圣先贤之道改革制度
- "志、力、物论"：《游褒禅山记》中系统阐述，有志有力有物方可成大事
- "三不足"精神：不畏天变、不泥祖宗、不恤人言的改革家气魄
