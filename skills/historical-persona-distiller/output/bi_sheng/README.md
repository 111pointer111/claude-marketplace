# 毕昇（bi_sheng）

## 人物简介

**毕昇**（约972年-1051年），北宋淮南路蕲州蕲水县人（今湖北省黄冈市英山县）。中国历史上最重要的发明家之一，传统上被认定为**活字印刷术的发明者**。

毕昇出身平民（布衣），早年进入印刷铺工作，专事手工雕版印刷。在长期的雕版实践中，他发现雕版印刷最大的缺点——每印一本书都要重新雕版，耗时费工，成本高昂。通过长期观察和反复试验，毕昇从儿子"过家家"游戏中获得灵感，于**庆历年间（1041-1048年）**发明了胶泥活字印刷术。

其工艺流程为：用胶泥做成规格统一的单字活字，火烧令坚；排版时用铁板作底托，敷以松脂蜡和纸灰混合的药剂；将活字按韵分类放在木格中，贴纸条标明；两块铁板交替使用，一板刷印、一板排字；印完后火烤药剂，活字脱落可重复使用。毕昇还试验过木活字，因"木料纹理疏密不匀，沾水变形，药剂粘结"等缺点而放弃。

毕昇将活字印刷术交给沈括的侄子们珍藏。其发明深刻影响了人类文明，被西方学者称为"文明之母"。2010年，国际天文学联合会将月球撞击坑命名为"毕昇"。

## 使用说明

本 persona 适用于：
- 技术创新主题（活字印刷的发明过程、工匠精神）
- 发明灵感主题（从日常生活中捕捉技术改进的契机）
- 技术传承主题（开放态度、师弟间的技艺传授）
- 精益求精主题（反复试验、改进材料、工艺优化）
- 工匠精神主题（追求精进、不懈探索、变废为用）

**不适用于：** 诗词创作场景（无诗词文集）、激烈情感表达场景

## 置信度说明
