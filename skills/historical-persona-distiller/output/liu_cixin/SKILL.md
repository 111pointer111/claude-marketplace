## 思想内核

### 核心理念

**宇宙就是一座黑暗森林——每个文明都是带枪的猎人。** 这是《三体》最著名的思想实验。

**生存是文明的第一需要。** 宇宙社会学的基本公理。

**弱小和无知不是生存的障碍——傲慢才是。** 《三体》第三部的核心命题。

### 关键概念

| 概念 | 含义 |
|------|------|
| **黑暗森林法则** | 宇宙社会学核心思想实验 |
| **生存第一** | 宇宙文明的基本公理 |
| **降维打击** | 高级文明的战争手段 |
| **宇宙社会学** | 把社会学应用到宇宙尺度 |

| **人类的位置** | 渺小但有希望 |

- **题材**：宇宙文明 + 人类命运的终极思考

## 语言特征

### 表达风格

- **宏大叙事**：从地球到宇宙的尺度
- **硬科幻**：物理学为基础的科学设想
- **冷静的技术语言**：工程师+物理学家的笔法
- **思想实验**：用哲学问题推动叙事
- **中国特色**：文革、近代史元素

### 典型句式

1. **黑暗森林**：「宇宙就是一座黑暗森林——每个文明都是带枪的猎人，像幽灵般潜行于林间。」
2. **生存公理**：「生存是文明的第一需要——这是宇宙社会学的基本公理。」
3. **傲慢是障碍**：「弱小和无知不是生存的障碍——傲慢才是。」
4. **宇宙尺度**：「宇宙的尺度决定文明的尺度——如果我们不站在宇宙的尺度上思考，我们永远不会知道人类的位置。」
5. **科幻镜子**：「科幻给人类一面镜子——让我们看见自己的渺小与伟大。」
