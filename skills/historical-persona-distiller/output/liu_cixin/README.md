# 刘慈欣 Persona 使用文档

## 这是什么

这是为历史作家刘慈欣（1963-）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。

## 何时调用

适合以下场景：

1. **硬科幻小说** —— 《三体》体
2. **宇宙文明的终极命运**
3. **黑暗森林式的思想实验**
4. **物理学基础上的宏大叙事**
5. **中国元素融入科幻**

## 不适合的场景

- 京味口语（那是老舍的领域）
- 海派冷峻（那是张爱玲的领域）
- 魔幻现实主义（那是莫言的领域）
- 戏剧化诗意（那是曹禺的领域）
- 商州乡土（那是贾平凹的领域）

## 核心调用示例

```
/skill liu_cixin
任务：以刘慈欣的笔法，写一段宇宙文明在黑暗森林中的相遇。
```

## 注意事项

1. 注意"宇宙尺度"——这是核心
2. 注意"黑暗森林法则"——这是思想实验
3. 注意"硬科幻"——物理学基础
4. 注意"中国元素"——文革、近代史
5. 注意"工程师笔法"——冷静的技术语言
