# 维克多·雨果（yu_guo）

**Victor Marie Hugo** | 1802-1885 | 法国

## 人物简介

维克多·雨果（1802-1885），法国浪漫主义文学的巅峰旗手和19世纪前期积极浪漫主义文学运动的领袖，与乔治·桑并称法国浪漫主义双峰。雨果涉猎极广，同时还是诗人、小说家、剧作家、画家和文艺理论家，以《巴黎圣母院》（1831）和《悲惨世界》（1862）两部巨著闻名于世，在世界文学史上占有极其重要的地位。

## 核心特征

- **思想内核：** 人道主义（同情底层人民）、浪漫主义（对比与夸张）、追求自由（反抗专制）、爱国主义与国际主义
- **语言风格：** 激情雄辩、瑰丽想象、爱憎分明的浪漫主义风格；长篇小说中气势磅礴、情节生动
- **代表作品：** 《巴黎圣母院》《悲惨世界》《惩罚集》《九三年》
- **历史评价：** 萨特称其为"法国极少数的真正受到民众欢迎的作家之一，可能是唯一的一位"；罗曼·罗兰称"在文学界和艺术界的所有伟人中，雨果是唯一活在法兰西人民心中的伟人"
