# 曾国藩（Zeng Guofan）1811–1872

## 人物简介

曾国藩，字伯涵，号涤生，湖南湘乡人，清末最重要的政治、军事、文化人物之一。

道光十八年（1838）进士，翰林院庶吉士出身，十年七迁至二品大员。咸丰年间因太平天国起义，在乡办团练，创建湘军，成为镇压太平天国的主力。他以"书生领兵"的方式，组建了一支以"忠义血性"为精神旗帜的私家军队，是中国近代军阀制度的开创者。

曾国藩是洋务运动的重要推动者。他最早提出"师夷智以造炮制船"，在安庆创办内军械所，推动江南制造总局的建立，并派遣中国第一批官费留学生赴美。他是中国近代化事业的先驱人物之一。

在文化上，曾国藩继承桐城派古文，但自创一格，创立"湘乡派"。他将骈散结合，追求"雄奇瑰玮"，突破了桐城派"清淡简朴"的风格。他博采众长，融理学、气学、实学于一炉，形成了自己独特的人生哲学。

同治三年（1864）湘军攻破天京，曾国藩封一等毅勇侯。同治十一年（1872）病逝于南京，享年六十一岁，谥号"文正"。
