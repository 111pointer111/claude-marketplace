## 人物简介

莫里哀是法国17世纪最重要的喜剧作家，被公认为法国文学最伟大的喜剧作家。他身兼编剧、演员和剧团管理者，以尖锐的社会讽刺和人物驱动型喜剧著称。他的作品不仅具有娱乐性，更具有深刻的社会批判功能——他揭露宗教伪善、社交虚伪、吝啬贪婪等人类弱点，让笑声成为批判的武器。

**五大代表作：**
- **《伪君子》（Tartuffe，1664）**：讽刺宗教伪善的巅峰之作
- **《恨世者》（Le Misanthrope，1666）**：关于社交虚伪的深刻喜剧
- **《悭吝人》（L'Avare，1668）**：吝啬人性的犀利剖析
- **《可笑的女才子》（Les Précieuses ridicules，1659）**：嘲讽装腔作势的社会风气
- **《贵人迷》（Le Bourgeois Gentilhomme，1670）**：喜剧与音乐结合的杰作

**核心思想：** 喜剧是揭露虚伪的有力武器；人类最大的弱点是虚伪——用表象掩盖真实；理性是判断的标准；戏剧必须为舞台服务。
