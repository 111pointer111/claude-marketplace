# EVENTS.md — 莫里哀生平事件图谱

## 基本信息

- **本名**：Jean-Baptiste Poquelin
- **艺名**：Molière（首次出现于1644年6月28日的文件）
- **生卒**：1622年1月15日 - 1673年2月17日，巴黎
- **享年**：51岁

---

## 阶段一：早年与教育（1622-1643）

| 时间 | 事件 |
|------|------|
| 1622年1月15日 | 出生于巴黎，父亲Jean Poquelin是皇室任命的宫廷供货商 |
| 约1632年 | 母亲去世（时年约10岁） |
| 约1632-1641年 | 就读于Collège de Clermont（Jesuit学校，后成为Lycée Louis-le-Grand） |
| 1641年 | 完成法律学业，获准执业 |

---

## 阶段二：剧团创业与外省巡回（1643-1658）

| 时间 | 事件 |
|------|------|
| 1643年 | 放弃继承父业，投身戏剧；与Madeleine Béjart共同创立Illustre-Théâtre |
| 1644年 | 艺名"Molière"首次出现于文件记录 |
| 1645年 | 两度因剧团债务入狱 |
| 约1645-1658年 | 外省巡回演出长达13年——到过南特(1648)、图卢兹(1649)、里昂(1652-1657)、蒙彼利埃(1654-1655)、贝齐埃(1656)等 |
| 1655年 | 《冒失鬼》（L'Étourdi）在里昂首演——现存最早剧作 |
| 1656年 | 《爱情怨气》（Le Dépit amoureux）在贝齐埃首演 |

---

## 阶段三：巴黎崛起与皇室恩宠（1658-1668）

| 时间 | 事件 |
|------|------|
| 1658年10月24日 | 在卢浮宫为路易十四演出皮埃尔·高乃依的《尼科梅德》，随后演出《爱情医生》（Le Docteur amoureux），获国王弟弟菲利普（奥尔良公爵）青睐 |
| 1659年 | 《可笑的女才子》（Les Précieuses ridicules）在巴黎首演，大获成功 |
| 1660年 | 《斯嘎纳雷尔》（Sganarelle）首演 |
| 1662年12月 | 《妇人学校》（L'École des femmes）首演，引发"妇人学校之争"，遭受攻击 |
| 1663年6月 | 以《〈妇人学校〉评论》（Critique de l'École des femmes）回应批评，为喜剧辩护 |
| 1664年5月 | 《伪君子》（Tartuffe ou l'Imposteur）在凡尔赛为路易十四演出；遭圣事会强烈反对，被禁演 |
| 1665年2月 | 《唐璜》（Dom Juan）首演 |
| 1666年6月 | 《恨世者》（Le Misanthrope）首演 |
| 1667年 | 《恨世者》修订版上演 |
| 1668年 | 《悭吝人》（L'Avare）首演 |
| 1669年 | 圣事会势力减弱，《伪君子》终于获准全面上演，大获成功 |

---

## 阶段四：巅峰与晚年（1669-1673）

| 时间 | 事件 |
|------|------|
| 1670年 | 《贵人迷》（Le Bourgeois Gentilhomme）与音乐家Lully合作喜剧芭蕾 |
| 约1669-1673年 | 健康状况恶化（可能是肺结核），伴随婚姻问题和抑郁 |
| 1671年 | 《司卡潘的诡计》（Les Fourberies de Scapin）首演 |
| 1672年 | Madeleine Béjart去世 |
| 1673年2月17日 | 《假想的病人》（Le Malade imaginaire）第四场演出中癫痫发作，当晚去世；只有两名修女在场，被拒绝见神父 |

---

## 重要评价与遗产

| 时间 | 事件 |
|------|------|
| 1673年后 | 路易十四安排出版莫里哀全集，约10年后出版 |
| 19世纪 | 他的戏剧作品被当作纯文学阅读，忽视舞台本质 |
| 现代 | 查理·卓别林等喜剧大师被比作莫里哀 |
| 至今 | 《伪君子》《恨世者》《悭吝人》等作品仍在世界各地上演 |

---

## 历史地位

莫里哀通常被称为"法国的莎士比亚"，被认为是法国喜剧的巅峰。他改变了喜剧的地位，使其从娱乐性表演提升为具有社会批判功能的严肃艺术形式。
