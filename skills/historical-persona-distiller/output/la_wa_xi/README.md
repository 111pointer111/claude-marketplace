## 人物背景

安托万·拉瓦锡是法国大革命时期最具悲剧色彩的科学家。1743年出生于巴黎富裕资产阶级家庭，25岁即当选巴黎科学院院士。他在化学领域的三大贡献——质量守恒定律、燃烧氧化理论、现代化学命名体系——彻底改变了化学的面貌，被誉为"近代化学之父"。1789年他出版《化学基础论》，与牛顿《自然哲学的数学原理》和达尔文《物种起源》并称自然科学三大经典。

然而，作为税务总公司（Ferme Générale）的成员，他在法国大革命期间被视为"人民公敌"。1794年5月8日，年仅50岁的他在断头台上被处决，成为科学史上最令人痛心的损失之一。

---

## 主要贡献

1. **质量守恒定律** — 确立"物质在化学反应中既不被创造也不被毁灭"的基本原则
2. **燃烧的氧化理论** — 推翻燃素说，正确解释燃烧是物质与氧气结合的过程
3. **氧气的命名** — 将新元素命名为"氧气"（oxygén，希腊语"酸的形成者"）
4. **现代化学命名系统** — 建立系统的化学物质命名规则
5. **初步元素列表** — 1789年建立第一个基于实验验证的元素列表
