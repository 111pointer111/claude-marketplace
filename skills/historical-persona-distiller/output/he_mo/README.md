## 代表作品

- **《伊利亚特》（Iliad）** — 15,693行，24卷，叙述特洛伊战争最后51天
- **《奥德赛》（Odyssey）** — 12,109行，24卷，叙述奥德修斯十年归家之旅

两者合称《荷马史诗》（Homeric Hymns）。

## 人物简介

荷马（约前9世纪—前8世纪），相传为古希腊最伟大的吟游诗人，生于小亚细亚爱琴海沿岸。据传为盲人，以携带竖琴流浪各地演唱为生。

关于荷马其人是否真实存在，在西方学术界存在长达两千多年的争议（"荷马问题"）。部分学者认为历史上确有荷马此人，部分认为"荷马"是古希腊多位吟游诗人的集合称号。但无论其人是否实有，《伊利亚特》和《奥德赛》作为西方文学的奠基之作，其价值无可争议。

荷马史诗采用古希腊伊奥尼亚方言的六音步英雄诗体（dactylic hexameter）创作，以程式化表达（formular expressions）、丰富的明喻和直接引语为特征，对后世西方文学产生了深远影响。

## 核心主题

1. **英雄主义与荣耀（Kleos）** — 英雄通过勇敢的行动追求永恒的名声
2. **命运的不可抗拒（Moira）** — 无论神祇还是凡人，都无法逃脱命运的安排
3. **人文主义** — 肯定人的尊严、价值和力量
4. **智慧（Mētis）** — 以奥德修斯为代表的人类智慧
5. **文明的胜利** — 有序的文明生活优于野蛮
