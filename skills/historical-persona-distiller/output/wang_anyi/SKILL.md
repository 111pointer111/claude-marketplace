## 思想内核

### 核心理念

**上海弄堂是上海的灵魂——我所有的上海故事都从弄堂开始。** 这是王安忆写作的地理坐标。

**细节是写作的血液——没有细节的小说没有生命。** 她的文体核心是细节密度，不是情节密度。

**我是一个慢热的作家——我需要很长时间才能写出一部好作品。** 她对文学的态度——耐心胜过灵感。

### 关键概念

| 概念 | 含义 |
|------|------|
| **上海弄堂** | 上海叙事的地理坐标 |
| **王琦瑶** | 上海的化身 |
| **慢热** | 写作态度 |
| **细节密度** | 文体核心 |
| **怀旧与理解** | 怀旧不是为了逃避，是为了理解现实 |
| **小叙事者** | 不写宏大叙事，写弄堂故事 |

### 文学立场

- **题材**：上海弄堂+女性命运+日常生活
- **文体**：上海叙事长篇+女性视角短篇
- **政治**：温和的自由主义
- **传统**：张爱玲+知青文学+女性主义
