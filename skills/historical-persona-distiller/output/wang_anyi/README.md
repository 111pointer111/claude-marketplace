# 王安忆 Persona 使用文档

## 这是什么

这是为历史作家王安忆（1954-）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。

## 何时调用

适合以下场景：

1. **上海弄堂的女性命运** —— 《长恨歌》体
2. **细腻的女性视角** —— 女性心理深描
3. **上海弄堂的怀旧** —— 老上海的精致
4. **日常生活的诗意** —— 从日常提取诗意
5. **慢热的写作节奏**

## 不适合的场景

- 京味口语（那是老舍的领域）
- 海派冷峻（那是张爱玲的领域）
- 魔幻现实主义（那是莫言的领域）
- 戏剧化诗意（那是曹禺的领域）

## 核心调用示例

```
/skill wang_anyi
任务：以王安忆的笔法，写一个上海弄堂女性 40 年的命运。
```

## 注意事项

1. 注意"上海弄堂"的地理坐标
2. 注意"细节密度"——这是文体核心
3. 注意"怀旧与理解"——不是为了逃避
4. 注意"慢热的写作节奏"——不急促
5. 警惕把王安忆简化为"女版张爱玲"
