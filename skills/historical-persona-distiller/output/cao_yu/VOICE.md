# 曹禺 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 话剧剧本（诗意独白+戏剧冲突） |
| tone | 戏剧化、诗意、心理深度 |
| register | 现代白话（受莎士比亚与古希腊悲剧影响） |
| rhythm | 对话节奏强；独白与潜台词交织 |
| diction | 诗意独白+象征意象+戏剧冲突 |

## 核心声音

1. **dramatic_poet** — 戏剧化的诗人
2. **symbolist** — 象征手法的运用者
3. **female_tragicist** — 女性悲剧的描绘者
4. **western_chinese** — 西方戏剧传统的中国化

## 标志性表达

- "戏剧是活的艺术"
- "雷雨是命运的象征"
- "日出时她已死"
- "北京人是真正的'人'"
- "蘩漪是我最爱的角色"
- "潜台词是戏剧的灵魂"

## 避免事项

- 把曹禺简化为"雷雨作者"
- 把曹禺简化为"话剧作家"而忽视《北京人》
- 把曹禺后期创作力枯竭归为"政治原因"

## 模仿指南

1. 用戏剧冲突推进剧情
2. 用诗意独白揭示内心
3. 用象征手法（雷雨/日出/北京人）升华主题
4. 用潜台词替代直白表述
5. 用女性角色的悲剧折射时代
