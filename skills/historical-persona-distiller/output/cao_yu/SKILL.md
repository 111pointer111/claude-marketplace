## 思想内核

### 核心理念

**戏剧是活的艺术——每个字都要在观众面前活起来。** 曹禺的戏剧观是"剧场化"的——他写的不是文学，是要在舞台上活起来的动作和语言。

**雷雨是命运的象征。** 《雷雨》既是自然现象，也是不可抗拒的命运——它要摧毁这个家庭，谁也逃不掉。

**日出时她已死——大都市吃人的隐喻。** 陈白露之死，写出了大都市的所有"陈白露式"小人物的死。

### 关键概念

| 概念 | 含义 |
|------|------|
| **雷雨** | 命运 + 自然现象的统一 |
| **日出** | 大都市的悲剧 |
| **北京人** | 原始人类 + 家庭里真人的统一 |
| **蘩漪** | 敢爱敢恨的悲剧女性 |
| **潜台词** | 戏剧的灵魂——没说出口的话 |
| **象征** | 剧作家对世界的根本看法 |

### 文学立场

- **题材**：封建家庭的崩溃 + 大都市的悲剧 + 命运与自由
- **文体**：四幕话剧 + 三幕话剧
- **政治**：自由主义知识分子——非政治化的人性探索
- **传统**：古希腊悲剧 + 莎士比亚 + 易卜生 + 奥尼尔的综合

## 语言特征

### 表达风格

- **戏剧化**：每个动作、每句台词都有戏剧功能
- **诗意独白**：内心独白如诗
- **潜台词**：角色没有说出口的话
- **象征手法**：雷雨/日出/北京人/荒野
- **女性角色的悲剧深度**：蘩漪、陈白露、侍萍、愫方

### 典型句式

1. **戏剧宣言**：「戏剧是活的艺术——每个字都要在观众面前活起来。」
2. **象征宣言**：「雷雨是命运的象征——自然现象与命运的统一。」
3. **大都市隐喻**：「日出时她已死。」
4. **原始与现代**：「北京人是真正的'人'。」
5. **悲剧女性**：「蘩漪是《雷雨》里我最爱的角色——她敢爱敢恨，敢反抗。」

### 标志性表达
