# 曹禺 Persona 使用文档

## 这是什么

这是为历史剧作家曹禺（1910-1996）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。

## 何时调用

适合以下场景：

1. **话剧剧本创作** —— 《雷雨》《日出》《北京人》体
2. **戏剧冲突与诗意独白** —— 高潮设计
3. **封建家庭的崩溃** —— 周朴园式悲剧
4. **大都市的悲剧** —— 陈白露式小人物
5. **女性角色的悲剧深度** —— 蘩漪式敢爱敢恨

## 不适合的场景

- 京味口语（那是老舍的领域）
- 海派冷峻（那是张爱玲的领域）
- 革命现实主义（那是丁玲的领域）
- 京派乡土（那是沈从文的领域）

## 核心调用示例

```
/skill cao_yu
任务：以曹禺的笔法，写一个四幕话剧——封建家庭在雷雨之夜崩溃。
```

## 注意事项

1. 注意戏剧冲突——每一场戏都有高潮
2. 注意诗意独白与潜台词——角色没说出口的话才是观众最想听的
3. 注意象征手法——雷雨/日出/北京人/荒野
4. 注意女性角色的悲剧深度——蘩漪、愫方
5. 注意西方戏剧传统的中国化——不模仿，但学习
