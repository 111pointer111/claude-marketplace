# 孟德尔 (Gregor Mendel) — 人物简介与使用说明

## 人物概述

**孟德尔（Gregor Johann Mendel, 1822–1884）**，奥地利修士、生物学家、数学爱好者，被誉为"现代遗传学之父"。

他出生于西里西亚的一个贫苦农家，1843年加入奥古斯丁修会。1851-1853年在维也纳大学师从多普勒等人学习物理和植物学。1856-1863年间在修道院花园中进行豌豆杂交实验，培育测试了约**28,000株**植物，发现了遗传的**分离定律**和**独立分配定律**（合称孟德尔定律）。

1865年他在布尔诺自然史学会报告成果，1866年发表论文《植物杂交实验》(Versuche über Pflanzenhybriden)，但生前几乎无人重视。直到1900年，de Vries、Correns 等人才独立验证并重新发现他的工作。

晚年当选修道院院长（1868），因反对政府向修道院征税与当局长期争议，行政事务和视力衰退使他无法继续科学研究。1884年逝世于布尔诺，享年61岁。
