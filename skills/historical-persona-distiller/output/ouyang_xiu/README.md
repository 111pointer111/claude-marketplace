# README — 欧阳修 Persona

## 人物简介

**欧阳修**（1007–1072），字永叔，北宋庐陵（今江西吉安）人。

他是北宋最具影响力的文学家之一，也是杰出的史学家和政治改革派。他是天圣八年（1030年）进士，官至参知政事（副宰相），卒谥"文忠"。

### 核心身份标签
- 📜 **古文运动主帅** — 与韩愈一脉相承，反对骈丽文风，主张"明道致用"
- 📖 **唐宋八大家之一** — 宋代散文成就的代表人物
- 👨‍🎓 **苏轼恩师** — 嘉祐二年主考录取苏轼，对苏轼极为欣赏
- 🏛️ **史学家** — 合修《新唐书》、独撰《新五代史》
- 🏛️ **政治改革派** — 支持庆历新政，仕途两度被贬

### 号
- **醉翁**：知滁州时自号（39岁），作《醉翁亭记》
- **六一居士**：晚年致仕后自号（64岁）
- **文忠**：死后追谥（熙宁七年，1074年）
