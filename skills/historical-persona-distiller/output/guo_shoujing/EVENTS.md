# 郭守敬 EVENTS.md — 生平事件图谱

## 时间线

### 早年（1231-1251）

| 时间 | 事件 | 类型 |
|------|------|------|
| 1231年（金正大八年） | 生于邢州邢台县 | 出生 |
| 约1246-1247年（15-16岁） | 自制竹篾浑仪进行天文观测 | 成就 |
| 约1246-1247年（15-16岁） | 据燕肃石刻莲花漏图自学计时仪器原理 | 成就 |
| 1251年（蒙哥汗八年） | 祖父郭荣将其送入刘秉忠门下深造 | 教育 |

### 青年时期（1251-1276）

| 时间 | 事件 | 类型 |
|------|------|------|
| 约1251-1260年 | 参与邢台河道整治工程，疏浚水泽、挖出埋没近30年石桥 | 工程 |
| 约1251-1260年 | 文学家元好问为邢州新石桥写记（郭生即郭守敬） | 事件 |
| 1260年（中统元年） | 随张文谦到大名路进行水利调查 | 仕途 |
| 1260-1262年 | 在大名召集匠人鼓铸宝山漏 | 成就 |
| 1262年（中统三年） | 受张文谦推荐，在开平府受忽必烈召见，面陈水利六条 | 仕途 |
| 1262年（中统三年） | 被任命为提举诸路河渠 | 仕途 |
| 1263年（中统四年） | 升副河渠使，加授银符 | 仕途 |
| 1264年（至元元年） | 前往西夏视察河渠水道 | 工程 |
| 1264年（至元元年） | 修浚唐来渠（400余里）、汉延渠（250余里）及大小渠系 | 工程 |
| 1265年（至元二年） | 升任都水少监 | 仕途 |
| 1265年（至元二年） | 自西夏返回，考察黄河漕运，上奏忽必烈 | 工程 |
| 1267年（至元四年） | 建议设水上驿站，获批10处 | 工程 |
| 1275年（至元十二年） | 奉命视察河北、山东可通舟行船之处 | 工程 |

### 天文时期（1276-1293）

| 时间 | 事件 | 类型 |
|------|------|------|
| 1276年（至元十三年） | 任工部郎中，受命研订新历 | 仕途/成就 |
| 1276年（至元十三年） | 提出"历之本在于测验，而测验之器莫先仪表" | 主张 |
| 1279年（至元十六年） | 太史局扩建为太史院，任同知太史院事 | 仕途 |
| 1279年（至元十六年） | 提议进行四海测验，获批准，率队开展27地点天文测量 | 成就 |
| 1280年（至元十七年） | 《授时历》完成 | 成就 |
| 1281年（至元十八年） | 王恂去世，承担太史院全部工作，整理《推步》《立成》等 | 成就 |
| 1286年（至元二十三年） | 升任太史令 | 仕途 |
| 1276-1286年 | 创制简仪、高表、仰仪等16件天文仪器 | 成就 |

### 水利时期（1291-1293）

| 时间 | 事件 | 类型 |
|------|------|------|
| 1291年（至元二十八年） | 勘查滦河、浑河航运可行性，提出通惠河方案 | 工程 |
| 1291年（至元二十八年） | 重置都水监，任领都水监事 | 仕途 |
| 1292年（至元二十九年） | 春，通惠河动工，忽必烈命丞相以下官员听其指挥 | 工程 |
| 1292年（至元二十九年） | 开辟白浮堰，解决大都运河水源问题 | 工程 |
| 1293年（至元三十年） | 七月，通惠河成，忽必烈赐名并赐钞12500贯 | 工程/荣誉 |

### 晚年（1294-1316）

| 时间 | 事件 | 类型 |
|------|------|------|
| 1294年（至元三十一年） | 任昭文馆大学士，兼知太史院事 | 仕途 |
| 1298年（大德二年） | 正确预见山洪渠道宽度不足，工程后果然泛滥 | 成就 |
| 1298年（大德二年） | 成宗叹道"郭太史真是神人"，为其未听劝谏自责 | 事件 |
| 1303年（大德七年） | 成宗下诏官员70岁退休，唯郭守敬不准退休 | 事件 |
| 1316年（延祐三年） | 去世，享年86岁 | 去世 |

## 核心人际关系

| 人物 | 关系 | 说明 |
|------|------|------|
| 郭荣（祖父） | 抚育者/启蒙者 | 号鸳水翁，金元之际著名学者，精通五经天文算学水利 |
| 刘秉忠 | 师从 | 元世祖重要谋士，精通经学和天文学 |
| 张文谦 | 师从/提携者 | 忽必烈左丞，引荐郭守敬入仕 |
| 忽必烈 | 君主/支持者 | 元世祖，多次采纳郭守敬建议 |
| 王恂 | 同事 | 太史令，与郭守敬共同制定授时历 |
| 许衡 | 同事 | 领太史院事，共同制定授时历 |
| 齐履谦 | 后世记述者 | 撰《知太史院事郭公行状》，为最重要一手文献 |
