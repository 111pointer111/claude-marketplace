# 郭守敬（guo_shoujing）Persona README

## 人物简介

郭守敬（1231-1316），字若思，元朝杰出的天文学家、数学家、水利工程专家和仪器制造家。邢州邢台县（今河北省邢台市）人。

他一生横跨金末元初，以实测和实践精神著称，主要成就包括：

1. **天文历法**：主持制定《授时历》（1280年），推算出回归年长度为365.2425日，与现行公历完全一致，比欧洲格里高利历早300年
2. **天文仪器**：创制简仪（世界上最早的赤道装置）、高表、景符等16件天文仪器
3. **水利工程**：修复西夏唐来渠、汉延渠（1264年）；主持开凿通惠河（1292-1293年），解决大都漕运问题
4. **地理学**：提出以海平面为基准测量海拔的概念，为中国地理学重要创举

**核心特质**：实测主义、躬行实践、精密数据导向、工程实效优先
