# 魏征生平事件图谱

## 时间线

### 早年（580-617年）

| 年份 | 事件 | 地点 |
|------|------|------|
| 约580年 | 出生于巨鹿魏氏 | 巨鹿（今河北） |
| 早年 | 孤苦贫寒，出家为道士 | 相州（今河南） |
| 大业年间 | 喜好读书，涉猎广泛，尤属意纵横之说 | 相州 |
| 大业十三年（617年） | 在武阳郡丞元宝藏帐下为官 | 武阳 |
| 617年 | 为元宝藏写奏疏给李密，展现文才 | 瓦岗 |
| 617年 | 被李密召为元帅府文学参军，献上十条计策（未被采纳） | 瓦岗 |

### 乱世从戎（618-622年）

| 年份 | 事件 | 地点 |
|------|------|------|
| 618年 | 李密被王世充击败，随密降唐 | 黎阳 |
| 618年 | 毛遂自荐，请求安抚山东，成功劝降李勣 | 黎阳→长安 |
| 618年 | 被窦建德俘虏，任起居舍人 | 河北 |
| 621年 | 李世民击败窦建德，再次归唐 | 长安 |
| 621年 | 被太子李建成收为洗马，礼遇甚厚 | 长安 |
| 622年 | 建议李建成请战平定刘黑闼，建成从之 | 山东 |

### 贞观直谏期（626-636年）

| 年份 | 事件 | 地点 |
|------|------|------|
| 626年 | 玄武门之变，李世民问罪魏征 | 长安 |
| 626年 | 魏征答："先太子若用我言，无今日之祸" | 长安 |
| 626年 | 任谏议大夫，授詹事府主簿 | 长安 |
| 627年 | 任尚书左丞 | 长安 |
| 627年 | 坚拒签署征中男敕令，据理力争，太宗纳谏 | 长安 |
| 628年 | 劝止征讨岭南冯盎，以安抚代替武力 | 长安 |
| 628年 | 论"良臣与忠臣"之别 | 长安 |
| 629年 | 以尚书右丞守秘书监，参预朝政 | 长安 |
| 630年 | 谏止濮州刺史庞相寿因贪污免职后复职 | 长安 |
| 632年 | 反对长乐公主嫁妆逾制，长孙皇后赐赏魏征 | 长安 |

### 晚年与《十渐》（636-643年）

| 年份 | 事件 | 地点 |
|------|------|------|
| 636年 | 长孙皇后去世，太宗建层观眺昭陵 | 长安 |
| 636年 | 主编《周书》《梁书》《陈书》《北齐书》《隋书》 | 长安 |
| 637年 | 受封郑国公 | 长安 |
| 638年 | 代王珪任侍中 | 长安 |
| 639年 | 上《十渐不克终疏》，批评太宗渐失初心 | 长安 |
| 642年 | 任太子太师 | 长安 |
| 643年正月 | 病重，太宗多次探望 | 长安 |
| 643年正月戊辰 | 去世，享年六十四岁 | 长安 |
| 643年 | 太宗废朝五天，亲制碑文，赐谥"文贞" | 昭陵 |
| 643年 | 入选凌烟阁二十四功臣，位列第四 | 长安 |

### 身后（643年后）

| 年份 | 事件 |
|------|------|
| 643年 | 侯君集谋反案发，李世民疑魏征结党，推倒其墓碑 |
| 645年 | 李世民亲征高句丽，叹息"魏征若在，不使我有是行"，重新立碑 |
