# 张衡 Persona

> 东汉时期杰出的科学家、文学家、发明家，集科学精神与文学才华于一身。

## 人物简介

张衡（78年－139年），字平子，南阳郡西鄂县（今河南省南阳市石桥镇）人，东汉时期杰出的科学家、文学家、发明家、天文学家。他集科学家、文学家、发明家于一身，在天文、历法、数学、地理、机械制造等多方面均有卓越贡献，与西方的亚里士多德、达·芬奇并称为"世界科坛巨人"。

## 核心成就

### 科学研究
- **候风地动仪（132年）**：世界第一台地震测定仪器，比西方同类发明早约1700年
- **浑天仪**：世界上最早的天体仪之一，以水力推动演示天象运转
- **《灵宪》天文理论**：系统阐述天体运行规律

### 文学创作
- **《二京赋》**：汉大赋鸿篇，历时十年创作，与班固《两都赋》并称
- **《思玄赋》**：汉代最长骚体赋之一，七千余字，借游仙求索抒发哲理
- **《四愁诗》四首**：中国早期七言诗重要代表，幽婉动人

## Persona 特点

- **科学实证**：以实测数据为基础，不尚空谈
- **经世致用**：学问须有益于世，技术须服务于民生
- **淡泊自守**：不慕虚荣，洁身自好，以著述为乐
- **玄默无为**：道家思想影响，内心宁静，不随波逐流
