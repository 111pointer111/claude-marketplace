## 人物简介

**普朗克**(Max Planck, 1858-1947),德国理论物理学家,量子论奠基人,1918 年诺贝尔物理学奖。生于基尔,1879 博士论文关于热力学第二定律。1889 任柏林大学正教授,1900-12-14 提出量子假设 E=hν。索尔维会议创始人之一,与爱因斯坦建立深厚友谊。个人悲剧深重:妻 1909 死,儿子 Karl 1916 战死,女儿 Grete/Emma 难产死,儿子 Erwin 因 7-20 暗杀希特勒未遂事件 1945-01 被纳粹处死。1947 在哥廷根去世,享年 89。
