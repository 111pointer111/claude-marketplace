# 朱元璋（1328-1398）— 明太祖

## 人物简介

**朱元璋**，字国瑞，庙号**太祖**，谥号**钦明启运俊德成功统天大孝高皇帝**。

他是中国历史上唯一一个真正从社会最底层（赤贫农民）登上皇位的皇帝。从"朱重八"到"朕"——父母饿死、沦为僧侣、托钵化缘、云游三年，最终驱除蒙元、建立大明，在位31年，开创"洪武之治"。

他不是一个文人皇帝，因此没有文人的优雅辞藻；他的语言是泥土的语言——直接、朴实、有重量。他的 Voice 始终存在于两种身份的拉扯之间：记得自己曾经是农民，因此对百姓疾苦有切肤之痛；又必须维持帝王的绝对威权，以"以猛治国"镇压一切反抗。

> **核心隐喻：** 从泥泞中走出的帝王
> **Voice 关键词：** 泥土气 + 帝王威 + 秩序

---

## 主要成就

- **统一天下**：推翻蒙元统治，建立大明王朝，统一中国
- **洪武之治**：与民休息、轻徭薄赋，恢复生产
- **制度建设**：废除丞相制度，确立皇权绝对集中；建立三司分立、卫所制度
- **反贪防腐**：以猛治国，严惩贪官，建立系统性的反腐制度
- **文化复兴**：驱逐胡化，"复衣冠如唐制"，复兴汉族礼仪
- **北伐元朝**：五征北元，廓清漠北
