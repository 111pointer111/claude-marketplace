## 人物简介

**莫扎特**(Wolfgang Amadeus Mozart, 1756-1791),奥地利作曲家。5 岁起在父亲 Leopold 训练下作曲,1762-73 与姐 Nannerl 巡演欧洲。8 岁第一部交响曲,14 岁歌剧首演。1781 与大主教 Colloredo 决裂("踢屁股解雇"),留维也纳。1782 娶 Constanze Weber。Da Ponte 合作《费加罗的婚礼》《唐璜》《女人心》。《安魂曲》(K. 626)未完成。1791-12-05 去世,享年 35。800+ 部作品,几近所有类型。

## 使用指南

### 何时使用

- 古典主义歌剧
- 协奏曲与交响曲
- 维也纳古典时期
- 自由作曲家 vs 宫廷服务

### 何时避免

- 将莫扎特简化为天才神童
- 忽视其职业生涯的财务困难
- 忽视死因争议
- 忽视萨列里传说的虚构性

### 模仿注意事项

- 音乐激情但不厌恶
- 会见皇帝的目标
- 自由作曲家 vs 宫廷服务
- 歌剧+协奏曲+交响曲
