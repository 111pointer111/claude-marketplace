# 老子（Lao Zi）

> 春秋时期道家学派创始人，"道法自然"的哲学奠基人

## 人物简介

老子（约公元前571年—约公元前471年），姓李名耳，字聃，春秋时期陈国苦县厉乡（今河南鹿邑或安徽涡阳）人。道家学派创始人，中国古代伟大的哲学家和思想家。

老子曾任周朝守藏室史，博览典籍，通礼乐之源，明道德之旨。孔子曾两次问礼于老子，称老子"其犹龙邪"。公元前485年左右，老子出函谷关，尹喜求其著书，老子遂著《道德经》上下两篇，共五千言。著书后骑青牛西去，莫知所终。

老子思想以"道"为核心，以"无为"为政治纲领，以"柔弱"为修身原则，构建了道家哲学体系。《道德经》与《易经》《论语》并称为对中国影响最深的三部思想巨著，被译为73种语言、1500余种译本，是世界上仅次于《圣经》流传最广的典籍。

## 核心思想

- **道法自然**：道是宇宙本原，道法自然是老子思想核心
- **无为而无不为**：顺应自然不妄为，无为可成就一切
- **辩证转化**：祸福相依，物极必反，有无相生
- **柔弱胜刚强**：水几于道，柔弱是道的本性
- **知足贵柔**：知足者富，不争之争

- 道家学派创始人，与庄子并称"老庄"
- 世界文化名人，西方人眼中的东方三大圣人之一

- 道教奉为道祖，称"太上老君"
