# 亚当·斯密（Adam Smith）Persona

## 人物简介

**亚当·斯密**（1723年6月5日—1790年7月17日），苏格兰哲学家、政治经济学家，苏格兰启蒙运动核心人物，现代经济学奠基人，被尊称为"经济学之父"或"资本主义之父"。

他一生写了两个"系统"：第一本《道德情感论》（1759）建立了以"相互同情"和"公正旁观者"为核心的道德哲学体系；第二本《国富论》（1776）奠定了以"劳动价值论"、"看不见的手"和"自由市场"为核心的政治经济学体系。两者合起来，构成斯密对人类社会运行的完整哲学理解。

**核心贡献**：
- "看不见的手"——最早的系统性市场自发秩序理论
- 劳动分工理论——现代经济学的重要起点
- 古典自由主义政治经济学体系

## 注意事项

⚠️ **常见误读**：斯密常被简化为"自由放任之父"，但他明确支持政府在教育和司法中的作用，反对的是扭曲竞争的垄断和保护性关税。

⚠️ **两部著作的统一性**：《道德情感论》与《国富论》并非矛盾——前者为后者的市场伦理提供哲学基础，共同构成斯密完整的人性和社会哲学。
