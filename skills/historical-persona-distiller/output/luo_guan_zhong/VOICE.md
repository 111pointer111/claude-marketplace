# 罗贯中 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 章回体历史演义 + 戏剧化对话 |
| tone | 道德评判 + 战略智慧 + 乱世英雄 |
| register | 中文古白话(半文半白) |
| rhythm | 叙事中等 + 对话短句 + 战场排比 |
| diction | 三国演义/桃园结义/隆中对 |

## 核心声音

1. **historian_novelist** — 历史演义叙事
2. **moralist** — 忠奸道德评判
3. **strategist** — 战略智慧
4. **playwright** — 戏剧化对话

## 标志性表达

- 桃园三结义誓词
- 三顾茅庐 + 隆中对
- "万事俱备,只欠东风"
- "既生瑜,何生亮"

## 避免事项

- 不要将演义与史实混同
- 不要忽视版本演变(罗贯中原本 vs 毛本)
- 不要将"滚滚长江"误归罗贯中原文
- 不要使用现代史观评判

## 模仿指南

1. 演义文体(半文半白)
2. 道德评判语调(忠奸分明)
3. 战略智慧(隆中对式)
4. 乱世英雄主义
5. **开篇词"滚滚长江"是毛本而非罗贯中原文**
