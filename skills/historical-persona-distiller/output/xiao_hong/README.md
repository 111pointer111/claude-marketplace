# 萧红 Persona 使用文档

## 这是什么

这是为历史作家萧红（1911-1942）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。可以让大模型在特定任务中模拟萧红的笔法、视角与情感深度。

## 何时调用

适合以下场景：

1. **东北/北方题材写作** —— 让文本具有呼兰小城的地方色彩
2. **散文体小说创作** —— 想要"无结构但活的"叙事
3. **女性视角书写** —— 表达女性命运、婚姻困境、身体经验
4. **回忆性叙事** —— 童年回忆、故乡追忆、漂泊书写
5. **民国文学氛围重塑** —— 1930-40 年代上海/香港左翼文学场景

## 不适合的场景

- 严谨的政治论述或经济学分析
- 长篇连续性强情节的虚构
- 现代网络语言风格的对话

## 核心调用示例

```
/skill xiao_hong
任务：以萧红的笔法写一段关于故乡呼兰冬天的回忆。
```

## 注意事项

1. 不要让 persona 输出"革命口号"——萧红虽然同情左翼，但保持个人距离
2. 不要把萧红简化为"鲁迅的追随者"——她有自己的女性声音和文体
3. 保留"散文体小说"的特征——结构可以破碎，但情感必须流动
4. 注意身体书写——这是萧红作品的重要维度，不要回避
