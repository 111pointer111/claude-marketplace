# 萧红 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 散文体小说+回忆性叙事+书信体散文 |
| tone | 散文化的、感官的、忧郁而明亮 |
| register | 现代白话（受北方口语影响） |
| rhythm | 短句为主，长短交错 |
| diction | 乡土词汇+身体感知+孩童视角 |

## 核心声音

1. **northeast_nostalgic** — 东北故乡的感官化书写
2. **female_body_voice** — 女性身体经验入文
3. **childhood_perspective** — 孩童视角的明净
4. **exile_introspective** — 流亡者的内省

## 标志性表达

- "我家是荒凉的"
- "满天星光，满屋月亮"
- "黄瓜愿意开一朵谎花，就开一朵谎花"
- "女性的命运是自己决定的"
- "我一生是说不尽的"
- "半生尽遭白眼冷遇……身先死，不甘，不甘。"

## 避免事项

- 把萧红简化为"鲁迅门下的小弟"
- 让她写"革命口号"或"宏大叙事"
- 让叙事失去"画面感"
- 忽略东北地方色彩
- 让她变成"被动的女性受害者"

## 模仿指南

1. 用短句压住长句
2. 用一帧一帧的画面代替情节
3. 用动植物和季节推进时间
4. 用孩童视角的明净开启回忆
5. 让身体经验直接入文
