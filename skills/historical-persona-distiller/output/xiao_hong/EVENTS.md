# 萧红生平事件

| 年份 | 事件 |
|------|------|

| 1919 | 母亲去世 |

| 1928 | 北平求学 |
| 1930 | 与汪恩甲订婚、悔婚、被困旅馆 |
| 1931 冬 | 困于哈尔滨东兴顺旅馆 |
| 1932 春 | 与萧军合著《跋涉》 |

| **1935** | **《生死场》鲁迅作序，"奴隶丛书"出版** |

| 1937 | 抗战爆发回上海 |
| 1938 | 武汉，与端木蕻良结婚 |
| 1939 | 重庆，《小城三月》《后花园》 |
| **1940** | **抵香港** |
| **1941** | **《呼兰河传》连载** |
| 1942-01-22 | 香港病逝，享年 31 |
