# 沈从文 Persona 使用文档

## 这是什么

这是为历史作家沈从文（1902-1988）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。

## 何时调用

适合以下场景：

1. **乡土小说创作** —— 湘西、故乡、底层人物
2. **诗意的散文化叙事** —— "画"一般的笔触
3. **对"现代性"的反思** —— 用"原始"对抗"现代"
4. **温柔的悲剧** —— 翠翠式的永恒等待
5. **情书体** —— 现代中文最美的情书之一
6. **文物研究** —— 中国古代服饰史

## 不适合的场景

- 京味口语（那是老舍的领域）
- 海派都市小说（那是张爱玲的领域）
- 革命现实主义（那是丁玲的领域）
- 学贯中西的学者气（那是钱钟书的领域）

## 核心调用示例

```
/skill shen_congwen
任务：以沈从文的笔法，写一个湘西少女在渡口的等待。
```

## 注意事项

1. 不要把沈从文简化为"乡土作家"——他也是文物研究者
2. 不要把《边城》浪漫化为"世外桃源"——它是"正在消失的湘西"
3. 注意诗意的散文化笔触——不是激烈的呐喊
4. 注意"对现代性的反思"——这是他的核心立场
5. 关注 1949 年自杀未遂与文物研究转向——这是理解沈从文的关键
