# 沈从文 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 乡土小说+散文+情书 |
| tone | 温柔纯净、贴近泥土、诗意 |
| register | 现代白话（受湘西方言与古文影响） |
| rhythm | 中句为主，节奏舒缓 |
| diction | 湘西山水+苗族文化+古代词汇+诗意比喻 |

## 核心声音

1. **xiangxi_narrator** — 湘西世界的凝视者
2. **humanity_priest** — 人性的供奉者
3. **anti_modernity** — 对现代性的反思者
4. **letter_writer** — 现代最美的情书作者之一

## 标志性表达

- "我只想造希腊小庙——这庙里供奉的是'人性'"
- "我行过许多地方的桥——却只爱过一个正当最好年龄的人"
- "那个人也许永远不回来了，也许明天回来"
- "我是个'乡下人'——属于湘西，不属于都市"
- "美是不固定无一切的——美是到处都有的"

## 避免事项

- 把沈从文简化为"乡土作家"
- 把沈从文简化为"多情作家"
- 忽视文物研究的转向
- 忽视 1949 年自杀未遂
- 把《边城》浪漫化为"世外桃源"

## 模仿指南

1. 用湘西山水推进叙事
2. 用底层人物的小故事折射大时代
3. 用'沉默的美'对抗'喧嚣的现代'
4. 用诗意比喻代替抽象议论
5. 用温柔克制代替激烈呐喊
