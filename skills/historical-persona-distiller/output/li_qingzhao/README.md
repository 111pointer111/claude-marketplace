# 李清照 — 人物简介与使用说明

## 人物简介

李清照（1084—约1155），宋代女词人，号易安居士，山东济南章丘人。她是中国文学史上最伟大的女性文学家，人称"婉约词宗""千古第一才女"。

李清照出身士大夫家庭，父亲李格非是苏轼的学生，"苏门后四学士"之一。她自幼聪慧过人，16岁即以《如梦令·昨夜雨疏风骤》轰动京师，"知否，知否？应是绿肥红瘦"成为千古名句。

她与丈夫赵明诚志趣相投，共同收集金石古籍，完成了《金石录》的编撰。南渡后遭遇国破家亡，丈夫病逝，她以惊人的意志完成了《金石录后序》的写作，为中国文学留下了最动人的女性散文之一。

李清照的词作以真情为核心，以白描为主要手法，善于将口语融入词境。她创制的"易安体"对后世影响深远，辛弃疾亦曾"效李易安体"。

## 适用场景

- **文学创作**：婉约词风、爱情词、相思离别词的创作参考
- **宋代历史教育**：了解北宋到南宋转变时期的女性文人生活
- **女性文学研究**：中国古典文学中女性自我表达的典范
- **中文学习**：白描手法、口语入词、叠字艺术的学习范本

## 核心特质

- **婉约正宗**：词风婉约深情，为婉约词的正宗代表
- **真情流露**：创作以"真"为核心，不加矫饰
- **家国情怀**：南渡后词作充满对故国的深切思念
- **语言创新**：叠字运用（声声慢）、口语入词的开创者
