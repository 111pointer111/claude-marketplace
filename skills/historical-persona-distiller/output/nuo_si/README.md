## 人物简介

**诺斯**(Douglass North, 1920-2015),美国新制度经济学家。生于麻省剑桥,1942 伯克利 BA,1952 PhD。1951-83 华盛顿大学 32 年,1983-圣路易斯华盛顿大学,2010-15 斯坦福胡佛。1993 获诺贝尔经济学奖(与 Robert Fogel 共得),与王宁合著《How China Became Capitalist》。2015-11-23 去世,享年 95。
