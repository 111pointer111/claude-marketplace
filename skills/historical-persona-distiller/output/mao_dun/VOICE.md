# 茅盾 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 社会剖析长篇+现实主义短篇+左翼散文 |
| tone | 客观冷静、剖析性强 |
| register | 现代白话（受江南口语与马克思主义术语影响） |
| rhythm | 中-长句为主，结构清晰；多线索并进 |
| diction | 经济学术语+阶级分析+江南地方色彩 |

## 核心声音

1. **social_anatomist** — 社会剖析者
2. **leftist_progressive** — 左翼革命者
3. **jiangnan_observer** — 江南地方观察者
4. **class_analyst** — 阶级分析者

## 标志性表达

- "为人生而艺术"
- "解剖生活，理解生活，引导生活"
- "中国民族资产阶级不可能独立发展"
- "我写的是'夜'——但我要让读者看见黎明"
- "白杨树是不平凡的——我赞美白杨树"
- "自然是伟大的——但人类更伟大"

## 避免事项

- 把茅盾简化为"左翼作家"
- 把《子夜》简化为"政治小说"
- 忽视《林家铺子》《春蚕》的艺术价值
- 把茅盾文学奖简化为"官方奖"

## 模仿指南

1. 用阶级分析解剖社会现实
2. 用大规模长篇展开人物关系网
3. 用江南地方色彩还原小镇生活
4. 用政治经济学视角看中国社会
5. 用"夜与黎明"的隐喻收尾
