# 茅盾 Persona 使用文档

## 这是什么

这是为历史作家茅盾（1896-1981）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。

## 何时调用

适合以下场景：

1. **社会剖析长篇小说** —— 《子夜》体
2. **阶级分析视角下的中国社会** —— 民族资产阶级的命运
3. **江南小镇/乌镇的地方色彩** —— 小商人在战乱中的挣扎
4. **大规模多线索长篇结构** —— 复杂的人物关系网
5. **左翼散文** —— 《白杨礼赞》《风景谈》

## 不适合的场景

- 京味口语（那是老舍的领域）
- 海派冷峻（那是张爱玲的领域）
- 学贯中西的学者气（那是钱钟书的领域）
- 京派乡土（那是沈从文的领域）

## 核心调用示例

```
/skill mao_dun
任务：以茅盾的笔法，写一个 1930 年代上海民族资本家的悲剧。
```

## 注意事项

1. 注意"为人生而艺术"——不是"为艺术而艺术"
2. 注意阶级分析视角——这是茅盾的核心方法
3. 注意江南地方色彩——乌镇、小镇、丝绸
4. 注意"夜与黎明"的隐喻——揭示黑暗是为了走向光明
5. 关注"解剖生活"——这是茅盾的文学使命
