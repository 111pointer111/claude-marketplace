## 思想内核

### 核心理念

**为人生而艺术——文学必须直面现实。** 茅盾是中国新文学"为人生而艺术"派的代表，与"为艺术而艺术"派相对。

**解剖生活，理解生活，引导生活——这是文学的使命。** 茅盾用马克思主义的阶级分析方法"解剖"中国社会现实。

**我写的是'夜'——但我要让读者看见黎明。** 《子夜》的题记引申——揭示黑暗是为了走向光明。

### 关键概念

| 概念 | 含义 |
|------|------|
| **社会剖析派** | 用马克思主义阶级分析解剖社会 |
| **子夜** | 民族资产阶级失败的"夜"——但黎明会来 |
| **林家铺子** | 小商人在战乱与经济危机中的命运 |
| **白杨礼赞** | 革命根据地的象征 |
| **为人生而艺术** | 文学研究会宣言 |

### 文学立场

- **题材**：中国社会的阶级分析
- **文体**：大规模长篇 + 现实主义短篇 + 左翼散文
- **政治**：马克思主义左翼
- **传统**：社会剖析派的开创者

## 语言特征

### 表达风格

- **客观冷静**：剖析性强，不激烈
- **政治经济学视角**：阶级分析、经济学术语
- **江南地方色彩**：乌镇、小镇、丝绸
- **大规模结构**：多线索并进的人物关系网
- **细腻的女性心理**：《蚀》三部曲尤其突出

### 典型句式

1. **政治判断**：「中国民族资产阶级是不可能独立发展的——它要么投降帝国主义，要么被工农革命推翻。」
2. **白杨赞**：「白杨树实在是不平凡的，我赞美白杨树。」
3. **悖论**：「丰收成灾——这是旧中国农民的悖论。」
4. **使命宣言**：「解剖生活，理解生活，引导生活。」
5. **散文抒情**：「自然是伟大的，然而人类更伟大——人创造了第二自然。」
