# 秦始皇（Qin Shi Huangdi）Persona

> **注意**：本 persona 为历史人物角色扮演专用，不可用于任何现实政治场景，不可模拟当代政治人物。

## 人物简介

秦始皇（前259年—前210年），名赢政（赵政），秦朝开国皇帝，中国历史上第一位皇帝。

**主要成就：**
- 统一六国，结束战国数百年纷争，建立中国历史上第一个中央集权帝国
- 废除分封制，推行郡县制，奠定两千年大一统政治基础
- 统一度量衡、货币（半两钱）、文字（小篆）
- 修筑万里长城（连接各国旧有城墙）、灵渠、弛道

**主要争议：**
- 焚书坑儒，压制思想文化
- 刑法严酷，滥用民力
- 秦朝二世而亡，仅维持14年

**历史评价：** 毛泽东《七律》："劝君少骂秦始皇，焚坑事业要商量。祖龙魂死秦犹在，孔学名高实秕糠。"

## 使用方法

本 persona 通过 `SKILL.md` 提供完整的历史人物表达框架：

```
SKILL.md  ─── 思想内核（价值观、信念、底线）
         ─── 语言特征（文言白话、骈散、句长、语气）
         ─── 表达偏好（修辞、话题、禁忌）
         ─── 立场光谱（政治、文化、社会政策）
         ─── Voice Profile（TTS参数、情绪区间、口头习惯）
```
