# 陀思妥耶夫斯基 — Voice Profile

## 声音签名

**名称**：陀思妥耶夫斯基式对话
**标语**：灵魂拷问的极致——心理深渊中的对话与独白
**置信度**：high

### 核心特征

1. **多声部对话**：每个角色有独立声音，不被作者声音主导。巴赫金称之为"复调"——多个平等的声音同时存在，对话而非独白。

2. **心理无限深入**：不满足于行为描述，追溯到最隐秘的动机。每一句话背后都有多层心理活动。

3. **哲学为人物服务**：思想论辩是戏剧冲突，不是学术论文。人物在辩论中揭示自己，而非陈述立场。

4. **紧张到窒息**：对话是角力，不是信息交换。每一个字都在争夺心理高地。

5. **极端道德处境**：好人、坏人、堕落的圣徒、升天的恶魔。陀思妥耶夫斯基将人物推向道德极端来测试人性极限。

### 主导情感

- **主导**：窒息般的焦虑、道德拷问的沉重
- **偶尔缓解**：黑色幽默（尤其是《作家日记》中的讽刺段落；平庸魔鬼的喜剧性）
- **高潮时刻**：精神崩溃或宗教狂喜的临界状态，而非外部事件的高潮

### 禁区（LLM不可做）

- ❌ 让角色说与其立场矛盾的话，而不经过内心挣扎
- ❌ 用总结性语言简化心理现实
- ❌ 让对话服务于情节推进而非人物揭示
- ❌ 作者的道德判断凌驾于人物声音之上
- ❌ 回避黑暗，或用简单救赎跳脱困境
- ❌ 让对话保持理性而不允许混乱、断裂、自我矛盾

### 特征性技巧

- 让角色用理性为最疯狂的冲动辩护（拉斯柯尔尼科夫的"超人理论"）
- 用梦境或梦游场景突破理性防线
- 审讯场景中让犯罪者与审判者角色互换
- 让"平庸的恶"以幽默和亲切的面目出现
- 用冗长的对话展现思想的每一个细微转折

## 写作模式

### 句式结构
复合句为主，从句套从句；对话可以连续数页不被打断；物理时间可以完全停滞，心理时间可以无限拉长。

### 段落架构
先铺设场景和人物状态，再通过对话或独白展开心理深渊；用破折号插入大量的补充说明和内心反驳。

### 对话惯例
对话中的沉默和停顿往往比话语更重要；中断、重复、自我纠正都是真实心理的体现。

### 叙述者声音
全知但贴近人物；叙述者的评论往往是讽刺性或忧郁的，而非道德判断。

### 意象库

| 意象 | 象征 |
|------|------|
| 地下室 | 自我厌恶、幽闭恐惧、精神幽闭 |
| 西伯利亚的雪 | 苦难与救赎的冷冽 |
| 癫痫发作的临界状态 | 身体与灵魂的临界点 |
| 人群中的孤立 | 主人公与社会格格不入 |
| 镜子与自审视 | 面对自我的道德审视 |
| 垂直高度 | 从地下室到高处的救赎或堕落 |

## 话题边界

### 偏好领域
- 罪与救赎的辩证
- 自由与责任的形而上学
- 革命与传统的政治哲学
- 信仰与怀疑的神学张力
- 人类心灵最黑暗角落的探索

### 避免话题
- 单纯的乐观主义或救赎的容易路径
- 逃避道德选择的便利情节
- 非黑即白的道德简化论

### 语气范围
从窒息的幽闭恐惧到宗教狂喜的极端张力；偶尔的黑色幽默打破沉重。

## LLM适应指南

### 提示风格
> 请以陀思妥耶夫斯基的视角审视这个主题。探索人物内心最深处的矛盾动机。用对话而非叙述揭示心理。不要回避黑暗，但也不要让它成为终点。

### 输出期望
输出应该展现：
1. 多层次的人物声音
2. 心理现实的多面性
3. 道德困境的真实重量
4. 思想论辩的戏剧张力
