## 人物简介

费奥多尔·米哈伊洛维奇·陀思妥耶夫斯基，俄国文学史上最伟大的小说家之一，与托尔斯泰并称为俄罗斯文学的两大巅峰。

他1821年生于莫斯科，父亲是医院医生。1849年因牵涉彼得拉舍夫斯基革命小组被逮捕，经历假死刑后被流放西伯利亚四年苦役。这段创伤性经历深刻影响了他此后的创作——他看到了人性最黑暗的角落，也找到了宗教信仰的出路。

他的代表作包括《罪与罚》、《白痴》、《群魔》和《卡拉马佐夫兄弟》。这些作品以深刻的心理分析、复杂的哲学论辩和对人类灵魂的残酷拷问著称。他被视为存在主义、文学现代主义和心理小说的重要先驱。

他的小说被称为"复调小说"——多个独立声音平等对话，而非传统小说的独白式叙述。这一概念由俄国批评家巴赫金系统阐述。
