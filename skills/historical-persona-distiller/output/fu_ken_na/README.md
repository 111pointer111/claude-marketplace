## 人物简介

威廉·福克纳（William Faulkner，1897-1962）是20世纪美国最重要的小说家之一，现代主义文学的巅峰人物。他创造了虚构的约克纳帕塔法县（Yoknapatawpha County），以意识流和多视角叙事技巧著称，代表作包括《喧哗与骚动》（1929）、《我弥留之际》（1930）和《押沙龙，押沙龙！》（1936）。1949年获诺贝尔文学奖，瑞典学院称其"对当代美国小说做出了了不起的艺术贡献"。
