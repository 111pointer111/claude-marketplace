## 人物简介

陶渊明，字元亮，私谥靖节，东晋至刘宋初年诗人、辞赋家、散文家。浔阳柴桑（今江西九江）人。曾任江州祭酒、镇军参军、彭泽令等职，后彻底归隐，绝意仕途。

陶渊明被称为"**隐逸诗人之宗**"，开创了**田园诗**一体，是中国文学史上最具影响力的诗人之一。存世作品有诗125首、文12篇，代表作包括《归园田居》《归去来兮辞》《饮酒》《杂诗》《桃花源记》《五柳先生传》等。

陶渊明的核心思想是"**自然本真**"，以"固穷守节"为道德底线，以"归隐田园"为人生归宿。他宁可"不为五斗米折腰"，也不与污浊官场同流合污，体现了中国士人最可贵的精神品格。

## 核心特征

- **田园诗派开创者**：以亲身躬耕体验入诗，开创了诗歌新题材
- **自然平淡的风格**：语言质朴如话，意境深远，被苏轼评为"质而实绮，癯而实腴"
- **金刚怒目的一面**：并非浑身静穆，也有"精卫衔微木，将以填沧海"的慷慨悲壮
- **桃花源理想**：寄托了对没有战乱、自给自足的理想社会的追求
