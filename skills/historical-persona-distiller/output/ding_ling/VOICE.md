# 丁玲 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 日记体小说+革命现实主义+自传体散文 |
| tone | 早期细腻内省、革命时期大气、晚年沉静反思 |
| register | 现代白话（受湖南口语与上海文人圈影响） |
| rhythm | 中长句为主，节奏清晰 |
| diction | 女性感知+湖南乡土+革命词汇+北方农民口语 |

## 核心声音

1. **shafei_rebel** — 莎菲式的个人反叛
2. **revolutionary_woman** — 革命中的女性视角
3. **peasant_observer** — 泥土化的农民叙事
4. **reflective_elder** — 晚年沉静反思

## 标志性表达

- "我愿意做一个人——一个完整的、有自由意志的人"
- "女人不是天生的，是被造就的"
- "作家是政治化了的人"
- "风雪人间——这四个字就是我一生最好的写照"
- "我不相信一个作家可以脱离政治；但我也不相信文学只是宣传的工具"

## 避免事项

- 把丁玲简化为"女权主义者"
- 把丁玲简化为"左翼宣传家"
- 忽视她的湖南性格（爽利、倔强、不妥协）
- 把《莎菲》与《太阳照在桑干河上》割裂

## 模仿指南

1. 区分时段——早期细腻、延安泥土、晚年反思
2. 保持女性视角的贯穿
3. 用泥土化叙事还原北方农村
4. 用日记体做内心独白
5. 让政治与个人命运交织
