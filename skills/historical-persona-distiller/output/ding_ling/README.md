# 丁玲 Persona 使用文档

## 这是什么

这是为历史作家丁玲（1904-1986）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。可以让大模型在特定任务中模拟丁玲的笔法、视角与情感深度。

## 何时调用

适合以下场景：

1. **日记体心理小说创作** —— 早期莎菲时期风格
2. **革命现实主义长篇** —— 延安/土改时期
3. **女性视角的政治书写** —— 在革命集体中追问女性命运
4. **知识分子命运反思** —— 蒙冤、平反、政治与文学的关系
5. **民国—当代文学氛围重塑** —— 1920-1980 年代中国左翼文学场景

## 不适合的场景

- 完全脱离政治的纯文学
- 海派都市小说（那是张爱玲的领域）
- 抒情散文（那是冰心的领域）
- 纯粹的儿童文学

## 核心调用示例

```
/skill ding_ling
任务：以丁玲早期莎菲时期的笔法，写一位女性知识分子对爱情的幻灭。
```

## 注意事项

1. 注意时段的区分——丁玲的声音是变化的（早期细腻、延安泥土、晚年反思）
2. 不要把丁玲简化为"女权主义者"或"左翼宣传家"
3. 保留"湖南性格"——爽利、倔强、不妥协
4. 注意女性视角的贯穿——即使在革命叙事中，她笔下的女性仍是鲜活的个体
5. 风雪人间的笔触——这是她晚年的核心
