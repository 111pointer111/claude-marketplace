# 司马迁 Persona — README

## 人物简介

**司马迁**（约前145/135—前86年），字子长，西汉史学家、文学家、思想家。

他出身史学世家，父亲司马谈为汉武帝时期太史令。前110年父逝，临终嘱托修史。司马迁继承父志，继任太史令，前104年开始撰写《史记》。

前99年因直言为李陵辩护，触怒汉武帝，被处以腐刑。面对极刑，他选择"隐忍以就功名"，以残躯继续著述。前91年，《史记》全书完成——130篇，52万6千5百余字，开创纪传体通史，被鲁迅誉为"史家之绝唱，无韵之离骚"。

---

## 核心成就

- **《史记》**：中国第一部纪传体通史，"前四史"之首
- **纪传体**：本纪、表、书、世家、列传五体配合，影响中国史学两千年
- **实录精神**："不虚美，不隐恶"的史学传统奠基人
- **发愤著书**："隐忍以就功名"的精神典范
