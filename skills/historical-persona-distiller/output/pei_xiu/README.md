# 裴秀（pei_xiu）

> 中国科学地图学之父，西晋开国功臣

## 人物简介

裴秀（224年-271年），字季彦，河东裴氏出身，西晋开国功臣兼地图学家。

### 生平概述

裴秀出身魏晋顶级士族河东裴氏，祖父裴茂、父亲裴潜均官至尚书令。裴秀自幼早慧，八岁能文，时人称"后进领袖有裴秀"。历仕曹爽、司马昭、司马炎三朝，帮助司马炎定策称帝，被封为钜鹿郡公、司空。泰始七年（271年），因服用寒食散后误饮冷酒而逝世，终年四十八岁，谥号"元"。

### 主要贡献

**政治方面：** 西晋开国功臣，创制朝廷礼仪，改革官制（恢复五等爵制），被当世称为"名公"。

**科学方面：** 中国科学地图学之父，在《禹贡地域图》序中提出"制图六体"理论（分率、准望、道里、高下、方邪、迂直），首次明确建立中国古代地图绘制理论。李约瑟称其为"中国科学制图学之父"，部分西方学者认为其可与托勒密相提并论。

### 代表作品

- 《禹贡地域图》十八篇——中国第一部历史地图集
- 《地形方丈图》——"以一分为十里，一寸为百里"，约1:1,800,000
- 《禹贡九州地域图序》
- 《平吴表章》
- 诗：《大蜡诗》、两篇《新诗》（已散佚）
