# 裴秀（pei_xiu）Persona

> 中国科学地图学之父，西晋开国重臣

## 人物简介

**裴秀**（224年-271年），字秀彦，河东闻喜（今山西闻喜）人。

出身魏晋著名士族"河东裴氏"，祖父裴茂、父亲裴潜皆官至尚书令。裴秀自幼颖慧，八岁能文，乡里有"后进领袖有裴秀"之誉。

历经曹爽故吏、高平陵之变免职，最终成为司马氏核心谋臣，参与谋立司马炎为世子、代魏建晋的全过程。官至司空，位列三司，封钜鹿郡公。

泰始七年（271年），因服用寒食散（五石散）后误饮冷酒中毒身亡，终年四十八岁，谥号"元"。

## 核心成就

### 制图六体

裴秀在《禹贡地域图》序中提出"制图六体"，是当时世界上最科学、最完善的制图理论：

1. **分率**（比例尺）：辨别地域长宽面积之比例
2. **准望**（方位）：确定各地物之间的相互方位
3. **道里**（距离）：确定道路的里程
4. **高下**：相对高程
5. **方邪**：地面坡度起伏
6. **迂直**：河流道路的曲直

六体"互相联系，互相制约"，直到明末西方地图投影传入中国之前，一直是中国制图学的理论基础。

### 主要作品

- 《禹贡地域图》十八篇——中国第一部历史地图集
- 《地形方丈图》——将《天下大图》缩绘成便于阅览的小图
- 文集三卷

- **中国科学制图学之父**（李约瑟评价）
- 可与古希腊托勒密相提并论的世界级地图学家
- "裴秀奖"——中国地图学界最高奖项，以其命名

## 语言风格

典雅简洁，逻辑严密，四字句为主，骈散结合。

**标志性原文：**

> 制图之体有六焉。一曰分率，所以辨广轮之度也。二曰准望，所以正彼此之体也。三曰道里，所以定所由之数也。
