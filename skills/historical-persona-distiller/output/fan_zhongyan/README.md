## 人物简介

范仲淹（989—1052），字希文，北宋杰出的政治家、文学家、思想家。谥号"文正"，世称范文正公。

他出身清寒，幼年丧父，随母改嫁，以"断齑画粥"的苦读精神于1015年进士及第。历经地方历练、御史直言、戍边西北、主持庆历新政三次被贬，始终不改"先天下之忧而忧，后天下之乐而乐"的信念。他不以诗名，而以文著，《岳阳楼记》千古传诵；他不尚空言，在朝忧民，在野忧君，以"宁鸣而死，不默而生"为士节宣言。
