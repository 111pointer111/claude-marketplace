# EVENTS.md — 涂尔干生平大事年表

## 基本信息

- **姓名**: Émile Durkheim (大卫·埃米尔·涂尔干)
- **生卒**: 1858年4月15日 — 1917年11月15日
- **国籍**: 法国
- **领域**: 社会学、哲学、教育学、人类学、宗教研究
- **地位**: 现代社会学的奠基人之一，与马克思、韦伯并称社会学三大家

---

## 年表

### 形成期 (1858–1887)

| 年份 | 事件 |
|------|------|
| 1858 | 出生于法国埃皮纳勒（Épinal），虔诚的犹太教拉比世家 |
| 童年 | 早年丧父 |
| 1870s | 进入巴黎高等师范学校（École Normale Supérieure），与亨利·柏格森、让·饶勒斯同届 |
| 求学期间 | 师从历史学家福斯蒂尔·德·库朗日，撰写关于孟德斯鸠的拉丁论文；研读孔德和赫伯特·斯宾塞 |
| 1882 | 通过哲学教师资格会考（因前一年重病，只考了倒数第二名） |
| 1882–1887 | 在多所省级中学教授哲学 |
| 1885–86 | 赴德国留学，在马尔堡、柏林、莱比锡大学研究社会学；特别欣赏威廉·冯特（Wilhelm Wundt）的实验心理学 |
| 1887 | 与路易丝·德雷福斯（Louise Dreyfus）结婚 |

### 奠基期 (1887–1902)

| 年份 | 事件 |
|------|------|
| 1887 | 受任波尔多大学社会学与教育学讲师，成为法国第一个正式教授社会学的人 |
| 1893 | 出版博士论文《社会分工论》（*De la division du travail social*）——机械团结与有机团结理论 |
| 1895 | 出版《社会学方法论》（*Les Règles de la méthode sociologique*）——社会学研究的方法论宣言 |
| 1895 | 在波尔多大学建立欧洲第一个社会学系 |
| 1896 | 创办《社会学年鉴》（*L'Année sociologique*） |
| 1897 | 出版《自杀论》（*Le Suicide*）——用定量方法研究社会自杀率的开创性著作 |

### 鼎盛期 (1902–1914)

| 年份 | 事件 |
|------|------|
| 1902 | 任索邦大学教育学教授 |
| 1906 | 成为索邦大学正教授 |
| 1912 | 出版《宗教生活的基本形式》（*Les Formes élémentaires de la vie religieuse*）——关于澳大利亚原住民宗教的社会学理论；提出"集体欢腾"概念 |
| 1913 | 正式获封教育学与社会学教授席位 |

### 哀痛期 (1914–1917)

| 年份 | 事件 |
|------|------|
| 1914–1917 | 第一次世界大战期间，学生纷纷奔赴战场，许多阵亡 |
| 1915 | 独子安德烈（André）在战争中战死——这是涂尔干一生最致命的打击 |
| 1917 | 11月15日因中风在巴黎去世，享年59岁；留下未完成的遗作《道德》（*La Morale*） |

## 核心概念

- **社会事实**（Social Fact）: 存在于个人身外并具有强制性的行为方式、思维和感觉
- **集体意识**（Collective Consciousness）: 社会作为一个整体的共同信仰和情感
- **失范**（Anomie）: 社会规范对个人欲望失去约束力的状态
- **机械团结**（Mechanical Solidarity）: 基于相似性的传统社会团结机制
- **有机团结**（Organic Solidarity）: 基于劳动分工和相互依赖的现代社会团结机制
- **集体欢腾**（Collective Effervescence）: 仪式过程中个体被集体情感裹挟而超越自我的状态

---

## 历史影响

涂尔干被视为现代社会学的奠基人之一。他确立了社会学的学科地位，建立了实证主义方法论，并留下了丰富的概念工具（失范、集体意识、社会团结等），至今仍是社会科学的核心资源。他与马克思、韦伯并称社会学三大家。
