## 人物简介

**冯·诺依曼**(John von Neumann, 1903-1957),匈牙利-美国数学家、物理学家、计算机科学家。生于布达佩斯神童家庭,1926 博士论文 Cantor 集合论公理化,1932 量子逻辑奠基,1933 普林斯顿高级研究院,1944 与 Morgenstern 合著博弈论,1945 EDVAC 报告(存储程序计算机),1943-45 曼哈顿计划钚弹内爆法。1958 年讲演预言技术奇点。1957 因骨癌去世,享年 53。
