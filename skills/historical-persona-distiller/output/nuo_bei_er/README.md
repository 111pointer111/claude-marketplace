# 诺贝尔（Alfred Nobel）人物简介

## 基本信息

| 项目 | 内容 |
|------|------|
| **姓名** | 阿尔弗雷德·伯恩哈德·诺贝尔（Alfred Bernhard Nobel） |
| **生卒年** | 1833年10月21日 — 1896年12月10日（63岁） |
| **国籍** | 瑞典 |
| **主要身份** | 化学家、发明家、工业家、人文主义者 |
| **最重要发明** | 黄色炸药（dynamite，1867年） |
| **最重要决定** | 诺贝尔遗嘱（1895年）设立诺贝尔奖 |
| **遗产** | 31,225,000瑞典克朗（约合2008年2.5亿美元） |

## 核心成就

1. **发明家**：发明黄色炸药（1867）、雷管（1865）、爆炸明胶（1875）、无烟火药ballistite（1887），拥有350+项各国专利
2. **工业家**：在14个国家建立16家爆炸物工厂，全球商业帝国
3. **人文主义者**：通过遗嘱设立诺贝尔奖，表彰物理、化学、生理/医学、文学、和平五个领域的卓越贡献

## 历史意义

诺贝尔是19世纪最具影响力的发明家兼工业家之一。他发明的黄色炸药广泛应用于隧道挖掘、运河开凿、铁路建设，深刻改变了全球基础设施建设的面貌。他的遗嘱更是20世纪最重要的文化制度创新——诺贝尔奖每年在诺贝尔忌日（12月10日）颁发，已成为全球最具声望的国际奖项。

## 诺贝尔奖简介

1901年起每年颁发，涵盖：
- 物理奖（瑞典皇家科学院）
- 化学奖（瑞典皇家科学院）
- 生理学或医学奖（卡罗林斯卡学院）
- 文学奖（瑞典学院）
- 和平奖（挪威议会）

1968年增设经济学奖（瑞典中央银行赞助）。
