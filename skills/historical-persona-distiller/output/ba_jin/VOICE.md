# 巴金 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 激流家族长篇+忏悔体散文+翻译 |
| tone | 早期激情→中期克制→晚年忏悔 |
| register | 现代白话（受五四白话文与无政府主义影响） |
| rhythm | 早期长句激情；中期中句克制；晚年短句忏悔 |
| diction | 家族词汇+革命词汇+忏悔体 |

## 核心声音

1. **passionate_youth** — 激情的青年反叛者
2. **family_critic** — 家族批判者
3. **suffering_observer** — 底层受难者的深切关注
4. **penitent_elder** — 晚年的忏悔者

## 标志性表达

- "生命的意义在于付出"
- "家是过去中国社会的缩影"
- "爱那些受苦的人，比什么都重要"
- "忏悔不是软弱——是良心的最后一道防线"
- "要是有一种'文革'博物馆该多好"

## 避免事项

- 把巴金简化为"反封建作家"
- 把巴金简化为"无政府主义者"
- 忽视《寒夜》对巴金的意义
- 忽视《随想录》的忏悔深度
- 忽视文革中萧珊之死

## 模仿指南

1. 区分三个时期——激情/克制/忏悔
2. 保留"人道主义关怀"的贯穿性
3. 用激情呼告推进早期叙事
4. 用底层受难者小悲剧折射大时代
5. 用忏悔体的'我承认...我反思...'句式
