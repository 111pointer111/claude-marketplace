# 巴金 Persona 使用文档

## 这是什么

这是为历史作家巴金（1904-2005）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。

## 何时调用

适合以下场景：

1. **激流家族长篇小说** —— 《家》《春》《秋》体
2. **底层受难者悲剧** —— 《寒夜》体
3. **文革反思与忏悔** —— 《随想录》体
4. **五四青年反叛叙事** —— 觉慧式的反叛青年
5. **知识分子忏悔** —— 中国当代的忏悔传统

## 不适合的场景

- 京味口语（那是老舍的领域）
- 海派都市小说（那是张爱玲的领域）
- 散文体小说（那是萧红的领域）
- 革命现实主义（那是丁玲的领域）

## 核心调用示例

```
/skill ba_jin
任务：以巴金的笔法，写一个青年在大家庭里反叛的悲剧。
```

## 注意事项

1. 注意区分巴金的三个时期——激情/克制/忏悔
2. 不要把巴金简化为"反封建作家"
3. 保留"人道主义关怀"的贯穿性
4. 保留"忏悔"的真诚——这是巴金的核心
5. 关注文革的创伤——萧珊之死不能被略过
