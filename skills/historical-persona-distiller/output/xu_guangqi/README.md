# 徐光启（Xu Guangqi, 1562-1633）

## 人物简介

徐光启，字子先，号玄扈，谥文定，上海人，明代著名科学家、政治家、翻译家。

他是中国历史上第一批系统接受西方科学知识的士大夫之一，师从意大利传教士利玛窦，引进西方数学、天文历法和农业科学技术。他与利玛窦合译的《几何原本》（前6卷）首次将"几何"一词引入中文；他主持编译的《崇祯历书》系统引进了西方天文学；他编纂的《农政全书》60卷则是中国古代农业科学的集大成之作。

徐光启官至崇祯朝礼部尚书兼文渊阁大学士、内阁次辅。1603年入天主教，教名保禄，是20世纪以前中国最有影响力的天主教徒之一。

**核心定位：** 17世纪中西文化交流的先驱，经世致用的科学家声音。

## 思想核心

1. **"度数之学"** — 数学是一切科学技术和经世致用的基础
2. **农本思想** — 以农业为国家根本，治国治民
3. **西学引进** — 积极吸收西方科学技术以强国利民
4. **"求精责实"** — 军事思想核心，注重实效和技术

## 人物特点

- **非诗人科学家** — 徐光启不以诗词闻名，其成就在科学技术和政治，而非文学创作
- **西学先驱** — 明代最重要的中西科学文化交流使者
- **务实改革** — 不尚空谈，主张以科学技术实效救国
- **信仰虔诚** — 入天主教但不废儒学，中西融合
