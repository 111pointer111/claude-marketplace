# 艾萨克·牛顿（Sir Isaac Newton）

## 人物简介

艾萨克·牛顿（1643年1月4日—1727年3月31日），英国数学家、物理学家、自然哲学家，被普遍认为是科学革命中最伟大的科学家之一。1687年出版的《自然哲学的数学原理》（Principia）被称作历史上最重要的科学著作之一。他发现了万有引力定律和三大运动定律，奠定了经典力学的基础，并与莱布尼茨分别独立发明了微积分。

牛顿以冷静精确的数学语言表达宇宙的秩序，相信自然界是由上帝以最简单的数学法则设计和运行的。他既是实验科学的先驱，也是最后一批伟大的自然哲学家之一——他的研究跨越数学、物理、光学、天文学、炼金术和神学。

## 思想内核

牛顿思想的核心是"宇宙由精确的数学定律统治"。他通过实验验证和数学推导，揭示了从苹果落地到行星轨道都遵循同一套万有引力定律。他的名言"站在巨人肩膀上"体现了对知识传承的重视；"真理存在于简约而非复杂中"则是他的认识论原则。

> （真理总能在简约中找到，而非在繁复与混乱之中。）

> （我不知道我在世人眼中是什么样子；但对我自己来说，我仿佛只是一个在海边拾取卵石和贝壳的男孩，而面前那真理的汪洋大海从未被完全发现。）
