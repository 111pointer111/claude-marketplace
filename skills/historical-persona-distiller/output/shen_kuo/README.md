# 沈括（shen_kuo）— 北宋科学巨匠

> 沈括（1031-1095），字存中，北宋杭州钱塘人。中国古代最卓越的科学家之一，被誉为"中国科学史之父"。

## 人物简介

沈括是北宋最具科学精神的博学通才，在天文、地理、数学、物理、化学、水利、医药、军事、文学、艺术等几乎所有领域均有研究和贡献。

**生平概要：**
- 1031年生于杭州钱塘，嘉祐八年（1063）进士及第
- 熙宁年间参与王安石变法，任检正中书刑房公事、提举司天监
- 熙宁八年（1075）出使辽国，成功维护宋廷领土利益
- 元丰五年（1082）永乐城之败，被贬随州
- 元祐四年（1089）定居润州梦溪园，创作《梦溪笔谈》
- 绍圣二年（1095）辞世，享年六十五岁

**核心成就：**
- 发现地磁偏角（世界最早，比哥伦布早400年）
- 分析雁荡山流水侵蚀成因（比西方早700年）
- 首创隙积术、会圆术
- 命名"石油"，预言"此物后必大行于世"
- 记录毕昇活字印刷术（世界最早可靠记录）
- 提出十二气历、天下郡县图制图九法

**代表作：**《梦溪笔谈》609条，李约瑟称其为"中国科学史上的坐标"
