# 蔡伦（cai_lun）

## 人物简介

**蔡伦**（约61年-121年），字敬仲，东汉桂阳郡人。中国历史上最重要的发明家之一，传统上被认定为**造纸术的发明者**。

蔡伦出身工匠之家（约金官或铁官），约永平末年（75年）入宫为宦官。章帝建初年间升任小黄门，章和二年（88年）升迁为中常侍，成为传达诏令、参预政事的高等宦官。后兼任尚方令，永元九年（97年）主持监造皇室器物，所造器械"莫不精工坚密"。永元十四年（102年），蔡伦受皇后邓绥之命改进造纸术，使用树肤、麻头、敝布、鱼网等原料，经十余年反复试验，于**元兴元年（105年）**成功改进造纸术并献于汉和帝，和帝下令全国推广——史称"**蔡侯纸**"。

元初元年（114年）封龙亭侯，三百户；不久任长乐太仆，成为邓太后近侍。元初四年（117年）主持东观校正经史典籍。建光元年（121年）邓太后逝世，安帝亲政，追查建初年间宋贵人一案，蔡伦耻于受辱，沐浴更衣后服毒自尽，封地收回。

蔡伦的造纸术是人类历史上最重要的发明之一，被美国《时代》周刊评为"有史以来最佳发明家"之一，在麦克·哈特《影响人类历史进程的100名人排行榜》中排名第七位。
