## 关键事件详解

### 1. 被俘阉割（约1381年）
云南被明朝军队征服，10岁的马三宝被俘并阉割，送入军中。这一事件彻底改变了他的命运——从一个普通的穆斯林农家子弟，变成了皇帝身边的宦官。

### 2. 靖难之役与获赐姓名（1402-1403年）
燕王朱棣通过三年内战从侄子建文帝手中夺取皇位。作为燕王亲信，马三宝在政变中立下功劳，被赐姓"郑"，从此成为"郑和"。这一赐姓标志着他在新皇帝身边获得的最高宠信。

### 3. 七下西洋（1405-1433年）
郑和七次率大规模船队下西洋，是人类历史上规模最大、时间跨度最长的航海探险之一。船队最多时有317艘船、62艘当时世界上最大的"宝船"、约28,000名船员。其足迹遍布东南亚、南亚、波斯湾、阿拉伯半岛和东非海岸。

### 4. 俘虏锡兰王（约1409年）
第二次航行期间，锡兰国王阿拉加纳卡拉试图抢劫郑和船队。郑和果断以武力反击，俘虏锡兰王并带回南京。这一事件展示了明朝的军事实力，维护了国家尊严。

### 5. 石碑铭文（1432年）
郑和在福建竖立石碑，留下他唯一存世的自我表达文字。碑文以壮阔的笔调描述了他的航海经历："观夫海洋，洪涛接天，巨浪如山……"这是研究郑和本人思想和语言风格的最重要一手史料。

### 6. 病逝古里（1433年）
郑和在第七次航行途中病逝于印度古里，遗体被送回中国安葬。作为一名宦官，他没有子嗣，一生奉献给了皇帝的使命，最终在最后一次航行的途中走完了人生旅程。

## 地理轨迹

```
云南昆阳（出生）→ 北方军营（被俘入军）
→ 北平/燕王府（崛起）→ 南京（靖难之役）
→ 北京（永乐朝廷）→ 七下西洋起点（江苏太仓）
→ 东南亚（第一次至第六次中间停靠）
→ 南亚印度（卡利卡特/科钦）
→ 波斯湾霍尔木兹
→ 阿拉伯半岛（亚丁/吉达/麦加）
→ 东非海岸（摩加迪沙/马林迪/蒙巴萨/桑给巴尔）
→ 南京（守备任上）
→ 福建（石碑立文）
→ 第七次航行 → 印度古里（病逝）
→ 南京（安葬）
```
