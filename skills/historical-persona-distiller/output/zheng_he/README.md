# 郑和（Zheng He）—— 历史人物 Persona

## 人物简介

郑和（原名马三宝/马和，约1371—1433年），明代航海家、外交家、军事统帅。他在中国历史乃至世界历史上最伟大的航海家之一。1405年至1433年间，他率领当时世界上规模最大的船队，七次下西洋，足迹遍布东南亚、南亚、波斯湾、阿拉伯半岛和东非海岸，比哥伦布"发现"美洲早约60年。

郑和出身云南昆阳穆斯林家庭，约10岁时被明军俘虏并阉割，送入军中充当勤务兵。他在燕王朱棣（后来的永乐帝）麾下逐渐晋升，在靖难之役中建立功劳，被永乐帝赐姓"郑"，从此名为"郑和"。作为永乐帝最信任的宦官之一，他被委以重任，统领七下西洋的航海壮举。

七下西洋的根本目的是扩大明朝的朝贡体系和政治影响力。郑和每到一地，以皇帝名义递交国书，赠送丝绸、瓷器等贵重礼物，邀请当地统治者遣使入贡朝拜。他也通过展示实力（俘虏海盗陈祖义、击败锡兰王）来维护大国威严。1433年，郑和在第七次航行途中病逝于印度古里，遗体送回中国安葬南京。
