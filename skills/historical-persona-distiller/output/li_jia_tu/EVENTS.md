# 大卫·李嘉图 生平事件图谱

## 早年（1772-1791）

| 年份 | 事件 | 类型 |
|------|------|------|
| 1772年4月18日 | 大卫·李嘉图生于伦敦 | 出生 |
| 约1784年（12岁） | 被父亲派到荷兰商业学校留学两年 | 教育 |
| 约1786年（14岁） | 随父从事证券交易业务 | 事业 |
| 1793年（21岁） | 与贵格会信徒Priscilla Anne Wilkinson结婚，改信唯一神教派，与父亲决裂 | 个人 |
| 1793年（21岁） | 独立开展证券交易活动 | 事业 |

## 致富期（1794-1814）

| 年份 | 事件 | 类型 |
|------|------|------|
| 约1798年（26岁） | 以800镑为资本独立经营证券业务 | 事业 |
| 1799年（27岁） | 在温泉胜地养病期间偶然阅读亚当·斯密《国富论》，对政治经济学产生兴趣 | 学术 |
| 1809年（37岁） | 发表第一篇经济学文章《黄金的高价是银行纸币贬值的验证》，讨论货币问题 | 学术 |
| 1810年（38岁） | 发表《答博赞克特先生对金价委员会报告的实际观感》 | 学术 |
| 1815年（43岁） | 发表《论低价谷物对资本利润的影响》，反对《谷物法》 | 学术 |

## 学术巅峰期（1815-1823）

| 年份 | 事件 | 类型 |
|------|------|------|
| 1815年（43岁） | 结识詹姆斯·穆勒，在其督促下开始写作《政治经济学及赋税原理》 | 学术 |
| 1817年（45岁） | 《政治经济学及赋税原理》出版，提出比较优势理论 | 学术 |
| 1819年（47岁） | 当选Portarlington上议院议员 | 政治 |
| 1820年 | 与马尔萨斯持续论战，出版《政治经济学原理》后马尔萨斯回应 | 学术 |
| 1821年（49岁） | 投票支持废除伪造罪死刑 | 政治 |
| 1823年（51岁） | 在东印度公司会议上反对奴隶制发言 | 政治 |
| 1823年9月11日 | 因耳部感染引发败血症去世，享年51岁 | 去世 |

## 学术关系

| 人物 | 关系类型 | 说明 |
|------|----------|------|
| 亚当·斯密 | 学术影响 | 《国富论》是李嘉图经济学研究的起点 |
| 詹姆斯·穆勒 | 师友与导师 | 督促李嘉图完成《政治经济学及赋税原理》 |
| 杰里米·边沁 | 哲学影响 | 功利主义哲学是其理论出发点 |
| 托马斯·马尔萨斯 | 终身论敌兼朋友 | 在价值理论等问题上长期争论，但保持深厚友谊 |
| 约翰·穆勒 | 学术后继 | 后来系统阐述李嘉图理论 |

## 理论贡献时间线

```
1772  出生
  ↓
1786  14岁开始证券交易（金融天才期）
  ↓
1793  21岁与父决裂，独立创业
  ↓
1799  27岁读《国富论》（学术觉醒）
  ↓
1809  37岁发表第一篇经济学论文
  ↓
1815  43岁《谷物法论战》+结识穆勒（学术转折点）
  ↓
1817  45岁《政治经济学及赋税原理》出版（学术巅峰）
  ↓
1819  47岁当选上议院议员
  ↓
1823  51岁去世
```

## 关键转折点

1. **1799年读《国富论》**：从金融家转向经济学研究
2. **1815年《谷物法论战》**：从实践经济学转向理论建构
3. **1817年《政治经济学及赋税原理》出版**：确立古典经济学集大成者地位
4. **1819年当选议员**：从学者转变为公共政策推动者
