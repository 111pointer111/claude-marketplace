## 人物简介

**大卫·李嘉图**（David Ricardo，1772年4月18日—1823年9月11日），英国古典政治经济学家，古典学派的最后一名代表，最有影响力的古典经济学家。

### 学术地位
李嘉图是英国古典政治经济学的集大成者，他以劳动价值论为基础，建立了包括分配论、货币理论、税收理论和比较优势贸易理论在内的完整理论体系。他的比较优势理论至今仍是国际贸易理论的基石。

### 人生轨迹
1. **金融天才期（14-27岁）：** 14岁随父从事证券交易，21岁与父亲决裂独立创业，26岁以800镑资本起家，42岁退休时资产达160万镑
2. **学术觉醒期（27-43岁）：** 27岁读《国富论》，37岁发表第一篇经济学论文
3. **学术巅峰期（43-51岁）：** 43岁与穆勒结交并开始写作，45岁出版《政治经济学及赋税原理》，47岁当选上议院议员，51岁因耳部感染去世

### 核心贡献
- **比较优势理论：** 国际分工与贸易的利益，即使一国在所有商品生产上均处于劣势
- **劳动价值论：** 商品价值取决于生产所必需的相对劳动量
- **分配论：** 工资、利润、地租三分配及相互关系
- **税收理论：** 赋税损害积累，最好财政计划是节流
