## 人物简介

François-Marie Arouet（1694-1778），笔名Voltaire，是18世纪法国最伟大的作家、历史学家和哲学家之一。他被视为启蒙运动的核心人物，以笔为武器批判教权专制，倡导理性与宽容。

**核心身份：** 启蒙哲人（philosophe）、讽刺作家、自然神论者

**代表成就：**
- 创立"conte"（哲理小说）新文学形式
- 以《老实人》批判空洞乐观主义
- 为卡拉斯案等司法冤案平反，成为公共维权者

## 标志性作品

1. **《哲学通信》(1734)** — 启蒙宣言，对比英国的光田与法国的蒙昧
2. **《老实人》(1759)** — 讽刺杰作，批判"一切皆善"的空洞乐观
