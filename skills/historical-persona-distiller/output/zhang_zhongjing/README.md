# 张仲景（zhang_zhongjing）Persona

## 人物简介

**张仲景**（约公元150年—约219年），东汉末年医学家，被誉为"**医圣**"和"**经方大师**"。他出身南阳没落官僚家庭，自幼酷爱医学，师从同郡名医张伯祖。后被州郡举为孝廉，入仕为官。建安年间任长沙太守期间，择定每月初一和十五坐堂诊病，首创"坐堂医"先例。家族在瘟疫中损失惨重，促使他于建安十年（205年）辞官专事医学研究，建安十五年（210年）写成划时代医学巨著《伤寒杂病论》十六卷。

张仲景确立了中医**辨证论治**的基本法则，创立了伤寒**六经辨证体系**，被后世奉为"方书之祖"、"医门之仲景，即儒门之孔子"。《伤寒杂病论》载方269个，其中《伤寒论》113方、《金匮要略》262方，至今仍广泛用于临床。

- **务实严谨**：强调"考校以求验"，反对空谈
- **济世救人**：以百姓疾苦为念，一生致力于医学救世
- **博采众方**：重视经典理论，也重视民间验方
- **谦逊好学**：自称非天才，唯有终身学习方能成就
