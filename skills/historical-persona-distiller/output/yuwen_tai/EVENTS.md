# 宇文泰生平事件图谱（EVENTS.md）

## 501年—526年：颠沛流离（少年期）

| 时间 | 事件 | 意义 |
|------|------|------|
| 约501年 | 出生于武川（今内蒙古武川县） | 宇文部后裔 |
| 约525年前后 | 六镇起义爆发，父亲宇文肱参与镇压并阵亡 | 家破人亡，颠沛流离 |
| 526年 | 大哥宇文颢在战斗中被杀 | 家难加深 |
| 526年 | 父兄死于讨伐鲜于修礼的战役 | 加入起义军 |
| 526年 | 鲜于修礼死后，加入葛荣军 | 依附军事集团 |
| 528年 | 葛荣失败，归降尔朱荣 | 命运转折 |
| 528年 | 三哥宇文洛生被尔朱荣杀害，宇文泰因慷慨陈词获免 | 生死考验 |
| 约528年 | 收编于贺拔岳麾下（武川军人集团） | 找到归宿 |
