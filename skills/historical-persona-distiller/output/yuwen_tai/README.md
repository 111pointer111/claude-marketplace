# 宇文泰（yuwen_tai）人物简介

## 基本信息

- **姓名：** 宇文泰
- **字：** 孝卿
- **民族：** 鲜卑族
- **生卒：** 约公元501年—556年
- **朝代：** 北魏→西魏→（追尊北周文皇帝）
- **主要身份：** 军事统帅、政治改革家、西魏权臣、北周奠基人

## 一句话概括

宇文泰是北魏末年六镇起义中崛起的鲜卑族军事和政治领袖，以"知人善任、用人不疑"聚合了一大批关陇英豪，在东西魏对峙中以少胜多创建了西魏/北周的军事霸权，并创建府兵制、推行六条诏书、仿周礼改官制，为隋唐政治制度奠基。

## 历史地位

宇文泰与高欢并称"两大权臣"，分别控制西魏和东魏，最终两人所奠定的基础分别演变为北周和北齐。宇文泰的贡献包括：
- **府兵制**：开创性的军事制度，影响深远
- **六条诏书**：系统治国纲领
- **均田制**：经济基础
- **胡汉融合**：民族政策超越华夷之辨
- **六官制**：为隋唐三省六部制之渊源

## 使用说明

本 persona 适合用于：
- 模拟军事战略家、政治改革家的决策风格
- 分析府兵制、均田制等历史制度
- 讨论胡汉民族融合的历史经验
- 理解"知人善任"领导力模型
- 以少胜多的军事战例分析

## 局限性

- 宇文泰不是诗人，没有诗词作品，其 persona 以政治/军事语料为主
- 主要语料来源为百度百科，传记细节丰富但无一手古典文献
