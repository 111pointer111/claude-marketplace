## 核心关系图

```
家庭：
- 父：索佛洛尼斯科斯（Sophronicus，雕刻匠）
- 母：费纳瑞特（Phaenarete，助产婆）
- 妻：赞提佩（Xanthippe，上层女性）
- 子：三子（名字不详，据色诺芬记载才智平平）

师承/影响：
- 受教于巴门尼德、芝诺（爱利亚学派）
- 受教于女祭司狄奥提玛（Diotima，爱的哲学）

弟子：
- 柏拉图（Plato，最著名弟子，后创柏拉图学园）
- 色诺芬（Xenophon，历史学家，记录苏格拉底）
- 阿尔西比亚德斯（Alcibiades，政治家，后叛雅典）
- 斐多（Phaedo，陪伴临终）
- 安提斯泰尼（Antisthenes，犬儒学派创始人）
- 亚里斯提卜（Aristippus，居勒尼学派创始人）
- 齐诺（Zeno of Citium，斯多葛学派创始人）
```

## 思想发展脉络

```
Stage 1-2: 早年学艺 → 从军经历（质疑传统德行）
     ↓
Stage 3: 德尔菲神谕 → 诘问式对话哲学（"认识你自己"）
          "美德即知识"
          "助产术"/"产婆术"
          "牛虻比喻"
     ↓
Stage 4: 审判 → 拒绝逃亡 → 饮毒而死
          以身殉道，为思想自由树立千古丰碑
```

## 历史背景

- **希波战争**（公元前500-449年）：雅典黄金时代的背景
- **伯罗奔尼撒战争**（公元前431-404年）：雅典由盛转衰，苏格拉底思想的现实土壤
- **三十僭主时期**（公元前404-403年）：斯巴达占领后扶持的傀儡政权，后被推翻
- **民主制恢复**（公元前403年后）：清算与反清算的政治氛围，导致苏格拉底被审判
