# 哥白尼（尼古拉·哥白尼）

> Nicolaus Copernicus | 1473–1543

## 人物简介

**尼古拉·哥白尼**（拉丁语：Nicolaus Copernicus，波兰语：Mikołaj Kopernik，1473年2月19日－1543年5月24日），波兰文艺复兴时期天文学家、数学家、教会法博士、日心说（太阳中心说）的系统化提出者。

他提出的**日心说**将太阳置于宇宙中心而非地球，彻底改变了人类对宇宙的认识，标志着**科学革命**的开端。其主要著作《天球运行论》（De revolutionibus orbium coelestium）在1543年临终时才出版，影响了开普勒、伽利略和牛顿等后世科学家。

---

## 核心贡献

1. **日心说系统化**：太阳而非地球是宇宙的中心，地球和其他行星围绕太阳运行
2. **地球双重运动**：每日自转一周，每年公转一周
3. **行星排列**：从太阳起：水星、金星、地球、火星、木星、土星
4. **科学方法论**：强调数学推理与天文观测的结合
