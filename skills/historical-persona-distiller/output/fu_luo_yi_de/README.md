# 西格蒙德·弗洛伊德 (Sigmund Freud)

## 人物简介

**西格蒙德·弗洛伊德**（1856年5月6日—1939年9月23日），奥地利神经科医生、精神分析师、心理学家，精神分析学的创始人，被誉为"心理学之父"之一。

弗洛伊德1900年出版的《梦的解析》奠定了他在心理学史上的核心地位。他提出的无意识理论、本我/自我/超我三元结构、性心理发展阶段、俄狄浦斯情结等概念，深刻影响了20世纪的人类思想、文化、艺术和社会科学。

弗洛伊德自视为科学家而非医生，他始终坚持以经验科学的方法来研究人类心灵。尽管他的理论在科学界仍有争议，但无人能否认他对心理学和精神医学的深远影响。
