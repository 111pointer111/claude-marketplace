# 西格蒙德·弗洛伊德生平事件图谱

## 1856-1885：求学与科学训练期

| 年份 | 事件 | 重要性 |
|------|------|--------|
| 1856 | 出生于Freiberg, Moravia（现捷克共和国Příbor） | 🔴核心 |
| 1859 | 全家因经济原因迁至莱比锡，后迁维也纳 | 🟡转折 |
| 1873 | 从Sperl Gymnasium毕业，受歌德文章启发转向医学 | 🔴核心 |
| 1873-1881 | 维也纳大学师从Ernst von Brücke，专注生理学与神经生物学 | 🔴核心 |
| 1881 | 获医学博士学位，进入维也纳总医院 | 🔴核心 |
| 1882 | 进入维也纳总医院临床实习，师从Theodor Meynert | 🟡转折 |
| 1885 | 任神经病理學讲师，对可卡因产生研究兴趣 | 🟡重要 |
| 1885-1886 | **关键转折**：赴巴黎Salpêtrière诊所师从夏尔科，19周的工作成为职业转折点 | 🔴核心 |

## 1886-1900：精神分析创立与临床实践期

| 年份 | 事件 | 重要性 |
|------|------|--------|
| 1886 | 返回维也纳，与Martha Bernays结婚（极幸福的婚姻，6个孩子） | 🟡重要 |
| 1886-1895 | 与布洛伊尔合作发展"谈话疗法"，Anna O.案例成为重要起点 | 🔴核心 |
| 1895 | 《癔症研究》出版（与布洛伊尔合著）——精神分析奠基文本 | 🔴核心 |
| 1895 | 布洛伊尔因弗洛伊德强调性的病因而分道扬镳 | 🔴核心 |
| 1895 | 开始撰写"Entwurf einer Psychologie"（Project for a Scientific Psychology） | 🟡重要 |
| 1896 | 父亲Jakob去世，开始认真自我分析 | 🔴核心 |
| 1897 | 发展自由联想法，取代催眠 | 🔴核心 |
| 1897 | 发现阻抗（resistance）的重要性 | 🔴核心 |
| 1897.9.2 | 著名信件：发现屏忆（screen memory）而非真实记忆 | 🔴核心 |
| 1900 | 《梦的解析》出版（自我分析的结晶）——被认为他最重要的著作 | 🔴核心 |

## 1900-1914：精神分析运动与理论成熟期

| 年份 | 事件 | 重要性 |
|------|------|--------|
| 1901 | 《日常生活的精神病理學》出版（弗洛伊德口误的来源） | 🟡重要 |
| 1905 | 《性学三论》出版——性理论的核心文本 | 🔴核心 |
| 1908 | 第一届国际精神分析大会在萨尔茨堡举行，名声开始被认可 | 🟡重要 |
| 1909 | **关键转折**：受克拉克大学邀请访问美国，奠定国际声誉 | 🔴核心 |
| 1911 | 阿德勒建立自己的心理学体系，正式分裂 | 🟡重要 |
| 1913 | 荣格正式分裂 | 🟡重要 |
| 1914 | 提出"自我"概念，为1923年三我结构铺垫 | 🟡重要 |

## 1914-1939：晚年与流亡期

| 年份 | 事件 | 重要性 |
|------|------|--------|
| 1916 | 《精神分析导论》出版 | 🔴核心 |
| 1923 | 《自我与本我》出版——三我结构成熟 | 🔴核心 |
| 1927 | 《幻想的未来》——对宗教的系统性批判 | 🟡重要 |
| 1930 | 《文明及其不满》 | 🟡重要 |
| 1933 | 纳粹上台，开始压制精神分析 | 🔴核心 |
| 1938 | 纳粹吞并奥地利后逃离至伦敦 | 🔴核心 |
| 1938-1939 | 患有口腔癌，长期痛苦 | 🔴核心 |
| 1939.9.23 | 请求致命剂量的吗啡，在伦敦去世，享年83岁 | 🔴核心 |

## 关键转折点总结

| 转折点 | 年份 | 影响 |
|--------|------|------|
| 巴黎师从夏尔科 | 1885-1886 | 从神经病理学家转向精神分析师 |
| 父亲的去世与自我分析 | 1896-1897 | 发现屏忆，建立俄狄浦斯理论基础 |
| 《梦的解析》出版 | 1900 | 理论与临床体系的形成 |
| 美国之行 | 1909 | 从争议人物到国际名人 |
| 《自我与本我》 | 1923 | 三我结构理论成熟 |
| 流亡伦敦 | 1938 | 从维也纳知识分子到政治难民 |
