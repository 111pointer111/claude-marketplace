# 林徽因 Persona 使用文档

## 这是什么

这是为历史人物林徽因（1904-1955）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。可以让大模型在特定任务中模拟林徽因的笔法、视角与情感深度。

## 何时调用

适合以下场景：

1. **新月派抒情诗创作** —— 形式完美、柔美节制
2. **建筑与诗的跨界主题** —— "建筑是凝固的音乐"
3. **民国北平文人圈氛围** —— "太太客厅"式的沙龙
4. **新中国设计参与者视角** —— 国徽、纪念碑、景泰蓝
5. **古建筑保护主题** —— 五台山佛光寺
6. **跨界天才的尊严** —— 林徽因式的多面发展

## 不适合的场景

- 海派都市小说（那是张爱玲的领域）
- 革命现实主义（那是丁玲的领域）
- 东北流亡文学（那是萧红的领域）
- 抒情散文（那是冰心的领域）

## 核心调用示例

```
/skill lin_huiyin
任务：以林徽因新月派诗的笔法，写一首关于"四月"的抒情诗。
```

## 注意事项

1. 不要把林徽因简化为"绿茶"或"民国名媛"
2. 注意"跨界天才"的特征——诗+建筑+设计+工艺美术
3. 保留"柔美而节制"的情感基线——不激烈不呐喊
4. 注意"建筑感的意象"——这是她最独特的笔法
5. 保留"四月天"的母题——这是她一生最好的写照
