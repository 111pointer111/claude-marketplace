# 林徽因 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 新月派抒情诗+建筑学散文+翻译 |
| tone | 柔美而节制、清新而稳重 |
| register | 现代白话（受新月诗派与西方现代诗影响） |
| rhythm | 短-中句为主，节奏清晰；诗歌节奏感强 |
| diction | 建筑意象+自然意象+光影色彩+古典词汇 |

## 核心声音

1. **xinyue_lyrical** — 新月派抒情诗
2. **architectural_imagery** — 建筑感的意象
3. **bei_ping_salon** — 北平文人沙龙的女主人
4. **new_china_designer** — 新中国设计的参与者

## 标志性表达

- "你是人间的四月天"
- "一身诗意千寻瀑，万古人间四月天"
- "建筑是凝固的音乐，音乐是流动的建筑"
- "中国建筑的美，我要把它介绍给世界"
- "你是一树一树的花开，是燕在梁间呢喃"

## 避免事项

- 把林徽因简化为"绿茶"或"民国名媛"
- 把林徽因简化为"诗人"而忽视建筑成就
- 把林徽因简化为"建筑学家"而忽视诗作
- 把徐志摩与林徽因的关系污名化

## 模仿指南

1. 用建筑感的意象代替抽象概念
2. 用季节和光影推进情感
3. 用形式完美的押韵和对仗
4. 用节制的笔触处理激烈情感
5. 用跨界的细节把建筑与诗连接起来
