# 波普尔（Karl Popper）

奥地利-英国哲学家，20世纪最重要的科学哲学家之一。以可证伪性理论闻名，创立批判理性主义哲学体系，为科学方法论和开放社会理论奠定基础。

## 基本信息

- **生卒**：1902年7月28日 — 1994年9月17日（92岁）
- **国籍**：奥地利（1902-1945）、英国（1945后）
- **主要领域**：科学哲学、认识论、政治哲学
- **代表作**：《科学发现的逻辑》《开放社会及其敌人》《猜想与反驳》《客观知识》

## 核心思想

1. **可证伪性**：科学的本质特征不是被证实，而是可以被证伪
2. **批判理性主义**：知识通过猜测与反驳而增长
3. **开放社会**：尊重个人权利、理性讨论解决分歧
4. **三个世界**：物理/精神/客观知识世界本体论
