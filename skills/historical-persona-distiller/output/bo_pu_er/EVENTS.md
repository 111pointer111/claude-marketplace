# 波普尔（Karl Popper）生平事件图谱

## 时间线

### 1902年：出生
- 7月28日出生于奥匈帝国维也纳
- 父母均为犹太裔，已皈依路德宗
- 父亲Simon Siegmund Carl Popper是律师，母亲Jenny是钢琴家
- 成长于"明显的书卷气"环境

### 1902-1918年：早年与教育
- 在Realgymnasium接受教育，后因不满教学标准而离开
- 1918年开始在维也纳大学旁听课程
- 1919年加入社会民主党学生协会，曾短暂信仰共产主义
- 1919年6月警察枪击事件后放弃马克思主义

### 1919-1937年：学术觉醒期
- 接触弗洛伊德和阿德勒的精神分析理论
- 参加爱因斯坦相对论讲座，发现关键差异
- 学习木匠手艺，获得小学教师资格（1925年）
- 1928年获心理学博士学位
- 1930年与Josefine Anna Henninger结婚
- 1934年出版《科学发现的逻辑》

### 1937-1945年：流亡新西兰
- 1937年移居新西兰基督城坎特伯雷大学学院
- 1938年奥地利被吞并后转向社会政治哲学
- 1945年出版《开放社会及其敌人》
- 1946年二战结束后移居英国

### 1946-1969年：LSE时期（学术鼎盛）
- 1946年任伦敦经济学院逻辑学讲师
- 1949年成为逻辑和科学方法教授，创建哲学系
- 1958-1959年任亚里士多德学会主席
- 1959年出版《科学发现的逻辑》英译本
- 1963年出版《猜想与反驳》
- 1965年被封为骑士
- 1969年退休但继续活跃

### 1969-1994年：晚年
- 1985年返回奥地利，妻子同年去世
- 1986年返回英国，定居肯利
- 1992年获京都奖
- 去世前两周仍在工作
- 1994年9月17日于肯利去世，享年92岁

## 关键转折点

| 年份 | 事件 | 影响 |
|------|------|------|
| 1919 | 警察枪击事件 | 放弃马克思主义，转向批判性思想 |
| 1919 | 爱因斯坦讲座 | 发现科学理论与精神分析的根本差异 |
| 1934 | 《科学发现的逻辑》出版 | 建立证伪主义理论基础 |
| 1938 | 奥地利被吞并 | 促使转向社会政治哲学 |
| 1945 | 《开放社会及其敌人》 | 建立开放社会理论 |
| 1959 | 《科学发现的逻辑》英译本 | 英语世界确立学术地位 |

## 核心关系

- **导师**：卡尔·布勒（心理学博士导师）、莫里茨·施利克（论文委员会）
- **影响来源**：爱因斯坦（科学方法）、休谟（归纳问题）、康德（批判哲学）
- **学术传承**：约瑟夫·阿加西、伊姆雷·拉卡托斯、保罗·费耶拉本德（学生）
- **政治同道**：弗里德里希·哈耶克、米尔顿·弗里德曼（蒙特佩林学会）
- **批评对象**：维特根斯坦、弗洛伊德、马克思主义、历史决定论
