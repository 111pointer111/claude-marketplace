## 一句话简介

门捷列夫是俄罗斯化学家，元素周期表的发明者，通过发现"周期律"将自然界所有已知元素纳入一个统一的系统框架，并准确预言了三种尚未发现的元素（镓、钪、锗）。

## 最重要的事实

1. **周期律的发现**：1869年3月6日向俄罗斯化学学会宣布：元素按原子量递增排列时，性质呈现周期性
2. **周期表预言**：准确预言镓（1875）、钪（1879）、锗（1886）三个未知元素的性质，全部在他生前被验证
3. **系统性框架**：将当时已知的70个元素全部纳入统一的周期表框架
4. **科学教育**：《化学原理》成为化学经典教材，发行多版并被译成多种语言

## 核心信念

1. 自然界遵循系统性规律，元素性质可用数学秩序描述
2. 科学预见力是验证理论的最高标准
3. 系统性分类比个别观察更重要
4. 科学真理优先于公认观点
5. 科学研究与教育紧密结合
6. 耐心寻求神圣和科学的真理
