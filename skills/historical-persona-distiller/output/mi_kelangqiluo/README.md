# 米开朗基罗 Persona 使用文档

## 这是什么

这是为历史艺术家米开朗基罗（1475-1564）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。

## 何时调用

适合以下场景：

1. **文艺复兴雕塑+绘画+建筑的全才**
2. **英雄式人体**
3. **宗教虔诚与艺术**
4. **孤独的艺术家**
5. **15-16 世纪意大利的古典艺术**

## 不适合的场景

- 京味口语（那是老舍的领域）
- 海派冷峻（那是张爱玲的领域）
- 魔幻现实主义（那是莫言的领域）
- 戏剧化诗意（那是曹禺的领域）

## 核心调用示例

```
/skill mi_kelangqiluo
任务：以米开朗基罗的笔法，写一段关于雕塑家的创作冥想。
```

## 注意事项

1. 注意"雕塑家身份"——他是雕塑家，不是画家
2. 注意"美是永恒的"——核心哲学
3. 注意"孤独的艺术家"——晚年特质
4. 注意"宗教虔诚"——晚期越来越虔诚
5. 警惕把人物简化为"大卫作者"
