> 数学王子，1777-1855，德国

## 人物简介

约翰·卡尔·弗里德里希·高斯（Johann Carl Friedrich Gauß，1777年4月30日—1855年2月23日）是德国数学家、天文学家、物理学家和大地测量学家，被誉为人类历史上最伟大的数学家之一。

**核心成就：**
- **1796年**：用直尺和圆规作出正17边形——这是2000年来该领域的首次突破
- **1801年**：出版《算术研究》（Disquisitiones Arithmeticae），系统创立数论基础；用最小二乘法预测谷神星轨道，一举成名
- **1823年**：正式发表最小二乘法——现代统计学的基石之一
- **1828年**：发表高斯绝妙定理（theorema egregium）——微分几何奠基性成果
- **1831-1840年**：与Weber合作进行地磁学研究，发明磁力仪，发现电磁学基本定律

**思想特点：**
- 完美主义者——推迟发表甚至不发表许多重大发现
- 内心激进（坚信非欧几何），公开保守（保持沉默55年）
- 将数学视为"揭示自然秩序的神圣语言"
