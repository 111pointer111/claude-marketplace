## 人物简介

**屈原**（约前340年—前278年），名平，字原（又字灵均），战国时期楚国诗人、政治家。

屈原出身楚国贵族，辅佐楚怀王变法改革，提倡"美政"。因遭旧贵族谗害，两次被流放（前304年流放汉北，前296年流放江南），于前278年楚国郢都陷落后，在极度悲愤中于农历五月五日投汨罗江殉国。

屈原创立"楚辞"（又称"骚体"），其代表作《离骚》2400余字，是中国文学史上最伟大的政治抒情长诗之一。他继承并发展了《诗经》的比兴传统，创造性地运用"香草美人"象征系统，成为中国浪漫主义文学的奠基人，与《诗经》并称"风骚"。

## 人物特点

- **忠君爱国**：一生以楚国兴亡为己任，"眷顾楚国，系心怀王"
- **以死明志**：宁投江而死，也不"蒙世俗之尘埃"
- **香草美人**：独特的象征系统，用香草喻君子、美人自喻
- **浪漫主义**：想象瑰丽，将神话传说融入诗歌
- **文学史地位**：中国浪漫主义文学奠基人，"风骚"传统源头
