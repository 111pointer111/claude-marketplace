# 莫言 Persona 使用文档

## 这是什么

这是为历史作家莫言（1955-）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。

## 何时调用

适合以下场景：

1. **魔幻现实主义长篇** —— 高密东北乡的笔法
2. **代际叙事** —— "我爷爷""我奶奶""我父亲""姑姑"
3. **极端意象** —— 酷刑、生育、战争、饥饿
4. **诺奖文体** —— 2012 年诺奖
5. **当代中国乡土场景** —— 1980 年代的农村

## 不适合的场景

- 京味口语（那是老舍的领域）
- 海派冷峻（那是张爱玲的领域）
- 学贯中西的学者气（那是钱钟书的领域）
- 戏剧化诗意（那是曹禺的领域）

## 核心调用示例

```
/skill mo_yan
任务：以莫言的笔法，写一段高密东北乡的红高粱场景。
```

## 注意事项

1. 注意"感官化叙事"——颜色、声音、气味的极度铺陈
2. 注意"代际叙事"——我爷爷、我奶奶、姑姑
3. 注意"极端意象"——酷刑、生育、战争
4. 警惕"高密东北乡"的方言色彩
5. 关注"种的退化"——这是莫言对现代中国最深切的担忧
