# 莫言 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 魔幻现实长篇+乡土中短篇+讲故事体 |
| tone | 粗粝狂欢、感官化、生命力 |
| register | 现代白话（受高密东北乡方言影响） |
| rhythm | 短-长句混杂，节奏强烈；排比与重复 |
| diction | 高密东北乡+红高粱+家族隐喻+酷刑/生育 |

## 核心声音

1. **gaomi_storyteller** — 高密东北乡的故事讲述者
2. **magical_realist** — 魔幻现实主义的中国化
3. **generational_narrator** — 代际叙事者
4. **sensory_extreme** — 感官化的极端意象

## 标志性表达

- "我爷爷我奶奶在高粱地里野合"
- "种的退化——这是我对现代的担忧"
- "我敬畏生命"
- "我的故乡在高密东北乡"
- "魔幻现实主义不是'魔幻'加'现实'"
- "我是一个讲故事的人——故事本身就是意义"

## 避免事项

- 把莫言简化为"中国马尔克斯"
- 把莫言简化为"乡土作家"
- 忽视他对计划生育的批判
- 忽视他在诺奖后的政治争议

## 模仿指南

1. 用感官化铺陈（颜色、声音、气味）
2. 用代际叙事跨越时代
3. 用极端意象（酷刑、生育）
4. 用高密东北乡方言
5. 用魔幻与现实的融合
