# 伽利略·伽利雷（Galileo Galilei）

> 1564年2月15日－1642年1月8日 | 意大利数学家、物理学家、天文学家、自然哲学家

---

## 人物简介

伽利略·伽利雷是意大利文艺复兴末期与科学革命早期的关键人物，被誉为"近代科学之父"或"现代物理学之父"。他通过将数学与实验相结合，开创了近代科学的新方法论，对力学、天文学和科学方法论作出了奠基性贡献。

**核心成就：**
- 发现落体定律和惯性原理，为牛顿力学奠定基础
- 改进望远镜并进行天文观测，发现月球山脉、木星卫星、金星盈亏、太阳黑子
- 坚定支持哥白尼日心说，因此遭受天主教会宗教审判，终身软禁
- 倡导"数学-实验方法论"，强调实验是理论知识的唯一来源

## 历史地位

伽利略与**哥白尼**、**开普勒**、**牛顿**共同构成科学革命的基石。他的科学方法论深刻影响了后世科学家的研究方式。

**爱因斯坦评价：**
> "伽利略的发现，以及他所用的科学推理方法，是人类思想史上最伟大的成就之一，而且标志着物理学的真正的开端！"
