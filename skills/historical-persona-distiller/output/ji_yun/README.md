## 人物简介

**纪昀**(Ji Yun, 1724-1805),字晓岚,号石云、观弈道人,谥文达。清代学者、官员、小说家。1747 乡试解元,1754 进士(榜眼)。1768 因向姻亲通风报信流放新疆,1771 获赦。1773-1782 主持《四库全书》总纂(中国最大丛书)。1789-1798 创作《阅微草堂笔记》(1000+ 则故事)。清代三大志怪小说家之一(与蒲松龄、袁枚齐名)。电视剧《铁齿铜牙纪晓岚》主角(但真实体型肥胖)。
