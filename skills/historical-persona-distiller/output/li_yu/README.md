# 李煜（南唐末代君主·千古词帝）

> 李煜，五代十国南唐末代君主，字重光，初名从嘉，号钟隐、莲峰居士。在位15年（961-975），国破降宋，封违命侯，后因作《虞美人》被宋太宗鸩杀，年四十二。

---

## 人物简介

李煜是"千古词帝"，在词学上的成就超越了同时代所有词人。王国维在《人间词话》中称其"词至李后主而眼界始大，感慨遂深，遂变伶工之词而为士大夫之词"，又称之为"以血书者"，与释迦、基督担荷人类罪恶之意相比肩。

李煜一生经历从帝王到囚徒的巨大转折。975年南唐灭亡，李煜被俘至汴京，从此沦为囚徒。在生命的最后三年，他以词抒发亡国之痛、故国之思，留下了《虞美人》《浪淘沙令》《相见欢》等不朽名作。

## 代表作品

### 《虞美人》
春花秋月何时了？往事知多少。小楼昨夜又东风，故国不堪回首月明中。雕栏玉砌应犹在，只是朱颜改。问君能有几多愁？恰似一江春水向东流。

### 《相见欢》
无言独上西楼，月如钩。寂寞梧桐深院锁清秋。剪不断，理还乱，是离愁，别是一般滋味在心头。

### 《浪淘沙令》
帘外雨潺潺，春意阑珊。罗衾不耐五更寒。梦里不知身是客，一晌贪欢。独自莫凭栏，无限江山，别时容易见时难。流水落花春去也，天上人间。
