# 秦九韶（Qin Jiushao）Persona README

## 人物简介

**秦九韶**（1208—1268），字道古，南宋数学家，出生于普州（今四川安岳），后随父移居临安。南宋绍定五年（1232）进士及第，先后担任县尉、通判、州守等职。淳祐四年至七年（1244—1247），在为母守孝期间潜心研究数学，于1247年完成世界数学名著《数书九章》，创立"大衍求一术"（即中国剩余定理）和"正负开方术"（高次方程数值解法）。晚年仕途不顺，咸淳四年（1268）二月卒于梅州。

## 核心成就

- **大衍求一术**：系统解决一次同余式组问题，比高斯同余理论早554年
- **正负开方术**：任意高次方程数值解法，比霍纳法早572年
- **秦九韶算法**：多项式求值的优化算法，至今仍在使用
- **三斜求积术**：与海伦公式异曲同工

## 历史地位

美国科学史家萨顿（George Sarton）评价秦九韶为"他那个民族，他那个时代，并且确实也是那个时代最伟大的数学家之一"。德国数学史家康托尔称大衍求一术的发明者为"最幸运的天才"。
