# 鬼谷子 生平事件图谱

## 隐居修道期（约前400年前后）

| 时间 | 事件 | 备注 |
|------|------|------|
| 约前400年 | 生于或隐居于东周阳城附近山谷 | 后称"鬼谷岭"，林木茂盛，幽不可测 |
| 早年 | 与墨翟一起采药修道 | "鬼谷子王诩则通天彻地，其学问之渊博，无人能及" |
| 某日 | 发现师父遗留的"无字天书"竹简 | 月光下显现金书，得纵横家书十三篇 |
| 夜夜读书 | 每夜读一遍，得一书（兵法、致富奇书、养性修真大法、推命相面术等） | "天上人间、治国安邦、仕途经济、天文地理、星命术数、丹药养生，无所不有" |

## 授徒讲学期（约前350-前320年）

| 时间 | 事件 | 备注 |
|------|------|------|
| 约前350年 | 在云梦山聚徒讲学 | "进山前他们都只是无名小卒，出山后个个大放异彩、名流千古" |
| 收徒 | 苏秦、张仪 | 《史记》明确记载："苏秦者，东周洛阳人也。东事师于齐，而习之于鬼谷先生。""张仪者，魏人也。始尝与苏秦俱事鬼谷先生，学术。苏秦自以不及张仪。" |
| 收徒 | 孙膑、庞涓 | 习兵法，"斩草为马，撒豆为兵"，在演兵岭训练排兵布阵 |
| 传说 | 收茅蒙、徐福为徒 | 仙道家，传说徐福后为秦始皇寻药 |
| 传说 | 收毛遂（存疑）、张良（存疑）为徒 | 策士 |
| 弟子五百 | 培养众多杰出人才 | "在世数百年，后不知所终" |

## 晚年及后世

| 时间 | 事件 | 备注 |
|------|------|------|
| 晚年 | 归隐云梦山，孝敬老母 | 挖鬼谷井，"朝看母亲十八九，暮观老母鬓如霜" |
| 不知年 | 谢世归天 | 传说为得道真人，"历数代而不老" |
| 身后 | 《鬼谷子》流传后世 | 最早见于《隋书·经籍志》，共二十一篇（转丸篇已失传） |
| 身后 | 入选国家级非物质文化遗产 | 2015年河南省"鬼谷子传说"列入省级非遗 |
| 身后 | 鬼谷子文化研究会成立 | 2015年山东省鬼谷子研究会成立 |
| 身后 | 国际学术论坛举办 | 2016年首届中国（蒙山）鬼谷子文化国际论坛 |

## 关键转折点

| 事件 | 意义 |
|------|------|
| 发现金书 | 从被动接受到主动研习，开启智慧的转折点 |
| 收苏秦、张仪 | 将学术思想转化为实际政治影响力的关键 |
| 教授孙膑、庞涓兵法 | 军事思想传承的重要环节 |
| 著述《鬼谷子》 | 将个人智慧系统化，为后世留下完整的思想体系 |

## 历史定位

**纵横家之鼻祖** — "一人之言，重于九鼎之宝。三寸之舌，胜过百万雄师。"苏秦佩六国相印，张仪两为秦相，左右战国政局。

**兵家之师** — 孙膑、庞涓习兵法，孙膑围魏救赵、庞涓魏国将军，"孙子兵法"与"鬼谷子兵法"一文一武相辅相成。

**道家真人** — 道教将其与老子同列，尊为"王禅老祖"，"玄微真人"。

**谋略家泰斗** — "潜谋于无形，常胜于不争不费"，为后世政治家、外交家、商人必读之书。

**教育家典范** — 原国防部副部长伍修权题字"天下第一军校"，培养了改变战国格局的弟子群。
