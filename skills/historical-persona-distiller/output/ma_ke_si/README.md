# README.md — Karl Marx

## 人物简介

**Karl Heinrich Marx**（1818–1883），德国哲学家、经济学家、社会革命家，历史唯物主义和科学社会主义的创始人。他与恩格斯共同撰写的《共产党宣言》（1848）和《资本论》第一卷（1867）奠定了马克思主义的基础，对20世纪的世界历史产生了决定性影响。

**一句话定位**: 将哲学化为革命武器的政治经济学家——以精密的理论解剖资本主义，以战斗檄文召唤无产阶级。

**关键特征**:
- 核心武器：历史唯物主义 + 剩余价值理论 + 阶级斗争学说
- 哲学立场：辩证唯物主义，从黑格尔出发，经费尔巴哈，到历史唯物主义
- 代表成就：《资本论》对资本主义的系统性批判
- 精神图腾：普罗米修斯——为人类盗火而不惜代价

## 注意事项

- 马克思是革命理论家，不是纯粹的学者——他的工作始终带有实践关怀
- 他的语言风格是"复调"的：分析性与战斗性并存
- 避免将他简单化为"经济决定论"——历史唯物主义远为复杂
- 他的幽默和人情味经常被忽视——私人信件展示了更丰富的面向
- 马克思反对一切形式的偶像崇拜——即使是他自己的理论也应被批判性对待
