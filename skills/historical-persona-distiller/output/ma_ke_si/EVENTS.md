# EVENTS.md — Karl Marx 生平事件图谱

## 核心时间线

### 1818-1835: 早年与教育启蒙
- **1818年5月5日**: 出生于普鲁士莱茵省特里尔城（Karl Heinrich Marx）
- **1816年**: 父亲海因里希·马克思改信路德宗（为职业原因）
- **1825年**: 母亲改信基督教
- **1830-1835年**: 就读于特里尔的弗里德里希·威廉中学（该校受警察监视）
- **1835年10月**: 进入波恩大学学习法律

### 1836-1841: 大学与博士阶段
- **1836年**: 转学至柏林大学，改修法律与哲学；与燕妮·冯·威斯特华伦秘密订婚
- **接触黑格尔**: 起初排斥，后成为青年黑格尔派成员
- **1841年**: 于耶拿大学获哲学博士学位，论文论德谟克利特与伊壁鸠鲁自然哲学之差异
- **普罗米修斯精神**: 博士论文中引用普罗米修斯"我宁愿被缚在崖石上，也不愿作宙斯的忠顺奴仆"

### 1842-1848: 新闻工作与革命转向
- **1842年10月**: 任《莱茵报》编辑
- **1843年3月**: 《莱茵报》被查封，辞职
- **1843年6月19日**: 与燕妮·冯·威斯特华伦结婚
- **1843年10月**: 迁居巴黎，接触法国社会主义思潮
- **1844年8月**: 在巴黎结识恩格斯，从此结为终生挚友
- **1845年**: 被法国驱逐，迁往布鲁塞尔；与恩格斯合著《神圣家族》
- **1845-1846年**: 撰写《德意志意识形态》，提出历史唯物主义
- **1845年**: 撰写《关于费尔巴哈的提纲》（包含"问题在于改变世界"）
- **1846年**: 创立共产主义者通讯委员会
- **1847年**: 加入共产主义者同盟；为正义者同盟起草纲领
- **1848年2月**: 与恩格斯共同发表《共产党宣言》

### 1849-1864: 伦敦流亡
- **1849年**: 被比利时驱逐；短暂返回巴黎后定居伦敦（此后40年再未离开）
- **1852-1862年**: 任《纽约每日论坛报》驻伦敦通讯记者
- **生活极度贫困**: 依靠恩格斯接济；多次因贫困和疾病陷入困境
- **1864年**: 参与创建第一国际工人协会（国际工人协会）

### 1865-1883: 《资本论》创作与晚年
- **1867年**: 《资本论》第一卷出版
- **1871年**: 巴黎公社期间写文章积极声援
- **1871年**: 与巴枯宁派在第一国际内部斗争
- **1875年**: 撰写《哥达纲领批判》，批评德国社会民主工党纲领
- **1878年**: 《反杜林论》出版（恩格斯著，马克思参与部分写作）
- **1881年12月**: 妻子燕妮去世
- **1881年**: 爱女燕妮（长女）去世
- **1883年3月14日**: 马克思在伦敦寓所去世，享年64岁
- **1883年3月17日**: 葬于伦敦海格特公墓；恩格斯发表墓前讲话

## 关键事件分析

### 思想转折点
1. **1843年《黑格尔法哲学批判导言》**: 提出"宗教是人民的鸦片"，完成从青年黑格尔派到革命社会主义者的转变
2. **1844年《经济学哲学手稿》**: 异化劳动理论，成为成熟历史唯物论的先声
3. **1845年《关于费尔巴哈的提纲》**: 标志历史唯物主义世界观的形成
4. **1848年《共产党宣言》**: 马克思主义正式诞生的标志性文件
5. **1859年《〈政治经济学批判〉序言》**: 对历史唯物主义最经典的表述

### 政治行动转折点
1. **1842年《莱茵报》时期**: 首次直接介入政治新闻工作
2. **1848年革命**: 亲历欧洲革命，在科隆创办《新莱茵报》
3. **1864年第一国际**: 从纯粹理论转向直接组织工人运动
4. **1871年巴黎公社**: 对无产阶级专政的首次实践给予理论总结

### 个人生活重要节点
1. **1843年与燕妮结婚**: 燕妮出身贵族，马克思出身平民，婚事曾遭燕妮家人反对
2. **1850年代贫困期**: 恩格斯多次给予经济援助，二人友谊经受住考验
3. **女儿们的回忆**: 燕妮、劳拉、埃莉诺三位女儿回忆起父亲为他们讲《荷马》、《堂吉诃德》的温馨场景
