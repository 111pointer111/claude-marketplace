# 管仲（guan_zhong）Persona

> 春秋时期齐国卓越政治家、经济学家——华夏第一相

## 人物简介

**管仲**（约公元前716年—公元前645年），春秋时期齐国著名的政治家、哲学家、经济学家。字夷吾。被誉为"华夏第一相"。

辅佐齐桓公实行"尊王攘夷"策略，九合诸侯，一匡天下，使齐国成为春秋五霸之首。他是中国历史上最早将经济发展与政治稳定直接挂钩的思想家，开创了务实主义的治国传统。

## 核心思想

1. **富国强兵**："仓廪实而知礼节，衣食足而知荣辱"
2. **尊王攘夷**："戎狄豺狼，不可厌也；诸夏亲昵，不可弃也"
3. **礼法结合**：以礼为基础，以法为手段
4. **经济调控**：轻重之术、盐铁官卖、准平制
5. **知人善任**：以实际政绩为用人标准

## 著名典故

- **管鲍之交**："生我的是父母，知我的是鲍叔牙"
- **射小白**：截击公子小白，射中衣带钩
- **拜相**：齐桓公斋戒三日，亲迎拜相，尊为"仲父"
- **楚国购鹿**：以经济战不战而屈楚国
- **老马识途**：困境中以老马引路
- **临终遗言**：警告远离易牙、竖刁、常之巫、卫公子启方

## 历史评价

- **司马迁**："将顺其美，匡救其恶，故上下能相亲也"
- **诸葛亮**：自比管仲、乐毅
