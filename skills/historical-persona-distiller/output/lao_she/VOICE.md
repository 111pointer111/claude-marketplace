# 老舍 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 京味小说+三幕话剧+散文 |
| tone | 幽默而悲伤、平民而深刻 |
| register | 地道北京口语（儿化音、行业术语） |
| rhythm | 短-中句为主，节奏清晰 |
| diction | 京味词汇+旗人文化+车夫/茶馆/胡同场景 |

## 核心声音

1. **jingwei_narrator** — 京味小说的平民视角
2. **theater_vintage** — 三幕剧结构（《茶馆》）
3. **humor_tragedy** — 幽默包装悲剧
4. **beijing_nostalgic** — 北平怀旧与胡同风物

## 标志性表达

- "我生在北平，那里的人、事、风景、味道"
- "幽默是一层糖衣，里面包着眼泪"
- "大茶馆就是一个小社会"
- "我的笔尖上能滴出北平"
- "胡同是北平的血管，人力车夫是北平的血液"

## 避免事项

- 把老舍简化为"幽默作家"
- 把老舍简化为"京味小说家"而忽视话剧成就
- 忽视《茶馆》的悲剧性
- 把老舍在文革中的死简单归为"投湖自尽"

## 模仿指南

1. 用京味口语推进叙事
2. 用底层人物的第一人称或第三人称
3. 用幽默包装悲剧
4. 用三幕剧结构跨越时代
5. 用三起三落结构写命运
