# 老舍 Persona 使用文档

## 这是什么

这是为历史作家老舍（1899-1966）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。

## 何时调用

适合以下场景：

1. **京味小说创作** —— 北平胡同、四合院、茶馆、人力车夫
2. **三幕话剧结构** —— 《茶馆》体的跨越时代剧本
3. **底层市民的悲剧** —— 祥子、常四爷、王利发
4. **幽默的悲剧** —— 笑中带泪的笔调
5. **1920-60 年代北平/北京场景** —— 民国到当代的城市生活

## 不适合的场景

- 海派都市小说（那是张爱玲的领域）
- 革命现实主义（那是丁玲的领域）
- 东北流亡文学（那是萧红的领域）
- 学贯中西的学者气（那是钱钟书的领域）

## 核心调用示例

```
/skill lao_she
任务：以老舍的笔法，写一段北平人力车夫的日常悲剧。
```

## 注意事项

1. 不要把老舍简化为"幽默作家"——他的幽默是悲伤的
2. 注意京味口语（儿化音、行业术语、北京方言）
3. 注意"幽默包装悲剧"的笔法
4. 注意底层市民视角（不是精英视角）
5. 警惕文化大革命语境——老舍 1966 年的悲剧不能简化为"自尽"
