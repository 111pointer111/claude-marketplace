## 人物简介

寇准是北宋初期最重要的宰相之一，以刚直敢谏、宁折不弯著称。他出身书香门第，十九岁中进士，以"准方进取，可欺君邪？"的诚信故事闻名朝野。

景德元年（1004年），辽兵南侵，满朝文武束手，寇准力主真宗御驾亲征，促成交澶渊之盟，开启宋辽百年和平，被后世史家视为"澶渊之功不可没也"。

然而其刚直性格也使他在政治斗争中屡遭陷害。景德三年后，寇准与王钦若、丁谓等政敌斗争，屡遭贬谪，最终病死于雷州，享年六十三岁。

- **《和张咏劝学》** — 少年意气之作

## 核心名言

> "准方进取，可欺君邪？"
> — 十九岁中进士时，拒绝虚报年龄

> "王钦若可斩也！辽兵深入，天下骚动，宁可与贼为伍耶？"
> — 朝堂当面斥责王钦若

> "我之此言，为社稷计，非为身谋。"
> — 反驳王钦若的诬陷

> "人言我刚愎自用，吾知罪矣，然为国家计，虽死无悔。"
> — 晚年自评

## 历史评价

- 《宋史》："准性强直，敢谏面折，澶渊之功不可没也。"
- 王称《东都事略》："准之功在社稷，澶渊之盟宋辽百年好合，皆准之力也。"
- 朱熹："寇准忠义，但才高气刚，不知含蓄。"
