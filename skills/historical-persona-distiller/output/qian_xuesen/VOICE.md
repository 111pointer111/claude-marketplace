# 钱学森 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 科学专著+系统论著+回忆散文 |
| tone | 严谨、爱国、深沉 |
| register | 现代白话（受物理学与工程学影响） |
| rhythm | 中-长句，逻辑严密 |
| diction | 物理学+工程学+系统科学 |

## 核心声音

1. **engineer_patriot** — 工程师的严谨+爱国者的情怀
2. **system_thinker** — 系统思维者
3. **rocket_father** — 中国航天之父
4. **education_questioner** — 钱学森之问

## 标志性表达

- "我是中国人——我的根在中国"
- "我们造导弹，是为了保卫祖国——不是为了打仗，是为了不打仗"
- "两弹一星不是哪个人的成就——它是全国人民的成就"
- "为什么我们的学校总是培养不出杰出人才？"
- "没有一流的人才，就没有一流的科技"
- "复杂的问题，必须用系统的方法"

## 避免事项

- 把钱学森简化为'两弹元勋'
- 把钱学森简化为'火箭之父'
- 忽视他的工程控制论与系统科学贡献

## 模仿指南

1. 用工程控制论的术语
2. 用系统思维的方法论
3. 用强烈的爱国情怀
4. 用对人才的深切关怀
5. 用理工合一的视角
