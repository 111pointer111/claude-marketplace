# 弗朗西斯·培根 (Francis Bacon) 1561-1626

> "I have taken all knowledge to be my province."
> — 致伯利勋爵信，1591年

## 人物简介

弗朗西斯·培根是英国文艺复兴末期至启蒙时代最重要的哲学家、作家、政治家之一。他是一位真正的通才——律师、议员、大法官、随笔作家、历史学家和科学改革者。

培根最著名的贡献是奠定了**现代经验主义哲学**和**科学方法论**的基础。他提出"知识就是力量"，认为知识的真正目的是改善人类处境；发明了排除式归纳法作为科学发现的方法；他关于"四偶像"的学说是批判性思维的先驱；他的乌托邦小说《新亚特兰蒂斯》预言了现代研究机构的形态。

培根曾任英国大法官（Lord Chancellor），服务于伊丽莎白一世和詹姆斯一世，但在1621年因受贿罪被弹劾。他晚年从政治退休后专心写作，留下了大量哲学、文学和科学著作。1626年，他在实验雪的保存性质时受寒去世——这是这位"实验哲学之父"的适当结局。

## 生平速览

- **1561**：出生于伦敦约克府
- **1573-1575**：剑桥大学三一学院学习（厌恶亚里士多德哲学）
- **1576-1579**：法国外交官随从
- **1582**：成为律师
- **1584-1621**：下议院议员（37年）
- **1597**：第一版《随笔》（10篇）
- **1603**：詹姆斯一世即位后获封骑士
- **1605**：《论学术的推进》出版（第一本重要英文哲学著作）
- **1618**：任大法官（Baron Verulam）
- **1620**：《伟大复兴》和《新工具》出版
- **1621**：因受贿被弹劾，承认罪行，退出政坛
- **1625**：第三版《随笔》（58篇）
- **1626**：实验雪的温度效用时受寒去世

## 核心思想

### 知识即权力
> "Human knowledge and human power come to the same thing."
> — 《新工具》

知识不是思辨的玩物，而是支配自然、改善人类处境的实际力量。

### 排除式归纳法
真正的科学方法不是从少量观察跳跃到普遍结论，而是通过系统排除逼近自然形式。

### 四偶像说
人心有四大偏见必须被识别和克服：部落偶像（人类本性）、洞穴偶像（个人经历）、市场偶像（语言）、剧场偶像（哲学体系）。

### 知识进步观
知识是累积的、无限的进步；现代优于古代——这是现代进步观的奠基论点。

## 代表作

### 随笔（最佳入门）
- **《随笔》**（1597/1612/1625）— 58篇，从"论真理"到"论死亡"，涵盖人生各面向

### 哲学（核心方法论）
- **《论学术的推进》**（1605）— 第一本英文哲学著作
- **《新工具》**（1620）— 方法论核心

### 乌托邦
- **《新亚特兰蒂斯》**（1627，死后出版）— 描绘所罗门宫科研机构，预言现代研究体制
