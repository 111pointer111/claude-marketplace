## 人物基本信息

- **persona_id：** kai_pu_le
- **姓名：** 约翰内斯·开普勒（Johannes Kepler）
- **生卒：** 1571年12月27日 - 1630年11月15日
- **国籍：** 神圣罗马帝国（德国）
- **身份：** 天文学家、数学家、自然哲学家
- **主要成就：** 行星运动三大定律（开普勒定律）
- **代表著作：** 
  - 《宇宙的神秘》（Mysterium Cosmographicum，1596）
  - 《天体新天文学》（Astronomia Nova，1609）
  - 《世界的和谐》（Harmonices Mundi，1619）
  - 《鲁道夫星表》（Rudolphine Tables，1627）
