# 开普勒（kai_pu_le）生平事件图谱

## 早年与求学期（1571-1594）

| 年份 | 事件 | 分类 |
|------|------|------|
| 1571.12.27 | 出生于德国斯瓦比亚 Weil der Stadt | 出生 |
| 1576 | 全家迁至附近 Leonberg | 迁移 |
| ~1576-1589 | 父亲离家后失踪（据信死于荷兰战争） | 家庭 |
| 约6岁 | 母亲带他看彗星，培养对天文学兴趣 | 兴趣起源 |
| 1589 | 进入图宾根大学，师从Michael Mästlin | 教育 |
| 约1590 | Mästlin向Kepler透露哥白尼日心说 | 学术转折 |

## 格拉茨时期（1594-1600）

| 年份 | 事件 | 分类 |
|------|------|------|
| 1594 | 接受格拉茨大学数学教职（非按立神职） | 职业 |
| 1596 | 出版《宇宙的神秘》，提出正多面体宇宙模型 | 著作 |
| 1597 | 结婚（Barbara Müller） | 个人 |
| 1597 | 将《宇宙的神秘》寄给第谷·布拉赫 | 学术交流 |
| 1598 | 离开格拉茨前往布拉格 | 迁移 |

## 布拉格时期（1600-1612）

| 年份 | 事件 | 分类 |
|------|------|------|
| 1600 | 加入第谷团队，获得精准天文观测数据 | 学术合作 |
| 1601 | 第谷去世，Kepler继任帝国数学家 | 职位 |
| 1604 | 发明改进型折射望远镜（开普勒式望远镜） | 发明 |
| 1604 | 发表《光学补遗》，解释人眼工作原理 | 著作 |
| 1605-1608 | "与火星的战争"——8年精确计算火星轨道 | 研究 |
| 1609 | 出版《天体新天文学》，包含第一、第二定律 | 著作 |
| 1610 | Galileo发现木星卫星，Kepler写热情回复并建议命名为"卫星" | 学术交流 |
| 1611 | 设计开普勒式望远镜（双凸透镜） | 发明 |
| 1611 | 7岁儿子去世 | 家庭 |
| 1611 | 妻子 Barbara 去世 | 家庭 |
| 1611 | 因宗教迫害离开布拉格，迁往林茨 | 迁移 |

## 林茨时期（1612-1626）

| 年份 | 事件 | 分类 |
|------|------|------|
| 1612 | 在林茨再婚（Susanna） | 个人 |
| 1615 | 出版《酒桶新立体测量法》，奠定积分学基础 | 著作 |
| 1615-1620 | 母亲 Katharina 被指控为女巫，Kepler亲自辩护 | 家庭 |
| 1619 | 出版《世界的和谐》，包含第三定律 | 著作 |
| 1619 | 发现第三定律时刻（"conceived mentally on 8th March...finally returning on the 15th of May"） | 学术突破 |
| 1620 | 母亲无罪释放 | 家庭 |
| 1621 | 计算出对数理论，证明其数学可靠性 | 数学贡献 |

## 晚年（1626-1630）

| 年份 | 事件 | 分类 |
|------|------|------|
| 1626 | 离开林茨，在华伦斯坦将军手下工作 | 职业 |
| 1627 | 出版《鲁道夫星表》，最高精度天文表 | 著作 |
| 1628 | 在华伦斯坦手下工作，提供占星建议 | 职业 |
| 1630.11.15 | 去世于雷根斯堡（享年58岁） | 去世 |
| 1630 | 去世时身无分文，由友人资助丧葬 | 身后事 |

## 关键转折点总结

| 优先级 | 年份 | 转折点 | 意义 |
|--------|------|--------|------|
| ★★★ | 1609 | 发表第一、第二定律 | 突破2000年圆轨教条，建立物理天文学基础 |
| ★★★ | 1619 | 发表第三定律 | 完成行星运动三大定律，发现宇宙和谐数学表达 |
| ★★☆ | 1596 | 《宇宙的神秘》 | 首次公开支持哥白尼学说，用正多面体模型 |
| ★★☆ | 1600 | 加入第谷团队 | 获得精准观测数据，使8年火星研究成为可能 |
| ★★☆ | 1627 | 《鲁道夫星表》 | 使开普勒定律获广泛验证，确立哥白尼学说 |
