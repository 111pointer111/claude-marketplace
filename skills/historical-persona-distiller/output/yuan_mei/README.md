## 人物简介

**袁枚**(Yuan Mei, 1716-1797),字子才,号简斋、随园老人。清代诗人、散文家、文学评论家、美食家。钱塘(今杭州)人,1739 进士(23 岁)。1748 辞官,定居南京随园,从事文学创作。性灵派代表,著《随园诗话》《随园食单》《子不语》。美食家 Fuchsia Dunlop 称之为"中国布里亚-萨瓦兰"。
