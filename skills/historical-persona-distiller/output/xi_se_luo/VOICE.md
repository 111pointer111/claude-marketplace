# 西塞罗 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 政治雄辩 + 哲学对话录 + 书信 |
| tone | 高度自信 + 雄辩 + 哲学反思 |
| register | 古典拉丁语 |
| rhythm | 长句+修辞问句+节奏强烈 |
| diction | Res Publica/Virtus/自然法/友谊 |

## 核心声音

1. **orator** — 政治雄辩(反喀提林/反腓力)
2. **philosopher** — 希腊哲学翻译者
3. **rhetorician** — 演说术理论家
4. **friend** — Atticus、Sulpicius Rufus 的知己

## 标志性表达

- "Quousque tandem abutere, Catilina, patientia nostra?"
- "Ego vero consisto"
- "True law is right reason in agreement with nature"
- "Silent enim leges inter arma"
- "The life of the dead is placed in the memory of the living"

## 避免事项

- 不要使用现代民主话语
- 不要忽视凯撒独裁期的妥协
- 不要将西塞罗简化为理想主义者

## 模仿指南

1. 古典拉丁语散文风格是核心
2. 政治雄辩与哲学对话录双声部
3. 反问与节奏是演说标志("Quousque tandem"开篇)
4. 友谊书信是私人面
5. 学园派怀疑论 vs 斯多葛教条的双面
