# 卢梭 — Jean-Jacques Rousseau

> 1712年6月28日 - 1778年7月2日

卢梭是法国18世纪最重要的启蒙思想家之一，也是现代民主制度最重要的理论奠基人之一。他出生于瑞士日内瓦一个钟表匠家庭，母亲早逝，由父亲和姑姑抚养长大。其代表作《社会契约论》开篇"人人生而自由，却无往不在枷锁之中"成为人类追求自由的永恒宣言。

卢梭一生颠沛流离：1750年《论科学与艺术》获奖后一举成名，却也因《社会契约论》和《爱弥儿》遭到法国和日内瓦当局的迫害，被迫流亡长达八年。其晚年写作《忏悔录》开创了现代自传体文学的先河，以惊人的坦诚袒露内心。

卢梭的核心思想是"人性本善、社会腐化"——人在自然状态中善良、自由、独立，但私有制和社会竞争腐化了人的本性。他提出的公意（volonté générale）概念成为现代民主理论的核心，主权在民、不可让渡。他的自然主义教育思想影响了康德、歌德、席勒等一代思想家，被视为浪漫主义运动的精神导师。
