# 卢梭生平事件图谱

## 早期（1712-1728）

| 年份 | 事件 |
|------|------|
| 1712 | 6月28日出生于日内瓦共和国，父亲依萨克·卢梭是钟表匠，母亲苏珊·贝尔纳 |
| 1712 | 母亲产后第九天去世，由姑姑抚养 |
| ~1722 | 父亲因纠纷逃往里昂避难 |
| 1722-1728 | 寄居舅舅家；在包塞学习古典语文、绘图、数学；在雕刻匠处当学徒 |
| 1728 | 从雕刻匠学徒家逃跑；在安纳西结识华伦夫人；改信天主教 |

## 青年漂泊期（1728-1749）

| 年份 | 事件 |
|------|------|
| 1728-1730 | 在都灵、洛桑等地做仆人；结识瓦瑟（戴莱丝）|
| 1731-1740 | 回到华伦夫人身边（尚贝里）；做土地测量工作；自学数学；阅读洛克、笛卡尔 |
| 1740 | 到里昂马布利神父家当家庭教师；结识孔狄亚克 |
| 1742 | 在巴黎科学院提出新音乐记谱法，被拒绝 |
| 1743 | 任威尼斯法国大使馆秘书；与蒙太居伯爵吵翻 |
| 1745 | 结识戴莱丝并与之同居；结识伏尔泰 |
| 1749 | 为《百科全书》撰写音乐部分；探望狱中狄德罗；读到第戎学院征文题目，顿悟"人性本善、社会腐化" |

## 启蒙高峰（1750-1766）

| 年份 | 事件 |
|------|------|
| 1750 | 《论科学与艺术》获第戎学院奖金，一举成名；开始与众多文人笔战 |
| 1752 | 歌剧《乡村卜师》成功上演；拒绝国王年金以保持独立 |
| 1754 | 赴日内瓦；恢复日内瓦公民权；重皈新教；起草《政治制度论》 |
| 1755 | 《论人类不平等的起源和基础》出版 |
| 1756 | 与瓦瑟迁往退隐庐（埃皮奈夫人房产）；享受乡村宁静生活 |
| 1757 | 因与乌德托夫人恋爱与狄德罗等人闹翻 |
| 1761 | 《新爱洛伊丝》出版，成为畅销书 |
| 1762 | 《社会契约论》和《爱弥儿》同年出版；遭到法国和日内瓦当局迫害；被迫流亡 |
| 1763 | 逃至普鲁士辖内纳沙泰尔；放弃日内瓦公民权 |
| 1764 | 出版《山中书简》；为科西嘉起草宪法 |
| 1765 | 《科西嘉宪法草案》出版；再次遭到驱逐 |
| 1766 | 在大卫·休谟带领下赴英国避难；与休谟发生冲突（被迫害妄想） |

## 晚年流亡与写作（1766-1778）

| 年份 | 事件 |
|------|------|
| 1766-1767 | 在英国写作《忏悔录》第一卷；被迫害妄想加剧 |
| 1767 | 匿名潜回法国；继续植物学研究 |
| 1768 | 与瓦瑟正式结婚 |
| 1770 | 重返巴黎；继续抄写乐谱维生 |
| 1771 | 在瑞典朗读《忏悔录》被禁止 |
| 1776 | 完成《对话录》；散步时被马车撞伤 |
| 1777 | 健康恶化，停止工作 |
| 1778 | 病逝于埃尔默农维尔庄园 |
| 1782 | 《忏悔录》上卷出版；同年《漫步遐想录》出版 |
| 1794 | 法国大革命期间灵柩迁葬巴黎先贤祠 |

## 关键关系

| 人物 | 关系 |
|------|------|
| 华伦夫人 | 早年保护人、朋友、情妇、精神上的母亲 |
| 狄德罗 | 启蒙运动同僚，后因论政分歧决裂 |
| 伏尔泰 | 论敌，曾用卢梭抛弃孩子事件攻击他 |
| 大卫·休谟 | 朋友，后因被迫害妄想决裂 |
| 瓦瑟（戴莱丝） | 伴侣，晚年结婚；为其生育5个孩子并送入育婴堂 |
| 康德 | 受其影响极大，读《爱弥儿》曾忘记散步习惯 |
| 罗伯斯庇尔 | 仰慕者，1778年曾拜访临终的卢梭 |
