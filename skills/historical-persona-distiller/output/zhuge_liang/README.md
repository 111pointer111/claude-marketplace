## 人物生平

诸葛亮出身琅琊望族，自幼父母双亡，随叔父辗转各地，后隐居荆州隆中。刘备三顾茅庐请其出山，诸葛亮以《隆中对》规划三分天下之计，此策成为此后数十年蜀汉的基本国策。

辅佐刘备取益州、建立蜀汉后，刘备去世，诸葛亮受托孤之重，独掌蜀汉大政。他依法治国，开诚心布公道，蜀汉政治清明；七擒七纵平定南中，五次北伐以攻为守，以弱抗强。最终病逝于五丈原，遗言令部下将其葬于汉中定军山。
