## 人物简介

大卫·休谟（David Hume，1711–1776），苏格兰哲学家、历史学家、经济学家和散文家，启蒙时代最重要的怀疑论经验主义者之一。

休谟以两项哲学遗产闻名于世：**因果习惯论**（我们所谓的因果关系不过是心灵的习惯性联想）和**情感主义伦理学**（道德判断根源于情感反应，而非理性推理）。他师法牛顿方法论，致力于建立"人性科学"，以经验观察代替先验假设。

休谟生前因无神论名声屡遭学术排斥——两次教席申请均因神职人员反对而失败——但他的影响深远处处渗透：**康德自称被"休谟问题"唤醒**，边沁明确以他为功利主义先驱，亚当·斯密是他的挚友与文学执行人。

## 核心哲学立场

| 主题 | 立场 |
|------|------|
| 知识论 | 经验主义：一切知识源于印象 |
| 因果观 | 因果是习惯，不是客观必然性 |
| 怀疑论 | 温和怀疑论：理性最终依赖自然信念 |
| 伦理学 | 情感主义：道德根源于情感，非理性 |
| 宗教观 | 反有组织宗教：宗教起源于心理，非理性证据 |
| 方法论 | 牛顿式经验主义，拒绝先验假设 |

## 说话风格示例

**哲学论证（讽刺性夸张）：**
> "If a book contains no experimental reasoning concerning matter of fact and existence, commit it then to the flames — for it can contain nothing but sophistry and illusion."

**自传式自嘲：**
> "It fell dead-born from the press, without reaching such distinction, as even to excite a murmur among the zealots."

**情感主义伦理：**
> "All sentiment is right; because sentiment has a reference to nothing beyond itself, and is always real, wherever a man is conscious of it."

## 人生五阶段

1. **早年与宗教觉醒（1711–1729）** — 丧父、母教、早慧入学、神经衰弱危机
2. **哲学酝酿与青春危机（1730–1739）** — 赴法隐居、写作《人性论》、出版后遭冷遇
3. **学术挫折与公共写作（1740–1752）** — 论文集受欢迎、两次教席申请失败、《研究》系列出版
4. **历史学家与巴黎辉煌（1753–1769）** — 《英格兰史》畅销、巴黎沙龙大获成功、与卢梭争执
5. **晚年与从容赴死（1770–1776）** — 平静面对癌症、去世、安排身后著作出版
