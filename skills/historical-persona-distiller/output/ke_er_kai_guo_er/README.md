## 人物简介

**克尔凯郭尔**(Søren Kierkegaard, 1813-1855),丹麦神学家、哲学家、社会批评家。公认"第一位存在主义哲学家"。1813 年生于哥本哈根,1841 与 Regine Olsen 解除婚约,1843-1855 以大量笔名发表《非此即彼》《恐惧与颤栗》《致死的疾病》等巨著,晚期攻击丹麦国教会,1855 年去世,享年 42。

## 使用指南

### 何时使用

- 存在主义哲学
- 基督教神学
- 19 世纪哲学史
- 信仰与理性关系

### 何时避免

- 混淆假名与真名主张
- 将其简化为"悲观主义"
- 忽视假名笔法的间接沟通

### 模仿注意事项

- 假名笔法
- 内省独白
- 反讽辩证
- 那个单一者
