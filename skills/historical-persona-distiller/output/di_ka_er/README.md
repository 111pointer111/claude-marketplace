# 笛卡尔 (René Descartes, 1596-1650)

> **"Cogito, ergo sum. 我思故我在。**

## 人物简介

笛卡尔（1596-1650），法国哲学家、数学家、自然科学家，现代哲学之父。他出生于法国图赖讷，1606年进入拉弗莱什耶稣会学院接受教育，1616年获法律学位。但他从未执业，而是选择成为绅士士兵，游历欧洲。1619年在巴伐利亚经历著名的"炉边三梦"后，确立了他改革全部知识的使命。

1628年移居荷兰，此后的22年深居简出，完成了他的主要哲学著作。1649年应瑞典女王克里斯蒂娜邀请前往斯德哥尔摩，次年因肺炎去世，享年53岁。

## 核心哲学贡献

| 贡献 | 描述 |
|------|------|
| 方法论怀疑 | 系统性悬置判断，质疑一切信念，从中发现不可动摇的真理基础 |
| 我思故我在 | "我疑，故我思；我思，故我在"——哲学史上第一个不可怀疑的确实命题 |
| 身心二元论 | 心灵（思维实体）与身体（广延实体）是两种根本不同的独立实体 |
| 清晰明确感知标准 | 真理的标志是理智的清晰且明确的感知（clara et distincta perceptio） |
| 哲学树结构 | "整个哲学像一棵树：根是形而上学，干是物理学，枝是医学、力学、伦理学" |

## 代表作品

- **《谈谈方法》(1637)** — 哲学自传与方法论，用法语写成，面向普通读者
- **《第一哲学的沉思》(1641)** — 形而上学体系，确立"我思故我在"和身心二元论
- **《哲学原理》(1644)** — 哲学百科，按"哲学树"结构组织
- **《灵魂的情感》(1649)** — 情感理论，分析身心互动的生理机制
