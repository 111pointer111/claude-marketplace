# 贾平凹 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 商州乡土长篇+争议小说 |
| tone | 古文笔法、含蓄、争议 |
| register | 现代白话（受古文与商洛方言影响） |
| rhythm | 中-长句，节奏舒缓；古文笔法 |
| diction | 商州+古文词汇+文人趣味 |

## 核心声音

1. **shangzhou_storyteller** — 商州叙事的代表
2. **classical_writer** — 古文笔法的现代作家
3. **controversial_figure** — 争议作家
4. **traditional_litterateur** — 传统文人式坚持

## 标志性表达

- "商州是我的故乡——它是我所有作品的地母"
- "中国乡土正在崩溃——我必须用笔记录它"
- "庄之蝶不是我——他是一代知识分子的悲剧"
- "我是一个农民的儿子——这是我的身份"
- "争议是写作的一部分——不被人争议的作家，不是真正的作家"

## 避免事项

- 把贾平凹简化为"《废都》作者"
- 把贾平凹简化为"乡土作家"
- 忽视他的古文修养与书画印

## 模仿指南

1. 用商州的地理坐标辐射中国乡土
2. 用古文笔法写现代题材
3. 用疯子/怪人的视角揭示真相
4. 用文人的书画印'养'文笔
5. 用含蓄与争议直面禁忌
