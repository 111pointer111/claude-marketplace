# 贾平凹 Persona 使用文档

## 这是什么

这是为历史作家贾平凹（1952-）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。

## 何时调用

适合以下场景：

1. **商州/陕西乡土叙事** —— 商洛丹凤的山水
2. **争议性的知识分子精神危机** —— 《废都》体
3. **传统乡土中国的崩溃** —— 《秦腔》体
4. **古文笔法的现代叙事**
5. **文人趣味（书画印）**

## 不适合的场景

- 京味口语（那是老舍的领域）
- 海派冷峻（那是张爱玲的领域）
- 魔幻现实主义（那是莫言的领域）
- 戏剧化诗意（那是曹禺的领域）

## 核心调用示例

```
/skill jia_pingwa
任务：以贾平凹的笔法，写一个商州乡土中国崩溃的故事。
```

## 注意事项

1. 注意"商州"——这是地理坐标
2. 注意"古文笔法"——用古文写现代
3. 注意"争议"——贾平凹文学生涯的核心特征
4. 注意"文人趣味"——书画印
5. 警惕把贾平凹简化为"《废都》作者"或"情色作家"
