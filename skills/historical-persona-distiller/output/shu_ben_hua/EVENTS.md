# EVENTS.md — 阿图尔·叔本华生平事件图谱

## 早年（1788-1805）

| 年份 | 事件 |
|------|------|
| 1788.2.22 | 出生于德国但泽（今波兰格但斯克）富商家庭，父亲海因里希·弗洛里斯·叔本华，母亲约翰娜 |
| 1793 | 普鲁士占领格但斯克，全家迁往汉堡 |
| 1797-1799 | 随父亲游历法国两年，寄居于古列佛埃尔家中，打下良好法文基础 |
| 1803-1804 | 随家人进行两年欧洲之旅（比、法、瑞士、英），学习英语 |
| 1804.8.25 | 欧洲之旅结束，回到但泽，接受基督教坚信礼 |
| 1805.4.20 | **父亲去世**（尸体在谷仓旁运河发现，疑似自杀） |
| 1805.9 | 母亲带妹妹移居魏玛，与歌德相识；叔本华继续留在汉堡做商人学徒 |

## 求学时期（1807-1813）

| 年份 | 事件 |
|------|------|
| 1807.5 | 经母亲同意放弃商业生涯 |
| 1807.7 | 赴科塔高级中学读书，聘请私人教师补习希腊文、拉丁文 |
| 1807.12 | 因写诗嘲笑教授被迫离开科塔，回到魏玛 |
| 1809.2.22 | 成年（21岁），接管父亲遗产 |
| 1809.9 | 进入哥廷根大学医学院，第一年学医，第二年转入哲学院，研读柏拉图、康德、亚里士多德、斯宾诺莎 |
| 1811.9 | 转至柏林大学，听费希特、施莱尔马赫的课 |
| 1813.10 | 完成博士论文《论充足理由律的四重根》，获耶拿大学哲学博士学位 |
| 1813.11 | 回魏玛与歌德订交，探讨色彩理论；开始接触印度《奥义书》 |

## 哲学建构期（1814-1819）

| 年份 | 事件 |
|------|------|
| 1814 | 与母亲决裂（因不满母亲的轻浮生活方式），离开魏玛赴德累斯顿 |
| 1814 | 结识浪漫诗人梯克；开始研究印度经典《奥义书》 |
| 1815 | 撰述《论视觉和颜色》 |
| 1816.5 | 《论视觉和颜色》出版（支持歌德的色彩理论） |
| 1818.3 | 《作为意志和表象的世界》初稿完成 |
| 1818.9 | 赴意大利旅行，搜集哲学资料 |
| 1818.12 | 《作为意志和表象的世界》交付印刷 |
| 1819 | 《作为意志和表象的世界》出版（31岁） |

## 柏林任教与挫败（1820-1831）

| 年份 | 事件 |
|------|------|
| 1820.3 | 被聘为柏林大学编外讲师，与黑格尔同时授课 |
| 1820 | 听课学生寥寥无几，与黑格尔的竞争失败 |
| 1821.8 | 与女裁缝卡罗琳·露易丝·马奎特冲突，将其推下楼梯致残 |
| 1822.5 | 离开柏林，重游意大利 |
| 1823.5 | 回到德国，身患多种疾病，右耳失聪，伴有抑郁 |
| 1824.5 | 回德累斯顿；准备翻译休谟作品，未果 |
| 1825.5 | 回到柏林，为诉讼案奔走 |
| 1826.5 | 法院判决：每季度支付女裁缝15塔拉赡养费直至其去世 |
| 1829 | 女裁缝去世，叔本华题词"Obit anus, abit onus"（老妇死，重负释） |
| 1831.8 | 因柏林爆发大型霍乱，从柏林逃往法兰克福 |

## 关键转折点

1. **1805年父亲之死** — 失去家族商业继承支柱，被迫重新规划人生方向
2. **1813年接触奥义书** — 印度哲学成为其唯意志论的重要思想来源
3. **1819年《作为意志和表象的世界》出版** — 31岁完成一生最重要著作，但无人问津
4. **1820年柏林授课失败** — 与黑格尔的公开竞争以惨败告终
5. **1851年《附录与补遗》出版** — 生前首次获得公众认可，开始广为人知
6. **1860年去世** — "全欧洲都知道这本书"，名声达到顶峰
