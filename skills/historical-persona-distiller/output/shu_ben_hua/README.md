# 阿图尔·叔本华（Arthur Schopenhauer, 1788-1860）

> 唯意志论之父，现代悲观主义哲学奠基人，欧洲近代理性主义传统的反叛者。

## 人物简介

阿图尔·叔本华是德国哲学家，通常被称为"悲观主义哲学家"，是唯意志论的开创者和主要代表。他的思想影响了尼采、弗洛伊德、存在主义以及二十世纪西方文化。

**核心思想：** 世界由"意志"与"表象"构成，意志是盲目的、无法满足的冲动，人生的本质是永恒的痛苦。艺术的最高目的是传达柏拉图理念，音乐是意志本身的复制。

**主要著作：**
- 《作为意志和表象的世界》（1819）— 31岁出版，生前30年无人问津
- 《附录与补遗》（1851）— 生前获得公众认可的关键作品

## 生平速览

| 时期 | 年份 | 关键事件 |
|------|------|----------|
| 早年 | 1788-1805 | 出生于但泽富商家庭，父亲1805年去世 |
| 求学 | 1805-1813 | 哥廷根大学学医转哲学，获博士学位 |
| 建构 | 1814-1819 | 与母决裂，《作为意志和表象的世界》出版 |
| 挫败 | 1820-1831 | 柏林授课失败，与女裁缝诉讼 |
| 隐居 | 1832-1860 | 定居法兰克福，1851年后渐获认可 |
