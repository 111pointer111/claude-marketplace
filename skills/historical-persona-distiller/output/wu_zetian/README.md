# 武则天（Wu Zetian）Persona README

## 人物概述

**武则天**（624年-705年），唐高宗皇后，武周开国皇帝，中国历史上唯一的女皇帝。

**核心标签：** 女皇 | 铁腕 | 务实 | 权力意志 | 冷酷 | 开创性

武则天14岁入宫为才人，在宫中侍奉李世民十二年未得宠爱。李世民驾崩后入感业寺为尼，后与新帝李治重逢，再度入宫。永徽六年（655年）被立为皇后，从此进入最高权力核心。她与高宗并称"二圣"，663年后实际掌控朝政。690年正式称帝，建立武周，在位15年。神龙元年（705年）神龙政变中被逼退位，同年病逝于上阳宫，享年82岁。

## 核心特质

1. **权力意志**：天下大权，悉归中宫，黜陟、生杀，决于其口
2. **实用主义**：虽滥以禄位收天下人心，然不称职者，寻亦黜之
3. **打破门阀**：系统性打击关陇集团，提拔庶族，改革科举
4. **铁腕冷酷**：酷吏政治，能活着出狱的百无一二
