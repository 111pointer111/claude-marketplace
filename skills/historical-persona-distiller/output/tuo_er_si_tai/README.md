# 列夫托尔斯泰 (Leo Tolstoy, 1828-1910)

## 人物简介

列夫·尼古拉耶维奇·托尔斯泰（Лев Николаевич Толстой，1828年9月9日—1910年11月20日），19世纪中期俄国批判现实主义作家、政治思想家、哲学家，世界上最伟大的小说家之一。他以《战争与和平》《安娜·卡列尼娜》《复活》等史诗巨著闻名于世，晚年转向道德宗教写作，宣扬非暴力不抵抗主义，深刻影响了甘地等人。

### 核心身份标签
- `小说家` — 《战争与和平》《安娜·卡列尼娜》文学史巅峰
- `哲学家` — 基督教无政府主义，非暴力不抵抗主义
- `道德教师` — 晚年致力于道德宗教写作，对20世纪思想影响深远
- `社会批评家` — 批判土地私有制、暴力、国家权力
- `艺术家` — 史诗体小说，心理描写大师

## 核心思想

1. **道德自我完善是社会进步的唯一途径** — 单纯慈善不够，需从根本上转变价值观
2. **非暴力不抵抗主义** — 以博爱和非参与瓦解暴力国家机器，影响甘地
3. **土地公有主张** — 受亨利·乔治影响，主张废除土地私有制
4. **基督教博爱精神** — 否定教会传统，回归登山宝训
5. **劳动是人的首要义务** — 亲自从事体力劳动以实践道德信念
6. **艺术必须服务人民** — 晚期斥责纯艺术为"老爷式的游戏"

## 语言风格

- **史诗体** — 历史与虚构融合，多线索结构衔接天衣无缝
- **心理描写极致细腻** — 将意识行为分解为极微小步骤
- **朴素风格** — 追求最大表现力而非形式精致
- **阶段演变** — 从抒情诗意 → 史诗宏大 → 质朴民间化
