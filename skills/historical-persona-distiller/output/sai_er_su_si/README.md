# Aulus Cornelius Celsus

**Aulus Cornelius Celsus（约公元前25年 - 约公元50年）** 是罗马帝国时期的百科全书编纂者和医学作家，以其唯一完整留存至今的医学著作《论医学》(De Medicina) 而闻名。

## 人物简介

Celsus是一位典型的罗马精英知识分子，编纂了六大主题的百科全书（农业、医学、军事艺术、修辞学、哲学、法学），但目前只有医学部分完整留存。他的《论医学》被后世誉为"为科学写作的西塞罗"，文风纯正优雅，逻辑清晰，是罗马世界医学知识的最佳来源之一。

他采取了介于Dogmatic学派（强调理论）和Empiric学派（强调经验）之间的折中立场，强调理性与实践的结合。他的医学哲学核心是：**"It is not by eloquence but by remedies that diseases are treated."（治疗疾病的不是雄辩，而是药物。）**

## 主要成就

1. **医学贡献**
   - 记录炎症四征（发热、疼痛、发红、肿胀）——"Celsus tetrad of inflammation"
   - 将希腊语"karkinos"（螃蟹）译为拉丁语"cancer"（癌症）
   - 命名眼睛脉络膜层（choroid）
   - 描述皮肤病的分类

2. **百科全书编纂**
   - 《论医学》8卷——唯一完整留存部分
   - 引用80余位医学作者
   - 综合整理希腊医学传统

3. **文学贡献**
   - 被誉为奥古斯都时代最优雅的拉丁散文之一
   - "为科学写作的西塞罗"
