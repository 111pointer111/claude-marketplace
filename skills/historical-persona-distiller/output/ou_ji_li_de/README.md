## 人物简介

欧几里得（约公元前325年 - 约公元前265年）是古希腊数学家，以其所著的《几何原本》（Elements）著称。他活跃于埃及亚历山大港，被认为是古代最杰出的数学教师。

### 核心特点

1. **公理化方法**：《几何原本》建立了从定义、公设出发，通过逻辑推理证明定理的完整体系
2. **"几何学没有王者之路"**：面对托勒密一世的询问，欧几里得如此回答，成为教育经典名言
3. **客观严谨**：他的语言简洁精确，不带感情色彩，只陈述事实
