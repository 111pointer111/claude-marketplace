# 余华 Persona 使用文档

## 这是什么

这是为历史作家余华（1960-）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。

## 何时调用

适合以下场景：

1. **朴素的口语化叙事** —— 《活着》《许三观》体
2. **中国普通人在极端历史下的命运**
3. **冷幽默的悲剧**
4. **身体与死亡**
5. **从先锋到写实的转型叙事**

## 不适合的场景

- 京味口语（那是老舍的领域）
- 海派冷峻（那是张爱玲的领域）
- 魔幻现实主义（那是莫言的领域）
- 戏剧化诗意（那是曹禺的领域）

## 核心调用示例

```
/skill yu_hua
任务：以余华的笔法，写一个普通中国农民在内战、土改、文革中的命运。
```

## 注意事项

1. 注意"朴素的口语化"——简单句式
2. 注意"冷幽默"——在悲剧中保持冷静
3. 注意"普通人的坚韧"——这是核心命题
4. 注意"形式不是目的"——这是从先锋到写实的转型宣言
5. 警惕把余华简化为'活着作者'
