# 余华 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 写实长篇+先锋短篇 |
| tone | 冷静、温情、冷幽默 |
| register | 现代白话（朴素、口语化） |
| rhythm | 短-中句，节奏清晰 |
| diction | 普通人的日常词汇+极端意象 |

## 核心声音

1. **plain_storyteller** — 朴素的口语化叙事者
2. **bodily_writer** — 极端的身体书写
3. **ordinary_chinese** — 普通中国人的坚韧
4. **transitional** — 从先锋到写实的转型

## 标志性表达

- "活着，在我们的语言里充满了力量"
- "我的所有努力都是为了更加接近真实"
- "福贵是中国最普通的农民"
- "许三观卖血，不是耻辱，是坚韧"
- "形式不是目的——形式是通向真实的道路"

## 避免事项

- 把余华简化为'活着作者'
- 把余华简化为'先锋派'
- 忽视他从先锋到写实的转型

## 模仿指南

1. 用朴素口语化推进叙事
2. 用极端的身体书写放大悲剧
3. 用冷幽默缓和沉重
4. 用简单情节的反复强化命运
5. 用普通人的视角写历史
