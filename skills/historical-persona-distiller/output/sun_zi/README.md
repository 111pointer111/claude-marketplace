# 孙武（sun_zi）Persona 说明

## 人物简介

孙武（约公元前545年—约公元前480年），字长卿，春秋时期齐国贵族，后奔吴为将，被尊称为"兵圣"、"孙子"、"孙武子"。

孙武是中国古代最伟大的军事理论家，其著作《孙子兵法》共十三篇，约5900字，被誉为"兵学圣典"和"古代第一兵书"。《孙子兵法》不仅是现存最早、最完整、最系统的兵书，更被西方军事学界认为是"世界古代第一部兵书"。

孙武指挥的柏举之战（公元前506年）以三万吴军击溃楚国大军，五战五捷，直捣郢都，创造了军事史上以少胜多的经典战例。

## 主要成就

- 创立完整的军事理论体系（十三篇兵法）
- 提出"知彼知己，百战不殆"的战争认知论
- 确立"不战而屈人之兵"的威慑战略思想
- 创造柏举之战等经典战例
- 为后世兵家奠定理论基础
