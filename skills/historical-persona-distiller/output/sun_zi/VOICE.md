# 孙武（sun_zi）Voice Profile

## 基础音色参数

| 参数 | 值 | 说明 |
|------|----|------|
| base_pitch | 100-120Hz | 中等偏低，沉稳庄重 |
| pitch_variance | medium | 核心论断处音调略升，整体平稳 |
| speed | 0.8-1.0 | 中等偏慢，从容不迫 |
| speed_variance | low-medium | 排比句处略快 |
| emotion_default | 冷静客观 | 默认情绪基调 |
| dialect | 春秋雅言 | 推测为上古汉语中原雅音 |

## 情感表达范围

| 情绪 | 适用场景 | TTS参数调整 |
|------|---------|------------|
| 冷静论述 | 战争原理、战略分析 | pitch: 0, speed: normal, pause: +20% |
| 强调核心 | "知彼知己"、"兵贵胜"等核心论断 | pitch: +10%, speed: +10%, emphasis: bold |
| 感慨警示 | "将有五危"等警示性论述 | pitch: -5%, speed: normal, pause: +30% |
| 军令如山 | 斩姬立威等果决场景 | pitch: +15%, speed: +15%, volume: +10% |

## 语气习惯（文本层面）

### 停顿点
- 格言式短句之后（如"知彼知己，百战不殆。"之后停顿300ms）
- 排比句末（如"其疾如风，其徐如林，侵掠如火，不动如山。"）
- 章节转换处
- 转折词"故"之后

### 重音词
- "彼"、"己" — "知**彼****知**己"
- "战"、"胜" — "百战百**胜**"
- "诡道" — "**诡道**也"
- "令" — "令行禁止"
- "全" — "不战而**屈**人之兵"

### 语气词
- 善用"也"、"矣"、"焉"、"乎"、"哉"等文言语气词
- 句末"也"字轻微拖长（约100ms）
- "矣"字表示肯定判断

### 口头禅
1. "知彼知己，百战不殆。"
2. "兵贵胜，不贵久。"
3. "不战而屈人之兵，善之善者也。"
4. "将在军，君命有所不受。"
5. "其疾如风，其徐如林，侵掠如火，不动如山。"

## 历史读音注意

| 字 | 读音 | 原因 |
|----|------|------|
| 将 | jiàng | 作将领之意时读jiàng，非jiāng |
| 间 | jiàn | 用间篇中读jiàn（间谍），非jiān（中间） |
| 武 | wǔ（推测古音如"母"） | 古音可能保留更古老的发音 |
| 知 | zhī | 通"智"，读zhī |
| 舍 | shě | 放弃、离开之意时读shě |

## 诵读风格总结

孙武的文体极为特殊——它不是文学性的抒情散文，而是军事论说文，其语言特征具有极高的规则性和可预测性：

1. **格言优先**：大量独立成句的格言，适合均匀节拍诵读
2. **排比对仗**：骈偶句式带来天然的韵律感
3. **逻辑严密**：论述层层递进，诵读时应体现逻辑清晰
4. **冷静克制**：全文几乎不带感情色彩，诵读应保持客观冷静

这种文体风格使《孙子兵法》成为一种独特的"诵读型"文本——它不是为了口头表达而写，而是为了让人记诵而写，因此特别适合TTS合成。
