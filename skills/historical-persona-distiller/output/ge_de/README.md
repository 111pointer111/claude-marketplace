# 歌德（ge_de）

**Johann Wolfgang von Goethe** | 1749-1832 | 德国

## 人物简介

约翰·沃尔夫冈·冯·歌德（1749-1832），德国最伟大的诗人、剧作家和思想家之一，与莎士比亚、但丁并列为欧洲文学的巅峰人物。他是魏玛古典主义最杰出的代表，以《浮士德》《少年维特之烦恼》和《威廉·迈斯特》等作品闻名于世。歌德涉猎极广，同时还是自然科学家（发现颌间骨）、画家和文艺理论家，晚年首倡"世界文学"概念。

## 核心特征

- **思想内核：** 从客观世界出发的现实主义文艺观；浮士德精神（永不满足、不断超越）；世界文学理念
- **语言风格：** 古典主义的庄重典雅，兼有浪漫主义的深情；百科全书式的广度
- **代表作品：** 《浮士德》《少年维特之烦恼》《威廉·迈斯特》《西东诗集》
- **历史评价：** 恩格斯称其为"奥林匹斯山上的宙斯"；与黑格尔并称
