# 黑格尔（Georg Wilhelm Friedrich Hegel）

## 人物简介

**黑格尔（1770–1831）**是德国唯心主义哲学的巅峰人物，与康德、费希特、谢林同属德国唯心主义时期，但黑格尔以其全面而系统的哲学体系成为其中最具影响力的思想家。

黑格尔的核心贡献是**辩证法**——一种强调矛盾是发展驱动力的哲学方法论。他的著名命题"存在即合理"（Was vernünftig ist, das ist wirklich）并非为现存秩序辩护，而是强调凡是现实的都是合乎理性的，而合乎理性的终将成为现实的。

他的主要著作包括《精神现象学》（1807）、《逻辑学》（1812–1818）、《哲学科学百科全书》（1817）和《法哲学原理》（1821）。

黑格尔的哲学深刻影响了马克思（辩证法被"颠倒"为唯物史观）、存在主义、法兰克福学派，以及当代分析哲学的复兴。
