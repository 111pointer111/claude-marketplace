# 黑格尔生平事件图谱

## 阶段一：青年时期——理想主义启蒙（1770–1800）

| 时间 | 年龄 | 事件 | 来源 |
|------|------|------|------|
| 1770年8月27日 | 0岁 | 出生于斯图加特（Württemberg，德国），父亲是税务官 | Britannica |
| 1788年 | 18岁 | 进入图宾根大学学习哲学与神学 | Stanford Encyclopedia |
| 1790年 | 20岁 | 图宾根大学毕业，获哲学学士学位 | Britannica |
| 1793年 | 23岁 | 离开图宾根，在伯尔尼担任家庭教师（3年），大量阅读吉本、孟德斯鸠 | Stanford Encyclopedia, Britannica |
| 1793–1796年 | 23–26岁 | 对正统神学感到不耐烦Certificate states he 'devoted himself vigorously to philosophy' but 'industry in theology was intermittent' | Britannica |
| 1796年 | 26岁 | 迁往法兰克福，重新与荷尔德林会合 | Stanford Encyclopedia |
| 1796–1800年 | 26–30岁 | 在法兰克福患忧郁症，通过更努力学习希腊哲学和现代历史政治来治愈 | Britannica |
| 约1800年 | 30岁 | 在荷尔德林和谢林影响下，兴趣转向康德批判哲学 | Stanford Encyclopedia |

## 核心关系图谱

```
家庭关系：
父亲 → 税务官（姓名未详）
母亲 → 从母亲处习得拉丁语基础

友谊关系：
荷尔德林（Friedrich Hölderlin） → 图宾根大学同学，后在法兰克福重聚
谢林（Friedrich von Schelling） → 图宾根大学同学，1803年友谊破裂

学术关系：
康德（Immanuel Kant） → 批判哲学继承与发展对象
费希特（J.G. Fichte） → 超越对象
马克思（Karl Marx） → 辩证法继承与"颠倒"
卢卡奇（Georg Lukács） → 批判理论继承
波普尔（Karl Popper） → 主要批评者
```
