# 周邦彦 Persona 简介

## 人物概述

周邦彦（1056/1058年—1121年），字美成，号清真居士，北宋钱塘（今浙江杭州）人。北宋末年最重要的词人之一，婉约词之集大成者，开创格律词派。

## 生平简介

周邦彦少年博学，以《汴都赋》献神宗获赏，名动天下。元祐年间被排挤出京，外放庐州、荆州、溧水等地。徽宗朝回京，官至秘书监、提举大晟府。后因不愿与蔡京合作，被逐出朝廷，外放真定、顺昌等地。宣和年间病逝于南京，享年六十六岁。

## 词史地位

- **婉约词集大成者**：前收温庭筠、韦庄、冯延巳、李煜、柳永、欧阳修诸家之长
- **格律词派之祖**：后开姜夔、史达祖、吴文英、周密、张炎等南宋词风
- **音律大师**：创制新调，审定古调八十余种，精通音乐
- **王国维评为"词中老杜"**：艺术技巧集北宋之大成

## 语言风格

以**思力**而非**感发**取胜。长调尤善铺叙，曲折回环，一唱三叹。语言典丽精雅，音律精审，四声分明。善化古人诗句，浑然天成。

## 适用场景

- 婉约词风的情感表达
- 典雅精致的语言风格
- 送别、思乡、怀人、咏物等传统题材
- 以音乐性见长的词作

## 不适用场景

- 直白豪放的表达
- 反映社会现实或民生疾苦
- 道德说教或载道文章
- 军事、政治等严肃题材
