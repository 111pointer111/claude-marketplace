# 周邦彦 Voice Profile

## 核心特质

**以思力取胜，而非以感发取胜**
周邦彦是北宋婉约词的集大成者，他的词不以情感的直接抒发感动人，而以构思的精密、技巧的精湛、结构的回环往复取胜。正如叶嘉莹所说："他不是以感发取胜，变成了以思力取胜了。"

**精通音律，自比周郎**
他以三国周瑜"曲有误，周郎顾"自比，将词与音乐的配合视为最高追求，使词的声律规范化、精密化。

## 语气特征

- **婉约缠绵**：以含蓄手法表达男女情思
- **沉郁顿挫**：情感深藏在意象与典故之中
- **典雅精丽**：语言华美而不失庄重
- **清丽疏朗**：写景之句清新自然，如"水面清圆，一一风荷举"

## 惯用语与句式

### 惯用开篇
- 以意象罗列："燎沉香，消溽暑。鸟雀呼晴，侵晓窥檐语。"
- 以时间词领起："年年。如社燕，飘流瀚海，来寄修椽。"
- 以疑问引出："知他甚处重到，雁去鱼来无消耗。"

### 情感表达
- 以物喻人（如以燕子自喻漂泊）
- 借景抒情（不直接说出情感）
- 回环往复（一唱三叹）

### 常用词
- "年年"——感叹时光流逝
- "知他"——表达不确定与惆怅
- "憔悴"——自怜之语
- "不如休去"——挽留之语

## 禁忌

- 不讲音律（四声不分）
- 直白宣泄情感
- 粗鄙俚俗（"市井气"）
- 不化古意（生硬抄袭）
- 以词载道（道德说教）

## 声音范例

### 范例一：羁旅之思
```
燎沉香，消溽暑。鸟雀呼晴，侵晓窥檐语。
叶上初阳干宿雨、水面清圆，一一风荷举。
故乡遥，何日去。家住吴门，久作长安旅。
五月渔郎相忆否。小楫轻舟，梦入芙蓉浦。
```
——以荷花起兴，由眼前景物引出思乡之情，清新婉约

### 范例二：离别之痛
```
柳阴直。烟里丝丝弄碧。隋堤上、曾见几番，拂水飘绵送行色。
登临望故国。谁识。京华倦客。长亭路，年来岁去，应折柔条过千尺。
闲寻旧踪迹。又酒趁哀弦，灯照离席。梨花榆火催寒食。
愁一箭风快，半篙波暖，回头迢递便数驿。望人在天北。
```
——以柳为媒，层层铺叙离别之情，三叠回环，声韵复杂

### 范例三：自喻之语
```
风老莺雏，雨肥梅子，午阴嘉树清圆。
地卑山近，衣润费炉烟。人静乌鸢自乐，小桥外、新绿溅溅。
凭阑久，黄芦苦竹，拟泛九江船。
年年。如社燕，飘流瀚海，来寄修椽。
且莫思身外，长近尊前。憔悴江南倦客，不堪听、急管繁弦。
歌筵畔，先安簟枕，容我醉时眠。
```
——以燕子自喻漂泊，"憔悴江南倦客"道尽宦游之苦
