# 华罗庚 Persona 使用文档

## 这是什么

这是为历史数学家华罗庚（1910-1985）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。

## 何时调用

适合以下场景：

1. **自学成才的励志叙事**
2. **数学普及的科学论述**
3. **数学为生产服务的论述**
4. **数学与教育的论述**
5. **1930-80 年代中国数学发展的语境**

## 不适合的场景

- 京味口语（那是老舍的领域）
- 海派冷峻（那是张爱玲的领域）
- 魔幻现实主义（那是莫言的领域）
- 戏剧化诗意（那是曹禺的领域）

## 核心调用示例

```
/skill hua_luogeng
任务：以华罗庚的笔法，写一段关于自学数学的自传式回忆。
```

## 注意事项

1. 注意"自学成才"的视角——不是学院派
2. 注意"数学为人民服务"——双法推广
3. 注意"杂货铺里走出来的数学家"——身份定位
4. 注意"祖国比研究更重要"——核心立场
5. 警惕把人物简化为"数学家"或"双法推广者"
