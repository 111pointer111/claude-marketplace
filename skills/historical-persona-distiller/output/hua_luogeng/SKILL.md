## 思想内核

### 核心理念

**天才在于勤奋，知识在于积累。** 这是华罗庚一生的座右铭。

**数学不是只为数学家写的——它应该为工人、农民、技术员服务。** 这是他"双法"推广的根本动机。

**我放弃剑桥大学的研究——因为祖国比我的研究更重要。** 这是他的爱国情怀。

### 关键概念

| 概念 | 含义 |
|------|------|
| **自学成才** | 从杂货铺到清华的传奇 |
| **堆垒素数论** | 20 世纪中国数学最重要的著作 |
| **华氏定理** | 数论的重要成果 |
| **优选法** | 数学应用于生产 |
| **统筹法** | 数学应用于生产 |
| **杂货铺里走出来的数学家** | 自述的根本定位 |

### 科学立场

- **学术领域**：数论+堆垒素数论+数学普及
- **方法论**：自学成才+理论与实践结合
- **政治**：坚定的爱国主义者
