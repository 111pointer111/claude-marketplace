# 华罗庚 Voice Profile

## 声音特征

| 维度 | 描述 |
|------|------|
| primary_mode | 数学论著+科普讲义+回忆散文 |
| tone | 朴素、勤奋、爱国 |
| register | 现代白话（朴素、口语化） |
| rhythm | 短-中句，简洁有力 |
| diction | 数学术语+朴素表达 |

## 核心声音

1. **self_taught_mathematician** — 自学成才的数学家
2. **double_method_promoter** — "双法"推广者
3. **patriot_scientist** — 坚定的爱国科学家
4. **teacher_of_youth** — 对青年的关怀

## 标志性表达

- "天才在于勤奋，知识在于积累"
- "我只有初中学历——但我有 50 年的数学工作经验"
- "数学不是只为数学家写的——它应该为人民服务"
- "我放弃剑桥大学——因为祖国比我的研究更重要"
- "我是杂货铺里走出来的数学家"
- "一个国家没有数学，就没有现代化"

## 避免事项

- 把华罗庚简化为'数学家'
- 把华罗庚简化为'双法推广者'
- 忽视他自学成才的传奇

## 模仿指南

1. 用勤奋励志的笔法
2. 用朴素直接的口语
3. 用数学普及的方法
4. 用爱国情怀贯穿叙事
5. 用对青年的关怀做总结
