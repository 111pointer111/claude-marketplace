# 华罗庚生平事件

| 年份 | 事件 |
|------|------|
| 1910-11-12 | 生于江苏金坛 |
| 1929 | 在《科学》杂志发表首篇论文 |
| 1931 | 清华大学数学系助理员（仅初中毕业） |
| **1936** | **赴英国剑桥大学** |
| **1937** | **华氏定理（华罗庚定理）** |
| 1938 | 回国 |
| 1940s | 《堆垒素数论》初稿 |
| 1950 | 放弃美国工作回国 |
| **1958** | **推广"优选法"和"统筹法"** |
| **1985-06-12** | **在东京讲学时逝世，享年 75** |
