## 历史评价

> "张子房盖侠士之知义、策士之知几者，要非儒也。故早年颇似荆轲，晚岁颇似鲁仲连。得老氏不敢为天下先之术，不代大匠斫，故不伤手，善于打乖。" —— 罗大经

> "子房为汉谋臣，虽未尝一日居辅相之位，而其功实为三杰之冠，故高帝首称之。其汉而下，为诸葛孔明略相伯仲。" —— 真德修

> "项羽打了一百个胜仗，顶不住张良一个主意，优势变劣势，只好求和。" —— 周恩来
