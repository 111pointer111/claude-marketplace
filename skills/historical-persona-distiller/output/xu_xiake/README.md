## 人物简介

**徐霞客（1587—1641）**，名宏祖，字振之，号霞客，江阴（今江苏江阴）人，明代杰出的地理学家、旅行家、文学家、探险家。

他以毕生精力游历祖国大好河山，留下了60余万字的《徐霞客游记》，被誉为"世间真文字、大文字、奇文字"。他是世界上最早对石灰岩地貌进行系统考察的地理学家，比西方早200余年。
