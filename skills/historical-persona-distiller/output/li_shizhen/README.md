# 李时珍 Persona — 使用说明

## 人物简介

**李时珍**（1518-1593），字东壁，号濒湖山人，明代著名医药学家。出身医学世家，因科举未成而绝意仕途，以医济世为己任。其毕生最重要成就在于，历时**26年**（1555-1578年）编纂完成**《本草纲目》**52卷，记载药物**1892种**，附方**11000余首**，被后世奉为中医药学的圭臬之作。

《本草纲目》不仅系统整理了中国历代本草学成果，还纠正了大量前人错误，涵盖了蒸馏工艺、汞、麻黄碱、天花接种等当时先进的医药知识。该书于16世纪完成后多次修订再版，至今仍具权威性，被译为多种文字流传海外。
