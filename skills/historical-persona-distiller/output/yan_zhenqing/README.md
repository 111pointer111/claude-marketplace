# 颜真卿 — 人物简介与使用说明

## 人物概述

**颜真卿**（709年—784年），字清臣，唐代名臣、书法家。封鲁郡公，人称"颜鲁公"，谥号"文忠"。

颜真卿出身颜氏家族（颜之推后裔），三岁丧父，由母亲殷夫人抚养成人。开元二十二年（734年）进士及第，历仕玄宗、肃宗、代宗、德宗四朝，官至吏部尚书、太子太师。

颜真卿以忠义名节著称于世。安史之乱时首唱大顺，任盟主，联络各郡起兵抵抗。晚年面对叛将李希烈的威逼利诱，宁死不屈，最终被缢杀于蔡州，年七十六岁。

颜真卿又为唐代最伟大的书法家之一，创"颜体"楷书，与柳公权并称"颜筋柳骨"。其行书《祭侄文稿》被誉为"天下第二行书"，《争座位帖》被誉为"天下行书第一"。

## 代表作品

- **书法**：《祭侄文稿》《争座位帖》《多宝塔碑》《颜勤礼碑》《颜氏家庙碑》
- **文章**：《论百官论事疏》《与郭仆射书》

## 历史评价

> 苏轼："诗至于杜子美，文至于韩退之，画至于吴道子，书至于颜鲁公，而古今之变，天下之能事尽矣。"

> 欧阳修："颜公书如忠臣烈士，道德君子，其端严尊重，人初见而畏之，然愈久而愈可爱也。"

> 文天祥："平原太守颜真卿，长安天子不知名……公死于今六百年，忠精赫赫雷当天。"
