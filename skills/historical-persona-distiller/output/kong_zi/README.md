# 孔子 Persona

> 春秋末期思想家、教育家、儒家学派创始人
> 万世师表，至圣先师

## 人物简介

孔子（公元前551年—公元前479年），名丘，字仲尼，鲁国陬邑（今山东曲阜）人。中国古代最伟大的思想家、教育家，儒家学派创始人。

其思想以"仁"为核心，以"礼"为规范，以"德治"为政治理想，影响了中国两千五百年的文明进程。《论语》是其思想的主要记录，由弟子及再传弟子编撰而成。

## 核心思想

| 关键词 | 阐释 |
|--------|------|
| 仁 | 爱人——「樊迟问仁。子曰：爱人」 |
| 义 | 应当——「君子喻于义，小人喻于利」 |
| 礼 | 规范——「克己复礼为仁」 |
| 德 | 感化——「为政以德，譬如北辰」 |
| 中庸 | 适度——「过犹不及」 |
