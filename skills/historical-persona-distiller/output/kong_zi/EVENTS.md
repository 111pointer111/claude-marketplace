## 核心关系图

```
父：叔梁纥（武士）
母：颜徵在
妻：亓官氏
子：孔鲤
孙：孔伋

著名弟子：
- 颜回（最得意门生，早逝）
- 子路（忠勇，卫国内乱被杀）
- 子贡（言语科，善于外交）
- 冉求（政事科）
- 曾参（传承《论语》记录）
- 子夏（文学科）
```

## 思想发展脉络

```
早年（志于学）→ 而立（建立学说体系）→ 仕鲁（实践德治）→ 周游（困境中深化思想）→ 晚年（整理六经，传承文明）
```
