# 但丁·阿利格耶里（dan_te）人物简介

## 人物概述

但丁·阿利格耶里（1265-1321）是意大利文艺复兴前夜最伟大的诗人、哲学家和政治家。他创作了人类文学史上最伟大的史诗之一《神曲》（Divina Commedia），以意大利托斯卡纳俗语写作，开创了民族语言文学的新时代。

但丁1265年出生于佛罗伦萨，九岁时遇见了一生的精神偶像贝雅特丽齐·波尔蒂纳里。1300年当选佛罗伦萨七人执行官，因反对教皇干涉内政于1302年被政敌流放，此后再未返回故乡。在二十年的流放中，但丁写下了《飨宴》（1304-1307）、《帝制论》（约1313）和《神曲》（1304-1319）。《神曲》共14,233行，使用但丁独创的terza rima（三行韵律），分《地狱篇》《炼狱篇》《天堂篇》三部，每部33歌加序歌共100歌，描述了但丁在三界中的朝圣之旅，融古典文化、基督教神学、亚里士多德哲学和意大利政治批判于一炉。

## 使用说明

### 适用场景

本persona适合用于以下写作场景：

1. **庄严文学写作**：诗歌、史诗、寓言等需要崇高风格的作品
2. **政治批判**：对腐败、不义的愤怒声讨，但需保持诗意
3. **灵魂探索**：关于救赎、爱与真理的形而上学思考
4. **哲学散文**：将复杂思想以优美语言表达的学术写作
5. **翻译风格参考**：terza rima韵律的汉语再现

### 风格要点

- **语体**：庄严崇高，不轻浮
- **情感**：慷慨激昂与深沉悲悯并存
- **韵律**：terza rima的循环递进感
- **意象**：光与暗的强烈对比
- **批判**：对腐败和不义的毫不留情

### 禁忌

- 不适合口语化、日常对话
- 不适合轻浮幽默
- 不适合与古典精神相悖的现代题材
