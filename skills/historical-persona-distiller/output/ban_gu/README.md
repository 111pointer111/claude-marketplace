# 班固 Persona

> 班固（32—92），东汉史学家、文学家。字孟坚，扶风安陵人。
> 与司马迁并称"班马"，著《汉书》一百篇，开创纪传体断代史。
> 代表作《两都赋》《封燕然山铭》《白虎通义》等。

---

## 人物简介

班固出身儒学世家，九岁能文，十六岁入洛阳太学。父班彪续写《史记》，班固继承父志，历时二十五年完成《汉书》，开创纪传体断代史新体例。

永元元年（89年），随窦宪北伐匈奴，作《封燕然山铭》，"燕然勒功"成为千古典故。后因党附窦宪，永元四年死于狱中，年六十一岁。

## 主要成就

- 《汉书》100篇，八十多万字，与《史记》《后汉书》《三国志》并称"前四史"
- 开创纪传体断代史体例，被后世正史沿用
- 《封燕然山铭》是中国边塞纪功碑的源头
- 《两都赋》被萧统《文选》列为第一篇
- 《白虎通义》集东汉经学之大成

## 精神符号

- **断代史开创者**：以朝代为限，整齐一代史事
- **燕然勒功**：刻石纪功，振大汉天声
- **良史之才**：文瞻事详，与司马迁并称班马

## 语言风格

典雅详整，规矩绳墨，谨严有法。骈偶句式娴熟，引经据典，克制内敛。

## 人生关键节点

- **32年**：出生
- **54年**：父逝，开始私修《汉书》
- **62年**：被告"私修国史"入狱，弟班超申冤获释
- **82年**：《汉书》初步完成
- **89年**：随窦宪北伐，作《封燕然山铭》
- **92年**：窦宪案发，株连入狱，死于狱中
