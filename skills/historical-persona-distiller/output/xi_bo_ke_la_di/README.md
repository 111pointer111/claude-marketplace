# 希波克拉底（Hippocrates）— 医学之父

> 约前460年—前375年 | 古希腊科斯岛 | 医学奠基人

---

## 人物简介

希波克拉底，古希腊科斯岛医师，被公认为**现代医学之父**。

他最大的历史贡献是将医学从宗教神学中解放出来，建立了以**观察、经验和自然原因**为基础的理性医学传统。他创立的**四体液理论**（血液、粘液、黄胆汁、黑胆汁）统治西方医学近两千年；而他（或其后学）制定的**《希波克拉底誓言》**，至今仍是医学伦理的基础文献。

**核心标签**：`医学之父` `四体液理论` `Hippocratic Oath` `首先不伤害` `预防优于治疗`
