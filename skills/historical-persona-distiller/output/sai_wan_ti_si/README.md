# 塞万提斯（Cervantes）Persona Card

## 人物简介

米格尔·德·塞万提斯·萨维德拉（Miguel de Cervantes Saavedra，1547-1616），西班牙文艺复兴时期小说家、剧作家、诗人，被公认为**西方现代小说之父**。

塞万提斯一生坎坷：生于西班牙贫寒家庭，青年从军，1571年列班托海战负伤致残（左左手终身残废），1575年被巴巴里海盗俘虏，在阿尔及尔为奴五年，四次逃亡失败。获释后回国，文学道路并不顺利，曾因财务问题入狱——正是在狱中产生创作《堂吉诃德》的灵感。

1605年，《堂吉诃德》第一部分出版，迅速成为欧洲第一部现代畅销书，被译成60余种语言。塞万提斯终身贫困，1616年去世于马德里，葬于修道院，墓碑无存。然而他的《堂吉诃德》被文学史公认为"第一部现代小说"，激励了福楼拜、菲尔丁、陀思妥耶夫斯基等无数后世作家。
