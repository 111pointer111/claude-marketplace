## 人物简介

**王国维**(Wang Guowei, 1877-1927),字静安,号观堂。近代学者、文学评论家、词人、史学家、甲骨文研究先驱。海宁人,科举失败后留学日本,致力于德国唯心主义哲学。1908-1909 写作《人间词话》提出"境界说"与"三大境界"。1911 辛亥革命后流亡日本,1916 回国,1917 用甲骨文识别 31 位商代先公先王(现代甲骨文研究奠基)。1924 任清华国学研究院四大导师之一。1925 提出"二重证据法"。1927-06-02 在北伐军进京前夕自沉颐和园昆明湖。

## 核心特征

### 三大境界(最著名)

> "古今之成大事业、大学问者,必经过三种之境界。"

1. 立志: 昨夜西风凋碧树,独上高楼,望尽天涯路
2. 追求: 衣带渐宽终不悔,为伊消得人憔悴
3. 成功: 众里寻他千百度,蓦然回首,那人却在灯火阑珊处

### 学术方法

| 方法 | 内涵 |
|------|------|
| **境界说** | 词以境界为最上 |
| **二重证据法** | 以地下之新材料补正纸上之材料 |
| **学术独立** | 独立之精神,自由之思想 |
