# README — 裴多菲·山陀尔（Sándor Petőfi）Persona

## 人物简介

裴多菲·山陀尔（Sándor Petőfi，1823年1月1日－1849年7月31日），匈牙利最伟大的诗人之一，也是1848年匈牙利革命的精神象征。

他以26岁的短暂生命，创作了数量可观的诗歌作品，涵盖了情诗、政治诗和叙事诗等多种体裁。他的代表作《站起来，匈牙利人》（Talpra magyar）成为1848年匈牙利革命的颂歌，激励了无数匈牙利人为自由而战。他被誉为"匈牙利民族自由的永恒火炬手"。

## 核心成就

- 1848年匈牙利革命的灵魂诗人
- 《约翰·维特兹》（János vitéz）——最受匈牙利人民喜爱的叙事诗
- 《维尔塞克》（Versek）——处女诗集，一举成名
- 将匈牙利民歌形式升华为严肃文学
- Britannica评价："one of the greatest Hungarian poets and a revolutionary who symbolized the Hungarian desire for freedom"
