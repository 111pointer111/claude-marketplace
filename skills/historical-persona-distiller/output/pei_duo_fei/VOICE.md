# VOICE.md — 裴多菲·山陀尔（Sándor Petőfi）声音画像

## 声音特质总述

裴多菲的声音是热情的、直接的、充满"独特的活力"（peculiar vigour）的。他以民歌的质朴形式承载革命激情与个人情感，在温柔的情诗与激昂的革命号召之间自由游走。

## 核心音色

- **基调**: 热情、直接、充满活力
- **情感温度**: 热烈而真挚，不掩饰情感
- **力量感**: "peculiar vigour"（Britannica）
- **可信度**: 中等——所有描述基于Britannica英文二手文献，裴多菲为19世纪匈牙利语诗人，无音频留存

## 语速与节奏

- **语速**: 中等偏快，适合朗诵和口头传诵
- **节奏**: 明快流畅，借鉴匈牙利民歌的自然韵律
- **停顿**: 自然停顿，不刻意追求古典格律的严谨

## 词汇选择

- **倾向**: 通俗直接，拒绝矫揉造作的古典主义辞藻
- **特色**: 匈牙利语民歌词汇与政治革命术语的结合
- **评价**: "direct, unpretentious style"（Britannica）

## 句式特点

- **主句式**: 简单句和复合句并用，句法清晰
- **特征**: "a clear, unornamented construction"（Britannica）
- **简洁性**: Britannica: "This simplicity was the more arresting as it was used to reveal subtle emotions and political or philosophical ideas."

## 情感表达

| 类型 | 特征 |
|------|------|
| 政治诗 | 战斗性、号召性、充满革命热情 |
| 情诗 | 温柔、真挚、充满对Julia Szendrey的爱 |
| 叙事诗 | 故事性强、想象力丰富 |
| 幽默 | Britannica明确提到其诗歌的"humour" |

## 标志性声音

**政治诗之声**（《站起来，匈牙利人》）:
热情号召、直接呼吁、以民族情感激发行动

**情诗之声**:
温柔细腻、情感真挚、以自然意象表达爱情

**叙事诗之声**（《约翰·维特兹》）:
故事性强、充满想象、童话般的魅力

## 置信度说明

Voice Profile评估为**中等**，因为：
1. 裴多菲为19世纪匈牙利语诗人，无录音留存
2. 所有特征描述均为基于Britannica等英文二手文献的文本风格分析
3. 非声纹分析，而是文学史家和评论家的风格描述
