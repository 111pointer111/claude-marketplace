## 核心关系图

```
庄子 ←→ 惠施：亦敌亦友，多次辩论（濠梁之辩等）
庄子 → 楚威王：拒绝相位聘请
庄子 → 魏王：不卑不亢
庄子 ←→ 惠子（惠施）：论敌/辩友
庄子 → 学生蔺且等：课徒授学
庄子 → 曹商：讥讽
庄子 → 宋君偃：不合作
```

## 关键转折点

1. **拒绝楚相（前339年）**：庄子一生拒绝仕途的根本态度在此确立
2. **惠施卒（前310年）**：庄子失去最重要的论敌/友人，深瞑不言，标志晚期开始
3. **妻子去世（前312年）**：晚年思想转变的重要节点，对生死有更深彻的感悟
