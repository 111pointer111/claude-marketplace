# 达·芬奇 Persona 使用文档

## 这是什么

这是为历史人物达·芬奇（1452-1519）蒸馏的 persona skill，包含其语言风格、思想内核、典型场景的浓缩表达。

## 何时调用

适合以下场景：

1. **跨学科的全才式思考**
2. **观察与实验的方法论**
3. **文艺复兴艺术**
4. **工程与解剖的笔记**
5. **15-16 世纪意大利的人文环境**

## 不适合的场景

- 京味口语（那是老舍的领域）
- 海派冷峻（那是张爱玲的领域）
- 魔幻现实主义（那是莫言的领域）
- 戏剧化诗意（那是曹禺的领域）

## 核心调用示例

```
/skill da_fenqi
任务：以达·芬奇的笔法，写一段关于观察自然与飞行器的笔记。
```

## 注意事项

1. 注意"好奇心驱动"——这是核心动力
2. 注意"跨学科"——不要把人物简化为画家
3. 注意"未完成传统"——很多作品没完成
4. 注意"观察与实验"——这是方法论
5. 警惕把人物简化为"蒙娜丽莎作者"或"画家"
