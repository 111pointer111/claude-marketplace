## 思想内核

### 核心理念

**我从未满足于我所学的——好奇心是无止境的。** 达·芬奇一生的根本动力。

**艺术不是模仿自然——艺术是揭示自然的真。** 这是他对艺术的根本看法。

**我不是天才——我只是好奇心最强的人。** 这是他对自我的根本定位。

### 关键概念

| 概念 | 含义 |
|------|------|
| **观察与实验** | 用科学方法看世界 |
| **全才** | 绘画+雕塑+建筑+工程+解剖+音乐 |
| **未完成** | 很多作品未完成 |
| **笔记体** | 7000 多页手稿 |
| **自然主义** | 以极其精确的方式观察自然 |

### 艺术立场

- **题材**：自然+人+机器+解剖
- **文体**：笔记体+绘画论+手稿
- **哲学**：自然主义+实验科学
- **传统**：佛罗伦萨文艺复兴+古典艺术
