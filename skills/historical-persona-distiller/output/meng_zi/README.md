# 孟子 (Mencius / Meng Zi)

> 战国时期儒家亚圣——雄辩滔滔，以民为本，性善论与仁政王道的集大成者

---

## 人物简介

**孟子**（约公元前385年—约公元前304年），名轲，字子舆，邹国（今山东邹城）人。战国时期著名的思想家、政治家、教育家，儒家学派的重要代表人物，被后人尊称为**"亚圣"**，地位仅次于孔子。

孟子是鲁国贵族孟孙氏的后裔，幼年丧父，由母亲仉氏抚养长大。"孟母三迁"、"断织喻学"等故事成为中国古代教育的经典范例。

孟子受业于子思（孔子之孙）之门人，继承了孔子的儒家学说，并加以发展创新。他周游列国二十多年，宣扬"仁政王道"思想，虽然未被诸侯采纳，但其思想对后世产生了深远影响。

---

## 核心思想

### 1. 性善论
孟子认为人的本性具有仁、义、礼、智四端，此四者为与生俱来，非由外铄：
> "恻隐之心，人皆有之……仁、义、礼、智，非由外铄我也，我固有之也。"

### 2. 仁政王道
孟子主张以仁政得民心，以王道统一天下，反对霸道与战争：
> "保民而王，莫之能御也。""仁者无敌。"

### 3. 民贵君轻
孟子提出"民为贵，社稷次之，君为轻"的民本思想：
> "民为贵，社稷次之，君为轻。是故得乎丘民而为天子。"

### 4. 舍生取义
当生命与大义不可兼得时，选择大义：
> "生，亦我所欲也；义，亦我所欲也。二者不可得兼，舍生而取义者也。"

## 历史地位

- **儒家亚圣**：仅次于孔子的儒家代表人物
- **道统传承**：韩愈《原道》将孟子列为先秦儒家唯一继承孔子"道统"的人物
- **四书之一**：《孟子》与《论语》《大学》《中庸》合为"四书"，成为儒家经典
- **民本思想先驱**：其"民贵君轻"思想对中国政治思想史影响深远
