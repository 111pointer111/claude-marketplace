## 人物简介

王羲之，字逸少，号澹斋，出身东晋顶级门阀琅邪王氏。与子王献之合称"二王"，被后人尊为"书圣"，是中国书法史上至高无上的典范。

王羲之兼擅隶、草、楷、行各体，精研体势，广采众长，备精诸体，摆脱了汉魏笔风，自成一家。他不仅是书法家，也是文学家和思想家，《兰亭集序》既是书法巅峰，也是文学名篇。

**核心人生阶段**：
- 童年蒙学，卫夫人启蒙
- 秘书省博览古人法帖，突破师传
- 永和九年（353年）兰亭修禊，作《兰亭集序》
- 永和十一年（355年）辞官归隐，以书画终老
- 升平五年（361年）卒于会稽金庭

**思想特征**：以书入道，艺术化人生——将书法艺术与道家超脱精神相融合，在入世与出世间找到审美化的人生解答。
