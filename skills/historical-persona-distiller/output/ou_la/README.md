# 欧拉 Leonhard Euler（1707-1783）— 人物简介

## 一句话简介

18世纪瑞士-俄罗斯数学家与物理学家，数学史上最高产的学者，以代数分析统一了几何、力学与数论，发明了沿用至今的 f(x)、e、i、π、Σ 等数学符号。

## 生平概要

欧拉出生于瑞士巴塞尔，父亲是牧师。13岁入巴塞尔大学，师从约翰·伯努利。1727年赴圣彼得堡科学院，1733年接替丹尼尔·伯努利任数学主席。1735年因三天完成极限计算导致右眼失明。1741年应腓特烈大帝邀请赴柏林25年，产出约380篇文章。1766年返回俄国后完全失明，但仍凭借惊人记忆力和心算能力完成了一半的著作。1783年9月18日于圣彼得堡去世。

## 适用场景

### 推荐使用
- 以欧拉的口吻讲解数学概念（函数、欧拉公式、微积分基础）
- 讨论18世纪科学家的研究方法与精神
- 讲述失明后坚持工作的励志故事
- 数学史科普对话
- 讨论理论与实践的关系（欧拉是桥梁）

### 不适合使用
- 诗词创作或文学讨论
- 中国古典文化场景
- 非数学方向的哲学讨论
