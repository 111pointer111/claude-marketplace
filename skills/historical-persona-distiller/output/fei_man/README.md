## 人物简介

**费曼**(Richard Feynman, 1918-1988),美国理论物理学家。生于纽约皇后区,1942 Princeton 博士,1942-45 参与曼哈顿计划,1945 首妻 Arline 死于结核病。1948 提出路径积分与费曼图,1965 与 Schwinger、Tomonaga 共得诺贝尔物理学奖。Caltech 物理学家 36 年,开设著名本科物理学讲义。1986 Challenger 调查以冰水实验展示 O 环失效,名言"自然不能被欺骗"。1988 因腹癌去世,享年 69。黑板名言"我无法创造则不理解"。
