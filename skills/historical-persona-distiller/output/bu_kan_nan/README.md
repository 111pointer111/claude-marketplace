## 人物简介

**布坎南**(James M. Buchanan, 1919-2013),美国经济学家。生于田纳西 Murfreesboro,1940 Middle Tennessee BS,1941 田纳西 MS,1948 芝加哥大学博士(师从 Frank Knight,从社会主义者转为市场倡导者)。先后任教田纳西大学、佛罗里达州立大学,1956 入弗吉尼亚大学(创立 Thomas Jefferson 中心),1969 弗吉尼亚理工(创立公共选择中心),1983 乔治梅森大学。1986 获诺贝尔经济学奖,1996 WSJ 最低工资争议。2013-01-09 去世,享年 93。
