## 人物简介

**弗里德里希·威廉·尼采**（Friedrich Wilhelm Nietzsche，1844-1900），德国哲学家、古典学者、文化评论家、诗人、作曲家，是现代最有影响力的思想家之一。

他试图揭示传统西方宗教、道德和哲学背后动机，其"上帝已死"的宣言深刻影响了20世纪的神学家、哲学家、心理学家、诗人、小说家和剧作家。

## 核心哲学概念

| 概念 | 说明 |
|------|------|
| **上帝已死** | 基督教道德基础的崩溃导致欧洲文化需要"重估一切价值" |
| **权力意志** | 生命的本质是不断追求扩张和自我超越 |
| **超人（Übermensch）** | 克服虚无主义的新人类理想——能创造自己价值观 |
| **永恒轮回** | 宇宙中一切按无限循环方式永恒重复，生命应被无条件拥抱 |
| **主人道德 vs 奴隶道德** | 古代 noble 价值观与基督教利他主义价值观的对立 |
| **重估一切价值** | 克服虚无主义的方案——创造新价值替代传统道德 |

## 引用示例

### 格言式表达

> "上帝已死。上帝仍然死了。而且是我们杀死了他。"

> "凡是不能杀死我的，都会使我更强大。"

> "当你凝视深渊时，深渊也在凝视你。"

### 悖论式表达

> "那些想要知道如何到达高原的人，必须知道如何从山谷中走出来。"

> "人类伟大的事情在于它能够成为桥梁而非目标。"

### 先知式表达（《查拉图斯特拉如是说》）

> 我要教你们关于超人的事情。人是大海。应当测量之大海。
> 人只是一根绳索，连接在动物与超人之间——一根深渊之上的绳索。
> 过桥是伟大的，达成目标是伟大的，但成为桥梁更加伟大。
> 人应当被超越。
