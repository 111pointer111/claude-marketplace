## 学术崛起（1869-1876）

| 年份 | 事件 | 影响 |
|------|------|------|
| 1869年 | 24岁任巴塞尔大学古典文献学副教授 | 史上最年轻的古典文献学教授 |
| 1869年 | 莱比锡大学授予博士学位（免试） | 学术地位确立 |
| 1870年 | 志愿医护人员参加普法战争，感染痢疾和白喉 | 健康严重受损，永久性影响 |
| 1872年 | 出版《悲剧的诞生》 | 从古典学术转向哲学，争议性的第一部著作 |
| 1872年 | Ulrich Wilamovitz-Möllendorff 激烈批评《悲剧的诞生》 | 古典学界排斥，但含深刻洞见 |
| 1875年 | 与瓦格纳友谊达到顶峰 | 共同的文化理想 |
| 1876年 | 瓦格纳转向基督教主题，尼采开始疏远 | 友谊出现裂缝 |
| 1876年 | 健康恶化，请假离开巴塞尔 | 健康问题开始影响学术生活 |
