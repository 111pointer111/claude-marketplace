## 人物简介

辛弃疾字幼安，号稼轩居士，山东济南人，南宋著名爱国词人、将领。他出生于金占区，22岁聚众两千参加抗金起义，曾在五万金兵中生擒叛徒张安国献俘南宋朝廷，一时震动朝野。然而"归正人"的尴尬身份使他长期仕途蹭蹬，壮志难酬。他在上饶带湖闲居22年，以词作抒发满腔报国之志，兼写田园词、山水词。

辛弃疾存词600余首，为两宋存词最多的作家。其词以豪放为主，兼具婉约，创造性地"以文为词"，打破了词的传统形式束缚。与苏轼并称"苏辛"，与李清照并称"济南二安"。
