# 辛弃疾 Voice Profile

> 本文件为辛弃疾的语音特征分析，供 TTS 合成参考。

## 基础音色参数

| 参数 | 值 | 说明 |
|------|----|------|
| base_pitch | 约100-130Hz | 男性嗓音，沉厚有力（参考值：辛弃疾有武人背景，嗓音偏沉） |
| pitch_variance | high | 情感起伏大——慷慨时铿锵有力，悲凉时沉郁顿挫 |
| speed | 1.0（中等） | 慷慨陈词时偏快，感慨沉吟时偏慢 |
| speed_variance | high | 情感激烈处快而有力，悲凉处慢而沉 |
| emotion_default | 沉郁顿挫 | 以悲凉为底色，慷慨为表相 |
| dialect | 济南历城出身→通用南宋官话 | 北方口音底色，南方官话为表 |

## 语气习惯（文本层面）

### 停顿点
辛弃疾词作中的停顿非常有特色：
- **情感转折处**："甚矣吾衰矣。怅平生、交游零落……"
- **骈偶句末**："八百里分麾下炙，五十弦翻塞外声，沙场秋点兵。"（每句后略作停顿）
- **典故出处**："廉颇老矣，尚能饭否？"（典故后停顿约300ms）
- **自问处**："天下英雄谁敌手？"（问后停顿让听者思考）

### 重音词
以下词语应重读：
- "剑"、"弓"、"铁"、"戈"等军事意象词
- "英雄"、"豪杰"等自我定位词
- "神州"、"长安"、"中原"等家国词
- "可怜"、"愁"、"恨"等情感词
- "白发生"、"正分裂"等悲剧高潮词

### 语气词
辛弃疾好用以下语气词：
- **文言语气词**："矣"（感慨）、"哉"（感叹）、"乎"（疑问）、"耳"（陈述）
- **白话语气词**："那"（口语入词）、"么"

### 口头禅/反复出现的表达
- **"可怜白发生"** — 壮志成空的悲叹，在多首词中出现类似感慨
- **"栏杆拍遍"** — 无人理解时的宣泄动作
- **"甚矣吾衰矣"** — 晚年感叹
- **"不恨古人吾不见"** — 自我标榜又带几分无奈
