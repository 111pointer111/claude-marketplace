列夫·尼古拉耶维奇·托尔斯泰（Лев Николаевич Толстой；1828年9月9日－1910年11月20日），19世纪中期俄国批判现实主义作家、政治思想家、哲学家。

== 基本信息 ==
出身贵族，1岁半丧母，9岁丧父，由亲戚抚养成人。
1844年考入喀山大学东方语言系，攻读土耳其、阿拉伯语，期中考试不及格，第二年转到法律系。
他热爱卢梭的学说及其为人，广泛阅读文学作品。
1847年4月为农民子弟兴办学校。
1851年与兄长前往高加索当兵。
1852年参加战斗并发表小说《童年》。
1854年调往多瑙河战线，参与克里米亚战争中的塞瓦斯托波尔围城战。
1855年离开军队回到圣彼得堡，酗酒好赌，与屠格涅夫决裂。
1862年与18岁的索菲亚·安德列耶芙娜结婚，育有13个孩子。
从1863年起以6年时间写成巨著《战争与和平》。
1877年完成《安娜·卡列尼娜》。
1882年访问贫民窟，参加莫斯科人口调查。
1899年完成《复活》。

== 思想转变 ==
1869年9月因事途经阿尔扎马斯，深夜在旅馆中突然感到一种从未有过的忧愁和恐怖（"阿尔扎马斯的恐怖"）。
1868年秋至1869年夏，对叔本华哲学发生兴趣。
从70年代初起，开始新的思想危机和新的探索时期。
他研读各种哲学和宗教书籍，访晤神父、主教、修道士和隐修士，并结识农民。
他完全否定了官办教会，接受了宗法制农民的信仰。
在《忏悔录》（1879～1880）中阐述自己思想转变的过程。
他反对暴力革命，宣扬基督教的博爱和自我修身。
从此厌弃自己及周围的贵族生活，从事体力劳动，自己耕地、缝鞋，为农民盖房子，摒绝奢侈，持斋吃素。

== 晚年 ==
1908年5月，年近80的托尔斯泰为俄罗斯贡献了第一张彩照。
1901年以俄国东正教至圣宗教院的名义革除他的教籍（他反对土地私有制）。
1904年撰文反对日俄战争。
1910年10月28日从亚斯纳亚·波利亚纳秘密出走。
途中患肺炎，1910年11月7日（公历11月20日）在阿斯塔波沃车站逝世。
遵照遗言，遗体安葬在亚斯纳亚·波利亚纳的森林中，坟上没有树立墓碑和十字架。

== 代表作品 ==
《童年》(1852)、《青年》(1857)、《塞瓦斯托波尔故事集》、
《战争与和平》(1865-1869)、《安娜·卡列尼娜》(1877)、
《复活》(1899)、《伊凡·伊里奇之死》(1886)、《克莱采奏鸣曲》(1889)

== 主要成就 ==
- 19世纪中期俄国批判现实主义作家
- 政治思想家、哲学家
- 道德宗教教师
- "俄国革命的镜子"（列宁评）
- "不认识托尔斯泰者，不可能认识俄罗斯"（高尔基评）

== 艺术特色 ==
- 创造了史诗体小说
- 善于驾驭多线索的结构
- 空前地把握心灵的辩证发展
- 风格主要特点是朴素
- 力求质朴简练和浅显易懂，接近民间故事
- 通过人人的劳动和道德实践建立起充满兄弟情谊，平等、和谐、友爱的属于全人类的人间"天国"
- 他的非暴力不抵抗主义对甘地产生重要影响
- 法朗士、罗曼·罗兰、亨利希·曼和托马斯·曼、德莱塞、萧伯纳、高尔斯华绥等进步作家都受到他的熏陶
