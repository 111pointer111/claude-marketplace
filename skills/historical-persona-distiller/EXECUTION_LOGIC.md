> 本文件是流水线的核心逻辑。OpenClaw 按此文件的步骤逐一执行。

## 预检查

```
## 预检查 — 每次执行前必做

1. 读取 DONE.md，找到「下一待处理」中的人物名（记为 TARGET）
2. 按 NAMING.md 规则，将 TARGET 转换为 persona_id（记为 PID）
   - 例：苏轼 → su_shi
   - 复姓：欧阳修 → ouyang_xiu
3. 检查 done/{PID}.done 是否存在：
   - 存在 → 跳过此人，重新从 backlog.md 取下一个 priority 最高的人物
   - 不存在 → 继续执行
4. 从 backlog.md 读取 TARGET 的完整条目：
   - priority
   - dynasty
   - lifespan
   - sources 列表（每个含 url / type / description）
5. 记录开始时间

6. 确定人物类型（用于 Stage 1.1 的回退链选择）

   根据 backlog.md 中的「朝代」和「备注」判断 TARGET 的人物类型：

   | 类型 | 判断规则 | 示例 |
   |------|---------|------|
   | chinese_poet | 朝代为中国朝代 + 备注含"诗人/词人/文学/诗/词"等 | 苏轼、李白、李清照 |
   | chinese_philosopher | 朝代为中国朝代 + 备注含"儒家/道家/法家/理学/心学/哲学" | 孔子、朱熹、王阳明 |
   | chinese_scientist | 朝代为中国朝代 + 备注含"科技/医学/数学/工程/天文/地理/发明" | 张仲景、沈括、蔡伦 |
   | chinese_emperor | 朝代为中国朝代 + 备注含"帝王/皇帝/统帅/军事/宰相/将" | 秦始皇、李世民、诸葛亮 |
   | chinese_artist | 朝代为中国朝代 + 备注含"书法/绘画/书家/画家" | 王羲之、颜真卿 |
   | western_philosopher | 朝代含"外国/西方/欧洲/古希腊/德国/法国/英国" + 备注含"哲学" | 康德、尼采、柏拉图 |
   | western_writer | 朝代含"外国/西方" + 备注含"文学/小说/戏剧/诗人" | 莎士比亚、雨果、但丁 |
   | western_scientist | 朝代含"外国/西方" + 备注含"科学/数学/物理/化学/天文/生物" | 牛顿、爱因斯坦、达尔文 |

   如果无法明确归类，标注 general 类型，按通用流程处理。
```

## Stage 1 — 资料采集

### Step 1.1：抓取原始文本

```markdown
## Stage 1.1 — 资料采集（含分类回退链）

### 第一步：从 backlog.md 的 sources 抓取

对 TARGET 的每个 source URL，按类型执行：

poetry_collection / collected_works / personal_essay →
    WebFetch 抓取全文，保存到 raw/{PID}/{type}_{序号}.txt
    标注每首诗/每段文字的写作年份（如有）

historical_record →
    WebFetch 抓取传记正文，保存到 raw/{PID}/传记.txt
    从正文提取所有引号包围的"直接引语"，保存到 raw/{PID}/引语.txt

commentary →
    WebFetch 抓取，保存到 raw/{PID}/后世评述.txt

### 第二步：分类回退链补充

根据预检查中确定的 TARGET 人物类型，按优先级依次尝试以下回退源。
每类源只需成功抓取 1-2 个即可，不必全部抓取。
成功标准：WebFetch 返回有效正文内容（非空、非 404、非验证页面）。

#### 🔗 中国诗人（chinese_poet）
按顺序尝试以下策略，任一成功即停止：
1. **百度百科传记** → baike.baidu.com/item/{人物名}
   → 如 URL 返回 404，尝试不带路径参数的重定向
   → 保存为 raw/{PID}/传记.txt，同时提取引语
2. **百度汉语/古诗文网 诗词** → 搜索"{人物名} 诗词 全文" 取 gushiwen.cn 或 baike.baidu.com 的诗词条目
   → 保存为 raw/{PID}/诗词_1.txt
3. **补充搜索** → SerpAPI: "{人物名} 诗词全集"（如 SerpAPI 不可用则 WebSearch）
   → 从结果中取前 2 个有效诗歌来源
4. **最终降级** → 如果以上均失败，以百度百科传记为主要语料，标注"诗词语料有限，以传记和评述为主"

#### 🔗 中国哲学家（chinese_philosopher）
1. **百度百科传记** → baike.baidu.com/item/{人物名}
2. **核心著作原文** → 搜索"{人物名} {核心著作名} 原文"（如"论语 原文"、"传习录 原文"）
   → 优先 ctext.org 或 baike.baidu.com 的著作词条
   → 保存为 raw/{PID}/著作_原文.txt
3. **Stanford Encyclopedia** → plato.stanford.edu（如有英文哲学条目）
4. **最终降级** → 以传记和后世评述为主要语料，标注"一手哲学著作原文有限"

#### 🔗 中国科学家/帝王/将相（chinese_scientist / chinese_emperor / chinese_artist）
1. **百度百科传记** → baike.baidu.com/item/{人物名}
2. **补充来源** → Britannica 或 World History Encyclopedia（如有英文条目）
   → 或 SerpAPI 搜索补充
3. **无需诗词语料**，以传记和后世评述为主要语料

#### 🔗 西方哲学家（western_philosopher）
1. **Stanford Encyclopedia of Philosophy** → plato.stanford.edu/entries/{英文名}/
   → 学术文献，含生平+核心思想+作品分析，50000字截取
2. **Britannica** → britannica.com/biography/{英文名}
3. **World History Encyclopedia** → worldhistory.org/{英文名}/
4. **百度百科** → baike.baidu.com/item/{中文名}（如有中文条目）
5. **无需诗词语料**

#### 🔗 西方文学人物（western_writer）
1. **World History Encyclopedia** → worldhistory.org/{英文名}/
2. **Britannica** → britannica.com/biography/{英文名}
3. **SerpAPI** 搜索补充作品分析、引语和后世评价
4. **百度百科** → baike.baidu.com/item/{中文名}（如有）
5. **无需诗词语料**，核心为传记+作品分析+引语

#### 🔗 西方科学家（western_scientist）
1. **Britannica** → britannica.com/biography/{英文名}
2. **Stanford Encyclopedia** → plato.stanford.edu（如有哲学/科学史条目）
3. **World History Encyclopedia** → worldhistory.org/{英文名}/
4. **无需诗词语料**

#### 🔗 通用流程（general — 无法归类时）
1. 先用 WebSearch 搜索 "{人物名} 传记" 取前 3 个有效结果
2. 再用 SerpAPI 搜索 "{人物名} 作品 全集" 补充
3. 继续执行标准的语料合并流程

### 第三步：语料完整性检查

所有来源抓取完毕后，执行以下检查：

1. **传记检查**：是否有 raw/{PID}/传记.txt？
   - 否 → 尝试搜索"{人物名} 生平"补充，如仍失败则标注"无系统传记，以零散语料为主"
2. **语料总量检查**：全部语料合计是否 ≥ 3000 字？
   - 否 → 标记 WARN，数据可能不足，但不阻断流程
3. **引语检查**：是否提取到直接引语？
   - 否 → 搜索"{人物名} 名言 语录"补充，保存到 raw/{PID}/引语.txt
4. **写作年份标注**：诗词/文集中尽可能标注年份，无法确定年份的统一标注"年份未详"

### 第四步：合并语料

将所有 txt 文件内容合并为 raw/{PID}/全部语料.txt。
（引语.txt 和 后世评述.txt 单独保留，不合并到全部语料中）
```

```markdown

## 核心要求

1. **跨越性**：思想内核必须是跨越其一生所有阶段都保持不变的核心
2. **原文佐证**：每条结论必须附 2-3 处原文引用，引用必须来自输入语料中的原句
3. **引用格式**：[原文] ——《{篇名}》
4. **诚实标注**：如果某方面语料不足，据实标注"语料有限，此为推测"

- {PID} 与同时代其他思想者/文人相比，最大的思想差异是什么？
- 他的思想在当时代是主流、异端、还是超前？
- 例如：苏轼的豁达与同时代贬谪文人（如柳宗元）的悲苦形成鲜明对比

### 2. 思想与实践的一致性
- 此人的言论主张与实际行为是否一致？
- 如果有矛盾：是"时代局限"还是"人格矛盾"？请明确判断
- 例如：白居易写《卖炭翁》讽喻时政，自己却蓄妓享乐

- 哪些价值观/立场是真正贯穿一生的核心？
- 哪些只是特定时期的侧重或权宜之计？
- 例如：杜甫的"忧国忧民"贯穿一生，但"致君尧舜"的期望在晚年明显减弱

### 4. 思想的内部张力
- 此人的思想体系中是否存在内在矛盾或张力？
- 这种张力是弱点，还是其思想深度的来源？
- 例如：陶渊明的"归隐"与"刑天舞干戚"的慷慨之间构成内在张力

请输出 JSON 格式：

{

      "描述": "{详细说明}",

      "实践一致性": "一致 / 部分矛盾 / 明显矛盾（附说明）"
    }
  ],
  "政治立场": {
    "描述": "{描述此人对皇权/改革/民生的综合态度}",
    "原文引用": ["[引用1]", "[引用2]"],
    "时代坐标": "{在当时代的独特之处}",
    "内在张力": "{如有矛盾，说明}",
    "confidence": "high/medium/low"
  },
  "底线清单": [
    {

  }
}

请开始分析。
```
