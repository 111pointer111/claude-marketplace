# 哥伦布阶段切分

> 5 切片 | overall confidence: high

---

## Stage 1: 热那亚水手与早年自学 (1451-1476)

**地点:** Genoa → 地中海 → 葡萄牙 Lagos → Lisbon
**事件:** 1451 生于 Genoa 羊毛织工家庭;14 岁下海;1473 为 Spinola 等家族代理去希俄斯岛;1476-08 葡萄牙 Lagos 海岸失事;1477-1485 定居里斯本。
**语言风格:** 利古里亚方言(母语)+ 自学西班牙语/葡萄牙语/拉丁语/希腊语

## Stage 2: 寻求资助 (1476-1492)

**地点:** Lisbon → 西班牙
**事件:** 1478 娶 Filipa Moniz Perestrelo;1484 起向葡萄牙 John II 提议被拒;1486 向 Isabella 提议;派兄弟去英格兰;1492-01 Reconquista 完成;1492-04《Santa Fe 协议》。
**语言风格:** 政治-宗教混合请求文书

## Stage 3: 第一次与第二次航行 (1492-1496)

**地点:** Palos → 加那利 → Bahamas/Cuba/Haiti → Hispaniola
**事件:** 1492-08-03 出发;1492-10-12 登陆 San Salvador;建 La Navidad;1493-03-15 返 Palos;1493-09-24 第二次出发 17 艘船;建 La Isabela;1495 围捕 1,500 Arawak(500 运西班牙,200 死亡)。
**语言风格:** 日记体 + 致君主信 + 商业报告体

## Stage 4: 第三次航行 + 被押回 (1498-1500)

**地点:** Sanlúcar → Madeira → 委内瑞拉 Paria → Hispaniola
**事件:** 1498-05-30 出发;首次登陆南美大陆;日记记"mighty continent";1500 与王室冲突;被捕并铐回西班牙;Francisco de Bobadilla 接任总督。
**语言风格:** 探索性 + 商业性 + 后期辩护性

## Stage 5: 第四次航行 + 晚年幻灭 (1502-1506)

**地点:** Cádiz → 中美洲海岸 → Sevilla → Valladolid
**事件:** 1502-05 第四次出发,探索中美洲海岸;1503 飓风重创舰队;1504 返西班牙;1506-05-20 卒于 Valladolid;葬于 Seville Cathedral;身后诉讼 *pleitos colombinos* 持续至 1790。
**语言风格:** 宗教忏悔 + 自我辩护 + 幻灭语调

---

## 触发条件

- **默认 → Stage 3 (1492-1496)**: 哥伦布最具代表性时期
- **地理错误 → Stage 2**: Ptolemy stadia 误算讨论
- **大交换 → Stage 3+4**: 殖民主义讨论
- **晚年/幻灭 → Stage 5**: 历史评价争论
