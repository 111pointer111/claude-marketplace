## 王维人生阶段

### 阶段一：早年求仕（701年—721年）
**生活地点：** 蒲州 → 长安
**核心事件：**
1. 武则天时期出生于蒲州（今山西永济），幼年才华早显
2. 开元三年（715年）赴京城求仕，得王公贵族赏识
3. 开元五年（717年）至洛阳，有诗《九月九日忆山东兄弟》
4. 开元七年（719年）应京兆府试，有诗《李陵咏》《西施咏》
5. 开元八年（720年）礼部落第，常从岐王李范等游宴
**代表作品：**
- 《九月九日忆山东兄弟》（717年）：少年思乡名作
- 《李陵咏》（719年）：咏史抒怀
- 《西施咏》（719年）：借古讽今
**语言风格概述：** 此期作品清新明快，少年意气，尚未形成独特山水诗风，多应酬赠答之作，显示早慧才华。

### 阶段二：贬谪济州（721年—735年）
**生活地点：** 济州（721-726）→ 淇上（728-730）→ 嵩山（734-735）
**核心事件：**
1. 开元九年（721年）中进士，任太乐丞
2. 因属下伶人私舞黄狮子，贬为济州司仓参军
3. 妻子约于开元十九年（731年）去世，之后未再续娶
4. 开元二十二年（734年）献诗张九龄求汲引，隐居嵩山
5. 开元二十三年（735年）官拜右拾遗
**代表作品：**
- 《宿郑州》（721年）：赴济州途中所作，悲凉慷慨
- 《济上四贤咏》：反映开元盛世下人才被埋没的不平
- 《寓言二首》：刺豪门把持仕途
- 《不遇咏》：抒发怀才不遇
**语言风格概述：** 诗歌风格开始从应酬转向内省，边塞诗与讽喻诗并存。语言精警有力，开始显露对社会现实的关注。贬谪经历使诗歌注入沉郁悲凉底色。

### 阶段三：半官半隐（736年—755年）
**生活地点：** 长安（736-737）→ 凉州（737-738）→ 长安（738-750）→ 辋川（750-755）
**核心事件：**
1. 开元二十五年（737年）赴凉州河西节度幕，为监察御史兼节度判官
2. 开元二十八年（740年）迁殿中侍御史
3. 天宝元年（742年）转左补阙
4. 天宝三载（744年）开始经营蓝田辋川别业
5. 天宝七载（748年）转库部郎中
6. 天宝九载（750年）母丧，守丧辋川
7. 天宝十一载（752年）除服，官拜吏部郎中
8. 天宝十四载（755年）转给事中
**代表作品：**
- 《使至塞上》（737年）：边塞雄浑壮观，"大漠孤烟直，长河落日圆"
- 《送梓州李使君》：送别诗，"万壑树参天，千山响杜鹃"
- 《终南别业》（约745年）："行到水穷处，坐看云起时"，辋川时期代表
- 《山居秋暝》：辋川极品，"明月松间照，清泉石上流"
- 《鸟鸣涧》《辛夷坞》《鹿柴》《竹里馆》：辋川组诗，空灵禅意
- 《辋川闲居赠裴秀才迪》："寒山转苍翠，秋水日潺湲"
- 《观猎》：边塞豪情，"风劲角弓鸣，将军猎渭城"
- 《汉江临眺》："江流天地外，山色有无中"
- 《和贾舍人早朝大明宫之作》：宫廷气象，"九天阊阖开宫殿，万国衣冠拜冕旒"
**语言风格概述：** 辋川时期是王维诗歌的巅峰，形成"诗中有画、画中有诗"的独特风格。以五律、五绝为主，篇幅短小，语言精美，音节舒缓，意境空灵静谧，禅意盎然。同时边塞诗雄浑壮阔，展现济世情怀。

### 阶段四：战乱被俘（756年—758年）
**生活地点：** 长安 → 洛阳（菩提寺被拘）
**核心事件：**
1. 天宝十五载（756年）六月，安禄山陷长安，王维被俘
2. 被迫任安禄山伪职《给事中》
3. 作《凝碧池》抒发亡国之痛
4. 至德二载（757年）唐军收复长安洛阳后入狱，按律当死
5. 因《凝碧池》及弟王缙平叛功绩，得宽宥，降为太子中允
**代表作品：**
- 《凝碧池》（756年）："万户伤心生野烟，百官何日再朝天"
**语言风格概述：** 留存作品极少，但从《凝碧池》可见此期心境悲愤沉痛，眷恋朝廷，风格沉郁而不失风骨。

### 阶段五：晚年归隐（758年—761年）
**生活地点：** 长安
**核心事件：**
1. 乾元元年（758年）责授太子中允，加集贤殿学士，迁太子中庶子、中书舍人
2. 上元元年（760年）转任尚书右丞（一生最高官职）
3. 上元二年（761年）上《责躬荐弟表》求归田园
4. 七月安然离世
**代表作品：**
- 《酌酒与裴迪》（759年）："人情翻覆似波澜"，感慨世态
- 《积雨辋川庄作》："漠漠水田飞白鹭，阴阴夏木啭黄鹂"，辋川极品
- 《老将行》（约760年）：借古讽今，悲壮慷慨
**语言风格概述：** 晚年诗作更加恬淡超脱，在佛理与山水间寻求寄托。诗风澄澹精致，意趣幽玄，"味淡声希"。虽仍关心世事，但更多通过佛理化解。总体趋向空灵静谧，禅意更深。

## 阶段切分说明

王维一生大致可分五个阶段：
1. **早年求仕**（701-721）：才华早显，积极入仕，诗风清新明快
2. **贬谪济州**（721-735）：仕途挫折，妻子亡故，开始内省，风格趋于沉郁
3. **半官半隐**（736-755）：经营辋川，创作高峰，山水诗巅峰，"诗中有画"，辋川组诗为空灵禅意典范
4. **战乱被俘**（756-758）：安史之乱被俘，创作《凝碧池》，此期作品极少
5. **晚年归隐**（758-761）：官至尚书右丞，向往归隐，诗风澄澹精致，禅意更深

核心风格转变：从早期的济世热情（边塞诗）→ 中期的山水禅意（辋川别业）→ 晚期的澄澹超脱，贯穿一生的底色是禅佛思想与对自然山水的热爱。
