# 李清照人生阶段

## 阶段一：少女时期（1084-1101）

**生活地点：** 山东济南章丘 → 东京（今河南开封）

**核心事件：**
1. 出生于济南章丘士大夫家庭，父亲李格非是苏轼学生、"苏门后四学士"之一
2. 6岁随父到东京开封，受到良好文化教育
3. 少年便有诗名，"自少年便有诗名，才力华瞻，逼近前辈"（王灼《碧鸡漫志》）
4. 16岁写出《如梦令·昨夜雨疏风骤》，轰动京师
5. 17岁作《浯溪中兴颂诗和张文潜二首》，展现政治见识

**代表作品：**
- 《如梦令·昨夜雨疏风骤》（约1099年）："知否，知否？应是绿肥红瘦"——一问世即轰动了整个京师
- 《如梦令·常记溪亭日暮》：少女时期的活泼清新之作
- 《浯溪中兴颂诗和张文潜二首》（1100年）："夏商有鉴当深戒，简策汗青今具在"

**语言风格概述：** 清新活泼，自然真切，以白描手法描写自然景物和生活细节。"绿肥红瘦"一语双关，既写景又传情。少女词作不受传统束缚，天真烂漫，灵动清新。

**confidence：** high（此期作品充足，《如梦令》二首均为代表作）

---

## 阶段二：新婚燕尔与党争风波（1101-1107）

**生活地点：** 汴京（今开封）

**核心事件：**
1. 建中靖国元年（1101），18岁与太学生赵明诚成婚
2. 夫妇"相对展玩咀嚼"碑文，"自谓葛天氏之民也"
3. 生活清贫但精神富足，"穷遐方绝域，尽天下古文奇字之志"
4. 崇宁元年（1102），父亲李格非被列入元祐党籍，罢官
5. 赵明诚之父赵挺之却一路升迁，李清照曾上诗救父未果

**代表作品：**
- 《一剪梅·红藕香残玉簟秋》："一种相思，两处闲愁。此情无计可消除，才下眉头，却上心头"
- 《醉花阴·薄雾浓云愁永昼》："莫道不销魂，帘卷西风，人比黄花瘦"
- 《凤凰台上忆吹箫》："惟有楼前流水，应念我，终日凝眸"

**语言风格概述：** 婉约深情，以女性的敏感细腻书写离别相思。意境优美，比喻新颖。"才下眉头，却上心头"尤为人称道。情感表达含蓄而有层次，深得婉约词之精髓。

**confidence：** high（此期作品丰富，《一剪梅》《醉花阴》等均为传世名作）

---

## 阶段三：青州隐居·归来堂时期（1107-1120）

**生活地点：** 青州（今山东青州）

**核心事件：**
1. 1107年，蔡京复相，赵挺之病卒，赵家被迫离开京师
2. 25岁在青州命室曰"归来堂"，自号"易安居士"（取义陶渊明《归去来兮辞》）
3. 夫妇在青州收集金石古籍，度过"平生少有的和美日月"
4. 政和七年（1117），协助赵明诚大体完成《金石录》
5. 宣和三年（1121），赵明诚赴莱州，李清照暂留青州

**代表作品：**
- 《蝶恋花·晚止昌乐馆寄姊妹》（1121年）：离别青州姊妹的惜别之情
- 《感怀》诗（1121年）：独处莱州的感怀
- 《多丽·咏白菊》："也不似、贵妃醉脸，也不似、孙寿愁眉。韩令偷香，徐娘傅粉，莫将比拟未新奇"

**语言风格概述：** 典雅凝重，借古讽今，用典精当。此期经历政治风波，词作更添沉郁。"易安居士"之名体现了对陶渊明的仰慕，隐逸情怀初现。

**confidence：** medium（此期作品较少，但《蝶恋花》等为代表）

---

## 阶段四：南渡漂泊·丧乱流离（1127-1132）

**生活地点：** 江宁 → 建康 → 浙东 → 越州 → 衢州 → 杭州

**核心事件：**
1. 靖康二年（1127），"靖康之变"，北宋灭亡，李清照整理遴选收藏南下
2. 押运15车书籍器物，历经战乱抵达建康
3. 建炎三年（1129），赵明诚罢守江宁，独自弃城而逃；八月病卒于建康
4. 李清照作《夏日绝句》："生当作人杰，死亦为鬼雄。至今思项羽，不肯过江东"
5. 作《祭赵湖州文》："白日正中，叹庞翁之机捷；坚城自堕，怜杞妇之悲深"
6. 绍兴元年（1131），书画在越州被盗，"一夕书画被盗"
7. 绍兴二年（1132），再嫁张汝舟，后因其虐待与营私舞弊而告官离婚，获释

**代表作品：**
- 《夏日绝句》（1129年）："生当作人杰，死亦为鬼雄"——借项羽讽喻南宋统治者
- 《武陵春·春晚》（1135年）："只恐双溪舴艋舟，载不动许多愁"
- 《题八咏楼》（1135年）："江山留与后人愁"
- 《声声慢·寻寻觅觅》（约1135年后）："寻寻觅觅，冷冷清清，凄凄惨惨戚戚"

**语言风格概述：** 凄婉悲凉，情感深沉凝重。"载不动许多愁""怎一个愁字了得"均是愁思极致之语。南渡后的词作充满家国之思与身世之感。叠词的运用开创了词学新境。沉郁悲怆，是乱世飘泊的真实写照。

**confidence：** high（此期作品最为丰富，《声声慢》《武陵春》等均为晚年代表作）

---

## 阶段五：晚年独居·成就《词论》（1132-约1155）

**生活地点：** 杭州（主要）

**核心事件：**
1. 经历再嫁离婚风波后，"生活的意志并未消沉，诗词创作的热情更趋高涨"
2. 绍兴三年（1133），作诗送韩肖胄使金："欲将血泪寄山河，去洒东山一抔土"
3. 绍兴四年（1134），完成《金石录后序》——中国文学史上最重要的女性散文之一
4. 作《打马赋》，借博弈之事讽喻时政，寄寓收复失地之志
5. 绍兴十三年（1143），将《金石录》校勘整理，表进于朝
6. 约绍兴二十五年（1155），在极度孤苦中悄然辞世

**代表作品：**
- 《金石录后序》（1134年）：自传性散文，回忆与赵明诚收集金石的经过
- 《打马赋》及《打马图经》（1134年）："借谈论博弈之事，引用大量有关战马的典故和历史上抗恶杀敌的威武雄壮之举"
- 《永遇乐·落日熔金》（晚年）：回忆"中州盛日"的京洛旧事，寄托故国之思

**语言风格概述：** 悲凉中有倔强，孤独中见坚毅。散文成就极高，《金石录后序》被后世推为"清新、词采俊逸"的佳作。词的风格依然是婉约为主，但融入了更深沉的家国情怀。晚年词作沉郁顿挫，"烈士暮年，感慨万千"。

**confidence：** high（此期作品丰富，《金石录后序》《打马赋》《永遇乐》等为代表）

## 阶段切分说明

李清照一生经历了从北宋到南宋的巨变，其人生轨迹深受时代变迁和个人遭遇的双重影响。

**划分依据：**
1. 地理迁移：山东章丘 → 开封 → 青州 → 莱州/淄州 → 江宁/建康 → 浙东 → 杭州
2. 政治变故：党争（1102）→ 金兵南侵（1127）→ 丈夫病逝（1129）→ 再嫁离婚（1132）
3. 精神状态：少女天真 → 新婚甜蜜 → 中年隐逸 → 晚年悲凉

**各阶段主要变化：**
- 少女期：以自然景物词为主，清新活泼
- 新婚期：以离别相思词为主，婉约深情
- 隐居期：典雅凝重，隐逸情怀初现
- 丧乱期：家国之思与身世之感融为一体，风格沉郁悲怆
- 晚年期：散文成就突出，词作融入更深沉的家国情怀

**语料丰富度评估：** 全期语料相对均衡丰富，各阶段均有代表作品，confidence整体评级为high。
