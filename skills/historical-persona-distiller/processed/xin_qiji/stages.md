# 辛弃疾 人生阶段

## 阶段一：北方成长（1140-1161年）

**生活地点：** 济南历城 → 燕京

**核心事件：**
1. 辛弃疾生于金占区济南，祖父辛赞仕于金朝，常带其"登高望远，指画山河"
2. 辛赞两度让辛弃疾至燕京参加进士科考试，借机侦察金人形势
3. 辛弃疾"两随计吏抵燕山，谛观形势"，亲眼目睹汉人在金统治下的屈辱
4. 与党怀英同窗受学于刘瞻，并称"辛党"
5. 青少年时期立下恢复中原、报国雪耻的志向，养成了燕赵侠义之气

**代表作品：** （无存世作品，此阶段属生平背景期）

**语言风格概述：** 此阶段无直接语料留存。据传记，辛弃疾青少年时期以武勇和侠义著称，尚未开始文学创作。其语言特质在此阶段尚未形成。

**confidence：** low（无直接语料，据传记推测）

---

## 阶段二：起义南归（1161-1163年）

**生活地点：** 山东 → 建康（南京）

**核心事件：**
1. 绍兴三十一年（1161年），22岁的辛弃疾"鸠众二千"参加耿京起义军，担任掌书记
2. 力劝耿京"决策南向"，接受南宋朝廷领导
3. 绍兴三十二年（1162年）正月，奉表南归，宋高宗在建康接见，任命为右承务郎、天平军掌书记
4. 得知耿京被害后，约五十人驰赴金营，在五万金兵中抓获叛徒张安国，献俘临安斩首
5. 洪迈评价："壮声英慨，儒士为之兴起，圣天子一见三叹"

**代表作品：**
- 《鹧鸪天》（"壮岁旌旗拥万夫"）：追忆少年抗金壮举——高亢慷慨，血性昂扬

**语言风格概述：** 此阶段辛弃疾以行动而非笔墨著称。现存作品极少，但《鹧鸪天》追忆作品显示其语言具有英雄豪侠之气，用词典雅而不失力度，"锦襜突骑""金仆姑"等军事意象贯穿。

**confidence：** medium（仅一首明确作品）

---

## 阶段三：仕宦沉浮与词学高峰（1163-1181年）

**生活地点：** 建康 → 滁州 → 江陵 → 江西 → 湖南

**核心事件：**
1. 1163-1168年：沉寂期，虽有北伐氛围但朝廷不采纳其抗金方略
2. 26岁（1165年）向宋孝宗上《美芹十论》（又名《御戎十论》），详述抗金战略，未被采纳
3. 后又向虞允文上《九议》，仍无果，陷入极度悲愤苦闷
4. 乾道四年（1168年）起，担任建康通判、滁州知州、江西提刑等职
5. 平定赖文政茶商军有功，知江陵府兼荆湖北路安抚使，成为封疆大吏
6. 创置"飞虎军"，妨碍权贵利益
7. 淳熙八年（1181年）十一月，被台臣王蔺以"用钱如泥沙，杀人如草芥"等罪名弹劾，罢去所有职务

**代表作品：**
- 《水龙吟·登建康赏心亭》（约1168年）："把吴钩看了，栏杆拍遍，无人会，登临意"——沉郁顿挫，悲愤难抑
- 《念奴娇·登建康赏心亭呈史留守致道》：吊古伤今，壮志难酬
- 《满江红·江行和杨济翁韵》：感慨生平
- 《水调歌头》等早期词作：慷慨激昂，忧国忧民

**语言风格概述：** 这是辛弃疾词学的重要形成期。语言以慷慨悲凉为主调，意象以剑、弓、吴钩等军事器物为核心，充满英雄失路的悲愤。句式长短交错，以长调慢词为主，悲壮沉郁。

**confidence：** high（作品丰富，约100首存世）

---

## 阶段四：乡居带湖与瓢泉（1181-1203年，约22年）

**生活地点：** 上饶带湖 → 铅山瓢泉

**核心事件：**
1. 淳熙八年（1181年）带湖新居落成，辛弃疾被罢官后回乡，自号"稼轩居士"
2. 在带湖时期写下大量描写田园风光、农村生活的词作
3. 淳熙十五年（1188年）冬，陈亮来访，两人于铅山长歌互答，称"第二次鹅湖之会"（辛陈之晤）
4. 绍熙三年（1192年）至绍熙五年（1194年），短暂出仕福建
5. 庆元二年（1196年）带湖庄园失火，举家迁铅山期思瓢泉
6. 此二十年间，与朱熹等学者交往密切

**代表作品：**
- 《青玉案·元夕》（约1174年）："众里寻他千百度，蓦然回首，那人却在，灯火阑珊处"——婉约含蓄，意境深远
- 《清平乐·村居》："茅檐低小，溪上青青草……最喜小儿亡赖，溪头卧剥莲蓬"——清新自然
- 《西江月·夜行黄沙道中》："明月别枝惊鹊，清风半夜鸣蝉"——明快爽朗
- 《丑奴儿·书博山道中壁》："少年不识愁滋味……而今识尽愁滋味，欲说还休"——对比鲜明，沉痛自知
- 《鹧鸪天·鹅湖归病起作》："不知筋力衰多少，但觉新来懒上楼"——晚年悲凉
- 《贺新郎·甚矣吾衰矣》（1188年）："我见青山多妩媚，料青山见我应如是"——旷达自适
- 《沁园春·带湖新居将成》："沉吟久，怕君恩未许，此意徘徊"——复杂心境
- 《菩萨蛮·书江西造口壁》（1188年）："青山遮不住，毕竟东流去"——悲中有壮

**语言风格概述：** 这是辛弃疾创作最丰的时期。风格发生重要转变：既有《青玉案》《清平乐》等婉约清新之作，也有《丑奴儿》《鹧鸪天》等悲凉感慨之作。"以文为词"的特点更加成熟，散文化的句式大量出现。语言时而通俗口语（"些底事，误人那"），时而典故密布。田园词清新明快，怀古词沉郁顿挫。

**confidence：** high（这是辛弃疾存词最多的时期，约400首）

## 阶段五：晚年再起与壮志未酬（1203-1207年）

**生活地点：** 绍兴 → 镇江 → 铅山

**核心事件：**
1. 嘉泰三年（1203年），64岁的辛弃疾被韩侂胄起用，知绍兴府兼浙东安抚使
2. 嘉泰四年（1204年）晋见宋宁宗，认为金国"必乱必亡"，差知镇江府
3. 登临北固亭，写下《永遇乐·京口北固亭怀古》——"千古江山，英雄无觅孙仲谋处"
4. 开禧元年（1205年）三月，被谏官攻击，降职，辞不就
5. 开禧三年（1207年）秋，朝廷再授枢密都承旨，已病重，九月初十（10月3日）逝世
6. 临终大呼"杀贼！杀贼！"

**代表作品：**
- 《永遇乐·京口北固亭怀古》（1204年）："千古江山，英雄无觅孙仲谋处……凭谁问：廉颇老矣，尚能饭否？"——壮心不已，悲凉慷慨
- 《生查子·题京口郡治尘表亭》："悠悠万世功，兀兀当年苦"——晚年感慨
- 此阶段创作数量较少，多为晚年感慨之作

**语言风格概述：** 晚年词作感慨最深，悲壮与超脱并存。代表作《永遇乐》用典密集，气象阔大，"佛狸祠下，一片神鸦社鼓"暗含讽刺。《南乡子》"天下英雄谁敌手？曹刘。生子当如孙仲谋"则是壮心不已的豪迈表白。语言苍劲老练，炉火纯青。

**confidence：** medium（作品数量有限但质量极高，约50首）

## 阶段切分说明

辛弃疾一生可划分为五个阶段：

1. **北方成长**（无创作期）：1140-1161年，青少年时期，为后面的创作积蓄了强烈的爱国情感和人生阅历
2. **起义南归**（创作萌芽期）：1161-1163年，短暂的军旅生涯，《鹧鸪天》追忆作品已显示英雄本色
3. **仕宦沉浮**（豪放词奠基期）：1163-1181年，任官各地，《水龙吟》等代表作奠定其豪放词风基础
4. **乡居隐退**（创作巅峰期）：1181-1203年，二十多年乡居，创作量最大，风格最多样
5. **晚年再起**（词艺化境期）：1203-1207年，词风老辣苍劲，《永遇乐》等作品炉火纯青

其词风最大特点：以豪放为主，但不拘一格，沉郁、明快、激励、妩媚兼而有之。"以文为词"是其最大创新——用散文句式入词，打破传统词的形式束缚。
