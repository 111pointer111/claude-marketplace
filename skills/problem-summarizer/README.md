# 问题总结记录 Skill

将开发过程中遇到的问题和解决方案自动记录到固定文档，构建可检索的项目知识库。

## 功能

- **记录问题**：自动提取问题描述、根因、解决方案、相关文件
- **查看记录**：按时间倒序展示所有记录
- **搜索问题**：按关键词搜索历史记录

## 使用方式

直接告诉 Agent：

- `记录这个问题` — 将当前对话中的问题记录到知识库
- `查看问题记录` — 列出所有历史记录
- `搜索问题 API` — 搜索包含"API"的记录

## 记录位置

```
docs/DEV-LOG.md
```

该文件在 `.gitignore` 中，仅作为本地开发知识库。
