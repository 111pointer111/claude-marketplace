# AI 音乐生成专业手册

> 本文件是 `minimax-music` skill 的核心知识库。基于 Suno/Udio/MiniMax 等主流 AI 音乐生成器的实际行为模式编写。
>
> **核心发现**：
> 1. **Hook-First**：先设计副歌记忆点，再倒推全曲结构
> 2. **约束叠加**：2-3 个相互制约的风格条件 > 单一大词
> 3. **结构标签直接识别**：AI 认 `[verse]`/`[chorus]`/`[bridge]` 标签
> 4. **迭代比一次到位更可靠**：固定核心变量，每次只变一个

## 一、歌词创作方法论

### 1.1 Hook-First 创作法

**传统写法**：先写主歌 → 再写副歌 → 结尾
**AI 音乐写法（Hook-First）**：先设计 hook → 再补充 verse → 交给 AI 执行

```
Hook 设计优先级：
1. 一句话记忆点（≤7个字，中文优先双音节）
2. 画面感强（"路灯下影子变短了" 比 "时间流逝" 更好）
3. 情绪精准（克制表达，不过度煽情）
4. 押韵位置明确（偶数句押韵 or 每句尾音上口）
```

### 1.2 歌词创作四轴法则

| 维度 | 问题 | 差例子 | 好例子 |
|------|------|--------|--------|
| **叙事性** | 有故事/场景吗？ | "我很孤独" | "深夜三点，厨房只有冰箱嗡嗡响" |
| **意象性** | 有感官细节吗？ | "时间过得好快" | "校服还穿着，只是领口松了两颗扣" |
| **韵律感** | 唱出来顺口吗？ | "我/爱/你/啊/爱/情/啊" | "月光落在左手边"（双字收尾） |
| **情感共鸣** | 听众能感同身受吗？ | "我的心好痛" | "把音乐声调大到没人能听见" |

### 1.3 歌词创作 SOP

```
Step 1 — 确定主题：一句话：谁+在什么情境+经历什么
Step 2 — 设计 Hook：副歌最后一句（≤7字），必须是一个具体画面或动作
Step 3 — 确定情绪走向：verse: 平静叙述 → pre-chorus: 情绪积蓄 → chorus: 爆发/释然
Step 4 — 写出 Verse 场景：2-3 个具体生活细节（物品+动作+声音）
Step 5 — 对照四轴检查：有一项不满足 → 重写该段落
```

## 二、AI-Priority Prompt 工程

### 2.1 核心规律

**规律 1：约束叠加 > 单一描述**
```
# ❌ 弱 prompt
--prompt "一首关于爱情的歌"

# ✅ 强 prompt（2-3个相互制约的条件）
--prompt "中文抒情ballad，钢琴主导，副歌弦乐突然加入，
         关于分手后第一周清晨，克制叙事不过度煽情"
```

**规律 2：先固定结构变量，再迭代细节**

**规律 3：--lyrics-optimizer 的 prompt 必须包含场景**
```
--prompt "
主题：分手后的第一个月圆之夜。
场景：阳台上的旧拖鞋还摆着，月光把两个人的影子连成一片。
情绪：从平静到突然哽咽，克制不过度。
歌词要求：
  - 用生活物品替代情绪词
  - 副歌hook: '月亮还是那个月亮，只是少了一扇窗'
  - 禁止：'心痛'、'眼泪'、'悲伤'等抽象词
"
```

### 2.2 人声音乐 prompt 结构

```yaml
基本信息:
  语种: "中文"
  流派: "民谣/流行/电子/中国风..."
  情绪: "忧伤/温暖/力量..."

场景设计:
  时间节点: "故事发生在哪个时刻（初见/热恋/离别/多年后）"
  空间细节: "阳台/便利店/凌晨的马路/大学校门口"
  核心物品: "一件具体物品串联全篇（手机相册/凉掉的咖啡）"

Hook设计:
  副歌核心句: "≤7字，具体画面，双字收尾"
  韵律要求: "偶数句押韵，句尾上口"

歌词禁止项:
  - 抽象情绪词: "心痛、悲伤、难过、崩溃"
  - 陈词滥调: "支离破碎、遍体鳞伤、飞蛾扑火"
  - 说教结尾: "要珍惜当下、把握今天"

--vocals 人声描述：
  主歌(verse): "chest voice 叙事，接近说话"
  预副歌(pre-chorus): "mixed voice 渐强蓄力"
  副歌(chorus): "belt 爆发，高音有穿透力"
  桥段(bridge): "head voice 轻唱，脆弱感"
  尾副歌(final): "strong belt + vibrato 高潮"
  整体质感: "男高音/女中音，温暖/沙哑/清亮..."
```

### 2.3 器乐 prompt 结构

```yaml
音色层次:
  铺底层: "ambient pads / 弦乐长音 / drone"
  节奏层: "底鼓+军鼓 / fingerpicked guitar"
  主旋律层: "钢琴solo / 吉他riff / 弦乐主旋律"
  点缀层: "钟琴 / 自然音效 / 电子特效音"

情绪曲线:
  开头: "soft / building / intense"
  发展: "steady / evolving"
  结尾: "fade / climactic / sudden stop"

制作要求:
  - 动态范围: "大动态（电影感）/ 小动态（背景音）"
  - 空间感: "近距感（intimate）/ 宽广感（spacious）"
  - 禁止项: "--avoid"
```

### 2.4 mmx 关键参数最佳实践

| 参数 | 最佳实践 | 常见错误 |
|------|---------|---------|
| `--lyrics-optimizer` | prompt 需包含场景+情绪+禁止项 | 纯主题词 |
| `--vocals` | "有呼吸感的人声，不过度修音" 优于 "女声" | 只写 "女声" |
| `--structure` | 用标准标签: `verse-chorus-verse-chorus-bridge-chorus` | 结构混乱或无结构 |
| `--avoid` | 具体元素: "auto-tune, 过量电子音效, 突然爆音" | 太抽象或太宽泛 |
| `--tempo` | 中文歌曲优先 moderate，慢歌 ballad pace | 过快或过慢 |

## 三、歌曲结构与标签系统

### 3.1 标准结构

```
[intro]           4-8小节，建立氛围，可纯器乐
[verse 1]         8小节，叙事开场，视角建立
[pre-chorus]     4-8小节，情绪积蓄，准备进入高潮
[chorus]          8小节，核心情绪+Hook，歌词精华
[verse 2]         8小节，叙事递进，视角/时间变化
[chorus]          重复，保持记忆点
[bridge]          4-8小节，情绪转折，歌词哲理化
[final chorus]    最终高潮，可升调或加配器
[outro]           4小节，逐渐淡出
```

### 3.2 标签识别规则（MiniMax mmx）

| 标签 | AI 识别效果 | 备注 |
|------|------------|------|
| `[verse]` | ✅ 高 | 主歌，叙事节奏 |
| `[chorus]` | ✅ 高 | 副歌，hook 段落 |
| `[bridge]` | ✅ 高 | 情绪转折/哲理 |
| `[pre-chorus]` | ⚠️ 中 | 蓄力段，有时被合并到 verse |
| `[outro]` | ⚠️ 中 | 结尾段 |
| `[hook]` | ✅ 高 | 记忆点强调 |

**标签使用原则**：标签写在单独一行，标签后立即跟歌词内容，不要空行。

## 四、中文歌词特殊技巧

### 4.1 普通话声调与上口度

| 技巧 | 说明 | 反例 | 正例 |
|------|------|------|------|
| 双字收尾 | 偶数句尾音成双 | "走（1声）" | "走了（轻声）" |
| 四声搭配 | 避免连续同声调 | "天是天还是天" | 1-4声搭配 |
| 口语化 | 说话感 | "吾之内心甚为忧愁" | "心里闷得慌" |
| 轻声优先 | 句尾轻声更顺 | "飞机" | "飞机儿" |

### 4.2 中文歌词意象体系

| 抽象词 | ✅ 替代意象 |
|--------|------------|
| 孤独 | 一个人的餐桌 / 空座位 / 手机没有新消息 |
| 思念 | 月亮 / 背影 / 旧照片 / 名字念出声 |
| 告别 | 最后一班地铁 / 月台背影 / 没说出口的话 |
| 成长 | 种子破土 / 航线偏移 / 校服变肥 |
| 失落 | 凉掉的咖啡 / 没回的消息 / 音乐声调大 |

### 4.3 中文歌词叙事禁忌（--avoid 指南）

```
必须禁止的词汇/表达：
抽象情绪词：心痛、悲伤、难过、崩溃、绝望、心碎
陈词滥调：飞蛾扑火、遍体鳞伤、沧海桑田、海枯石烂
说教结尾："要珍惜当下"、"把握今天"、"勇敢前行"
过度煽情：直接哭喊、直接说"我爱你爱到死"
```

**克制表达示范**：
```
❌ "我好想你，想得心都碎了"
✅ "输入法还记得你的名字"
✅ "窗台上的绿萝又长了一寸，没人再问我要不要浇水"
✅ "把音乐声调到最大，这样就没人能听见我"
```

### 4.4 中文歌词韵律技巧

- 偶数句押韵（最安全）
- 句尾：双音节词 > 单音节词（更上口）
- 换韵时机：段落转换时
- Hook 韵律设计：短句重复（4-6字一句，重复 2 次）、顶真手法、画面感

## 五、人声与演唱技巧体系

### 5.1 声音类型速查

#### 按音域分

| 类型 | 中文俗称 | 特点和适合场景 |
|------|---------|--------------|
| **Soprano** | 女高音 | 清亮华丽，高音穿透力强。适合情歌高音、大气高潮 |
| **Mezzo-soprano** | 女中音 | 温暖厚实，情感丰富。适合民谣、爵士、叙事 |
| **Alto** | 女低音 | 低沉有磁性。适合 R&B、灵魂乐 |
| **Tenor** | 男高音 | 明亮有力，高音能爆发。适合摇滚副歌、热血高潮 |
| **Baritone** | 男中音 | 最普通的男声，温暖有力量。通吃型，适合叙事 |
| **Bass** | 男低音 | 极低深沉。适合爵士、气场铺垫 |

#### 按音色/质感分

| 描述词 | 含义 | 像谁 |
|-------|------|------|
| 清亮（bright） | 高频突出，通透 | 王菲、张韶涵 |
| 温暖（warm） | 中频饱满，圆润 | 陈奕迅、蔡琴 |
| 沙哑（raspy） | 带颗粒感和年龄感 | 杨坤 |
| 气声（breathy） | 漏气感，柔弱亲密 | 王菲慵懒感 |
| 磁性（husky） | 低频丰富，有吸引力 | 毛不易、李荣浩 |
| 金属感（metallic） | 明亮带穿透力 | 邓紫棋高音、林志炫 |
| 醇厚（mellow） | 成熟圆润，无尖锐感 | 张学友 |
| 空灵（ethereal） | 虚幻感，像远处传来 | 王菲 |

---

### 5.2 核心演唱技巧速查（AI 可识别）

| 术语 | 中文 | 说明 | 用在哪儿 |
|------|------|------|---------|
| **Chest voice** | 胸声（真声） | 厚实有力，中低音区 | 主歌叙事，低沉部分 |
| **Head voice** | 头声 | 轻、高、空灵 | 桥段高音，柔和段落 |
| **Mixed voice** | 混声 | 胸声+头声混合，过渡平滑 | 全场衔接，预副歌蓄力 |
| **Belt / Belting** | 强混声爆发 | 用胸声唱高音，有力量感 | **副歌高潮，最需要这个** |
| **Falsetto** | 纯假声 | 比头声更轻更虚 | 点缀句尾，脆弱感 |
| **Vibrato** | 颤音 | 尾音自然抖动 | 尾音收束，增加情感 |
| **Melisma** | 转音/花腔 | 一字多音 | R&B 装饰，句尾 |
| **Slide** | 滑音 | 从一个音滑到另一个 | 句尾处理，增加味道 |
| **Ad-lib** | 即兴自由唱 | 结尾自由发挥 | 尾段高潮升华 |

### 5.3 歌曲各段落的人声策略

主歌 ▸ 预副歌 ▸ 副歌 ▸ 桥段 ▸ 尾副歌
→ 收 ▸ 蓄 ▸ 放 ▸ 收 ▸ 最放

| 段落 | 发声方式 | 力度 | 目的 |
|------|---------|------|------|
| **Verse** | 胸声/弱混声，像说话 | 轻-中轻 | 叙事，让听众进入故事 |
| **Pre-chorus** | 混声，慢慢加入头声 | 逐渐变强 | 积蓄能量，铺垫高潮 |
| **Chorus** | **Belt/强混声**，高音区 | 强-很强 | 情感爆发，hook 出来 |
| **Bridge** | 头声/假声，脆弱感 | 回落到中轻 | 情绪转折，制造对比 |
| **Final Chorus** | Belt + 头声，最有力 | 最强 | 最高潮，升华 |

**核心原则：对比创造戏剧性** — 主歌收着唱 → 副歌才有爆发力；桥段轻下来 → 尾副歌才推得上去；全程同一个力度 = 无聊。

### 5.4 各曲风声乐风格速查

| 曲风 | 声乐特点 | 关键词 |

| **爵士** | 慵懒，即兴，swing | `lazy, improvisational, swing` |

### 5.5 古风歌曲专项指南

#### 古风声乐特点

| 特点 | 说明 | `--vocals` 写法 |
|------|------|----------------|
| 民族唱法（head-dominant） | 头腔共鸣为主，圆润不刺耳，咬字靠前 | `民族唱法，头腔共鸣，圆润明亮` |
| 戏曲韵味 | 加入戏曲腔调，尾音上扬 | `带戏曲腔，尾音上扬，转音细腻` |
| 气声控制 | 古风常需要半声唱法，轻而不虚 | `半声唱法，气声控制，轻而不飘` |
| 咬字清晰 | 每个字都咬清楚，字正腔圆 | `咬字清晰，字正腔圆，句尾收音干净` |
| 高音清亮不破 | 古风高音追求清亮通透，不是力量感 | `高音清亮，head voice 通透，不嘶吼` |

#### 古风歌词特点

| 维度 | 做法 | 示例 |
|------|------|------|
| 意象体系 | 古代器物+自然景物 | 竹简/油纸伞/檀香/烛火/青山/流水/残月/柳絮 |
| 用词风格 | 文言感但不晦涩 | "当时明月在" > "昨夜明月仍悬于苍穹之上" |
| 修辞手法 | 对仗、顶真、互文 | "你研墨，我泼墨" / "山水穷，穷途又见一城" |
| 叙事方式 | 古代背景，但情感要现代人共鸣 | 不是真讲古代故事，是用古代场景讲现代情感 |

#### 古风 `--vocals` 完整示例

**仙侠/武侠风**：
```
--vocals "女高音，民族唱法，头腔共鸣圆润明亮。
          主歌半声轻唱，像山间流水；
          预副歌加入头声，渐亮；
          副歌清亮展开，高音通透不嘶吼；
          桥段气声轻吟，带一丝惆怅；
          尾副歌高音走上去，余韵悠长 fade out"
```

**古风叙事（器乐+人声）**：
```
--vocals "男中音，带一点戏曲腔，咬字靠前。
          主歌 chest voice 叙事，低沉念白感；
          预副歌 mixed voice 蓄力，尾音略带戏曲上行；
          副歌 warm belt 展开，不过度用力的古风唱法；
          桥段 head voice 轻唱，气声为主；
          尾副歌回归 chest voice 收尾，余韵"
```

## 六、器乐 BGM 专业制作

### 6.1 音色层次设计

```
铺底层：ambient pads / 弦乐长音 / drone
节奏层：底鼓+军鼓 / fingerpicked guitar
主旋律层：钢琴solo / 吉他riff / 弦乐主旋律
点缀层：钟琴 / 自然音效 / 电子特效音
```

### 6.2 情绪曲线设计

| 情绪描述 | tempo | bpm 范围 | 典型乐器 |
|---------|-------|---------|---------|
| 冥想/入睡 | very slow | 50-65 | 弦乐长音、钢琴、单音合成器 |
| 沉思/孤独 | slow | 60-75 | 钢琴、吉他、环境音效 |
| 温暖/治愈 | moderate-slow | 70-85 | 钢琴+弦乐、温暖合成器 |
| 怀旧/怅惘 | moderate | 80-95 | 原声吉他、钢琴、民谣风格 |
| 活力/希望 | moderate-fast | 95-115 | 打击乐、吉他、昂扬弦乐 |
| 力量/激昂 | fast | 120-140+ | 电吉他、鼓组、电子音色 |
| 紧张/悬疑 | slow-fast交替 | 不规则 | 低频 drones、钢琴不协和音 |

## 七、每日生成迭代策略

### 7.1 生成前的 checklist

```
[ ] --lyrics-optimizer prompt 是否包含：时间/地点/人物/情绪走向/禁止项
[ ] --vocals 是否描述了人声质感（不止性别）
[ ] --structure 是否明确（AI 在无结构时会乱来）
[ ] --avoid 是否列出明确不需要的元素
[ ] --tempo 是否与情绪匹配
[ ] 器乐：音色层次是否描述清楚（铺底/节奏/主旋律分开）
```

### 7.2 迭代方向决策树

```
生成效果不好？
│
├─ 歌词太抽象 → 强制加"禁止抽象词" + 指定3个具体物品
├─ 副歌不够记忆点 → 单独设计 hook prompt，重新生成
├─ 编曲太乱 → 加 "--avoid heavy mixing, cluttered"
├─ 人声被器乐盖住 → prompt 加 "人声突出，器乐退让"
├─ 结构乱 → 加 --structure 标签，或去掉让 AI 自由发挥
└─ 节奏不对 → 明确 --tempo 或 --bpm
```

## 八、质量自检清单

### 歌词自检（生成后对照）

```
[ ] 有没有具体的时间/地点/人物/物品（至少各1个）？
[ ] 有没有直接出现"心痛/悲伤/难过/崩溃"等抽象词？（出现即失败）
[ ] 副歌hook是否≤7字且有画面感？
[ ] 有没有陈词滥调（飞蛾扑火/遍体鳞伤等）？
[ ] 韵律是否顺口（跟读一遍）？
[ ] 情绪是否有变化（不是从头一个情绪）？
[ ] 克制程度如何（用细节暗示 vs 直接说出情绪）？
```

### 音乐自检（生成后对照）

```
[ ] 器乐：音色层次是否清晰（铺底/节奏/主旋律分得开）？
[ ] 人声歌曲：人声是否突出（没被器乐压住）？
[ ] 情绪曲线是否合理（有没有突然的断层）？
[ ] 动态是否与场景匹配（背景BGM不应太抢戏）？
[ ] 结构是否完整（有没有某段落突然消失）？
```
